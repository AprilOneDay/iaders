<br/><img src="http://ww2.sinaimg.cn/crop.0.0.1400.786.1000/006prEoojw1fb8hvl9mqxj312w0q4q8u.jpg" /><br/><br/>我想每个城市都有一些像我这样的青年，没有什么理想，也没有什么目标，游荡在城市之中存活，为的的只是一日三餐和吃饱喝足后的娱乐项目，当然这里说的娱乐项目因人而定，或许是约朋友开黑游戏，或许是约朋友吹牛打牌，地点不停的变换，但身边的总是一群熟悉的面孔。不为任何人而活，只为让自己显得不那么孤独。<br/><br/><!--more--><br/><br/>我手头通常拮据，每到月底甚至泡面都无法维持，饥肠辘辘的我会腆着脸回到父母身边。这是一个月中最落魄的几天，也是操碎了心的家长最得意的几天。?<br/><br/>在这几天里，会吃到很多平时舍不得买的水果和美食，但这并不是白来的，因为要无时无刻不承受白眼，训话甚至谩骂，所以拿到工资的第一件事通常是，逃出家，周次往复。<br/><br/>此时家庭处在一种很微妙的平衡中，在之后的几年会一直持续到一方妥协，或者一方爆发。爆发之后家庭关系会变得更加复杂，紧张又敏感，互相关心又互相敌视，就像两个怄气的小孩子，简单的一句话甚至一个眼神都可以让对方大发雷霆。<br/><br/>这种现象总是随着青春的消逝，责任的增加而逐渐消失，变成成家立业后的一句“小时候不懂事”。不过总有一些家长不愿意去等，于是，治疗所因此而生。<br/><br/>治疗所的经历就像病历上的红戳一样，已经刻在我记忆里，在那一块不愿触碰的记忆里，反正我肯定不会用炫耀的口气告诉别人：“我曾经在精神病院住了三个月哦~”? 当然如果相亲遇到不喜欢的对象可以拿这个当做理由，百试不爽。<br/><br/>如果问我疗效怎样，我会告诉你，这个地方让人学会虚伪，让人成长？或许在里面这两个词语是一个意思。在这里你很难坚守本心，就像理智在治疗仪面前一样不堪一击，那种扭曲的环境会在不知不觉中侵蚀你的内心，它让你变得现实，变得市侩，甚至变得扭曲。它让你学会逆来顺受，让你忘记反抗，让你丢掉自尊，让你变得不再像一个“人”。<br/><br/>主任经常说过一句话：“以不自私的手段达到自私的目的。”<br/><br/>虽然这句话一般用于忽悠那些有钱家长慷慨解囊，但并不影响我觉得这句话有道理，尤其是我以虚伪的手段达到不虚伪的目的―――离开治疗所时。<br/><br/>&nbsp;<br/><br/>&nbsp;<br/><br/>那两道铁门后。<br/><br/>是青春的墓地。<br/>首次转载时间: 2017-02-11 13:59:20"
