涉及人体的医学研究必须遵从的三大原则：</h1><br/><h2>? ? ? ? ? ? ? ? ? ? ? ? ? ? ? ?不伤害原则?、知情同意原则、合法性原则。</h2><br/>------赫尔辛基宣言<br/><h2>1. ?不伤害原则</h2><br/>（1） 自2009年卫生部紧急叫停“电休克”治疗“网瘾”之后，临沂网戒中心采用了低频电子?脉冲治疗仪。而相关专家已指出，这是换汤不换药，孩子一样会感受到点击时身体刺激而带来的痛苦、害怕。本人的亲身经历完全可以证实这一点。通过已经出院的盟友了解，甚至在某些程度上，新的低频电子脉冲治疗仪在痛苦程度上更胜以前的电休克治疗仪。<br/><br/>这种疗法被称为电刺激厌恶疗法，?而这种疗法有严格规定，必须遵守相关的程序，在经过相关检查后谨慎使用。而在临沂网戒中心里，每位入院的盟友中很大的一部分在入院第一天在未经过任何相关检查的情况下（简单的体温和血压检测或全无）被送进治疗室接受电击治疗以达到“下马威”的作用。除电击治疗之外，还有两种体罚方式：<br/><br/>抱沙袋<br/><br/>顾名思义，就是抱着十几公斤的沙袋保持军姿。通常用于大点评课，点评课上，盟友和家长分别坐在南北两侧，而被罚抱沙袋的盟友则站在中间迎接所有人的目光持续数个小时的点评课，还要抱着沉重的沙袋保持军姿，不管是肉体还是精神上，都是很大的摧残。<br/><br/>磕操<br/><br/>此惩罚动作完全仿照西藏地区朝圣路上的动作。从站立到跪下，跪下到完全趴在地上，再跪起来，最后站立完成一套动作。体力消耗巨大，但是此惩罚也会持续整节点评课，除此之外还有磕操程序，在规定时间内做满规定数目的磕操动作。曾经有一位女生被杨永信惩罚磕一万个操，从中午一直做到深夜，最后体力透支晕了过去......在这里我想问一下，西藏地区同胞朝圣做这个是因为心中信仰和对神明的尊重，那么网戒中心推崇这个是何目的？<br/><br/>（2）药物滥用<br/><br/>每位入院的盟友都必须食用精神类药物，以“达到”听话的目的。?在未经过相关体检和精神评测的情况下直接给未成年使用精神类药物是否符合相关规定？是否会对患者造成不可挽回的创伤？造成的后果谁来承担？<br/><br/>（3）精神摧残<br/><br/>在临沂网戒中心自尊心是一种完全要抛弃的东西，因为里面的各种规定表面，只需要你听话，不需要你有自主想法，对已经思想相对成熟的成年人还好，对心智不成熟的未成年而言，这是一种很残酷的事情，再加上家长过于看重成绩，被灌输的思想就是：只要你学习就好，别的都不重要。这真是一种正确的教育方式吗？我不是教育学专家，我不作任何评价。甚至很多孩子因为对学习的兴趣不够浓厚，而被家长以“学习动力不足”的罪名上报问题，结果就是孩子去做电击治疗。<br/><br/>除此之外，在里面是没有任何个人隐私而言的，你能控制的就是你的思想，因为你想什么他们是看不到的，所有的隐私都会被揭露出来，不管之前你之前做了怎样难以启齿的事情都必须晾晒出来。据一位已经出院的盟友回忆，有一天的点评课专门点评女盟友，杨永信强令他们说出之前和男生交往的事情。过程我不用多说，该盟友笑称，就像听了一节课的小黄文。<br/><br/>在网戒中心精神压力巨大，出院后一样不能幸免。<br/><p class="picbox"><img src="http://ww4.sinaimg.cn/large/006prEoogw1f7l217vwjmj30ah0j3q4e.jpg" alt="小光出院后的经历" /><span class="picinfo">小光出院后的经历</span></p><br/>? ? ? 小光是前几年从网戒中心“改变”回家的盟友，据他描述，离院后生活并不快乐。小光在上学的时候在学校里被老师当着全班的同学挖苦，再加上学习压力大，就不想去上学了，然后被家长送进了网戒中心。<br/><br/>网戒中心的经历我也不用多说，大家大致都有一个了解，可是离院并没有结束他的梦魇，离开网戒中心后，小光选择了退级，但是并没有改变别人对他的看法。在网戒中心的宣传照中赫然出现了小光的照片（已涉嫌侵害他人肖像权），很短的时间内整个小镇都传开了，乡亲们都知道了他是一个“网瘾少年”，小光也失去了大部分的朋友，因为家里不让自己的孩子和小光玩。<br/><br/>除了乡亲们冷漠的目光之外，在学校的状况也不容乐观，小光被调到最后一排的座位上，自己坐一个桌子，连个同桌都没有。在发小的透露下，小光才知道，老师在小光不在场的时候当着全班的面说小光是个“网瘾孩子”，千万不要和他玩，不然你一辈子就瞎了。网戒中心，精神病院和“网瘾少年”的帽子压在他的头上，成了一个人生的污点，怎么洗都洗不掉，小光就这么卑微的活着。<br/><br/>随着时间的推移，小光也重新认识了几个朋友。也许是幸运女神的眷顾，小光喜欢上班里一个女生，但是那个女生学习成绩很高，也经常鼓励他。小光才渐渐走出了阴影，开始用心学习。让我感到讽刺的是“戒网”的后果还需要“早恋”来弥补心灵的创伤。现在小光已经大学毕业，正准备成为一名人民教师，因为他不想还有孩子因为老师走自己的老路，在这里我祝你以后的生活充满阳光。<br/><br/>其实像小光这样的情况在离院盟友中并不罕见，很多年龄大一点的，在离院后就外出工作，到现在都没有回过家，难道这就是所谓的“治愈”？<br/><br/>―――――――――――――――――――――――――――――――――――――――――――<br/><h2>2. 知情同意原则</h2><br/>在这一方面我毫不隐瞒，当时家里要让我去网戒中心的时候是告诉了我去哪的，但是并不是每个人都“知情”，里面很大比例的盟友是被哄骗甚至强行带过去的。家里长辈生病，出去旅游....各种理由五花八门，但是最终的目的地都是临沂网戒中心。<br/><br/>在网戒中心笔记本前面简介的家长守则中有这样一条：早预防，早发现，早处理。<br/><br/>就是一有不好苗头要赶紧向杨永信报告，并且在孩子不知情的情况下尽早带到网戒中心，如果孩子有抵触情绪，可以求助于“爱心家长”，强制带回网戒中心。此条款不仅适用于未成年盟友，同样适用于成人盟友。<br/><br/>而且在相关规定中，电击治疗是需要本人同意的，而在网戒中心里是“被同意”，在新盟友的入院“体验”中点评师会问你，“你能不能好好配合在这里治疗”？如果你回答的是”不能“那么你将面对的是无休止的”治疗“，直到你同意为止。个人认为已经侵害人权。<br/><br/>而且网戒中心的制度就是进去容易，出来难，入院基本不需要任何检查，只需要家属签一份入院协议，就可以入院成为一名新盟友。而成为盟友之后，你就完全丧失了人身自由和言论自由，不然面对的就是杨永信的电击治疗。<br/><br/>想必大家都看了我之前的《我在临沂网戒中心的真实经历》，里面的情况我也不多说，而且，曾经因为逃跑而跳楼受伤的不胜奇数，个人认为，临沂网戒中心的行为已经严重违反知情同意原则，并且已经侵害人权。<br/><br/>―――――――――――――――――――――――――――――――――――――――――――<br/><h2>3. 合法性原则</h2><br/>本人并非法学专业，但是也咨询了很多的律师和法学院的大学生。<br/><br/>就”网瘾“而算，医学界还没有共同的共识，我也不作深究，但是游手好闲，逃学厌学，早恋同居，同性恋，淡漠亲情，甚至晚婚都算精神类疾病吗？ 如果不算，那么网戒中心收治此类是否已经违反法律？把所谓的戒网瘾科室挂靠在精神病院是否合理？如果上述情况都算精神类疾病，那么临沂网戒中心是否有收治的资格？<br/><br/>据我所知精神病院的相关规定很多，比如治疗室需要有监控设备，而网戒中心的监控设备全在走廊，作用更像是监控盟友行动，而不是公开治疗。而且收治精神类病人需要严格的体检和测试，不经任何体检和测试直接入院，是否符合标准？<br/><br/>此外本人的病例中写的是认知和冲动障碍，希望相关专家给予解释，此障碍是否为精神类障碍？<br/><br/>因为本人对相关的法学知识实在薄浅，只能提出此类问题，希望相关人士证实。<br/><br/>―――――――――――――――――――――――――――――――――――――――――――<br/><br/>此上内容中关于临沂网戒中心内所发生事情都为本人亲身经历或离院盟友讲述，并已存好相关证据<br/><br/>―――――――――――――――――――――――――――――――――――――――――――<br/><br/>在之前《锵锵三人行》中，社会学专家周老师提到此机构有邪教性质，希望政府方面尽快解决，并完善此方面规定，并且引导青少年正确健康接触网络，希望我们的社会明天更美好！<br/><h1>后记</h1><br/>自本人发出《我在临沂网戒中心的真实经历》之后，家庭和个人均收到各方面的威胁，对是否杨永信煽动家委、家长对本人进行人身威胁保持怀疑态度，对众多家长，家委非法侵入本人家中，语言威胁和人身攻击的行为保留上诉的权利。<br/>首次转载时间:
