山东临沂精神病医院杨永信自创“醒脑疗法”被卫生部紧急叫停</span><br/><br/>国内网瘾治疗流派混乱专家称大多不能治根反而会造成二次伤害　电刺激厌恶疗法，就是将被治疗者的不良行为与电刺激之间建立起条件反射，一旦这一不良行为出现就予以电刺激，使被治疗者产生厌恶体验。<!--more--><br/><br/>卫生部日前发出通知称，“电击治疗网瘾”技术的安全性尚不确切，暂不宜应用于临床，因此要求停止该疗法的临床应用。这一事件缘起山东临沂市精神病医院医生杨永信利用电击治疗网瘾所引发的诸多争议。有亲历者形容，这一治疗方法“生不如死”。<br/><br/>如何治疗网瘾才是合适的、有效的方法？卫生部叫停了电击治疗方法，其他的方法是否仍然有效？“这也是新问题，我们也在探索和研究之中。”卫生部新闻发言人毛群安昨天对早报记者说，这些问题请专家来解释更合适。<br/><table border="0" cellspacing="0" cellpadding="0" align="left"><br/><tbody><br/><tr><br/><td>&nbsp;</td><br/></tr><br/></tbody><br/></table><br/>《中国青年报》不久前的三篇文章，把“全国戒网专家”杨永信的治疗网瘾模式拉到了聚光灯下。<br/><br/><strong>网友形容“生不如死”</strong><br/><br/>5月，中国青年报《一个网戒中心的生态系统》、《谁都想在网瘾治疗市场分杯羹》、《“戒网专家”杨永信电击治网瘾引发争议》连续见诸报端，掀起有关“电击治网瘾”风波。<br/><br/>山东临沂的这家网戒中心全名叫“中国杨永信网络成瘾戒治中心”。杨永信，临沂本地人，1962年出生，已在第四人民医院(前身为临沂精神病医院)工作20余年，专业是心理精神卫生。<br/><br/>报道中称，杨永信独创的“醒脑疗法”是用1毫安～5毫安的电流通过脑部，这种刺激治疗网瘾，会造成疼痛，但非常安全，不会对孩子造成任何伤害。具体的操作是，他在网瘾孩子太阳穴或手指上接通电极，以电流刺激脑部进行治疗。<br/><br/>据说，目前有100多名孩子留在杨永信所在的山东省临沂市第四人民医院网瘾戒治中心接受这种治疗，不能擅自离开。治疗方式为电击，并服用精神类药物，孩子入院时大多被强行押制。在这里“治疗”过的“网瘾孩子”，据杨永信介绍，已经近3000人。<br/><br/>虽然杨永信称“醒脑疗法”很安全，但经历过这种疗法的网友却形容“让人生不如死”，更有网民形容杨永信的电击治疗网瘾方法虐待患者毫无人道。“电头的感觉就像用毛线针从太阳穴一边扎进去，再从另一边扎出来。”网友腾飞在百度“杨永信吧”里讲述被电击时的痛苦，他用“非人的日子”形容自己在网戒中心的经历。<br/><br/>曾有在网戒中心“表现出色”的孩子离开后，在网上发帖陈述自己的痛楚，用文字、图片、视频等方式表达对杨永信的憎恨，有的甚至扬言要报复。在“杨永信吧”1万多个主题、8万多篇帖子中，有一半以上帖子恶搞、咒骂杨永信和他的网戒中心，且言辞激烈。<br/><br/>据《重庆晚报》报道，不少曾接受治疗的患者在接受采访时也表达了类似看法，形容网戒中心“和传销一样”，如同地狱。<br/><br/>13日，卫生部以“安全性有效性不确切”为由，叫停“醒脑疗法”。<br/><br/>据了解，卫生部要求山东省卫生厅停止辖区内电刺激治疗网瘾技术的临床应用，若开展科学研究，应按规定申报，经批准后，须尊重受试者知情权和选择权，不得收取相关费用。<br/><br/><strong>　临沂已停“电击疗法”</strong><br/><br/>早报记者昨天下午致电山东省临沂市精神病医院，据介绍，目前该医院已经停止了电击治疗网瘾。而据《新京报》报道，山东省临沂市精神病医院宣传部负责人杨淑云称，之所以停止电击治疗网瘾，是因为舆论压力大，但是戒网中心仍在招收戒网者。杨淑云称，戒除网瘾治疗是包括药物、心理干预等多方面的工作，电击只是其中一部分。该报记者以消费者身份向网瘾治疗中心咨询，工作人员说，如果治疗网瘾，每人的治疗住院费每月约5500元。至于是否使用电击疗法，她纠正说应该叫“脉冲疗法”，只有在特殊情况下才可能使用。<br/><br/>昨天，杨永信在接收到早报记者电话后发来短信：“对不起，我不方便接电话，请发短信好吗？”不过，当早报记者把相关问题发给他时，并没有收到答复的短信，手机仍然无人接听。当早报记者多次拨打手机后，对方回复短信说：“对不起，我是他的同事，他现在正忙。”此后手机同样无人接听。<br/><br/>在中国青少年网络协会的专家顾问名单里，杨永信位列其中。其名单里专家顾问有8位，包括华中师范大学特聘教授陶宏开、北京军区总医院网瘾咨询中心主任陶然等。<br/><br/>在中国青少年网络协会对杨永信的介绍里，有这样一段意味深长的话：对神经病学研究深入的他，却对网瘾问题研究欠佳。在对网瘾少年的治疗中，依然采用治疗神经病的手段和方式。甚至滥用电击、殴打、下跪、监禁等方法。严重侵害了孩子的人权，受到很多家长的抗议，也涉及了相关法律问题。2008年7月2日《战网魔：谁把天才变成了魔兽》播出，杨永信开始“走红”。<br/><br/>介绍的最后一句是：“从早年优异的工作表现，到晚年偏执所犯下的大错。大起大落的人生，更使他成为了公众的焦点以及最有争议的人物之一。”<br/><br/><strong>孩子的皮肤被烧焦</strong><br/><br/>国家注册心理咨询师，多年从事网瘾治疗的上海专家郭铁军昨天在接受早报记者采访时称，电击疗法被卫生部叫停，可能有三方面原因：一是被叫停的山东这家医院的电击疗法，并未向相关部门报批，程序上不合规定。<br/><br/>其次，目前尚无任何根据可以证明电击疗法对治疗网络成瘾有效。郭铁军介绍，电击疗法在其他的精神类疾病中是一种较为成熟的医疗手段，简单点说，其原理是通过刺激大脑中主管人情绪的某些区域，以改变或者抑制患者的某些情绪状态。“但是网瘾比较特殊，现在业界对网瘾的界定都不统一。有可能使用电击疗法的机构按照自己的标准觉得这样治疗有效，但是它也可能伤害到被治疗者。”<br/><br/>再次，即使该单位报批了，电击疗法带有实验性，必须征得当事人同意，而且不能收费，而实际操作中被叫停的单位可能涉及收费问题。<br/><br/>昨天正在北京出差的华中师范大学特聘教授陶宏开也接受了早报记者的采访。陶宏开，被誉为“中国戒网瘾第一人”，面对面挽救了500多名网瘾未成年人。<br/><br/>“看了这样的治疗方法，我真的很愤怒很愤怒。”谈到电击治疗网瘾，他说：“这样能戒掉网瘾吗？不仅不能戒掉，还会造成第二次伤害，第三次伤害。不仅如此，他们和父母的关系还会更加恶化。”<br/><br/>他谈到了他遇到的一个例子。临沂某个县的官员的儿子，曾被两次送进杨永信的网戒中心。最后家长无奈之中，又找到了陶宏开。<br/><br/>“这个孩子出来时，太阳穴的皮肤都烤焦了，皮肤红得发烫。可是父母呢，还高兴得不得了。为什么高兴？每一个出来的孩子，杨永信都要求他们见父母面时下跪、抱头痛哭，否则马上再进行电击治疗。如此一来，哪个孩子还敢不和父母抱头痛哭？”陶宏开说，他曾经劝说这位家长到电视台和他一起做节目，但遗憾的是家长有所顾虑没有参加。
