class="origin_image zh-lightbox-thumb lazy" src="https://pic4.zhimg.com/80/775c9219eb36db9a12850d5e058b2b4d_hd.jpg" width="1206" data-rawwidth="1206" data-rawheight="799" 拍摄/触乐</i><br/>“我们上去吧。”抽完一支烟，王安对我说。在2008年离开网戒中心后，他仍留在本地生活。8年中，他无数次需要经过这里，始终选择绕更远的路。<br/><br/>我开始有点担心，周围会不会有家长的暗哨在看守。我们是今天早上从北京飞来的，两个人，到达临沂后，我们在四院附近住下。中午，我和同事出门，在四院门口，拿出相机拍了两张照片，突然就被不知从哪里冒出的家长们围住了，他们要求查看证件，并删除照片。直到报警才得以逃脱。<br/><br/>那之后，从四院门口经过，始终有眼睛在暗中注视着我们。<br/><br/>幸运的是，晚上并没有暗哨。我们在楼底下转了转，楼后面是一片空地，盟友们每天早上在这里跑操，王安说这里当年有个小屋，是电脑室，现在被拆了。他向我们指出二楼一扇没有开灯的窗户，那里漆黑一片。<br/><br/>“这就是13号室。”他说。<br/><figure><img class="origin_image zh-lightbox-thumb lazy" src="https://pic1.zhimg.com/80/f7f61da8def69572d188644b8e3df1a2_hd.jpg" width="3264" data-rawwidth="3264" data-rawheight="2448" data-original="https://pic1.zhimg.com/f7f61da8def69572d188644b8e3df1a2_r.jpg" data-actualsrc="https://pic1.zhimg.com/f7f61da8def69572d188644b8e3df1a2_b.jpg"></figure><br/><i>2楼两扇亮着的窗户中间，是漆黑的13号室 拍摄/触乐</i><br/>我们开始上楼。楼梯拐角贴着网戒中心的宣传画，最大的一张照片上，杨永信正在带领当地小学生们参观“网瘾孩子”的宿舍，周围的小照片里，杨永信在各种大学和政府的报告会上做着演讲。其中最吸引我的一张照片，“网瘾孩子”和小学生们面对面站在一起，一条分明的线隔开了他们，照片下的解说词写着“网瘾孩子和临沂晚报小记者们亲密互动”。<br/><figure><img class="origin_image zh-lightbox-thumb lazy" src="https://pic4.zhimg.com/80/4099bb3964468e01fc28895218fae90b_hd.jpg" width="3264" data-rawwidth="3264" data-rawheight="2448" data-original="https://pic4.zhimg.com/4099bb3964468e01fc28895218fae90b_r.jpg" data-actualsrc="https://pic4.zhimg.com/4099bb3964468e01fc28895218fae90b_b.jpg"></figure><br/><i>“杨永信带领小学生们参观网瘾孩子的宿舍”</i><br/>二楼和三楼在入口和楼梯之间被两道铁门隔开，铁门里是宿舍，最外侧的铁门上有方形的窗户，我们向里看，两扇门之间摆着两把椅子，可以想象，这里平时有两个人守着。透过铁门上的窗户可以看到更里面的情况，少年们正列队站着，站在最外侧的少年突然侧头向外看了一眼，我们的目光对视，我从他的脸上看不到表情。<br/><br/>四楼是普通的精神诊室，只有一道铁门。<br/><br/>我们停留了十分钟，担心会被人发现，开始下楼。走出大楼后，我们慢慢走回车里，王安点了一支烟，他沉默了一会，对我说，“刚刚从四楼走下来的时候，我的腿一直在抖。”<br/><br/>王安曾以“网瘾患者”之名，<a class="internal" href="https://www.zhihu.com/question/24183980/answer/116288345" data-za-detail-view-id="1043">在知乎回答自己的经历</a>。以下为采访实录。<br/><br/><strong>触乐</strong>：您是哪一年进被送到网戒中心的？<br/><br/><strong>王安</strong>：2008年，那时候我还上高中，当时属于他鼎盛的时期，一共有80多名学员。<br/><br/><strong>触</strong>：生源大部分来自本地还是自外地？<br/><br/><strong>王</strong>：外地，本地的很少，当时网瘾这个概念比较火，杨永信和陶宏开他们两个人都很出名，陶宏开那时候也很厉害。<br/><br/><strong>触</strong>：是因为玩什么游戏被送进去的吗？<br/><br/><strong>王</strong>：不仅是玩游戏，我那时候还上高二。玩游戏还在其次，主要是我那时候不怎么回家，和我当时女朋友在外边同居，当然也不怎么去上课了，主要是因为这些。玩游戏？也玩，玩玩魔兽。<br/><br/><strong>触</strong>：就您所知，当时的学员在里面最长待过多长时间？<br/><br/><strong>王</strong>：每个人进去，时间都不会太长，一般都是两三个月。但是有反复进去的，我也反复进去过，进入了，出院了，感觉不行了，再进去。然后每次送你去就是更厉害了。<br/><br/><strong>触</strong>：更厉害是指？<br/><br/><strong>王</strong>：被管得更厉害，待的时间也更长了。当时有个人在里面应该待了半年多吧，我进去的时候，他已经待了半年多了，我出院之后他还在。<br/><br/><strong>触</strong>：怎样才能出去？<br/><br/><strong>王</strong>：一般按时间，两三个月，最主要的还是杨的评价，一般到待了3个月的时候，家长会去找他。<br/><figure><img class="origin_image zh-lightbox-thumb lazy" src="https://pic3.zhimg.com/80/acf5c0bd898a7b5c270d2735899bf556_hd.jpg" width="690" data-rawwidth="690" data-rawheight="459" data-original="https://pic3.zhimg.com/acf5c0bd898a7b5c270d2735899bf556_r.jpg" data-actualsrc="https://pic3.zhimg.com/acf5c0bd898a7b5c270d2735899bf556_b.jpg"></figure><br/><i>学生参观网戒中心 照片来自杨永信微博</i><br/><strong>触</strong>：据我所知的话，应该是家长跟孩子一起陪护，家长也住在陪护中心吗？<br/><br/><strong>王</strong>：对。医院里有那种陪护床，你知道吗？就是一个小床，病床旁边有一个折叠的那种椅子。晚上放倒了，孩子在椅子上睡，家长在床上睡。<br/><br/><strong>触</strong>：家长可以去电击室观看电击吗？<br/><br/><strong>王</strong>：我第一次去的时候就不允许家长进，他们好奇想进去看，不行。<br/><br/><strong>触</strong>：你觉得家长会知道你们接受电击时是怎样的感受吗？<br/><br/><strong>王</strong>：哪怕是我这样跟你说你也体会不到，我来路上我还在想，就是你说你要进去看一下，我在想，您不去体会一下，我确实是没法给你表达的，没法用语言去表达。<br/><br/><strong>触</strong>：有家长曾经因为后悔，然后带着孩子走吗？是因为什么原因？经济负担比较大？<br/><br/><strong>王</strong>：经济上一般没有，因为在咨询的时候他肯定就确认过，基本上没有经济上的问题。因为治疗也不多，主要是因为他不认同杨那一套。<br/><br/>因为家长不了解在这里是什么感觉，哪怕孩子说给他，就像我现在给你这样说，您也体会不到，我跟你说我操他妈太疼了，你也没法体会那种操他妈是怎样一种操他妈。<br/><br/><strong>触</strong>：那是你一生当中经历过的最疼痛的感受？<br/><br/><strong>王</strong>：它不仅是疼痛，他妈的那是心灵打击。就是让你觉得，嗯，我操，我确实错了，单纯那种疼，就像纹身之类的那种是可以忍受的，我觉得可以忍受，后来我去纹身，我觉得可以忍受。那个是你说1+1等于3，我操你妈你说的对。<br/><br/><strong>触</strong>：在你的眼里面杨永信是一个什么样的人？<br/><br/><strong>王</strong>：我一年前一直会觉得它就是一个坏人。但是一年前我开始接触心理学，我很认同的一句话就是存在即合理。那天我又去翻他的微博，看他所有的东西，他真的就是一本道，一本正经在那胡说八道，他自己可能真的就觉得他是神，他来拯救我。<br/><br/>而且你别忘了，他每天就生活在那样的环境下，所有人供奉他，就把他奉为神，你要知道人久而久之，在这样的环境下，难免它会自己膨胀，所以其实我也挺同情他，他明明在干坏事，就像希特勒一样。<br/><br/>我住院的时候，发生了一件事，他没有出名之前接触的大多数学生都是我们本地的，后来有一次袭击未遂，我还挺清楚，这么多年我也清楚那场点评课，我记得他讲着讲着，他妈逼自己哭了。我操，他说我为了你们为了这件事，我现在自己受到威胁，就这种感觉，然后自己哭了，然后他带着整个教室，当时160个人一起哭吧<br/><figure><img class="origin_image zh-lightbox-thumb lazy" src="https://pic2.zhimg.com/80/9765b5db58ba4e3e01dc235b699a905d_hd.jpg" width="690" data-rawwidth="690" data-rawheight="459" data-original="https://pic2.zhimg.com/9765b5db58ba4e3e01dc235b699a905d_r.jpg" data-actualsrc="https://pic2.zhimg.com/9765b5db58ba4e3e01dc235b699a905d_b.jpg"></figure><br/><i>网戒中心的孩子们在这里庆祝过年 照片来自杨永信微博</i><br/><strong>触</strong>：你当时是哪一种？是被氛围感染，还是不得不跟他哭？<br/><br/><strong>王</strong>：两者都有，每节课差不多有两个小时。而且他这么多年实践之后，他对心理的把握和引导非常在行，他去引导你。如果你真的放下戒备心去听他的引导，你很容易信服他。我也就是这样一次一次一点一点洗脑，然后洗到最后，其实第一次出院之前，有那么一段时间我基本上认同了他。<br/><br/><strong>触</strong>：当时确实觉得自己做错了，需要被电击？<br/><br/><strong>王</strong>：有那么一段时间，确实是觉得。<br/><br/><strong>触</strong>：但现在不这么认为了？这其中发生了什么？<br/><br/><strong>王</strong>：我出来之后，以前的朋友全不联系了，女朋友也更不能联系了，基本上和以前断了关系，我家里人给我重新换了一个学校，你接触所有人都要拿那种，啊，我操，好学生，我要好好学习――其实所有都是装的，你明明知道我不是那种人，但你每天都要去做那种事情，包括在学校，在班里，再回到家的时候。<br/><br/>其实我觉得越到回家的时候越应该当自己，在家人面前你没有必要的，结果回家之后装得更厉害。从到家到睡觉之前，基本上都不是自己，然后就慢慢慢慢的感觉这样不行。<br/><br/>进去之前，我除了没动手，基本上什么都干了。出来之后基本上都改了，有很多次都忍住了。但是不是因为别的，就是因为害怕。怕他们再因为我吵架然后再把我抓进去。<br/><br/><strong>触</strong>：之后你就离家出走了？<br/><br/><strong>王</strong>：对。我在出走之前的时候，已经偷偷联系了女朋友，跟她一起走。那时候拿了家里1万块钱，然后我去打过很多工，最后还是没钱了。但是我女朋友也没有身份证，我女朋友她妈给她打钱，当时他妈还比较清楚，然后也比较阔绰，她给我们打了2000块钱。<br/><br/><strong>触</strong>：你离家出走了多久？<br/><br/><strong>王</strong>：半年，也不是打工一次都没成功，你有的时候也能拿到工资，只不过很少，不足以支撑我们两个人在一起生活。<br/><br/><strong>触</strong>：然后你就直接回到家里了？<br/><br/><strong>王</strong>：没有，那段时间，我操，现在想起来比我私奔之后更难受。我就在临沂，随时都可能再被抓，那段时间，其实……我操，我后来不想再想起来这件事，就是那段时间是最难受的。因为没有钱，我们一直住在她家在郊外的一套别墅里面。每天，我需要把所有的门全部锁起来，所有窗子全都锁上，别墅一共3层，上面带一个小阁楼，每天都要去阁楼里边睡，因为我觉得那下边有声音的话我能听到。阁楼没有空调，反正就是各种受罪觉得都没问题，只要让我觉得安全一些。<br/><br/><strong>触</strong>：这个状态一直持续了多长时间？<br/><br/><strong>王</strong>：具体多长时间不太清楚了，反正就差不多一两年，然后又被逮进去了。<br/><br/>其实那时候他们尝试过给我发短信，我也想过我要不要回去，他们给我发短信说你回来上课。现在我是可以上学，但当时不行。哪怕我女朋友跟我说让我回去上课，不行，我去上课，更容易随时就把我带走了。哪天表现不好，比如期末考试考不好，就完了，说不定我期末考试刚考完最后一场，他们就在教室门口等我，该放暑假了，再进去蹲两天吧。<br/><br/>那时候基本这么一折腾就折腾到高三下半学期的时候，然后快高考了，那时候别墅也不住了，哪也不去了，就去网吧。有时候会求助我表哥，他们会给我打点钱过来。<br/><br/>他们也会让我去他们那边住，我不去，最多就是去他们那拿钱的时候，哪怕是去我哥家拿钱去，说好一块喝茶。我操，我都先和他约好了，我们俩在a见面，然后快到的时候，我说咱到b去，就这样，就没有任何人是我信任的。<br/><br/><strong>触</strong>：你对自己父母的感情，过了这么长时间，感觉会有变化吗？<br/><br/><strong>王</strong>：上星期我姐，我有个亲戚，给我打电话说咱爸妈是不是吵架了，我就说吵就吵吧，反正家庭关系很微妙。就是不想回去了，然后我宁愿在外边打地铺，我睡大街我也不要回去。<br/><br/><strong>触</strong>：他们会请求你回去吗？<br/><br/><strong>王</strong>：会。<br/><br/><strong>触</strong>：他们后来对自己当时把你送进去是什么看法呢？会跟你说吗？<br/><br/><strong>王</strong>：有一次我说过，我喝多了，我那天晚上喝多后给他们打电话说这事。他说我们当时也没有办法，基本上就是这一句话，我们也没有其他办法，如果有其他办法，也不会把你送进去。<br/><br/>但我说，你至少应该，你如果稍微对我负责一点的话，你应该了解清楚里边怎么样，再把我送进去，因为我父亲在临沂也认识一些人，我的意思是，我爸如果想了解清楚的话，很简单，但他就直接把我送进去。<br/><figure><img data-original="https://pic1.zhimg.com/01fe4260e53f7231455c2b49cd6c82a5_r.jpg" data-actualsrc="https://pic1.zhimg.com/01fe4260e53f7231455c2b49cd6c82a5_b.jpg"></figure><br/><i>杨永信接受家长们送上的锦旗 照片来自杨永信博客</i><br/><strong>触</strong>：谈谈在网戒中心里面的事情吧，你觉得网戒中心给你带来的最大影响是？<br/><br/><strong>王</strong>：我觉得他破坏了我和家里人的信任，这种感情。有一次我犯了事儿被电，那个房间在2楼，我妈就在楼下，我看到她，她就在那，她在那喊，当然她也生气了，她说让杨永信加大剂量电死我。<br/><br/>我到现在为止……其实有的时候我再看看我妈，我觉得她挺可怜的，她没犯什么错，这个决定也不是她做的，她不想把我送去，但我还是不能原谅她。就因为这些，我觉得你不应该在这个时候告诉他说电死我，所以你说我能原谅他吗？不能（沉默很久）。<br/><br/>很多这样的事就是说，有人刚进去不太懂规则，他们都很聪明，会想我表现好了就能走。有那种很聪明的孩子，过了一个星期，就会给他爸妈说，我差不多了，我以后听话，然后就想走，然后他妈就去找杨永信，会告诉他怎么回事，然后回来就把那个孩子拉去了。电完了，问你是不是想走，你是不是跟你妈说什么什么。<br/><br/>他明明当时偷偷给他妈说，我受不了，你们带我出去，我回家听话，还是偷偷给他妈说的，我觉得这些关系全部是破坏了孩子与家长最基本的信任，我明明偷偷地告诉你的事，然后你去给杨永信说了，杨反过来就拉走他们，你觉得他以后还能信任家长吗？<br/><br/><strong>触</strong>：这个问题可能有些冒犯，您说您也是班委，那么您也曾经协助电击其他学员？<br/><br/><strong>王</strong>：会，后来我成为班委之后，这些都会，人到时候来了，包括大晚上来了，凌晨来了，基本上后来每次都是我们。<br/><br/><strong>触</strong>：你当时是什么感觉？<br/><br/><strong>王</strong>：没感觉，就是确实是没有，甚至到后来之后，就是出来之后，我学了心理学之后我才会想那是有多可怕。有一个学员在里面，我们一般会有那个护牙的，就是那个让你牙咬着，一个橡胶的那种。那个学员也挺倔，他是刚刚去当学生，就觉得就是怎么样都不咬。<br/><br/>然后他把牙咬碎了，他把自己的牙咬碎了，那是我按着的，当时我们没有一个觉着这事很严重，我当时的感觉就是，傻逼，我让你咬你不咬。反正就是没有同情心，就不会觉得他这是很可怜，并不会觉得他这是，就觉得你是自找的，并不觉得杨有错。<br/><br/><strong>触</strong>：所以看起来你的确被网戒中心改变了一些。<br/><br/><strong>王</strong>：会，我第一次出院，我也觉得挺好，刚出院一两个星期那种就像打鸡血一样。那时候我也觉得挺健康的，但是那种状态很脆弱，你经不起打击，我能一直获得成功的话，就能一直坚持住这个状态，但是一旦遇到一点失败，一点一点打击，我就……甚至有吸毒的冲动。<br/><br/>一开始的时候，一开始几个月，身体锻炼的很好，然后呢，出来几个月后就白搭了，那就是噩梦。包括我现在也会做噩梦，09年的时候，我忘了在哪，反正是在一个酒店里边，是看电视的时候，无意看到那个，看到柴静那个报道，就真的是无意间，我换台的时候就看这张脸，然后我就愣住了。我操，脑子就，真的我从来就……一开始也没体会过，从那之后，我也没再体会过什么叫放空自己。我靠，那是真的，就看到这张脸我就放空了，然后等我回过神来之后，发现我把电视砸了。<br/><br/>那时候已经过了一年了，一年多了，等我再有意识的时候，这些事已经干了，我就已经把电视砸完了，然后赔了酒店一个电视，然后也就是从那个时候，我操，以前我的睡眠出奇的香，从那之后就完全白搭。<br/><br/><strong>触</strong>：现在还会失眠吗？<br/><br/><strong>王</strong>：不仅会失眠，至少一个月还会梦见一次。越是有事的时候，就比以前更想去逃避了，不仅是逃避那么简单，哪怕现在工作之后，越是不顺心的时候，我也没法跟家里说，也没法跟朋友说，我所有朋友都不知道我去过那，我拒绝和别人提起这个，所以他们也没法感同身受。<br/><br/><strong>触</strong>：尝试接触过心理医生吗？<br/><br/><strong>王</strong>：去过，但是临沂这个城市太小了，医生都没法感同身受。我去过一趟北京，在医院门口的时候，所有挂号那些手续全都弄完之后，到门口我转身就走了，还是不太想。<br/><br/>我是无意间看到知乎这个问题的，我在食堂吃饭，刚打完饭端着盘子，刚放下，然后翻知乎的时候第一个就是。我点开看了一眼，看着看着就不行了。<br/><br/>我当时饭都没吃完，那整个晚上就我在食堂，我操，你说我让人看见一米九高个，在那就哭得不行了，就那时候，我都已经……我在一家事业单位，很封闭，这种事会传得很快，所有人都会知道你不知道因为什么事在食堂哭了，我那时候也顾不了，我操，我就吃到一半就止不住了，把手机关了也不行，然后把饭倒掉之后就上楼去了，整个晚上都恢复不过来。如果不是因为看到这个问题，哪怕是我知道你们来，我也不会联系你们。<br/><br/><strong>触</strong>：你曾经想过从法律程序上解决这个问题吗？<br/><br/><strong>王</strong>：我想过，我甚至咨询过了一些，其中有两个律师，但是可能是我没找对或者怎样，我觉得这种事小律师解决不了，那种纯粹为了钱的，为了谋生的律师，我去找他，他不会用心，除非那种非常有责任感，就觉得这件事我哪怕不要钱我也要去做的那种……我觉得现在还在灰色地带里边，你说大家看法，他犯法，但说不犯也没什么问题。<br/><br/><strong>触</strong>：假设说杨永信最后真的被人告倒了，或者网戒中心关掉了，然后他也被制裁了，你会觉得好受一点吗？<br/><br/><strong>王</strong>：反正我觉得至少，至少如果在我家里有矛盾的时候，我觉得我能挺直腰板了。<br/><br/><strong>触</strong>：你的父母除了说当时自己没选择之外，他们有为这件事情直接道歉过吗？如果他们直接说自己当时做错了，你也会好受一点吗？<br/><br/><strong>王</strong>：好受难受，我现在应该不太在意。我只是觉得这件事永远我也没法原谅他们。8年前是这样的态度，现在还是。<br/><br/>其实从那之后改变了，我操，确实是把我整个人都改变了，我以前是内向，现在也不能说外向，反正就是真正的双重性格，我见人的时候确实挺外向的，然后回家了对着家人，比较亲近的人，就是不爽。一般人都是刚见面的时候生分，熟了之后逗逼，我正好调过来。<br/><br/>所以我朋友不多，他们觉得我对他们不自觉就生疏了，他们觉得我觉得他们其实并不重要了，就这样。然后我不想要孩子，做事非常集中，很多事我会去赌，我会把所有筹码都拿去赌。比如说一个东西，拿一点进去蹲两三年，拿所有必死无疑，我肯定会把所有东西全部拿走，要不就不做。不怕死了，必须要把你带走。这个想法从来没有实践过，但就是死之前一定把你带走，然后等死。<br/><br/>注：为保护当事人，王安为化名。<br/><blockquote><strong>专题：“被杨永信改造的孩子们”</strong>・<a
