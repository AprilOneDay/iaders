<br/>请求对于临沂网戒中心涉嫌犯法一事的信访处理结果进行复查<br/><br/>本人于十月二十号收到了信访的回复, 因信访的回复对于我投诉的许多地方没有作出任何回应, 且内容明显转移话题甚至歪曲事实, 又不能给出令人信服的解释, 本人不服本次信访的结果, 依据信访条例的规定, 现在向山东省卫计委提出复查申请.<br/><br/>原因如下:<br/><br/><!--more--><br/><br/>1. 对智力残障, 同性恋, 自闭症乃至在因在恋爱, 工作学习等方面没有听从父母的孩子进行所谓的治疗, 请问这是否合理?<br/><br/>2. 根据精神卫生法第四十条, 精神障碍患者在医疗机构内发生或者将要发生伤害自身, 危害他人安全, 扰乱医疗秩序的行为, 医疗机构及其医务人员在没有其他可替代措施的情况下, 可以实施约束, 隔离等保护性医疗措施.?实施保护性医疗措施应当遵循诊断标准和治疗规范, 并在实施后告知患者的监护人. 禁止利用约束, 隔离等保护性医疗措施惩罚精神障碍患者.<br/><br/>根据精神卫生法第七十五条,, 医疗机构及其工作人员有下列行为之一的, 由县级以上人民政府卫生行政部门责令改正, 对直接负责的主管人员和其他直接责任人员依法给予或者责令给予降低岗位等级或者撤职的处分; 对有关医务人员, 暂停六个月以上一年以下执业活动: 情节严重的, 给予或者责令给予开除的处分, 并吊销有关医务人员的执业证书:<br/><br/>(一)违反本法规定实施约束、隔离等保护性医疗措施的;<br/><br/>(二)违反本法规定，强迫精神障碍患者劳动的;<br/><br/>(三)违反本法规定对精神障碍患者实施外科手术或者实验性临床医疗的;<br/><br/>(四)违反本法规定，侵害精神障碍患者的通讯和会见探访者等权利的;<br/><br/>(五)违反精神障碍诊断标准，将非精神障碍患者诊断为精神障碍患者的.<br/><br/>根据精神卫生法第七十八条, 违反本法规定, 有下列情形之一, 给精神障碍患者或者其他公民造成人身, 财产或者其他损害的, 依法承担赔偿责任:<br/><br/>(一)将非精神障碍患者故意作为精神障碍患者送入医疗机构治疗的;<br/><br/>(二)精神障碍患者的监护人遗弃患者，或者有不履行监护职责的其他情形的;<br/><br/>(三)歧视、侮辱、虐待精神障碍患者，侵害患者的人格尊严、人身安全的;<br/><br/>(四)非法限制精神障碍患者人身自由的;<br/><br/>(五)其他侵害精神障碍患者合法权益的情形.<br/><br/>根据中华人民共和国刑法第238条规定, 非法拘禁罪为非法拘禁他人或者以其他方法非法剥夺他人人身自由的行为. 在认定非法拘禁罪时, 在客观方面拘禁行为必须具有强制性, 即指违背他人意志, 强行使他人处于被管束之中. 主要表现为使用足以剥夺人身自由的强制性手段, 如实施绑架, 关押, 禁闭等.<br/><br/>而网戒中心切切实实存在强迫治疗的行为, 严重限制人身自由, 禁止外出, 通信, 甚至包括内部的交流, 并且在治疗的过程中不乏各种多变残忍的惩罚措施. 无论对于未成年人还是成年人, 这是否违背了医学伦理, 是否是对于人权乃至人格的公然侮辱?<br/><br/>3. 杨永信并不具备职业药师资格, 针灸师资格, 请问其是否有权开发售卖精神类药物, 并进行针灸等治疗行为?<br/><br/>4. 根据精神卫生法第七十六条, 有下列情形之一的, 由县级以上人民政府卫生行政部门, 工商行政管理部门依据各自职责责令改正, 给予警告, 并处5000元以上10000元以下罚款, 有违法所得的, 没收违法所得; 造成严重后果的, 责令暂停6个月以上1年以下执业活动, 直至吊销执业证书或者营业执照:<br/><br/>(一)心理咨询人员从事心理治疗或者精神障碍的诊断、治疗的;<br/><br/>(二)从事心理治疗的人员在医疗机构以外开展心理治疗活动的;<br/><br/>(三)专门从事心理治疗的人员从事精神障碍的诊断的;<br/><br/>(四)专门从事心理治疗的人员为精神障碍患者开具处方或者提供外科治疗的.<br/><br/>没有医师资格证的护士与心理咨询师是没有权利行医的, 请问杨永信手下的护士和心理咨询师为什么能够与杨永信一起进行治疗行为?<br/><br/>5. 杨永信的治疗模式中明显存在极其不正常的"个人崇拜"的迹象, 且其中的"举报"与"加圈"等机制明显有违人性, 这些做法颇有邪教作风, 为什么能够被用于治疗?<br/><br/>6. 杨永信会与盟友的别动队从公共场所抓人. 那么杨永信有从公共场所强行将人带走治疗的权利吗? 家委会会对"再偏"者进行抓捕, 他们又有这个权利么?<br/><br/>7.?根据精神卫生法第十七条医务人员开展疾病诊疗服务, 应当按照诊断标准和治疗规范的要求, 对就诊者进行心理健康指导; 发现就诊者可能患有精神障碍的, 应当建议其到符合本法规定的医疗机构就诊.<br/><br/>结合上文中的第七十五条与第七十八条, 将非精神障碍患者故意作为精神障碍患者送入医疗机构治疗的行为与对违反精神障碍诊断标准，将非精神障碍患者诊断为精神障碍患者的行为是被命令禁止的.<br/><br/>而网戒中心对于网瘾的治疗依据是什么, 目前国际上对于网瘾一词本身就尚无定论, 网瘾尚未被列入精神障碍诊断标准之中, 无论是国际通用的精神疾病诊断与统计手册, 还是中国精神障碍分类与诊断标准, 都未将网瘾列入其中. 国际上针对网瘾的试验也宣告失败, 证明其并不是精神疾病.<br/><br/>那么杨永信到底在用什么标准? 他的所谓诊断与治疗真的科学可靠吗? 对于初到的"患者", 杨永信会根据家长的一面之词直接判断孩子有网瘾并直接带走治疗, 请问这么简单粗暴的治疗方式真的科学可靠吗?<br/><br/>8. 根据杨永信的说法, 网瘾是一种极其严重的精神疾病, 那么他是怎么做到在初次见到患者不久就在仅仅四十分钟内, 把他口中所谓的严重的, 性格刚烈的"患者""治"得服服帖的呢? 这真的是奇迹吗, 请问医学会相信奇迹吗?<br/><br/>9. 杨永信等人从前自称的2/58/108种治疗手法环环相扣缺一不可, 那么他是怎么做到将他收治的自闭症, 多动症, 乃至网瘾等层出不穷得患者纳入同一种治疗方法的, 请问杨永信是万能的吗?<br/><br/>10.回复称低频脉冲治疗仪是合法的, 但是治疗手法与治疗用途是否同样合法?<br/><br/>①. 对于治疗仪的超额, 并联使用, 势必会对患者造成极大的痛苦. 请问, 杨永信等人的治疗手法真的合理合法, 不会违背医学伦理吗?<br/><br/>②. 低频脉冲治疗仪是中医用以缓解肝病症状, 调理身体的, 作用为刺激肌肉组织, 促进局部血液循环以及镇痛. 精神科医生使用其进行所谓"开创性"治疗用途, 真的安全吗?<br/><br/>③. 治疗以前不会做详细的检查, 请问这种明显违反规范的做法是否安全合理?<br/><br/>11.网戒中心对外从不公开其治疗的任何细节信息, 请问我们又如何了解患者到底是不是被动屈服的?<br/><br/>12.网戒中心的收费是否合理, 对家长是否存在强迫收费? 请问收费最终流向了何处, 网戒中心是否具有强迫收费的权利?<br/><br/>13.临沂市网络成瘾戒治中心作为一家医疗机构, 并没有办学资质, 然而却有授课行为, 并且孩子们"按照医院要求", 在治疗期间与家长留在医院, 这势必会严重耽误学业. 那么请问中心有权进行授课吗? 中心有权迫使未成年人放弃九年制义务教育吗?<br/><br/>14.根据精神卫生法第二十六条, 精神障碍的诊断, 治疗, 应当遵循维护患者合法权益, 尊重患者人格尊严的原则, 保障患者在现有条件下获得良好的精神卫生服务.<br/><br/>医疗工作者必须严格为咨询者保密, 而在"点评课"与采访中却在逼迫咨询者泄露自己的隐私并极力描述自己的种种不堪, 请问这种完全无视人格的行为真的没有违背医德吗?<br/><br/>15.同样根据精神卫生法第二十六条, 精神障碍分类、诊断标准和治疗规范，由国务院卫生行政部门组织制定.<br/><br/>杨永信等人自称"认真探索开发网瘾的诊疗模式", 请问这真的不是无视无视标准以"本土化标准"的名义行胡诊乱治之实?<br/><br/>16.临沂市网戒中心并不存在医学伦理委员会这个组织, 请问这合理吗?<br/><br/>以上内容全部经过查阅自述, 可信可靠的纪实视频, 视频采访资料, 采访文稿,
