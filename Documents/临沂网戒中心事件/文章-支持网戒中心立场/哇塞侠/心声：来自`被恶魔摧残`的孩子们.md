/><br/>听听孩子们的声音<br/><code><!--more--><br/><br/><strong>心声：来自一个“被恶魔摧残”的孩子</strong><br/><br/>在不知不觉间，自己来到临沂网戒中心已经五个月有余了。相比于一百六十多天前的自己，同样也是翻天覆地的变化。想起从前所发生的一切都还是历历在目，在我如潮的记忆中却有如最难以回首的往事一般印刻在脑海中难以磨灭。<br/><br/>那段日子中，自己如同一个失去了灵魂的木偶，自己的躯体还健在，但支持自己不断前行的方向与目标却早已丢失。在自己的脑海中所充斥的仅仅只有自己的欲望，它如同一个巨大的黑洞般在一点点的蚕食着我的青春、健康乃至生命。一叶障目的我就像身处在一片伸手不见五指的黑暗中一般，眼前充斥的仅仅是一片虚无，那盏指引我前行的、代表着希望的灯早已失去了最后的一丝光华，被我遗弃在不知名的角落，任凭时间将它淹没。身旁与四周间一种难以言喻的迷茫与恐惧弥漫着，笼罩着仅有的希望与温暖，将我引上了一条通往深渊的道路。<br/><br/>那时的自己早已休学在家，整日与虚拟的网络世界为伴，自己享受着那份虚无与空洞的快乐并深陷其中不可自拔。但一旦脱离其中自己所感受到的却仅仅是生活的冰冷与现实的残酷，巨大的落差感让我感受到的是一阵阵心悸。自己原本拥有的美好与幸福，在自己偏执的认知下早已磨灭殆尽，原本那个充满着和谐与欢笑的家庭也在自己的放纵下支离破碎，如同在狂风暴雨中被打落的浮萍一般在巨大的冲击下随波逐流，又如同在风雨飘摇中的蜡烛般转瞬即逝。充斥在整个家庭中的只有痛苦与冰冷。我，爸爸，妈妈，每个人都如同有一块巨石压在心口让人难以喘息，在空气中飘扬的也不再是欢笑与幸福，而是愈演愈烈的吵闹与饱含着无奈与痛苦的泪水......而这一切的一切的原因都是因为自己走偏，致使整个家庭蒙受了如此巨大的灾难。<br/><br/>2016年2月某日，这是我一生中永远难以忘怀的日子。这一天，我来到了中心，也结束了那段痛苦而又迷茫的日子，开始迎接我崭新的人生。而五个多月后的今天的我，在握笔写下这些的时候，心中又何尝不是感慨万千：如果没有中心，没有杨叔，没有在这里这么多关心帮扶我的叔叔阿姨，又怎能有我的今天。或许此时的我已经休学在家继续折磨着父母，亦或在昏黄的网吧里彻夜不归，甚至可能在某个清晨被人察觉早已倒在了电脑面前，以这种可笑的方式结束我的一生。<br/><br/>而今天的我之所以还能存活在这个世界上写下这些文字，正是因为我受益于中心，受益于杨叔。是杨叔用大爱唤醒了迷失在虚拟世界的自己，用声声教诲将我从网络的泥潭中拽出，让我拥有一个正常的人生。更是杨叔让我找回了与父母间丢失已久的亲情，让我明悟这世上最爱我的人始终是我身边关爱着我帮助着我的父母，让我在黑暗中迷失了方向的家庭也重见了希望与光明。是杨叔让我重新寻回了曾经失去的一切。<br/><br/>但如今，因为舆论的误导以及始作俑者的推波助澜，中心遭受到了巨大的压力。在这样生死存亡的关键时刻，作为中心百名盟友中的一员，更是作为阳光心态受益者的我，此时此刻必须站出来。吃水不忘挖井人，饮水亦要会思源。可能以前的我会以一副旁观者的姿态在一旁观望，但如今在杨叔的教导下，重新找回感恩之心的我绝不能袖手旁观，可能仅凭我一个人的微弱的言语与力量改变不了什么，但在中心大爱的合力下我必须尽上自己的一份力量。<br/><br/>我不清楚如今事态如何，也不知道在外对杨叔及中心又如何评价，我只希望大家能以事实说话。十年来在中心无数受益的孩子与家庭都是最好的证明，亦或以我的改变也同样能说明一切。我也希望那些没有认清事实的善良的网友们不要被网上别有用心的言论所误导，要有一个明辨是非的能力。不要因为一些谣言的误导便随波逐流，要看到这风雨十年来每一个成功改变的孩子用铁一般的事实击碎那些虚妄的说辞，事实将会证明一切。<br/><br/>十年前的那场风波只会成为我们中心发展的助力，今天这次也亦然。让我们众志成城，用真心来为中心作出我们的贡献，让中心变得更好，让更多的家庭与孩子受益。<br/><br/>――盟友小齐<br/><br/><strong>?愤怒</strong><br/><br/>大家好!我是盟友小兵，下面是我听了今天点评课之后的心声。<br/><br/>听说网上又有人写文章炒作、诋毁中心，我和妈妈一起看了那篇文章，作者把09年的负面报导又整理了一遍，又从杨叔的微博上截取文字与照片，断章取义，夸大治疗，攻击中心。<br/><br/>如果在没来中心以前，这样的新闻根本不会引起我的注意，因为与我没有任何关系；如果是在几个月前我来中心时，我一定会为此欢呼雀跃，心中暗自叫好！可现在，我的心中只有气愤。听说作者是一个外号叫雷斯林的游戏制作人，他发表这篇文章，不明真实情况的人一定会认为他站在道德的制高点上，因为他在维护祖国未来的花朵，所以很容易被他拐带，他把杨叔称作恶魔，说我们被摧残、被压迫，真的是这样吗？<br/><br/>他为什么要攻击中心？可能有很多种原因，因为他是搞游戏的，如果网戒知名度越来越高，他的利益就会受到威胁，这是一种；另外，他也可能会借此提高自己的知名度。<br/><br/>他说杨叔是恶魔，到底谁才是真正的恶魔？外界不了解我们，我们自己还不了解吗？<br/><br/>我们来中心之前是什么样子？厌学、逃学、早恋、同居、抽烟、喝酒、打架斗殴、赌博、吸毒，每天泡在网吧与酒吧，五毒俱全；在家中稍有不顺就打爹骂娘，不工作只会伸手要钱。可以说：我们就是这个社会的蛀虫！是什么把我们变成了那个样子？难道我们天生就是那样吗？不！人之初，性本善！我们本都是活泼可爱的，可自从我在初中时结实了损友，接触了网络游戏，一个学习数一数二的好学生不见了，一个人见人爱被父母视为骄傲的好孩子不见了，他去哪儿了？去了虚拟的世界！每天沉溺其中，从十四岁到了现在，八年了！我与社会已经严重脱轨了，这八年来，我一直活在梦里，一直逃避现实，因为现虚拟对比现实，简直是天堂，我再也不愿从虚拟中走出来，因为那里有我“想要”的一切！而现在的我们，一个个如同天使一样，可爱，善良，阳光自信，我们都是好孩子。是谁把恶魔变天使？是中心，是杨叔。我很清楚，如果不来中心，我们家真的会家破人亡！杨叔拯救了成千上万个孩子，和他们身后成千上万个家庭，没有中心就没有现在的我们。<br/><br/>而如今，有人诋毁中心，他说自己在维护祖国的花朵，是这样吗？他在维护恶魔！他是不折不扣的魔鬼，他想毁了中心，让我们万千家庭无家可归，他想让我们再回到过去，再成为社会的蛀虫，他想让我们家破人亡，他想让我们中国不攻自灭，他是孬种！他是整个社会的罪人！他利用那些不明实情的网民来维护他，来攻击我们中心，他试图毁灭我们的家，妄想！我们看似孤立无援，但我们不会屈服，我们坚信邪不压正，我们都受益于中心，如果没有中心，就不会有现在的我们，如果中心不存在了，我们该如何是好？所以我们必须站起来，齐心协力，一致对外，尽自己所能，全力以赴，维护我们共同的家。而我们盟友该怎么做？全力以赴改变，没有比事实更好的证据，我们用自己来向社会证明，到底谁才是真正的恶魔，到底哪里才是我们真正的家！<br/><br/><strong>致所有不明真相的人</strong><br/><br/>在那些负面的文章中，中心被称为是“奥斯维新集中营”，而杨叔则是他们口中的杀人犯、刽子手。你们也是受到了这样的报道的影响，在网上形成了一片漫骂声。可是你们有没有想过几个问题――文章的作者是谁？他们为什么要评论中心？如果中心和杨叔真的如文章中写的那样，家长们为什么要把孩子送来？而我，倘若真的被摧残，为什么还要写下这篇文章？<br/><br/>首先要告诉你们的是，那篇负面报道的作者是一名游戏开发商，而我们这里是什么地方？网瘾戒治中心。两者之间的关系显而易见，稍微再动一个脑筋就能想明白他们为什么要写这篇文章，因为中心触碰到了他的利益，既然中心能够触碰到他们利益，不恰恰说明了中心和杨叔的优秀吗？<br/><br/>他们对作者，对中心都不是很了解，你们的信息渠道只不过是在网上发布的一篇又一篇以偏盖全，断章取义的文章。但是现在舆论已经形成了两种不同观点，每一方看起来都有“合理”的道理，你们也很迷茫，你们也不知道究竟谁对谁错。或许，在你们的眼中，家长们也已经被“洗脑”。那就不妨听一听我们这些孩子内心最真实的想法。尤其是对你们最敏感的那个所谓的“电击”的认识。<br/><br/>不可否认，中心的确存在这个环节。因为我体验过，所以可能我最有发言权，疼是客观事实，但绝对没有文中描写的那样恐怖，并且我从来不痛恨这个环节。为什么？那么请问你们，打针疼不疼，开刀做手术疼不疼？但又有谁痛恨这些？因为我们都知道，如果没有打针，没有手术，我们的病情就会恶化，甚至危及生命。中心的这个环节亦是如此，像我们这等无法无天的孩子，打群架、抢劫都是常事，赌博吸毒也不在少数，如果不靠这个环节帮我们建立后果意识，等待我们的只有牢狱之灾。那个时候人生已经失去希望，后果意识的建立还有什么意义？能做的只有“后悔”罢了。几秒钟的疼痛换来一生的幸福，换还是不换？这道选择题我不知你们如何选择，我的答案是前者，而且换得很值。<br/><br/>或许，从表面上看，那篇负面文章的作者好像很好本事，不断的批判着中心的不是。似乎是帮助未成年人声讨人权。但是，他究竟能做到些什么？他能帮助我告别抑郁，走出那十个月没有走出的家门吗？他能帮助我重回校园，继续奋斗于追求梦想的路上吗？他能帮助我找回亲情，让家庭找回昔日的温暖吗？他能帮助我走出阴霾，重新奔跑在充满着希望的人生中吗？不能，都不能。既然连我都改变不了，何谈那些“十年漫漫求医路”，游走全国戒毒所但都没有结果的孩子？<br/><br/>但是，这些他所做不到的事情，杨叔都能做到，中心都能做到。这就是我要写下这篇文章的原因和最真实的中心。<br/><br/>――盟友小卫。<br/><br/>?<br/><br/>?<strong>感谢网友提供的资料。如果你也同样关心青少年，如果你了解真实的临沂网戒中心，欢迎把你的故事告诉我。――哇塞侠</strong><br/><br/><br/>首次转载时间:
