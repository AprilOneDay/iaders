/><br/>国内网瘾现状<br/><br/>《中国青少年网瘾报告（2009）》显示，我国城市青少年网民中网瘾青少年约占14.1%，人数约为2404.2万，当年网民规模达3.8亿。《第38次中国互联网络发展状况统计报告》显示，截至2016年6月底，中国网民规模达7.10亿，10岁以下占2.1%，10～19岁占24.5%，20～29岁占30.7%，按《中国青少年网瘾报告（2009）》网瘾青少年约占青少年网民的14.1%估算，人数约为5000万，在绝对数量上较2009年（2404.2万）增加了一倍多。值得重视的是，中国网络游戏用户规模达到3.68亿，使用率从2015年底的54.7%升至58.2%，中国手机网民规模已达到5.27亿，占网民总数的83.4%，首次超越80.9%的传统PC上网比例，手机游戏使用率为47.8%，较2015年增长4.7个百分点，规模增长3648万，网络的易获得性必定会产生更多的网瘾患者。<br/><br/><strong>国外网瘾现状（摘自人民日报）</strong><br/><br/>网络在带给人们方便快捷的生活及海量信息的同时，其“双刃剑”效应也日益显现：在一些国家，青少年沉迷网络正在成为令人担忧的社会问题。世界各国都在动员多方力量，采取限龄、限时等多种手段措施，为保障健康上网而不懈努力。<br/><br/><strong>新加坡　培训家长当协管</strong><br/><br/>新加坡染上网瘾需要接受心理治疗的人也越来越多。近年来，在新加坡曾发生过青少年因迷恋网络而荒废学业，甚至还有网瘾太大不能自拔，最后闹到要割腕自杀的案例。为从源头上防范网瘾，新加坡对互联网内容实行严格审查制度，同时鼓励父母参与教育和保护孩子。新加坡医学专家将网瘾归为心理疾病范畴，并对患网瘾的青少年进行积极干预治疗。新加坡卫生部专门开设的戒瘾康复教育中心，除了帮助戒酒瘾、毒瘾之外，也治疗网瘾患者。<br/><br/><strong>越南　坐禅戒网瘾</strong><br/><br/>为了让孩子尽快摆脱网瘾折磨，不少越南家长采取了切断网络、专人跟踪等办法，更有家长把孩子送到寺院坐禅，希望借助佛经禅法感化孩子，助其重归现实。<br/><br/><strong>埃及　网吧显示屏朝大门</strong><br/><br/>埃及政府在应对互联网不良信息方面，采取的是道德约束和法规管理并举的措施，网络主管部门严格规定，网吧内所有电脑必须把屏幕朝向大门方向，保证管理者“一目了然”，便于检查。检查由文化部官员和当地的治安警察联手进行，多采取突击抽查的方式，如发现网吧内有浏览色情、暴力或其他违规内容的，除对经营者实施经济上的重罚外，还要没收全部营业设备，吊销营业执照，永久性关闭网吧，并对网吧负责人实施2―3年的监禁。<br/><br/><strong>美国　少年上网过九成</strong><br/><br/>美国很早就注意到网络上瘾问题。1995年，纽约的心理病医生伊万・戈登堡创造了“网络上瘾症”一词，这一说法随即受到关注和研究。1996年，匹兹堡大学的心理学家金伯利・扬提出了“病理性上网”概念，并细分为两种表现方式：“特殊病理性网瘾”，指人们对于特定网络内容的依赖，如网络股票交易、拍卖或色情内容等；“一般病理性网瘾”，指人们仅仅为了打发时间、浏览、聊天等过度使用网络。<br/><br/><strong>拉美社会关注网瘾问题</strong><br/><br/>专家指出，家长和教育工作者的积极引导作用很重要，应当认识到网络已经深深扎根于青少年，家长和教育工作者应确保网络在可控制的范围内正常应用，不能对互联网给青少年带来的问题袖手旁观。<br/><br/><br/><strong>网瘾少年究竟该由谁买单？</strong><br/><br/>不管我们再怎么追究责任，无数网瘾少年的诞生已成为事实，这个问题需要解决，该由谁来解决？有几个选项供大家参考。<br/><br/>1.游戏开发商。按国家相关法律规定，现今市面上的游戏都必须有防沉迷系统。<br/><br/>什么是实名注册和防沉迷系统？<br/><br/>1、根据2010年8月1日实施的《网络游戏管理暂行办法》，网络游戏用户需使用有效身份证件进行实名注册。为保证流畅游戏体验，享受健康游戏生活，请广大腾讯游戏的玩家尽快实名注册。<br/><br/>2、保护未成年人身心健康，未满18岁的用户将受到防沉迷系统的限制：<br/>游戏过程，会提示您的累计在线时间。<br/>累计游戏时间超过3小时，游戏收益（经验，金钱）减半。<br/>累计游戏时间超过5小时，游戏收益为。<br/><br/>这个系统最致命的漏洞在于：只减少游戏收益而不会强制下线。<br/><br/>?举一个简单的例子，我们都玩过《掉线城与虚弱勇士》，这个游戏看似很良心，因为它有一个疲劳值系统，疲劳刷完了就不能再刷图了，而且根据上面的防沉迷系统，超过五个小时你就没有收益了，那么对于小学生来讲该怎么办？――去PK场呗，这不需要我再另外写一篇攻略教你们吧？所以疲劳值系统也罢，防沉迷系统也罢，从根本上解决不了未成年人沉迷网游的问题。何况你从家里找个身份证有多难？你去网吧找老板借一个身份证有多难？如果我是游戏开发商，我巴不得小学生都来玩我的游戏，别问我为什么――人傻、钱多、速来！<br/><br/>2.学校。实际上在上文当中我已经讲的很清楚了，学校是拿网瘾孩子没办法的。我就喜欢逃课上网你奈我何？抓不到算我走运，抓到了顶多让我写个检讨，再不然叫家长，你还有什么办法？劝退我？来啊，这样我就可以名正言顺地回家上网了。<br/><br/>3.社会。毒瘾有戒毒所，酒瘾有戒酒科，打架斗殴有少管所，违法犯罪有拘留所和监狱，网瘾呢？对不起，是沉迷网络呢？国家并没有这样的相关机构。<br/><br/>饶了一大圈我们又回到了原点――网瘾少年到底该由谁买单？<br/><br/>答案是：没人买单！<br/><br/><br/><strong>?谁在拯救？</strong><br/><br/>面对伴随网络信息化时代带来的新问题，世界范围内无论是政府、机构或个人都在尝试着寻找解决的方法，国家卫生部曾在2009年11月发布的《未成年人健康上网指导》指出：极少数上网者沉迷于网络，社会功能严重受损。表现为：不能正常学习和生活，身体发育和健康受损，出现各种反常行为和情绪问题，现实人际关系（包括亲子关系）恶化，与周围人交往困难、不合群。未成年人是否存在社会功能严重受损的情况，需要由精神卫生专业机构进行判断。对网络使用不当者中伴发明显焦虑、抑郁、强迫等精神症状的个体，应到医疗机构进行诊断，并依照有关临床诊疗规范进行治疗。<br/><br/>2013年2月，文化部等15部门出台网瘾防治方案，未成年人网络游戏成瘾综合防治工程的总体要求是：坚持未成年人保护优先原则，充分发挥各级网吧和网络游戏管理工作协调小组作用，以预防、干预、控制网瘾为主线，加强网瘾基础研究，抓紧明确网瘾干预机构及其从业人员的法律地位，完善相关管理制度，全面落实网吧和网络游戏市场的日常监管措施，依法打击违法违规经营活动，净化网络文化环境，减少网瘾对未成年人的危害。<br/><br/>同时在工作重点中着重强调：研制本土化网瘾预测和诊断测评系统。针对目前我国尚无符合国情的网瘾诊断测评量表的现状，要调动研究机构、精神卫生机构各方的力量，研制本土化的网瘾诊断测评系统，防止由于文化和地域差异等原因在使用外来量表过程中而导致的误诊和误判。同时，开创性地开展网瘾预测工具的研制工作，在未成年人出现网瘾症状前进行有效的事前干预，减少网瘾危害，降低诊疗成本。<br/><br/>主要措施中要求：开展网瘾综合防治的基础性和应用型研究。卫生、教育部门要依托精神卫生机构、高校等，开展网瘾防治的基础性和应用型研究。<br/><br/>令人遗憾的是，虽然政府相关部门也做出了积极的努力，但至今还没有正式的“网络成瘾诊断标准及治疗规范”发布，这也是从事网瘾干预研究或工作的机构少之又少的原因之一，加上青少年网瘾的成因复杂、戒治难度大、管理风险高、伤害性事件频发，许多机构或个人望而却步，导致2009年之前涉猎网瘾戒治的数百家机构渐无影踪。<br/><br/>现在看来，只剩下那些饱受我们诟病的戒网瘾机构了，而其中最热门的莫过于杨永信和他的网戒中心。社会对于其争议主要在于他所采用的治疗手段，质疑他在以“戒治网瘾”之名行“虐待孩子”之实。可奇怪的是，无论是2009年网戒舆论风波，还是眼下对杨永信的“挞伐”，其始作俑者都与网络有着密不可分的利益关系。<br/><br/>杨永信和他所创建的临沂网戒中心在这种暴风骤雨般的炒作下能不能生存下来并不重要，关键的是谁还会愿意顶着近似一边倒的社会舆论，接手网瘾这块烫手山芋呢？教育部？卫生部？文化部？可能在2013年之前他们还关心过这个话题，但是经历过“网瘾战争”之后很少听到他们的声音了。至于原因，仁者见仁，智者见智。<br/>首次转载时间:
