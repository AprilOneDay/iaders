/><br/><!--more--><br/><br/>近段时间以来，网上炒作临沂网戒中心甚凶，甚至一度出现舆论企图绑架政府决策的情形。我把我的孩子送到网戒中心那是我们的家事，现在说给大家听，只想用事实戳穿一些人的险恶用心，让一些不明真相的人不再被蒙蔽，也希望政府考虑这部分群体的内心诉求。<br/><br/>我的孩子从小乖巧懂事，上学以来除了学习跟不上，其他都很好。学习是他的弱项，他自身没有学习兴趣，我们也不会引导教育，特别是我的方式方法尤为简单粗暴。我和孩子妈妈都把工作看得太重要，对孩子的照顾太少，和他内心的沟通交流更少，孩子自己也不善于言辞，话不多。为了让孩子的学习跟上去，从小学三年级开始，我们给他选择了一所离家很远但条件及师资力量较好的学校，为了方便，让孩子寄宿在学校旁的学屋里。但事与愿违，孩子的学习一直没有跟上，我们家庭关系也不融洽，束手无策的同时，对孩子的批评变得变本加厉。孩子上初中后不久，因为和其他同学发生口角，我的孩子在中午放学时被七八个同学围住挨了一顿暴打，他妈妈在听到孩子的哭诉后没有站在孩子的角度，只是告诫孩子不要去惹别人，以后躲开他们，不要想着报复，最好的方法是告诉老师。孩子显然不满意妈妈的态度，愤怒的摔门离去，从那次起，孩子开始不再学习，而是开始拜大哥，拉帮派，每天想着和一些不三不四、不思进取的同学混在一起，以上网、打架、飙车、喝酒为乐。<br/><br/>孩子的堕落让我十分苦恼，也跟孩子坐下来谈过，当然也骂过他，也打过他，不仅亲子关系越来越紧张，孩子的性格也变得越来越极端。<br/><br/>让我震惊的事情发生了！孩子伙同其他同学勒索小学生钱财不成痛打小学生的视频在微信圈流传开来，并在省电视台播放了，致使学校领导一直在给主管部门写检查做检讨。这个消息对我和他妈妈来说，堪比五雷轰顶。在这之前，孩子偷拿了家里的钱买了一辆摩托车，在学校周围经常呼啸而过，发生这件事后，孩子自己也害怕了，随即离家出走。他没有地方住，有时在网吧过夜，有时在一个辍学的同学家里过夜，身边的朋友有社会混混，有比他大几岁的技校生，有辍学的，有吃喝嫖赌毒不务正业的各种消极人群。<br/><br/>哪个父母会让自己的孩子和这群人天天混在一起，天天泡在网吧？后来我们听孩子说，这群人里有好几个吸毒的，他妈妈听到这句话后几乎都站不住了。他不知道，监狱的大门正在徐徐为他敞开，如此下去，他人生的黑暗时光已经来临，结果不是进监狱就是死在街头。先前买的摩托车成了他炫耀的资本，每天在学校周围转几圈，也成了他走上作死之路的快车，经常半夜和一些社会闲散青年飚车，摔个粉身碎骨那是迟早的事，他哪里有一丁点的后果意识啊！<br/><br/>孩子离家出走的半个月让我度日如年，我也想离家出走，我也想了此一生，我也想找个寺庙对灯诵经……孩子毕竟还小，我作为父亲不能放弃他，最终我决定改变现状，先找到他再说。有一次半夜两点钟，我一直在寻找他，最终在一家肯德基店找到他，他很警觉，发现了我，飞也似的骑车就跑，我开车就追。半夜，路上没有几辆车，我的车快，我当时想撞倒他，就算残了，至少还能保住他这条命，要是跑了，我怕活不见人死不见尸啊！最终没有追他，让他拐入巷子跑了。我开始寻找其他的路，我开始想找一个改变现状的路。四处打听，我来到临沂网戒中心考察，尽管当时我并不认为这里能行，但我确实没有其他路可走了，没有比这更合适的路了。我找来家里亲戚四处寻找并抓到他，将他送到网戒中心，和网戒中心的不解之缘就这样开始了。<br/><br/>事实证明我的决定是正确的，在网戒中心的八、九个月里，改变的不只是孩子，首先是我们当家长的，杨永信医生经常告诉家长们：只有家长做出改变，孩子才有可能改变，才有可能改变家庭的现状。“家长是孩子的镜子，孩子是家长的影子”，“己所不欲，勿施于人”，“爱出爱返，福往福来”……耳熟能详的道理，但从未得到我们的重视，从未认真对待。而在中心，我们不仅要会背，还要照做，相互监督，共同改变。我们在这里开始学着从自己身上找原因，而非斥责孩子，要“严以律己，宽以待人”，勇于承担责任，不要试图让别人去分担；要明白管好自己，做最好的自己。我们家长和孩子一起，共同学习做人、做事，正确处理身边的人和事。在这里，孩子们作息规律，恢复了社会功能，做回正常人。在全方位学习的基础上，洗衣、拖地、打扫小室、清理走廊、训练健身、端正坐姿；在思想上、学习上、生活上，全方位改变自己，提升自己。<br/><br/>我的孩子已经出院一年有余，在这一年里，孩子不断的突破自我，战胜自己，学习成绩虽然有升有降，但总体是上升的，状态是向上的，思想是上进的，斗志是昂扬的，在家在外处理身边的事细致到位，老师认可，亲朋好友赞赏，我们家长别提有多高兴了。现在，我们真正算得上幸福的一家人，我要感谢杨永信医生，多么希望杨医生能救治更多的孩子，更多的家庭！但游戏商不高兴，游戏玩家不高兴，没有改变的个别孩子不高兴。<br/><br/>因为我受益，所以我支持网戒中心，支持杨医生，因为我知道，现实社会中还有成千上万的这样的孩子在迷途而未知返，我真心希望像临沂网戒中心这样的机构越多越好，解决社会难题，挽救问题孩子，拯救痛苦家庭。<br/><br/>我不理解这个社会到底是怎么了？我不理解游戏开发商怎么就有如此大的能量？我不理解怎么就有那么一部分人甚至是公众人物或所谓的“专家”不讲真话不求实事不讲职业操守？我不理解个别没改变的孩子整天满口脏话恶意诽谤，还有人给点赞？我不理解造福社会的杨医生和网戒中心哪里惹着他们了？<br/>?<br/><br/>盟友小磊爸<br/>首次转载时间:
