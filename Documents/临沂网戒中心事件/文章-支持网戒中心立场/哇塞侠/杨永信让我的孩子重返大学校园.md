/><br/><!--more--><br/><br/>中国五千年的文明史，形成了以传统道德为基准的模范型社会，“望子成龙”、“望女成凤”是现代家长的典型期盼。优秀的的性格、能力、气质等，都会成为一个人安身立命之本。孩子在成长过程中，有两个方面尤其需要培养。一个是人生观、价值观、世界观，这是基本人格，也是判断一个人是不是好人或者有没有服务于社会的公益心，这就是我们平常说的做个好人。另一个是有没有认知能力、沟通能力；有没有行动力、创造力和执行力；有没有抗压力、抗挫折和抗委屈能力；有没有自我管理、意志力等等可迁移的能力，这是成长中的遇到问题、思考问题和解决问题的重要能力。<br/><br/><br/>在沉迷于网络游戏或者其它不良嗜好的孩子们里，不否认有的孩子能够通过这种可迁移的能力，如:认知能力、沟通能力、自我管理、意志力等重新回归正常发展的轨道，并且能够逐步培养自己的人生观、价值观和世界观，实现自己人生的价值。这也是有的人觉得自己也曾经沉迷网络游戏，沉迷一些不良嗜好，可以通过自我调节而走向正轨，而对社会、学校和家庭对这种偏离主流价值观的干预不理解，甚至认为没必要对这种行为进行纠正。这种想法应该说非常的片面和错误，它仅仅能代表这些个体而不能代表整个群体。同样，也有相当数量的孩子在成长过程中，深陷网络游戏，深陷一些不良嗜好中不能自拔，认知存在严重问题，不能通过可迁移能力形成正确的人生观、价值观和世界观，不能回归到正常发展的轨道。他们进入消极的圈子，结交一些损友，逐步走向自卑、暴戾、抑郁、自闭等，慢慢发展成顽硬性格的“问题”孩子。这些“问题”孩子会对抗社会、学校和家庭对他们的教育和引导，漠视亲情，仇视亲人，并且做出一些有悖道德甚至违法犯罪的事情，给社会、家庭带来巨大的危害。面对这样一个群体，谁来拯救？<br/><br/><br/>     我的孩子就属于这样的“问题”孩子。在那段不堪回首的日子里，我的孩子结交损友，最开始只是旷课逃学，后来发展到天天泡在网吧里打游戏，一天只吃一顿饭甚至不吃饭，困了就趴在电脑前睡会儿，睡醒接着玩，荒废了学业，摧残了身体，丧失了目标。慢慢发展到除了要钱，不给我们打一个电话。在下着大雨的天气，我和生病的妻子一同去寻找孩子，换来的却是孩子跟我们的“躲猫猫”。我们的家庭陷入了深深的危机。<br/><br/>天下没有不爱孩子的父母，没有父母愿意看到自己的孩子这样沉沦下去。我们想办法对孩子进行说教、劝导，希望孩子能悬崖勒马；我们找学校去做孩子的工作，希望孩子可以幡然悔悟；我们去咨询心理医生，希望得到救治孩子的科学办法；我们在庙宇里拜佛求签，希望孩子得到神灵的庇护；我们甚至重修了祖坟，希望祖上保佑全家顺利度过难关......然而，我们的努力与网游的诱惑相比简直是蚍蜉撼树，孩子对学校和家庭的态度是脱离和对抗。一番折腾，换来的也只是更加恐惧和绝望，长期活在担惊受怕之中，我的妻子甚至几次梦见孩子猝死在网吧，走向了毁灭，那种感受真的是生不如死。对孩子的拯救是坚持还是放弃？坚持还是放弃？每每想到这个问题我们的心都在滴血！<br/><br/><br/>坚持不放弃！也许正是这份执着得到了上苍的眷顾，我们终于找到了临沂网戒中心。在一天的考察时间里，我们观摩了大点评课堂，认真听了点评师对孩子和家长的点评，与家长和点评师进行了交流和沟通，询问了临沂网戒中心救治“问题”孩子的主要方法和环节。通过这些了解，我们认为中心也许是能够拯救我的孩子最后的地方了，我们也没有任何的退路，哪里还能帮得到我们呢？于是，我们带着孩子来到了中心。<br/><br/>中心是一个充满正能量的家，它源自杨永信医生以及医护团队高度的责任心和事业心。杨永信医生几乎每天都工作到很晚，可以说是十余年如一日，作为一个团队的领头人和享受国务院政府特殊津贴的专家，这份执着令人钦佩。杨永信医生带领团队对每个孩子都仔细进行剖析，发现这些“问题”孩子在性格上的短板和缺陷，有针对性的对每个孩子进行性格重塑。在这里，杨永信医生教会孩子们成人做事的道理，引导孩子们形成正确的人生观、价值观和世界观，培养孩子们的认知能力、沟通能力；行动力、创造力和执行力；抗压力、抗挫折和抗委屈能力；自我管理、意志力等等可迁移的能力。杨永信医生从家庭教育出发，注重对家长心态的调节，让家长学会认识自己和认识孩子的方法，克服“尊卑观念”，改变过去那种简单粗暴的责罚和“机械”讲道理的方式，启发家长如何与孩子们有效沟通，走进孩子的内心世界，培养亲情，重建信任；引导孩子们理解父母，培养孩子们的责任心、感恩心、孝心等，家长和孩子需要共同为家庭加分，共建和谐幸福家庭，从根上解决家庭给孩子带来的问题。孩子们通过每天的日记课和演讲环节，把大点评课的内容和当天的行为进行梳理和总结，结合自身写出感悟，在感悟中找到前进的动力和方向。辩论赛和网吧课作业则培养了孩子们的逻辑思考能力，以及团队合作、协作能力、语言组织、表达能力和构思、写作能力等。<br/><br/>杨永信医生和他的团队在探索和追求目前很少有人涉足的领域，形成和完善了对“问题”孩子性格重塑的108个环节，这些环节丝丝相扣，发挥了巨大的作用。为孩子们重新回归校园、回归社会打下了坚实的基础。中心充满正能量的理念影响和教育着成千上万的家庭和孩子，成为孩子包括家长成长的动力和源泉。杨医生是医术精湛的医生，更是教育家和孩子成长中的导师。<br/><br/>刚到中心时，我的孩子很不理解，感觉玩游戏没啥问题，觉得我们是小题大做，对抗中心、对抗我们，但接下来发生的事情，使我的孩子改变了很多。由于长期玩游戏和不按时吃饭、作息，整个人很虚弱，营养严重不良并伴有心慌、焦虑等问题，需要打针补充营养，可血管瘪的扎10多次才能打上，当时情况可以说很危险，如果说不来中心救治，放任孩子在游戏里挥霍青春、透支生命，很有可能没有实现自身价值的机会了。八个月的时间，孩子在一天一天的变化，从最初的不理解逐渐开始理解；从消极、颓废逐渐变得乐观、自信；脆弱的内心开始变得强大；从毫无人生目标到逐渐形成正确的人生观、价值观和世界观。八个月的改变后，现如今我的孩子重新回到了大学校园，开始了崭新的生活。现在，我的孩子对人谦和有礼，在家中勤做家务，每天坚持锻炼身体，学习积极努力，阳光，自信。正是因为有杨永信医生这样无私奉献、不求回报的有责任心的良医，才成就了我的孩子、我们的家庭，给了我的孩子第二次生命，让我们全家重拾幸福、快乐！否则，等待我孩子的不是监狱就是地狱！感谢杨永信医生！<br/><br/><br/>十余年来，临沂网戒中心经历了风风雨雨，杨永信医生带领着医护团队在探索、求真的道路上奋进，已经成功改变了6000多例网瘾、性格扭曲等“问题”孩子。如今，在个别无良媒体和别有用心人的推动下，个别所谓的专家也加入了声讨的大军，在没有调查和核实的情况下，公然利用几个没有改好的孩子大肆渲染和攻击中心，他们采用选择性失明的方式，只引用没改好的孩子的言论，甚至有些是臆想的结论，对改好的孩子和家长们的发声一概视而不见，给中心带来极大的伤害。试问这样的无良媒体和专家，你们的职业操守在哪里？你们的社会公德在哪里？作为受益的家庭，我们要把我们的心声呐喊出来，让社会、学校、家庭多关爱我们的孩子，多关心我们孩子的成长。“正义有可能会迟到，但绝不会缺席。”我们相信政府一定会还临沂网戒中心、杨永信医生以及医护团队一个公道，不能让这样无私奉献的英雄流汗又流泪。
