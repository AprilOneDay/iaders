/><br/><!--more--><br/><br/>我儿子2015年2月经朋友介绍来到临沂网戒中心，于2016年5月离院，距今已有4月有余。与入院前相比，我孩子发生了天翻地覆的变化，面对如今阳光自信的孩子，我的内心百感交集，更多的是对杨永信医生的感恩之情。<br/><br/>九年前，孩子被几家权威医疗机构诊断为重度感统失调，即多动症。面对顽劣多动，一刻也静不下来的孩子，我们全家束手无策。小宇先天不足，从小顽皮好动，除了睡觉的时候，没有片刻安静，而且因感统不协调，时不时地受伤，至今手臂还留有严重疤痕，在老师的建议下，我们带着他去看医生，在医生的建议下，我们一直给孩子服用“利他林”。作为母亲的我更是倾尽全力，全方位地陪同孩子，事无巨细地照顾孩子的学习、生活，但是收效甚微。由于无法专注，孩子的学习成绩一路下滑，繁重的学习压力和药物反应让孩子身体也每况愈下。年复一年，日复一日的服药让孩子身心俱损，对药物产生了严重的依赖，日益加重的学业负担让孩子累得像一只奄奄一息的小鸟，再也无法飞行。<br/><br/>好在天无绝人之路，仿佛神灵指引，让我们有幸得到杨永信医生的救助，2015年元月二日我们全家来到了中心考察，在听了一堂杨医生的点评课后，我就被杨医生的课深深地震憾了。我是一名教师，我从教二十多年，给学生上过无数次课，也参加过许多的培训，然而真正打动我的还是杨医生的点评课，原来教育还可以这样做!经过两个多月的反复酝酿和筹划，我们一家终于走进了临沂网戒中心。<br/><br/>经过十五个月的坚持治疗，小宇的顽疾终于得到救治，我们原本只想让孩子戒掉服了八年的利他林，能安安静静地做个正常孩子，没想到，这十五个月的治疗不仅治好了他的多动症，而且让孩子得到了全方位的改变，获得了许多意外的收获，也让我和他爸爸得到了人格的提升和成长。杨医生不仅是孩子的救命恩人，更是我们家庭成长的领航者，我们感激不尽。特别是从去年9月，我从中心回到单位上班，半年多来穿梭于中心和社会，感受外界的炎凉冷暖，更觉中心世外桃源的和谐和温暖。当我们累了、困了、跑电了，回到中心陪几天儿子，顿时又精神焕发、斗志昂扬，中心的加油站作用彰显无疑。庆幸能在杨医生大爱的感召下，我们一直坚持坚持，终于看到了孩子和整个家庭的可喜的变化。当我们看到孩子安静地坐在点评课堂，一坐五六个小时；当我们看到孩子在学习组专心致志，头也不抬地学习；当我们看到儿子在春节联欢会上欢乐的韵律舞；当我们看到孩子“嗒嗒”连跳几十下的跳绳……，我们真的是惊喜万分。这些变化对别的家长来说可能是无所谓，但对我们来说，是十多年的梦寐以求。虽然孩子没有网瘾，但我们来到这儿，杨医生把我们一个多动症的孩子变得安安静静、动作协调，我们真是万分感激。<br/><br/>小宇离院四个多月了，离院后的表现特别好。念念不忘杨医生的大恩大德，对中心充满着怀念。孩子优秀的表现受到了亲戚朋友的一致认可。他一直在非常努力认真地做好每一件事情。中心十五个月的改变给了我们无比的动力和能量，让我们更加有理由和信心面对未来。小宇出院后报考了现代职教改革的3十3项目，那是直接与高校合作，是富士康科技集团投资的工业机器人项目。按照目前中国制造2025的前景和工业4.0的未来发展，工业机器人将是新兴的好专业。另外，合作的大学是我的母校，给孩子的成长提供了一个好平台，而且，如果学有余力的话，小宇还可以专升本。想想一路走来，坎坷波折，可一到关键时刻总是祥云再现，贵人相助。小宇在小学和中学碰到了特别好的班主任老师，对我特别理解，对孩子无微不至的关心，特别宽容他。在最无助的时候，遇见了杨医生，彻底让孩子摆脱了多动的困扰，让孩子变得阳光，自信，特别懂事，超出我对他的期望。孩子改变了，命运也改变了。当大多数孩子挤上独木桥，拼得你死我活时，幸运的橄榄枝又抛向了我们，我们又坐上了高校直通车，而且是最有前景的专业，未来更有老师和师兄的牵手。难道这就是杨医生所说的善报，我们以前所做的所有努力，所有的善行都在我儿身上得到了回报。更高兴的是学校决定让我担任这个班的班主任，感谢杨医生给了我全新的理念，让我有信心和能力带着孩子们有着健全的人格进军机器人工业制造的时代，我定当不负杨叔期望，做最好的老师妈妈。<br/><br/>小宇自从上次分享回来，表现更好，我深感长效机制的价值。我觉得孩子最大的亮点是坚持中心模式。一是每日坚持写日记，每天都是围绕一个主题，孩子的心中好像有永远写不完的故事，他的分析独到，句句在理。我把他的文章发给朋支看，朋友们都很惊奇他的变化，要知道，来中心之前，他是最怕写作，有时三两行一写，没下文了，即使写了也是凑字数，没有什么思想，现在有理有据，文笔流畅都是在中心十五个月浸泡的结果。改变孩子就像是咸鸭蛋的制作，只要放在好的盐水中，坚持浸泡，蛋黄出油是迟早的事。一个孩子能坚持每天写日记，从不落下，这本身就是一种坚持，有这一好习惯的坚持，孩子的改变是不用担心的。<br/><br/>二是坚持每日课操，早上200，晚上200。我很佩服孩子的毅力，我和他爸也尝试了几次，做起来并不那么轻松，可孩子却乐此不疲，从不懈怠。我的内心被孩子深深感动了。刚出院时，我最担心他会不坚持锻炼，哪知他比谁都明白，他知道此番锻炼定会感动天地，还他健康休魄，让他幸福快乐的。<br/><br/>三是坚持做善事。上次分享离开中心，我们在医院门口准备坐车去医院，他开始很想坐出租车，后来在我的引导下坐了三轮车而且坐了一辆很旧的中老年妇女的车，在车上我们讨论了为什么这样选择的道理，我告诉他，能开出租车的家中境况会好些，出来开三轮车定是生活所迫，一个年老的妇女出来开三轮，生活的窘迫可想而知了，我们应该尽可能帮助弱势群体。那个妇女一路上听我们的对话直流眼泪，说我的教育好，其实这都是杨医生教化有功。下车时儿子给她50元钱，说什么也不让找零，我走了很远，回头还看到那个女人站在那里对我们挥手，那天儿子特别高兴，他说妈妈，我怎么感觉做了好事会这么高兴。以前，我们给了他那么多，他从来没有这么高兴过。这就是赠人玫瑰，手留余香吧。自那以后，他每日买菜，总是到年老的农妇那边买，从不讨价还价，有时多给一元两元的，回来就说，妈，今天的菜又贵了些，我又帮助了一个弱势群体。中考我拿了400元监考费，我原本让他买菜持家锻炼，结果，走亲戚时，把钱塞给了一个穷亲戚，孩子又一次得到了更好的教育，他说妈妈，你很辛苦，忙了4天得来的监考费，一下子没了，可对舅奶奶太重要了，至少让他觉得在她身边有亲友帮她，让她温暖一点。孩子说这些话真是千金难买呀，孩子一生之中能有善心善行伴随，他的命运能差到哪里去呢?今天来中心，坐公交车，上来一个老爷爷，他连忙站起来，扶老爷爷坐下，车上的人都很感慨，现在年轻人低头一族，玩游戏玩手机，哪有看什么人上车，还让座，不抢座就很好了。<br/><br/>四是学习生存技能。每日去市场买菜，昨日暴雨，没让他去，菜场一个卖鱼的惊呼市场没见过这么小的孩子天天买菜，而且从不讨价还价，大家都很喜欢他，混熟了都愿便宜点卖给他，说他太懂事了。回来后洗鱼，切菜，洗菜，做饭，米多少，水多少，清楚明了，简单的菜会做，盐、油放多少都有数，还学会了烤虾。用料理机给家人打果汁，核桃红枣奶，吃完饭后，收拾碗筷，洗锅涮碗，奏响锅盆瓢碗交响曲。拖地，洗衣服，一开始洗自己的，现在把我和他妹的衣服全洗了。这种幸福的生活以前从来没有想过，家中骤然多了一家政工作者。我以前担心他不会照顾自己，让妹妹长大照顾他，没想到他成了照顾妈妈和妹妹的一把好手。<br/><br/>这些变化源自于儿子十五个月我们能坚持，而别人不能坚持的改变，源于我们相信杨医生，永不放弃的决心。千年铁树能开花，不信东风唤不回。我坚信杨医生定能云开见日，桃李花开，精品满园，事业节节攀高。<br/><br/><br/>小宇妈<br/>首次转载时间:
