/><br/><!--more--><br/><br/>这段时间以来，杨永信成了一名网红，从一篇文章开始，到各大门户网站的狂轰滥炸，再到中央电视台的两篇报道，让他在全国真正红了一把。我不知道作为游戏开发商，在网络上发表这样文章，对一位享受国务院特殊津贴的精神科医生发难，到底是何居心？也不知道各大网站疯狂转载到底有没有幕后黑手？更不知道作为国家喉舌的中央电视台、中青报等媒体站出来为他们摇旗呐喊到底有没有利益驱动？<br/><br/>在他们面前，我们所有的人显得那么渺小。不要说杨永信始终保持沉默；中心每一位点评师的联系方式都被居心叵测的人曝光了；就连我们这些看不过去的家长理智的回几个帖子，也换来他们无休止的谩骂。难道中央电视台和中青报为代表的社会公德和正义是站在了那些不会理智发声，只会侮辱、谩骂的人那一边了吗？我作为一个受益的家长，我良心没有泯灭，我想把我看到的、感受到的告诉不明真相的人，想让广大网民真正知道杨永信到底是一位什么样的人。<br/><br/>我当初也和大多数人的观点一样，认为在当今社会，网络具有无与伦比的功能，方便了我们的工作、学习和生活。所以在我孩子刚上小学的时候，我就教会他上网，让他在网上解决生活中的遇到的困惑，让他查询学习上遇到的不会做的问题，那时候我的想法是让他从小就知道网络的的作用，让他知道网络就是一部大百科全书。确实，在起初对他学习帮助非常大，他的成绩也一直非常优秀，从三年级开始，数学、英语就一直差不多是满分。他性格有些内向，但是我认为人的性格养成是一个非常复杂的过程，可能因为一个机缘巧合，特别是如果遇到一位喜欢的老师、一位要好的同学，他的性格就因为受他们的影响而发生变化，所以我虽然也在有意识的引导他，效果始终不太明显，但是也没有感到非常着急。他从3年级开始接触单机游戏，我认为，只要是不影响学习成绩，有限度的玩玩游戏，可以缓解学习紧张的情绪、开发智力、锻炼协调能力。好像是到了六年级的时候，他接触网络游戏，但他时间控制的非常好，平时不开电脑，到了周末、假期每天玩一两个小时，他游戏等级也很高，很多同学请他做代练，他也把这当做联系同学的纽带。成绩更是一直保持前茅，在1200人的一个年级稳居前二十，所以老师和同学都说他是另类，说没想到一个孩子可以把学习和游戏处理的这么好。我也为他感到骄傲，在周围的同事和同学面前一直非常有面子，以至于有孩子不听话的时候，他们还说，能不能让他们孩子到我家生活一段时间，让孩子跟着我儿子学习学习，也让我教育教育他们。<br/><br/>一转眼到了初二暑假，他那时候通过玩游戏，在网络上认识了一个辽宁沈阳的30多岁的理发师，五个人组成了一个战队，一个暑假玩的昏天黑地，白天玩游戏，晚上看直播。他说马上就初三了，这个假期他想放松放松。听他这样说，又因为我过于相信了他的自控力，也没有阻拦他。期中考试，他在全市8万人中考了前100名。没想到期中考试两周后，突然就给老师提出来不上学了，老师感到非常诧异。我把他带到家里，以为是学习压力大放松放松就好了，结果回家后他还是变本加厉的玩游戏，基本上是黑白颠倒，让他回学校就又哭又闹。这时候，我才意识到他已经失去自控力了，带他找过很多知名的心理专家咨询。每次都是满怀希望而去，结果每次都是失望而归。那时候我和他妈妈每天以泪洗面，可是他置若罔闻，还是沉迷在游戏中不能自拔，他说只有在游戏中他才能真正快乐起来，而且玩游戏完全可以生活的非常好（全国冠军就是几十万的奖金）。我劝他说，在学校中的学习，不仅仅是文化知识的学习，还可以学会怎么与人相处，德智体等各方面得到全面的发展。玩游戏是可以成功，可以衣食无忧，但是概率非常低，一共才有几个全国冠军？一年全国大学生录取数却是600万。而且如果他喜欢游戏，可以大学考相关的专业，开发一款游戏的奖金比电竞比赛要高的多。但是他那时候已经不再听我们说什么。这时候有朋友给我介绍起临沂市第四人民医院的网络成瘾戒治中心，我想在网络上搜索一下，结果铺天盖地的都是负面的信息，所以我断然决定还是求助心理专家，不能把我儿子带去那种灭绝人性的地方。<br/><br/>这样前前后后耽误了11个多月，到最后他几乎是全天除了睡觉、吃饭每天都是泡在网络游戏中，经常一个多月不洗刷、不下楼。这时候我才意识到如果不想办法挽救他，我们可能要失去这个孩子。那时候想到，哪怕是真的像网上说的有电击，只要是能救我儿子的命，我也愿意试试。结果到了中心才知道，远远不是那么回事，他们是通过心理点评课等环节来影响感化孩子，我们住院几个月，儿子的人生观、价值观得到重塑，重新树立了学习目标，再也不触碰网络游戏，从当初那么一个内向、胆小、自私、颓废的孩子，变成了今天这样一位开朗、阳光、自信的大男孩，我儿子也经常说，多亏了在网戒中心的经历，让他的人生少走了很多的弯路，他以后愿意像杨永信和他的医护团队那样，做一个对国家、对社会有贡献的人，不辜负他们对他的谆谆教诲。每天看着他快乐的生活、高高兴兴的上学，我和他妈妈感到非常幸福，从内心非常感激杨永信和他的医护团队，我们一直把网戒中心当做了我们命运的转折点、家庭的幸福港湾。<br/><br/>现在我儿子在我们县一中上高一，他在学习上浑身有使不完的劲。回到家里对我们体贴入微，经常和我一起到公园晨练，到学校打球，陪他妈妈逛超市、选服装，对别人有礼貌，热情帮助老人和孩子，赢得了所有亲人和朋友的赞誉。所以我想说，不管是别人说网游再怎么好，我只能说是它害了我的儿子，让他差一点过早的陨落。不管别人怎么诽谤杨永信，我想说是他救治了我的儿子，也挽救了我们全家。作为一个受益孩子的家长，我真的搞不清楚现在的社会怎么了，为什么这样对待一位热心帮助别人、救治孩子的一位医生？现在媒体的公信力又在哪里？（我儿子看过了中央台的两篇报道，就曾质问我，他们竟然这样信口雌黄，你还天天看新闻有啥用？那时候我只能给他解释，社会主流是好的，媒体也是有良知的，只不过可能有时候让某些人利用了。）<br/><br/>最后我想呼吁社会上有良知的人士，在这场风波面前要冷静思考，擦亮眼睛，有时间能真正多关心关心那些孩子，多关心关心我们这样的家长。即使不愿意为我们做什么，也别充当了别人的打手，别当了跳梁小丑，为那些居心叵测的人才摇旗呐喊，别让杨永信这样的以天下为己任的医生寒心。<br/><br/>小晨爸<br/>首次转载时间:
