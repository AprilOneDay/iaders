style="color: #ff0000;">当王哲醒来时，身边围着四五个穿迷彩服的人，他恍恍惚惚看到了“网戒中心”四个字。有人叫他进屋睡觉，他不顺从，穿迷彩服的人就把他抬起来了。</span><br/><br/>他被抬到一个房间，手脚被绑起来，他记得，足足有四根针扎进手上的虎口，连上电极，电流通过，身上一阵生疼。<br/><br/>这样一段极具视觉冲击力的文字算是报道的正式开头了。毫无铺垫，毫无预兆，恐怖的令人猝不及防，这哪是医院？这根本就是在描述只有电影中才能看到的酷刑！恐怖之后细细回味，感觉哪里有些不对，好像少了点什么似的，到底少了点什么呢？<br/><br/>少了的那一部分，自然是作者不愿展现给观众的部分――王哲为何会被父母送到网戒中心？他是上网成瘾？打架斗殴？打爹骂娘？偷摸抢骗？还是精神错乱？对于这位贯穿全文有着超多戏份超多台词的主角王哲，作者好像在刻意回避他的过去。这是为什么？<br/><br/>我们在大街上看到一个人骂另一个人，第一反应会觉得骂人者的人品有问题没素质，可走上前去听了听，原来被骂的那个人是小偷，转念又会想：光骂怎么行？得揍他呀！这就是作者“隐瞒”的原因，因为要努力将这些被父母送去临沂网戒中心的“问题孩子”塑造成“正常孩子”的形象，才能彰显出网戒中心的黑暗。<br/><br/>但明眼人一眼就能看出端倪：孩子当真如此正常的话，家长脑子有病送去网戒中心啊！<br/><br/>不仅仅是王哲，后文当中出现的几个孩子，都没有被描述过去有什么劣迹，与常人无异。<br/><h3><strong>2.谁在说谎？</strong></h3><br/><span style="color:
