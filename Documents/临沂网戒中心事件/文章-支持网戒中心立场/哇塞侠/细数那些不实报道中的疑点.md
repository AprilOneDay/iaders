/><br/>大家好，我是哇塞侠，新一期的“洗地小作文”又和大家如期而遇了。<br/><br/>读书百遍，其义自见。上一期我们对南都记者嵇石、向治霖的《无病少年三次被诊有病起诉网戒中心》进行了大致分析，今天我们一同来细细品读，看看还能有哪些意外收获。<br/><!--more--><br/><br/>1.周书是否有病？<br/><br/>在上一期的评论中，很多网友提问：什么是双向情感障碍？对于这类医学名词的解释，度娘比我说得好。<br/><br/>“双相障碍（又称双相情感障碍）属于心境障碍的一种类型，指既有躁狂发作又有抑郁发作的一类疾病。双相障碍的临床表现按照发作特点可以分为抑郁发作、躁狂发作或混合发作。”这是度娘给出的解释，大家可去自行检索。<br/><br/>王芳说，她最不能接受的是，周书“封闭自己”，关上门，拉上窗帘，在自己的房间里总不出来。<br/><br/>不过周书慢慢地恢复了原状，到2014年冬天，再度回到了“不上学，封闭自己”的状态，这让她始料未及。<br/><br/>上面这两段话中，王芳（周书的母亲）两次向记者提到了周书“封闭自己”，不上学，关上门，拉上窗帘，在自己的房间里总不出来。已经足以说明：周书并非正常孩子。而且他的种种异样表现非常符合抑郁。当然，报导中关于周书日常行为的描写太过稀少，我们也只能从只言片语中推测：如果他同时又有躁狂的表现，那么诊断双相障碍就变得合情合理。可惜记者对于周书的表现过于吝啬笔墨，这些证据我们无从得知。<br/><br/><br/>2.周书的父母为何要送他去网戒中心？<br/><br/>周书很快想起前不久的一次“恶意退款”，通过不停催促淘宝卖家退款，有的粗心的卖家会稀里糊涂地打钱过来，“一周前，刚好敲过一笔几十块的。”他说。<br/><br/>读到这里笔者有些惊讶。哇塞！除了上面的病，15岁的周书竟还会敲诈！而我们更不知道，周书是从几岁开始学会的敲诈？敲诈过几次？敲诈过多少？也许他认为自己年龄尚小，法不责幼；也许他认为父母宠他爱他，会为他摆平；也许他心怀侥幸，自欺欺人，掩耳盗铃。可如果父母不做阻拦，对周书放任自流，15岁的他就可以敲诈，那等到20岁、30岁的时候，他会改邪归正还是会愈演愈烈？<br/><br/>在浏览过所有关于网戒中心的负面消息后，山东枣庄市的王芳得出一个结论：这些负面消息都是不懂事的孩子写的，就像她15岁的儿子周书。<br/><br/>这里给我们提供了一个重要信息：原来周书家在山东枣庄。<br/><br/>山东临沂离山东枣庄有多远？不到200公里。自2009年炒作风波后，山东临沂的名气因网戒中心的存在而增色不少，当然，大多都是负面的，那周书的父母为何还要把自己的孩子送进这样一个被世人所诟病的机构中？我想他们一定是山穷水尽，拿周书毫无办法。<br/><br/>2013年12月20日的上午9点，刚醒过来的周书发现家中来了4个警察装扮的人，“你犯事儿了，跟我们到所里走一趟吧”，一名民警说。后来周书才知道，这四名警察是他母亲的朋友。<br/><br/>读到这里难免又有一点心酸。周书的父母，两个三四十岁的成年人，却拿一个15岁的孩子毫无办法，竟需要请来4位警察朋友协助。如果周书真的是一个正常的孩子，他会享受这样的待遇么？这件事一方面说明了父母教育的失败，另一方面说明了周书早已摆脱了父母的监管，在他父母眼中，周书是需要动用警察才能管制的。<br/><br/><br/>3.起诉威胁到底是谁的主意？<br/><br/> “父母也一度被‘洗脑’了，不相信我。”他告诉南都记者，对此，他将就此事对网戒中心提起诉讼。<br/><br/>在浏览过所有关于网戒中心的负面消息后，山东枣庄市的王芳得出一个结论：这些负面消息都是不懂事的孩子写的，就像她15岁的儿子周书。<br/><br/>从上面这两段话来看，周书与母亲的意见并不一致。周书一直强调自己没问题，父母是因为“被洗脑”才不相信自己的，并要起诉网戒中心。而周书的母亲认为是周书不懂事。可奇怪的是后面周书的话却很坚定，彷佛胜券在握。<br/><br/>“不出意外，我和父母一起在这个周末去北京，找律师。”周书说。<br/><br/>面对母亲不一样的态度，周书为何如此坚定？他会以何种方式说服母亲来达到自己的目的？我很希望周书与母亲间只是简单的语言沟通，千万不要把敲诈勒索那一套用到亲生父母身上，更不要拳脚相向，暴力相逼。希望这些只是我不必要的担心。此事若能再次见报，也希望记者能知无不言，别再遮遮掩掩。<br/><br/><br/>4.我们究竟该拿周书怎么办？<br/><br/>像周书这样的孩子在当今社会并不少见。那些不上学的孩子，那些不回家的孩子，那些脱离父母监管的孩子还有那些违法犯罪的孩子，我们究竟该拿他们怎么办？有网友说：我们国家有少管所、拘留所、监狱等地方可以监管这一类的孩子。对，这话没错，因为孩子不是你的。<br/><br/>试问哪个父母会愿意亲手把自己的孩子送进少管所？<br/><br/>试问哪个父母能做到亲手把自己的孩子送进少管所？<br/><br/>怕就怕，就算孩子进去了，父母也要千方百计地把孩子弄出来！<br/><br/>所以周书的父母，还有那些与他们一样拿孩子毫无办法的父母，宁可把孩子送到饱受争议的网戒中心。<br/><br/>还有网友质疑：把这样违法的孩子送到网戒中心，网戒中心岂不是犯了窝藏、包庇罪？其实我们无须担心，因为包庇周书敲诈勒索的父母和南方都市报的两位记者都没事，外人会有事么？<br/><br/><br/>PS：本期讨论话题：该拿周书这样的孩子怎么办？<br/>首次转载时间:
