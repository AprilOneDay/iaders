<br/>(2009-11-03 08:51:38)<br/><br/><strong>政府部门权利之争下的网瘾之殇</strong><br/><br/>蔺玉红&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr><!--more--><br/><br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr>&nbsp;新闻出版总署和文化部围绕网络游戏的监管又掀起了新一轮的权利之争。<br/><br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr>&nbsp;11月2日，新闻出版总署发出通知，终止《魔兽世界》(燃烧的远征)审批，退回关于引进出版《魔兽世界》的申请。原因是网之易公司在未经新闻出版总署审批同意的情况下，于9月19日擅自收费并提供新账号注册的行为，已经造成事实上的公开运营服务，严重违反了国家关于网络游戏上网前须经新闻出版总署前置审批和境外著作权人授权的互联网游戏作品须经新闻出版总署审批的规定。<br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr>&nbsp;很快，文化部文化市场司网络文化处处长刘强对外表示，文化部是网络游戏的主管部门，而新闻出版总署只是对出版物在出版环节有监管的权力，所以允许不允许网络游戏正常运营不在新闻出版总署的职责范围之内。<br/>刘强称，《魔兽世界》自7月30日以来属于正常运营，目前《魔兽世界》收费运营以及注册新账号均属于合法行为。<br/><br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr>&nbsp;网络游戏到底归谁管？应该怎么管？经过多年的争吵与纠结，直到现在，仍然各说各的理。在这场纷争中，我们看到的是，越来越多的青少年沉迷于网络游戏中，荒废了学业，怠误了青春。<br/><br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;中国青少年网络协会为了进一步研究网络游戏对青少年的影响，今年8月1日，专程到山东临沂第四人民医院网瘾戒治中心，对146名网瘾青少年和144名家长进行了问卷调查，相信大家看过这份调查结论后，对网络游戏的管理应该会有一个思路。<br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr>&nbsp;接受调查的这些网瘾孩子中，有132名男生，14名女生。年龄在13―31岁之间。在这些人中，高中以下学历的占88.52%。其中,66.80%的人已经休学或退学。<br/><br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr>&nbsp;89.86%的网瘾青少年表示，上网的主要目的是玩网络游戏。<br/><br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr>&nbsp;46.62%的人表示最喜欢角色扮演类游戏如《魔兽世界》<strong>。</strong><br/><br/><strong>&nbsp;<wbr>&nbsp;<wbr></strong>&nbsp;89.19%的人表示,他们最喜欢的网络游戏是《永恒之塔》。19.59%的人则认为,自己最喜欢的网络游戏是《魔兽世界》。&nbsp;<wbr><br/><br/><strong>&nbsp;<wbr>&nbsp;<wbr></strong>&nbsp;在回答“哪一款网络游戏使你的游戏时间最长”的问题时，23.65%的人首选《梦幻西游》。20.27%的人则选了《魔兽世界》。为玩这些游戏，64.86%的人选择了每天连续上网5小时以上。<br/><br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr>&nbsp;现有网络游戏做哪些改进能有效减少沉迷现象？25.67%的人认为，应减少血腥、色情、暴力画面。<br/><br/>&nbsp;<wbr>&nbsp;<wbr>&nbsp;<wbr>&nbsp;网瘾青少年的家长们在回答“为了缓解和解决网瘾问题，您认为网络游戏公司应该做出哪些改进”的问题时,78.47%的人认为，国家应该建立网游开发监管的标准和制度，网游公司必须开发符合国家标准的健康产品。78.47%的人认为，国家有关部门应该设立国家级网瘾预防和救助中心。<br/>首次转载时间: 2018-07-27 23:00:33"
