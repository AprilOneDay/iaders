<br/><img src="http://wx4.sinaimg.cn/crop.0.0.334.188.1000/0067hHJjly1fevrpxa85yj30a40583z2.jpg" /><br/><br/>关于杨永信风波中被恶媒谣记精心制造并反复炒作的几个所谓“热点”，本博已经在置顶文章《祝佳音，走在末日穷途的“拯救客”》 中作了集中说明，有喷子还在拿“抵触互联网”和所谓的“电击”做恶意说辞，或出口成脏，唯污言秽语之能事，本博一律拉黑！<br/>再次申明：本博为正义执言，文章均由亲历者所写，用事实说话，戳穿谎言，让“经营事件”的”恶媒“见光死！――<br/><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？<br/><br/><!--more--><br/><br/>??看完《和杨永信战斗的人们》这篇文章后，自己终于彻底弄明白临沂网戒中心为什么会被外界那么炒作、那么抹黑了。媒体们原来就是用这种无中生有、颠倒黑白的手段来博得人们的眼球，也真是枉费了他们的一片苦心啊！<br/><br/>这样的文章确实有一套，只要读者不细心去看，几乎是看不出什么毛病的。同时，由于大家不是中心的亲历者，很有可能就会相信这类文章的一面之词。而对于我，一位在临沂网戒中心改变的盟友，看完这篇文章后不禁感到愤懑不平，同时为这篇文章的漏洞百出感到可笑。所以我一定要站出来为临沂网戒中心正名，让大家对于中心和杨永信医生有一个正确的认识。<br/><br/>首先，文章中所说的“磕操”环节确实存在。但奇怪的是，在中心里我们都称其为“课操”，不知道为什么作者非要写成“磕头操”或“磕操”。我想，其目的就是为了引发读者遐想，让大家认为这是一种不人道的惩罚方式，或者说故意渲染成一种宗教仪式吧。而且文章中说“每人每天都要磕五百到一千”，这么大的数目我还真没做过，用一个月的时间做完还差不多。所以说作者简直就是无中生有。课操，只是帮助我们改变的一种身心锻炼方式而已，有和没有我们都会改变，但我本人更希望有，因为可以锻炼身体，稳定心态，促进思考，加速我们自身的改变。<br/><br/>其次，关于“盟友和家委会”。“盟友”是在临沂网戒中心住院的孩子们之间的一种称呼；“家委”，是在中心里起到一定的服务和管理作用，让中心更加有序和温暖的陪护家长们的志愿组织。然而他们没有任何特权也没有任何的回报。文章中说“家委会是为杨永信做事”，“逮捕”那些对网戒中心不利的盟友，但我想说：谁给他们的权力？他们有资格这么做么？如果这样做无异于犯法！如果盟友们都是这么被“逮捕”来的，网戒中心不会存留到现在，早就被司法机关给关门了，而且对于家委会有什么好处？谁去给盟友交住院费？他们的父母会同意吗？文章中所说的这一切真的说不过去，没有一点点逻辑可言。<br/><br/>当然，对于触乐网祝佳音之流来说，他们本来就是混蛋逻辑。<br/><br/>再就是关于杨永信医生。看到那么多负面文章，我个人真的为杨医生感到不平。这么一个大爱无私的人被说成了一个思想龌龊的人，一位拯救了万千青少年的仁心医者被炒作成了折磨青少年的恶魔，这个社会还有没有天理？媒体还有没有人性？读者们还分不分黑白？杨医生每天面对着我们这样一群问题孩子，十几年从没退缩，他一直在用自己的心灵去感化我们，教我们如何学习，如何生活，如何做人，给我们心灵的洗礼让迷途的羔羊重变天使，杨医生真的太不容易了。我自己很想为他做些什么，但也只能尽自己的一份微薄之力，希望用我的亲身感受为中心讨回公道。<br/><br/>“和杨永信战斗的人们”啊，请你们别再做无用的努力了，不要再被触乐网祝佳音蒙蔽欺骗做他们的炮灰了，邪恶永远战胜不了正义。<br/><br/>我们盟友和家长们会永远站在杨医生和网戒中心这一边，因为他们是我们的恩人。<br/><p align="right">盟友 晓谌</p><br/>首次转载时间: 2017-04-23 01:37:33"
