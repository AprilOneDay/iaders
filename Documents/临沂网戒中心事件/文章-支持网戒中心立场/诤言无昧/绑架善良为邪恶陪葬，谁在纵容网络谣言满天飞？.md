class="WB_editor_iframe"><br/><br/>??<b>无聊的看客与被绑架的善良</b><br/><br/>近日，重庆万州大巴坠江的最后一段视频让人感到心堵。司机与女乘客争吵互殴，而其他人则泰然若素，均以为和自己没有半点瓜葛，当大巴冲破护栏高高坠向江面的那一刹那，所有人才发出一声垂死求生的惨叫……<br/><br/>在正义面前，袖手旁观的看客固然可悲，然而，被邪恶绑架的善良则令人愤懑。<br/><br/>网上曾流行一个著名的段子，真实性无从考证，却颇具现实意义：<br/><br/>荒山僻野的山间公路上，几名歹徒盯上漂亮的女司机，强迫大巴停下，要带女司机下车玩玩，全车乘客无视女司机的呼救，噤若寒蝉。<br/><br/>一中年瘦弱男奋起，大呼全车人制止暴行，却被打翻在地，无人响应。半个时辰后，歹徒与衣衫不整的女司机归来。意想不到的是，女司机竟然声色俱厉要求该男下车。<br/><br/>“下车吧，我的车不拉你！”<br/><br/>“你这人怎么不讲道理，我想救你有错吗？”<br/><br/>“你救我？你救我什么了？”女司机一脸“嘲讽”，乘客哄笑。<br/><br/>瘦弱男气极，坚决不下。<br/><br/>女司机面无表情，态度坚定：“不下车，我就不开。”<br/><br/>没想到的是，刚才对暴行视若无睹的乘客，犹如刚刚睡醒，异口同声地劝那男子下车：你快下去吧，我们还有事呢，耽搁不起！有几位壮年乘客甚至想拖这男子下车。<br/><br/>歹徒们咧着嘴，得意地笑了，“哥们把她玩恣了！”“她是我对象，关你屁事！”男子被推搡而下，女司机平静的掠了一下头发，汽车又平稳地行驶在山路上，车到山顶，拐弯处右侧是百丈悬崖。汽车猛然加速......<br/><br/>第二天，当地报纸报道：伏虎山区昨日发生惨祸，一大巴摔下山崖，车上司机和十三名乘客无一生还。<br/><br/>两个故事，虚实之间写明了当今的两种社会生态――高高挂起的看客，被邪恶绑架的善良。<br/><br/>谁为正义执言？谁为邪恶陪葬？<br/><br/><b>野蛮生长的游戏与被妖魔化的戒网瘾英雄</b><br/><br/>游戏本无原罪，但其野蛮生长却很不正常。<br/><br/>十几年前，网瘾一度被视为洪水猛兽。一批批孩子因沉迷游戏而荒废学业，一个个长发披肩浑身恶臭的少年被家长从网吧里拽出，因长期沉迷游戏而引发的恶性案例屡见于媒体报端，网吧被视为藏污纳垢之地，网瘾被视为洪水猛兽，社会与媒体呼应，游戏如过街老鼠，戒网瘾机构应社会所需如雨后春笋。<br/><br/>即使是这样的环境，也未曾遏止游戏的野蛮生长，尽管在这个过程中社会为之付出了沉重的代价。<br/><br/>有钱使得鬼推磨，重金之下有勇夫。偶像明星纷纷为游戏代言，丰乳肥臀与铠甲英雄的广告无孔不入，2017年，中国游戏产业年创收2036.1亿元，手游收入占比57％，成为当之无愧的文化支柱产业，壮大了的资本也从社会边缘走向主流舞台。<br/><br/>经济基础决定上层建筑，经济利益决定阶级立场，很多媒体也在市场经济中迷失方向自甘堕落。<br/><br/>今天，即使全国几乎所有的基础教育学校都在禁止手机进校园，面对沉迷游戏荒废学业的孩子，面对与孩子抢夺手机导致的自杀自残的惨案和求助无门的家长，面对那些因深度沉迷游戏而导致精神异常的患者，媒体已经很少提及“网瘾”二字，那些所谓的专家学者面对媒体的镜头，几乎众口一词的把责任推向家庭和学校教育，鲜有人提及祸患的源头在哪里。<br/><br/>没有人统计：因痴迷游戏，公众的平均休息时间减少了多少？对全民健康的影响有多大？对工作绩效、经济发展、道德素养等带来的负面作用究竟如何？更没有人再去统计全国究竟有多少网瘾患者！<br/><br/>这，就是被邪恶绑架的善良。<br/><br/>谁都怕挨骂，怕被诅咒，因为网络游戏已经培养出数以亿计的玩家，据说，他们中的很多人已经把游戏作为人生信仰，非常忌惮“网瘾”，并擅长以游戏化的网络语言围殴支持“网瘾戒治”的人和事。<br/><br/>不仅如此，市场经济下，资本的发展已经把媒体发展为首要的利益同盟。<br/><br/>在这样的语境下，杨永信，曾成功救治数千例网瘾问题孩子，荣获国务院政府特殊津贴的精神科专家，备受家长拥戴的“救命恩人”，在一次次不可预期的炒作风波中，被黑成“电击狂魔”、“邪教教主”。<br/><br/>即使他殚精竭虑创建的网瘾戒治和性格缺陷矫治综合模式已经通过官方和高端专家组的鉴定，即使这个模式是有108个环节的系统工程，而媒体炒作所需要聚焦的，就是那个在精神科专业范围内的“低频脉冲”，和少数几个未成功改变的患者爆料。<br/><br/><b>媒体的臭脏黑与舆论监管的真空</b><br/><br/>和资本利益捆绑的媒体，正严重扭曲着社会心态和主流价值观。<br/><br/>近日，人民日报、新华社频频发文，剑指自媒体的“臭脏黑”乱象。文章指出，部分自媒体以发布负面信息为要挟，敲诈勒索收取企业“保护费”；把“拿钱办事”的恶习发挥到极致，只问报酬不问是非……其中有文章还一针见血的发问：世上只有两条赚钱之道，一条曰“合法”，一条曰“违法”，有多少自媒体“裸奔”在第二条道路上?<br/><br/>然而，臭脏黑的又何止是自媒体？<br/><br/><b>给钱就说话，有奶便是娘，这是媒体走向市场的怪象</b>。2009年，就在杨永信被主流媒体以戒网瘾英雄频繁报道，社会对网瘾关注聚焦时，5月7日，中青报记者白雪、王烨捷在中青报<b>“经济特刊”</b>专栏（请注意：是<b>“经济特刊”</b>）发表了三篇“力作”：《一个网戒中心的生态系统》、《谁都想在网瘾治疗市场分杯羹》、《“戒网专家”电击治网瘾惹争议》，拉开了妖魔化网瘾戒治的序幕。此后9年，媒体对网瘾和网瘾戒治的态度一再走偏，即使是前不久网瘾被世卫组织纳入精神疾病之后，恶媒谣记试图重挑争议为网瘾翻案的努力就一直没有停止过。<br/><br/>所有的网瘾戒治机构，不管此前成功救治了多少患者，不管积累了多少成功经验，只要出现一例安全事故，只要一经媒体曝光，就会被舆论群殴，网瘾戒治妖魔化，网戒机构见光死，几乎无一例外。<br/><br/>正常吗？这就是资本的力量，经济基础决定上层建筑，经济利益决定阶级立场，马克思的经典论述，就这样被现实一再验证为真理。<br/><br/><b>谣言制造如此离奇玄幻，舆论监管又在哪里？</b><br/><br/>近日，一篇“开网店被送杨永信的网戒中心，离开后这十年”的帖子在微博上风传，5天之内，阅读量突破千万，创网络谣言之最。<br/><br/><b>这是篇近乎玄幻的谣文，它打破了以往妖魔化网瘾戒治的写实风格而走向神话模式：杨永信不仅戒网瘾，还管着患者的生育权；杨永信不仅戒网瘾，还用“电击”帮反贪局破案，甚至想把“电击”模式大而化之到治理国家。最为“罪大恶极”的是，他竟然把“我”这个开网店做电商创“千万”巨富的成功人士也拉去戒网瘾了？！</b><br/><br/>这样的谣言居然挂在网上，而且引爆舆论，欺骗社会，强奸善良，围殴正义，抹黑英雄。<br/><br/>浏览跟帖评论，观看客百态，颇值得玩味：<br/><br/>一种是故意搅浑水，推波助澜煽风点火。这种人用语污秽粗鄙加表情图片，诅咒谩骂，一看就不是善类，他们是恶媒谣记的追随者，玩家中的败类，扰乱舆情的急先锋。<br/><br/>一种是被谣言蛊惑，不分青红皂白，不理性分析真相，随波逐流，宣泄“义愤”，表达“观点”，标榜“正义”、“悲悯”和“善良”。这种人大多以为“网瘾”就是喜欢上网，“戒网瘾”就是反互联网反游戏反电竞，并在不经意间成为邪恶的帮凶。<br/><br/>一种是明辨是非，支持正义，对谣言质疑辨伪。<br/><br/>而更多的人是忌惮网络暴力，怕引火烧身而选择沉默。<br/><br/><b>我们需要思考的是：是谁让这样的谣言在网络上如此放肆？</b><br/><br/><b>又是谁在纵容媒体无底线的“臭脏黑”？</b><br/><br/>当有一天社会价值观被彻底扭曲，当善良被邪恶绑架，没有人再为公平正义呐喊呼吁，没有人直面网瘾解决问题，后果如何？<br/><br/>社会是一辆公交车，这辆车上没有看客，愿公众不再被谣言蒙骗，愿政府不要被舆情绑架，让造谣诽谤等邪恶之徒如过街之鼠，人人喊打。<br/><br/>为了不再让善良为邪恶陪葬，惩治媒体“臭脏黑”刻不容缓。<br/><br/>中宣部，网信办，请挺直你正义的腰板！<br/><p
