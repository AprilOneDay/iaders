/><br/><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？（之二十）<br/><br/><!--more--><br/><br/>近日读过澎湃新闻《“无法告别”的临沂网戒中心：“杨叔”，“盟友”和家委会》，令人非常气愤，文章对临沂网戒中心的报道严重失实，而且我觉得这是记者故意所为。其中有几位所谓临沂网戒中心走出来的盟友，讲了好多在那里的经历，在我看来非常陌生。<br/><br/>作为2013年在临沂网戒中心改变过的一位盟友，就谈一谈我的亲身经历，以正视听。<br/><br/>我是山东泰安人，来临沂网戒中心之前，我已经走偏成了一个地地道道的魔鬼，每天无所事事，整日喝酒，纵欲，挥霍！对父母，只有一味的索取，父母辛辛苦苦经营的生意，也差点被我搞垮。对自己的老婆孩子，也是不管不问，自己在外面租了房子，拿着父母的血汗钱，养起了情人。在钱财挥霍得差不多以后，情人离我而去，自己回到家中，不思悔改，反而迁怒于生我养我的父母，终日里浑浑噩噩，行尸走肉一般，稍有不如意，轻则又摔又砸，重则对父母拳脚相加。现在想起来，那时候的所作所为就是一个畜生！<br/><br/>百般无奈之下，父母把我骗到了临沂网戒中心。<br/><br/>在临沂网戒中心，我看到的是处处洋溢着和谐与博爱，充满积极向上的氛围，每个人都很热情，很真诚，很友好。在这里，父母和子女找回了亲情，夫妻之间找回了爱情，父慈子孝，家庭和睦，人与人之间充满了理解和包容。刚开始陪我在中心改变的是妈妈，每天在道德讲堂，唱着《跪羊图》《父亲》《母亲》，背着《弟子规》，听着杨永信医生的谆谆教诲，让我突然明白了自己以前是多么的混蛋，作为人怎么可以那样来伤害生我养我的父亲母亲。<br/><br/>从此，我痛下决心，一定要痛改前非，好好孝顺自己的父母双亲。<br/><br/>在中心住了将近一个月，由于妈妈有糖尿病，我主动提出让媳妇来陪护，因为我觉得在中心好多人都找回了自己的爱情，找回了自己的家庭，我也想试试。后来我才知道，起初我媳妇是不愿意来的，因为她已经被我伤透了心，是杨永信医生一直在做她的思想工作，如果没有杨永信医生，等待我的就已经是支离破碎的家庭了。在媳妇来到中心之后，杨永信医生又跟中心其他点评师一起做我媳妇的思想工作，我跟媳妇之间的坚冰终于慢慢融化了，我濒临毁灭的家庭得救了。<br/><br/>在中心，我经历了7个月的改变。这期间，杨永信医生教给了我很多做人做事的道理，教会了我要对规则有敬畏，教会了我要对家庭对社会有责任，教会了我要百折不挠越挫越勇，教会了我要有阳光心态面对一切……这些，很多都是在学校里学不到的。<br/><br/>离开中心已经4年了，现在的我，又有了一个两岁多的儿子，四口之家，温馨而又甜蜜，爸爸每天打打乒乓球，最大的乐趣就是陪着孙子，享尽天伦之乐。妈妈参加了老年大学，每天唱歌跳舞，真正的老有所乐，而且心情好了，血糖也奇迹般的恢复了正常。媳妇跟我感情越来越好，我们两个人一起用在中心学到的知识来经营自己的家庭，教育自己的子女，全家人都受益匪浅。现在的我，戒烟戒酒，生活规律，接过了爸爸的公司，成为了一家注册资金三百万的运输公司法人，把公司经营得井井有条，将来，我更有信心将公司的事业发扬光大。<br/><br/>这些，都离不开在临沂网戒中心的改变，离不开杨永信医生的辛苦工作，而我，只是中心救助的6000多个走偏盟友中的一个。<br/><br/>“澎湃新闻”，我想问你们，为什么不来采访我们，而只是偏听偏信，去采访个别没有改变好的问题孩子，用他们的胡说八道来抹黑我们的网戒中心，你们不觉得这样很不客观很不厚道吗？这些人的话的可信程度，就不应该打上一个大大的问号吗？像“未消逝的青春2015”这样的，动辄就离家出走的人，动辄就可以弃自己的父母于不顾的自私到了极点的人，他说的话可信吗？如果只是考虑他个人的感受你们就这样来抹黑诋毁我们的网戒中心，试问，是不是你们也会因为罪犯的个人感受就抹黑诋毁我们的国家机关和我们的法律呢？你们媒体是足够强大，你们掌握着足够的舆论资源，你们拥有别人没有的话语权，但是你们用舆论来绑架政府，用黑暗来掩盖真相，最后也只能是搬起石头砸自己的脚！不要人云亦云，以讹传讹，更不要颠倒是非，混淆视听！你们要有最起码的职业操守！作为新闻人，你们的责任是报道真相，还是抹黑和掩盖真相？！<br/><br/>今天，我站了出来，明天，如果你们还不知悔改，会有6000多个家庭站出来，甚至更多，你们掩盖不了真相，乌云也遮不住太阳！我强烈要求你们对临沂网戒中心和杨永信医生正式道歉并及时删除你们之前的不正当言论，不然我们必然拿起法律的武器捍卫我们的网戒中心！<br/><p
