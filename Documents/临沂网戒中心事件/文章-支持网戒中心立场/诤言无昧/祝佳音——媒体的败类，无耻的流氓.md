。”<br/><br/>他再次向我道歉，说尽快处理此事。<br/><br/>不实微博一直没删除，3月24号14：31分接到15605**5490的电话，说再了解一些情况。我说：“那天我已解释的很清楚明白了，你微博还没删除，我们没有谈的必要。”<br/><br/>直到3月29号，我看到20:09分祝佳音发了一篇《被杨永信拯救的家长们》，尽管里面的采访内容被断章取义，歪曲了事实，以偏概全，但我依然看出这是来自对我的采访，而且违背了我的本意！<br/><br/>我打电话13028**1412跟他理论，此电话号码已不使用，打15605**5490，我说：“我们的聊天内容被你断章取义，歪曲了事实，违背了我的本意，你赶紧删除。”他随后挂断电话，再打不接，再打关机，我知道自己被骗了。<br/><br/>他，就是祝佳音――触乐网的创始人，一个毫无职业操守的骗子，一个卑鄙无耻的小人，是媒体的败类。这两次通话都有录音，如果必要，我会公布在网上，揭露他的丑恶嘴脸，让他臭名昭著。<br/><br/>我再次以一个被采访家长的名义声明：祝佳音的文章《被杨永信拯救的家长们》，采访内容被断章取义，严重失实，违背我的本意，不可信。请立即删除微博！<br/>首次转载时间:
