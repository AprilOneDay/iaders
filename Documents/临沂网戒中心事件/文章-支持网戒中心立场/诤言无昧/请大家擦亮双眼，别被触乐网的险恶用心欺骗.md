<br/><img src="http://wx1.sinaimg.cn/crop.0.0.263.147.1000/0067hHJjly1ff9macqu6aj307b045mxc.jpg" /><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？<br/><br/><!--more--><br/><div class="WB_editor_iframe"><br/><br/>??看到触乐网发布的这篇题为《盟友小辰：逃离临沂》的文章，作为临沂网戒中心的盟友，一位真相的见证者，我感到十分气愤，因为这篇文章歪曲事实，对临沂网戒中心这么多年拯救数千网瘾问题孩子的贡献只字未提，而写了许多不真实的地方，我在此提出，希望大家不要被不实报道所蒙蔽。<br/><br/>说实话，读触乐网的这篇文章，听小辰讲的“生死逃亡”，我觉得恍如隔世，就像看一个玄幻色彩的穿越故事。<br/><br/>1.文中所说的家委会去小辰家找他妈妈谈话，还说家委会是什么“别动队”，说得好像网戒中心是个可以非法拘人的地方，家委会有事没事就去抓人。然而我在中心长达几个月的经历，从来没有见过家委会出去抓人，家委会只是给住院病友和家长提供服务并照顾大家的住院生活，他们和杨医生一样和善热情，杨医生教我们做好人，家委会热情为大家服务，根本不是小辰口中的“别动队”。<br/><br/>2.文中小辰说，“如果留下来，就很可能遭黑手”。我想问问小辰，现在是什么社会？法治社会，文明社会，人人平等！请问小辰你编的故事究竟是哪个朝代，哪个国家？而且中心的家委会成员都是一些有文化素质和思想层次的人所担任的，请你不要把家委会说得跟流氓、混混一样。<br/><br/>3.小辰还说，“中心可以动用警力来跟踪盟友”，还有“卫星定位”，“监控系统”什么的。我想说的是公安和警察系统是国家的权力工具，不会也绝不可能成为私人的工具，小辰所描绘的完全是不可能的事情。<br/><br/>4.文中说，家长和盟友在住院时都“讨好杨医生”，甚至故意把自己的过去说的很坏，然后来衬托中心的疗效多么的好。我想说的是，家长全部是24小时陪护的，那么你的家长就没有自己的思想和见解么？如果中心真如文中所说那样，我想你的父母也不会待在那里，天下哪个父母会害自己的孩子呢？如果中心只靠吹捧，没有实际效果的话，爸妈也不会甘心给你花住院费的。<br/><br/>5.文中所说的“用电击控制盟友”，让他们变得像一个个木头人一样听从他人。这是一个完全不属实的信息，因为我从来没有见过一次这种所谓的治疗仪器，完全不属实。<br/><br/>以上几点，是我对文中内容所存在的质疑，所以我要在此发声，让更多不明真相的人了解真相，别被你们触乐网等人的险恶用心所欺骗利用。<br/><p align="right">盟友 新波</p><br/><br/></div><br/>首次转载时间: 2017-05-05 17:44:10"
