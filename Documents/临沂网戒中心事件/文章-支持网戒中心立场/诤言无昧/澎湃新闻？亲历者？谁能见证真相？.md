/><br/><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？（之十八）<br/><br/><!--more--><br/><br/>大家好，我是山东临沂网戒中心的一位受益者――曾经的问题孩子。我想跟大家分享一下我自己的经历，好让大家知道，网戒中心里的问题孩子是不是像澎湃新闻所描写的那么“正常”。<br/><br/>“照片”、“贷款”、“18岁”、“女大学生”，这四个常人心中很难联想在一起的词语，准确地概括了那段我不堪回首的往事。<br/><br/>大一的时候我特别贪玩，几乎每天都会和朋友们出去玩。那个寒假，正是我最自由的时候，从每天出去玩2、3个小时再到5、6个小时，最后到夜不归宿。后来，朋友给我介绍了一个男朋友，因为那是我第一次谈恋爱，所以看得几乎比生命都重要。但是后来才明白自己太天真了，世界上没有那么容易得到的美好爱情。分手以后，我就像变了一个人一样，从一个有文化、有素质的大学生变成了一个抽烟、酗酒、满嘴脏话的泼妇。<br/><br/>如果仅仅是这样还好，可越往后我的做法越不正常。<br/><br/>分手后，我觉得自己一定要变得优秀，一定要变得完美。但我这时心中的“完美”却是变态的完美，就是要吃好穿好玩好，让别人前呼后拥，前男友才会后悔抛弃我。从那以后，我的心理开始扭曲，开始看不起任何人，更害怕被别人看不起。我有一个朋友以前是干老鸨的，她认识很多社会上的有钱人。以前我最烦的就是她每天花枝招展地游荡在人群之中，毫无节操，但后来我开始慢慢向她靠近了，也天天跟着她去和有钱人一起玩，我就感觉没有人会看不起我了。那时候的我会在晚上睡觉的时候流泪，因为我自己都看不起这样的自己，但是到了第二天又忍不住想跟这群人一起吃喝玩乐，享受虚荣的生活。<br/><br/>记得有一次我过生日，很多男人请我喝酒，我感觉自己被宠上了天，扭曲地想着如果我的前男友看到我今天这么受欢迎，他一定后悔死。那一天我连续喝了好几个场子，喝的晚上回去头痛得都快要炸了，还一直吐酸水，第二天躺了一整天。<br/><br/>可我仍不知回头。<br/><br/>爱慕虚荣的我开始羡慕我这个老鸨朋友的高收入。于是，在一个朋友的忽悠下，我开始贷款干微商挣钱（也就是裸贷）。从那以后，我就开始活在了钱的世界里，淡漠亲情，每天都跟红了眼的疯子一样对身边的一切充满了怨恨。<br/><br/>在放款人一次次的强压下，我无力还钱，被逼得无路可走，最终把自己的“裸照”给了他们。但后来他们更加放肆，每个星期都会问我要2000多的利息，我根本无法用正常的工作去偿还这笔巨款。慢慢的，我把这种对他们的仇恨转移到了父母的身上，甚至几度想与父母拼个你死我活。后来就干脆不回家了，每天跟着朋友住酒店，半夜出来玩。整天走遍各处，想方设法去弄到钱来支撑、来弥补一家家的贷款。海南、济南等好几个城市我都去过，放贷人让我去哪里借钱，我就去哪里，只不过就是为了弥补我因一时糊涂犯下的错误，不断地出卖着自己的“人格”和“清白”。<br/><br/>妈妈每天都会打电话让我回家，那时候我感觉到的只有恶心与厌烦，因为长时间的巨大压力，我在家里也开始抽起了烟，要知道我爸爸是个滴酒不沾而且格外注重品德修养的一个人，但这些他都忍了。在一次妈妈打电话数落我的时候，我把所有的负面情绪都引爆了。我竟然自己去放贷人的地方住了四天！在第四天的时候给我妈妈打电话说：“这钱你要是不帮我还，你就别当我妈了！”父母眼里那个活泼善良的女儿突然发生了天翻地覆的变化，对他们来说是多么沉重的打击啊！<br/><br/>妈妈哭着求我回去，而当父母把钱还上的时候，我的手机被放贷人关机了！<br/><br/>当晚上爸妈从家里赶到很远的外地时，无论放贷人怎么狮子大开口我都不敢做声，因为我很清楚他们手里有我的把柄，我甚至还帮他们说话。刑警本来想把他们带走的，可看到我一副一厢情愿的奴才样，他们顿时也没有了理由。<br/><br/>回家后父母没有责怪我，还带我买了衣服。可是过了几天我又一次离家出走了，因为另一笔钱该还了。走的那天晚上，我姨就一直看着我妈，生怕她自杀。等我回来的时候，从小到大都没有动过我一指头的爸爸却狠狠地打了我，他说想打断我的腿，哪怕养我一辈子，也不愿意我出去鬼混。这一次爸妈又给我还了4万多，他们以为没有外债了，一家人就可以好好的过日子了。<br/><br/>可是，连我自己都不记清自己究竟借过多少钱了！<br/><br/>果然，又有一笔忘记还的钱发来了短信，不敢告诉父母的我，在心魔的驱使下又借了一笔补窟窿的钱，后来被我爸发现了。那天晚上从没有哭过的爸爸边哭边打自己的脸，说都怪他没有能力保护好自己的女儿。看着父亲的眼泪，我也知道自己错了，可那时候，我的“资料”已经被卖给了很多人，我没法回头。<br/><br/>当爸妈都跪在我面前求我别走的时候，我无法直视他们的眼睛，因为我已经陷进那种奢靡的生活里出不来了。<br/><br/>当我再一次要出去“借钱”的时候，父母把我关了起来。我哭闹了一晚上终于在凌晨三点多的时候睡着了，早上起来我找不到任何跑出去的路，无力地躺在了床上，眼看还款的时间就要到了，如果还不上，我的“照片”不知道会出现在多少人的电脑里，我绝望地想：等晚上爸妈睡觉的时候我就割腕自杀，我真的累了。<br/><br/>幸运的是，当天中午我就被爸妈送到了临沂网戒中心。<br/><br/>走进中心的那一天是我人生新的开始。当我来到中心时，我发现这里的人都特别善良。我没有经历过那些不实报道里所说的“低频脉冲治疗”，但我知道所有没来过这里的人都没有资格诋毁这里的一切。当看到澎湃新闻的那篇《“无法告别”的临沂网戒中心：“杨叔”，“盟友”和家委会》时，我特别气愤，也特别的心寒。那些不良媒体为了利益颠倒黑白，胡说八道，抹黑中心，一些问题少年盲目从众，不辨是非，对中心造成了很恶劣的影响。<br/><br/>我是中心的见证者和受益者，我应该有资格说话。<br/><br/>中心的108个环节，能把魔鬼变成天使。俗话说得好“冰冻三尺非一日之寒”，我们在这里改变的过程就是把我们那颗冰冷的心融化的过程。当我第一次给妈妈洗脚的时候，看着妈妈那受宠若惊的表情，突然鼻子酸酸的，想一想自己竟然从来没为妈妈洗过脚。后来，当别的孩子跟自己的父母很和谐地沟通交流时，我看到妈妈在旁边十分羡慕地看着他们，那时候的我突然想起自己已经好长时间没有坐下来与妈妈说说心里话了。别人都说女儿是母亲的小棉袄，可我是什么呢？<br/><br/>当然，我也有感到不适应的时候，特别想回去，当消极思想出现的时候我甚至打算出去之后就离家出走，继续过我随心所欲的生活再也不回家了。但是后来通过杨永信医生在心理点评课堂对我们一次次苦口婆心的教育，我明白了我现在所做的一切都是错误的。“家才是最温暖的港湾”，如果没有父母那么我又从何而来呢？我不但不感谢他们赐予我生命，还仇视他们，是多么的冷血啊！<br/><br/>如果不来到中心，我只有两条路可以选，一个是监狱，另一个则是地狱。<br/><br/>中心就像杨医生为我们这些问题孩子建的世外桃源，在这里的每一节点评课都能让我把内心中最丑陋的思想挖掘出来，一一净化。在这里我发现了好多自己存在已久，但却又不敢承认的缺点和短板；在这里我开始慢慢地不让自己受消极情绪控制，不让自己那么冲动；在这里我明白了什么事情该做什么事情不该做，做人一定要有最基本的底线，不能为一己私欲而去做害人害己的事，应该踏踏实实做事，规规矩矩做人。<br/><br/>这一切都是我之前闻所未闻的。如果不是杨医生每天费尽心思根据我们每个人心理的问题进行点评，那么我这辈子也不会明白这些道理，我就会一直在自己偏差的认知中沦陷。<br/><br/>毫不夸张的说，临沂网戒中心救了我的命！<br/><br/>可是，上海报业集团下属的澎湃新闻记者张维、贾亚男写的文章《“无法告别”的临沂网戒中心：“杨叔”，“盟友”和家委会》对中心进行了不真实、不客观的报道，污蔑中心，令人非常气愤！我不知道为什么这两个记者要如此煞费苦心地破坏这么好的地方，但是我明白“善乃福根，恶乃祸兆”，我一定要保护好我的“家”――临沂网戒中心。<br/><br/>家如果没了，我们这群问题孩子该何去何从？！<br/><br/>&nbsp;<br/>首次转载时间:
