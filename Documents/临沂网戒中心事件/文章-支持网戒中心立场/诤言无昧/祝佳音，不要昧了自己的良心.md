/><br/><br/>几起校园霸凌事件，引起社会的高度关注；一起精神病患者持刀杀人案，引起政府对监护制度的重视落实；网瘾肆虐十几年，问题少年无数，惨案充斥媒体报端，难道还不能引起我们足够的警惕？！我们的力量尽管微小，但我们有权力用良知向社会表白我们见证的真相！――<br/><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？<br/><br/><!--more--><br/><br/>读完《被杨永信拯救的家长们》这篇文章，自己内心最想说的三个字就是“不公平”。这篇文章中有很多有问题的地方，会给人带来一种颠倒黑白的感觉。如果我自己没去过临沂网戒中心的话，看完这篇文章后我会相信一些事情，可惜我是在临沂网戒中心改变过的孩子，所以我有资格也有责任说出事实的真相。<br/><br/>在文章中有一处地方说：他们的采访是和家长们在“大爱流淌”的氛围下进行的。我不反对这句话，但作者为什么要对四个字加上引号？因为加了引号之后容易让别人产生一种不真实甚至相反的感觉。还有一个地方说“他的孩子去了以后就出不来了”，这一点更是大错特错，在这里住院的每一个盟友都会面临出院，因为这个权力在家长手中，只要家长说走，杨永信医生是不会留的，也没有权利留。还有一处说：有大多数孩子是和小唐一样是因为看动画片被送进来的，这简直就是作者睁眼说瞎话，我敢说在社会当中的每一个人都看过动画片，为何他们没进来呢？因为他们没有因为看动画片耽误自己应该干的事情，更何况这里根本就没有单纯因为喜欢看动画片被送进来的病友。<br/><br/>在临沂网戒中心里根本没有教官，这里只有医生，家长，护士，盟友。医护人员根本不会动手打我们，而且在这里我们也没有必要为了出院而去讨好谁，因为在这里的每一个人都知道“改变为自己，付出为未来”。另外，杨医生在课上从来没有说过自己是专家、他具有权威性什么之类吹嘘自己的大话，他的点评课讲究以理服人、以德育人，以情感人，以爱化人。<br/><br/>在文章当中还提到了“磕操”，纠正一点，在中心里我们自己叫做“课操”，因为这是一门健康身体、净化心灵、修养身心的一门功课。但是不清楚为什么触乐网要写成“磕操”甚至“磕头操”。作者说在中心大部分人每天要磕1000个操，我笑了。在这里课操因人而宜，而且数目也远远达不到作者说的那么多。在这里的氛围是靠大家自己努力去维持的，如果这里对我们没用，我们也不会来到这里。<br/><br/>临沂网戒中心是一个功德无量，利国利民的地方，并不是作者说的那样邪恶。在临沂网戒中心处处充满了正能量，她让我们重塑自我，健康身心，无数问题孩子和患难家庭在这里受益，我们应该让这个地方不断的扩大，而不是让你们那些游戏商得逞。我们是正义的，而你们是邪恶的，我们坚信“正义永远会战胜邪恶”。<br/><br/>写到这里，我想介绍一下自己。我父亲是一个企业老板，可以说曾经我的家庭条件比较优越也特别幸福。但是正因为我的走偏，我的家庭变得“鸡犬不宁”，家人们都为我之前的所作所为感到绝望。<br/><br/>我彻底走偏时只有13岁。在2013年九月初，我接触了摩托车并深深痴迷，整日想着与那些损友一起飙车玩，可是自己的欲望越来越大，从而变得越来越消极。我开始抽烟，打架，喝酒，包小姐，玩摇头丸。当时的我喜欢上了“夜夜当新郎”的感觉，每天和不同的女生开房。那个时候心里从没有道德感，也从来都不害怕会得病，只感觉特别的享受。我还爱打架，可以说是特别的残忍，下手根本没有轻重，拿着匕首白刀子进红刀子出，根本没有任何害怕的感觉。那时的我用“武力”打下了自己的一片天，我带领着我的团伙到各个学校给别人“站场子”。还记得有一次去替初中生打架，动用了几十人，我们一次性赚了一万多块的“保护费”，拿到钱我就带着损友们去夜店找小姐。之前我从来没有感觉到自己做的那些事情是错的，对于那些管我的人，不管是父母还是老师我从来都置之不理，我感觉自己干的就是对的。<br/><br/>可是就在我的父母最无助的时候，也是我走偏最厉害的时候，我的父母四处打听，来到临沂网戒中心实地考察，他们考察了两次，最终决定将我送到了临沂网戒中心。一开始我特别反感这个地方，可是经过了几天的心理点评课，自己开始接受这里。从那时开始我的人生发生了巨大的转变。<br/><br/>我开始变得活泼开朗了，而且能够认识到自己的错误。来到这里，我干了很多我之前认为根本不可能的事，例如给父母洗脚，跟父母坐在一起好好沟通，改变自己的人生观世界观价值观，以及诸多不好的恶习，这些都是我之前认为不可能改变的，可是事实证明这里把我的整个人生都给挽救回来了。而我唯一的心情就是自责，我对不起我的家人，和那些被我欺负过的学生，我之前的所作所为给家人带来了多大的伤害，之前干过的坏事我都不知道如何去弥补。<br/><br/>可是现在的我想明白了，只有改变，做一个好人才能让自己以后的生活变得更好，我要在临沂网戒中心这个神圣的地方进行人生中的一次洗礼。我来到中心现在已经接近四个月了，在这四个月的时间里我学到了我这一辈子都没学到的东西，给自己的人生找到新的方向，我永远感激这里！<br/><br/>希望触乐网以及那些不怀好意的媒体悬崖勒马，不要破坏我们共同的家园，更不要拿着中心当作你们炒作的摇钱树。人在做，天在看，不要昧了自己的良心。<br/><p
