class="WB_editor_iframe"><br/><br/>???“抬头望见北斗星，心中想念毛泽东，迷路时想你有方向，黑夜里想你照路程……”<br/><br/>随着9月9日的临近，这些天看到微博上的好友们很多让人热血沸腾的文章，他们在用对开国领袖的怀念，对自己青春岁月的回忆和对当今诸多流弊的抨击，试图唤醒社会的良知，试图梦回他们曾经的那个虽然和今天相比显得清贫却充满公平正义、激情燃烧的时代。<br/><br/>他们在呐喊，用尚未被物欲泯灭的良知！<br/><br/>今天的祖国，正在接近中华历史的巅峰和辉煌，国家的快速发展，是任何人不能否认的现实。但是，毋容置疑的是，我们却少了一份本应让社会感知感恩的幸福，也缺失了本该有的维护社会公平正义的责任。<br/><br/>我深深的相信，这些微博好友怀念毛主席，怀念周总理，怀念朱老总，并不是怀念那个时代的清贫，事实上那个时代与今天相比虽然是清贫的，物质生活虽远远不能和今天相比，但历史是向前比的，要比起解放前全民逃荒要饭闯关东的年代，已经是了不起的进步了，而最重要的，是那个时代的社会洋溢的公平正义，是社会心理中实实在在的感恩和幸福感。<br/><br/>我深深的相信，他们怀念那个时代，不是否认今天国家在科技上的发达、军事上的崛起和综合国力的提升，他们之所以在今天怀念毛主席，是因为毛主席率领中国共产党缔造了一个新中国，缔造了一个空前公平正义、人心向善的伟大时代，为民族复兴和国家发展，为未来的历史开辟了一条充满光明的金光大道。<br/><br/>有些喷子可能会说，你若羡慕那个时代，你就回到从前好了。这其实就是一种悖论，因为我同样深深的相信，他们怀念毛主席，怀念那个时代，是源于忧国忧民的忧患意识。我清晰记得那个时代的典型场景：在春耕需要地里施肥的时候，某一天早晨你会发现，早春的田地里忽然多了满满一堆土杂肥，你却不知道是谁在夜深人静时做了这件好事；你丢了的钱包，某一时刻生产大队的喇叭里会响起让你去认领的声音，但你又不知道这是谁拾金不昧，除了感激这个火红的年代，你不知道具体去感谢谁；当你的家庭遇到困难，你不用开口就会得到好心人的帮助；一位大队会计仅仅因为偷用了几元钱，就以贪污之名受到审判；一位县物资局的女会计仅仅因为挪用12元钱，就被定为贪污而开除公职；而让我们倍感温暖而怀念不已的，是随时为你解决疾苦的赤脚医生和合作医疗，今天已成为永远的幸福记忆……<br/><br/>今天说起路不拾遗夜不闭户似乎是笑谈，但却是那个时代的真实写照。<br/><br/>今天，活买不起楼房，死买不起墓地，甚至于连掉一颗牙你都镶不起，这已经不是个别人无法回避而且必须接受的现实。<br/><br/>这还不是最可怕的，最可怕的是什么――<br/><br/>是社会价值观念的扭曲，是与大国崛起不相称的道德素养，是社会少了维护公平正义的价值认同。对金钱的崇拜已经成了一个社会的畸形信仰，而资本，俨然已经成为无所不能甚至操控国家意识形态的力量。<br/><br/>当碰瓷成为一种谋生的手段，当乞讨由谋生成为一种职业，你会不会为你的善良被欺骗而愤慨世风日下？但你又有没有反过来想一想是什么迫使他们用这种丧尽尊严的手段去谋生？<br/><br/>当全社会都在谴责官员的贪腐时，有没有人却在惋惜自己没有贪腐的机会？甚至有人还在想法攀附没有败露的腐败权贵而幻想鸡犬升天？<br/><br/>更为可怕的是，当国家崛起民族复兴需要强化意识形态唤醒社会良知的时候，我们的媒体却没有尽到历史的责任，甚至在资本利益的操控下操守沦丧，沦落为颠倒是非扰乱社会思潮的恶媒谣记。<br/><br/>本来，作为媒体和媒体人，之所以被冠以“无冕之王”的美誉，就是因为其天职就是捍卫真相，维护公平正义，而在媒体走向市场的今天，有些媒体却沦落为资本的走卒和利益的奴隶，把操守弃若敝履，打着公平正义的旗号颠倒黑白。<br/><br/>从21世纪传媒爆特大新闻敲诈案――昔日“新闻圣徒”成囚徒，到今天的丢肾门、纱布门、缝肛门、八毛门、绿茶门……门类之多，彻底撕去了不良媒体人的遮羞布，赤裸裸暴露了其利欲熏心造谣生事唯恐天下不乱的丑恶。<br/><br/>但是，切不要以为媒体操守的沦丧仅仅是市井巷陌之类的草根，甚至有些大媒腕媒也不例外。<br/><br/>在此，我想请教大家一个问题：自互联网普及以来，网络游戏一直高烧不退而且持续升温，游戏产业也水涨船高恶性膨胀甚至成为国家巨富的资本支撑，在全民撸起袖子加油干同筑中国梦的时候，游戏究竟给国家带来了什么？<br/><br/>进入21世纪以来，关于游戏与网瘾，一直是社会的热点话题，因为无法抵御游戏的诱惑，无数青少年因为网瘾不仅荒废学业、鬼混网吧、自卑自闭、焦虑狂躁、嗜血好斗，甚至为了装备升级更换皮肤不惜坑蒙拐骗，沉迷游戏已经成为青少年健康成长的头号杀手，学校为维护正常的教育教学秩序而到了不得不禁止手机进校园的程度，家长们更是把与游戏争夺孩子演变为一场战争。<br/><br/>而游戏商人，则在自信满满的看着无数家长血汗通过网瘾孩子手指神奇的变成源源不断的财源，游戏已然成为指尖上的印钞机。<br/><br/>媒体真的看不见么？当《王者荣耀》成为千夫所指的时候，当众多良知未泯的媒体纷纷谴责“游戏陷害人生”的时候，共青团媒为什么依然在否认网瘾，依然在吹捧网游能提高学生对历史的学习兴趣，依然把游戏美化成青少年人群的文化符号？马化腾为什么在关键时刻高调做客RM网？为什么全民声讨并未影响游戏大亨成为“万民景仰”的首富？<br/><br/>与此对比鲜明的是，长期呼吁健康上网，致力于网瘾戒治，十余年间成功让数千例网瘾问题孩子回归健康人生，让众多患难家庭重拾幸福的戒网瘾专家杨永信，却被恶媒谣记和共青团媒妖魔化为摧残青少年的“恶魔”！<br/><br/>没有恶媒谣记的张目，游戏巨贾的自信在哪里？！<br/><br/>全民游戏的时代，国家的未来在哪里？！<br/><br/>本来，民族复兴国家崛起与公平正义的社会生态是相辅相成的。<br/><br/>所以，“抬头望见北斗星，心中想念毛泽东”，不是对现实的否定，而是对社会生态理想化的民意诉求，是对公平正义的呼唤。<br/><p
