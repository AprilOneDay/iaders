/>?网游已经不再是一片净土。没有公平，只有赤裸裸的金钱、暴力、赌博和色情广告，可谓五毒俱全。盛大公司作为知名网络游戏厂商，用网络游戏到底怎样圈钱？近日，许峰、李斌等几个资深玩家向作为记者的我详细解释了其中的种种招数。?<wbr />?<wbr />?<wbr />?<wbr />?<wbr /><br/><b>?<wbr />?<wbr />?<wbr /></b>?<b>1.16</b><b>元人民币换一个网上“元宝”</b><br/>记者：盛大公司的《热血传奇》（玩家们简称为《传奇》）是一款什么样的游戏？<br/>玩家：是盛大2001年从韩国引进的。<br/>记者：他们是怎么收费的？<br/>玩家：以前《传奇》的经营模式只是收取一些点卡，起初一些玩家也无法接受，时间一长，也就自然接受了。可自从2003年一场知识产权案件之后，《传奇》进入了一个发展高峰，让它成为中国网游界的龙头。随后在2004年里，《传奇》出现了永久免费区，推出了虚拟货币。也就是说，进门不需要买门票了，但进去之后必须消费，不然在里面寸步难行，也就是从这时起出现了虚拟空间里的犯罪，并让很多人走向为现实生活中的犯罪。<br/><wbr />?<wbr />?<wbr />?记者：你认为，《传奇》圈钱的行为有哪些？<br/><wbr />?<wbr />?<wbr />?玩家：靠赌博圈钱。早期的《传奇》中有一种开箱子的功能，其实就是用钱买钥匙。后来，政府部门规定网络开箱子属于赌博行为，于是他们把开箱子改成了“神盒”，现在又叫“开天赐”，其实是和国家有关部门玩文字游戏。“开天赐”是一个宝箱的形状，用钥匙打开，打开后可以得到箱子里的一件随机物品。它就像老虎机一样，不停地转，里面有各种游戏人物所需要的装备。我希望转到我的装备，你就要买钥匙，才能不停地开。<br/><wbr />?<wbr />?<wbr />?记者：怎么买钥匙？花多少钱能买把钥匙？<br/><wbr />?<wbr />?<wbr />?玩家：要用“元宝”买钥匙。钥匙分三个档次，一个是一元宝，花1.16元人民币可买一个一元宝。另一个是两元定，此外，还有三元宝。不同档次的钥匙开出来的装备不一样。三元宝开的装备等级越好，要想得到最好的装备，就得不停地花钱，有时一花就是两三万元人民币。<br/><br/>记者：用元宝还能在网上买什么？<br/><wbr />?<wbr />?<wbr />?玩家：有好多可买的呢。游戏运营商在游戏里经常拍卖道具。游戏中的商品比现实中不知要贵多少倍。比如有的游戏人物的一件衣服就要花30多万元人民币。有的游戏玩家为了得到一枚《热血传奇》中的虚拟复活戒指，要花112万“元宝”买，按1：1.16换算，折合成人民币是130万元。<br/><wbr />?<wbr />?<wbr />?记者：从哪里买？<br/><wbr />?<wbr />?<wbr />?玩家：一是网上自带的买元宝的系统，通过网银可以直接购买。二是从元宝商人那买，他<br/><br/>们是盛大公司的代理商。还有报刊亭、网吧都能买到卡，有10块钱的，也有30、50、1000一张的。代理商销售得越多，拿到的折扣越多。<br/><wbr />?<wbr />?<wbr />?记者：他们开发票吗？<br/><wbr />?<wbr />?<wbr />?玩家：大多数玩家都是通过网银直接汇到代理商那里购买，从来没有人给我们开过发票。<br/><wbr />?<wbr />?<wbr />?记者：代理商还在网上干什么？<br/><wbr />?<wbr />?<wbr />?玩家：他们会担保赌博。比如在游戏中经常会丢出一个色子，让玩家们比大小。游戏中没法进行人民币交易，所以，如果是我输了，我就把人民币给副总代理商，代理商再转给你，但是，他要收手续费。另外，网上有一个交易元宝的地方，叫庄园。比如，我想买你这个元宝，但是，我不认识你啊，你可能对我也不信任，怎么办？这时，代理商也会出来担保，并收5%-8%的手续费。<br/><br/><wbr />?<wbr />?<wbr />?记者：盛大除了“开天赐”，还通过什么方式圈钱？<br/><br/><wbr />?<wbr />?<wbr />?玩家：通过藏宝阁圈钱。藏宝阁分为60个房间，而每个房间都会有突发事件发生,并且每间房间里都会有一个守卫，他会要求你给他一定数量的乾坤令，而乾坤令可通过探索密室中的秘室守护兽来获得。当达到守卫的要求之后，便可掷骰子，根据得到的数值来决定进入的房间。每间房间都将为我们带来不同的感受!在13、23、33、43、53号房间,骰子的数值会决定前进还是后退，而在10、20、30、40、50号房间，机关会给大家带来惊喜。到达60号藏宝阁后，将会得到一次掷3个骰子的机会，凭借着运气，就将有机会得到玛法大陆失传多年的镇龙盔、镇魔盔、镇魂盔。?<wbr />?<wbr />?<wbr /><br/><wbr />?<wbr />?<wbr />?记者：除了赌博，你认为网络游戏还有哪些不好？<br/><wbr />?<wbr />?<wbr />?玩家：鼓励你杀人。你玩《传奇》，杀的人越多越有奖励。《传奇》中还公然设置免费的PK值清洗员，而早在很久以前，如果在此游戏中杀人是要红名的，红名就意味着进不了城买不了药，被杀还要把身上的东西都掉出去，杀红了要在红名村挂好几个小时，现在竟然公然出现在这里。此外，还鼓励你抢劫，如果游戏中人物的装备从身上随机掉下来了，你可以随便抢，有的是上万块钱的装备，那就是上万块钱的人民币啊。在现实中，抢劫是要负刑事责任的。但在游戏中，你抢完了，却变成了一种炫耀。网络游戏商就是靠宣扬暴力来赚钱。一些职业玩家就是抢劫犯，抢完装备就能卖钱。现在有的玩家已经成立了一个小组，一起在游戏中“杀”人，把装备卖出后，马上就能分钱。<br/><wbr />?<wbr />?<wbr />?记者:玩家的级别和花费的人民币是直接相关的。<br/><wbr />?<wbr />?<wbr />?玩家：我们玩家分成了人民币、半人民币、非人民币用户。半人民币用户是像我这样花了四五十万的人，之上的就是人民币用户了。<br/><br/><wbr />?<wbr />?<wbr />?记者：要升级就要不断地花钱。<br/><br/><wbr />?<wbr />?<wbr />?玩家：是。在《传奇》中，闯天关就是用人民币换元宝，1元宝=1灵符。使用灵符作为凭证进入一个地图，有相关NPC对话刷出怪物，消灭掉以后随机得到经验或装备（可能是极品，也可能是垃圾）。如果这次闯关决定放弃的话，花掉的灵符是不退还的。网上有人已经玩到了999级，如果单算时间，每天24小时玩，五辈子也玩不到999级。我玩到46级，花了三年时间，越到后面越难升级。我们推算，不算装备，不算英雄等级，不算英雄装备，不算内功所需的钱，不算金针，不算买白金积分等，只算升级要花的药钱和闯天关要花的元宝钱，玩到999级估计花了1000万。现在回过头去看，我们其实就是盛大公司的提款机。?<wbr /><br/>记者：传奇中有没有色情平台？<br/><br/><wbr />?<wbr />?<wbr />?玩家：有。国家明文规定不允许宣传色情内容以及视频裸聊，但是《传奇》公然发布色情广告无人清理，这与玩家在游戏里抗议被拉进小黑屋形成鲜明对比，他们不是没看到，是不管。<br/><br/><wbr />?<wbr />?<wbr />?记者：国家要求网络游戏中都要预装防沉迷系统，《传奇》做到了吗？<br/>玩家：据了解，游戏一般只能定在三到五个小时的时间，系统会自动的关闭。而第一款《传奇》中没有装，后来出现的《传奇1》、《传奇2》、《传奇3》都没有，一些玩家长时间在电脑旁一坐就几天几夜，出现过休克、昏迷至到死亡的都有。<br/><br/><b>?<wbr />?<wbr />?<wbr />?花出去的钱什么也没留下，连痕迹都没有</b><br/><wbr />?<wbr />?<wbr />?记者：毁档是什么意思？<br/><wbr />?<wbr />?<wbr />?玩家：网游企业可以随意更改数据甚至消灭数据，我们叫毁档。网民的虚拟财产怎么保护？我们曾经直接找到盛大公司去投诉，甚至睡到他们的投诉大厅里，但都没有解决问题，感到很无奈。我们玩家看起来像是一个很大的群体，其实就是一只只小蚂蚁。<br/><br/>记者：为什么很多玩家知道被骗，仍不放弃呢？<br/><wbr />?<wbr />?<wbr />?玩家：如果一个月薪只有1500元的打工仔为此游戏花了上万元，他会轻易放弃不玩了吗？不能。如果是一个生活费只有1000元的学生或者一个月零花钱只有两三百元的高中生能轻易的放弃吗？不会。如果一个私营业主花了几十万他会轻易放弃吗？还是不会。为什么？因为放弃了就什么都没有了，花出去的钱什么也没留下，连痕迹都没有。所以只能象吸毒一样周而复始。有很多人，在现实生活中很失意但无奈地在网络中找自尊自信，象渴望水的鱼。但是现在的网游更赤裸裸，比现实更可怕。<br/><wbr />?<wbr />?<wbr />?记者：只是盛大如此吗？<br/><br/><wbr />?<wbr />?<wbr />?玩家：不是，所有所谓的免费网游都这样。<br/><br/><wbr />?<wbr />?<wbr />?记者：你们有什么希望？<br/><br/><wbr />?<wbr />?<wbr />?玩家：第一，要盛大为自己的行为负责，向所有玩家道歉。第二，呼吁政府净化网游环境，规范网游管理，实行分级，并能推行行之有效的防沉迷系统，以及网游立法。第三，呼吁网游公司自我收敛并约束自己的行为，尽社会责任。第四，成立玩家自己的维权组织，由此组织监督网游公司。<br/>首次转载时间: 2018-02-26 21:15:12"
