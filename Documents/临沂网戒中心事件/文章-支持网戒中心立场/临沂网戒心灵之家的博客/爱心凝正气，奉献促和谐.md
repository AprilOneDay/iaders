/>――我们心中的杨叔<br/><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><br/><br/>这是一个特殊的群体――被网瘾孩子的厌学、早恋、离家出走、偷摸抢骗、自杀暴力折磨的痛不欲生的父母们，他们哭过天跪过地，烧过香拜过佛，不惜父母的尊严给孩子下过跪。甚至想用结束生命来唤醒孩子那颗扭曲的心灵。然而，各种办法都用尽了无济于事，连最伟大的母爱在网魔面前都是那样的苍白无力，正当他们悲观绝望，啼血哀嚎，大声疾呼――救救孩子的时候，一个让他们感恩一生的人，走进他们的心中。他，就是临沂市第四人民医院网络成瘾戒治中心主任，网络成瘾戒治专家――杨永信。在网瘾的挑战面前，他是一位勇敢的斗士，万丈豪情勇往直前；在家长眼里他是一名灵魂的守护者，春风化雨润物无声；在孩子心中他是一位忘我的父亲，疲惫的身躯，托起一个个希望的梦想；他用执着唤醒了千百颗迷途的心，他用真情开启了一扇扇紧闭的心门，他用爱心挽救了一个又一个网瘾孩子的家庭，他用博大的爱，温暖的手为孩子们重新点燃心中那久已熄灭的希望之灯；他精湛的医术、高尚的医德感动临沂、感动山东、感动中国！<br/><br/><span lang="EN-US" xml:lang="EN-US">20</span>多年来，凭借一颗治病救人的心，成长为一名出色的精神卫生专家，为了挽救无数痴迷于网络的孩子，他创建了国内第一家戒瘾学校，第一家戒瘾网吧，第一个戒瘾同盟，第一家戒瘾专业网站。中心从?<span lang="EN-US" xml:lang="EN-US">2006</span>年<span lang="EN-US" align="center"><b>点评课上泪雨纷飞</b></p><br/>心理点评是戒网瘾的一个重要环节，也是每天的必修课，由于长期沉迷网络孩子的性格扭曲，脾气暴躁，自闭自杀现象时有发生。青少年的违法犯罪中，<span lang="EN-US" xml:lang="EN-US">80%</span>―<span lang="EN-US" xml:lang="EN-US">90%</span>与网络有关。<br/><br/>“当我有一天走进了虚拟的世界，我就再也找不到自己的方向，当别人给我带来关爱的时候，我却把他当作了一世的仇人，在魔幻的虚拟世界里，封闭了真实的自我，我没有了生命的色彩，亲爱的朋友啊，从网梦中醒来吧，让我们用新的生命，乘着爱的翅膀起飞，追求新的希望，追求爱的梦想。”这是一首杨叔作词，戒网瘾同盟班会的成员每天都要唱的《网梦醒来》，每当唱起这首歌，不少孩子和家长都流下了泪水，因为歌词就是他们的心声，泪水背后是一个个心酸的往事。<br/><br/>魏<span xml:lang="EN-US">16</span>岁一个文文静静的男孩，由于迷恋网游，性格扭曲，为了上网，在父亲病危之际，都不去医院探望，直到死父亲也没能见上亲生儿子一面，遗憾的离开了人世。父亲的死未能唤醒儿子迷失的心灵，为了弄到上网的费用，他变卖了家中的电视、冰箱、洗衣机，连纯净水的桶都卖了，后来又拆卖了家中的门窗。没办法母亲租了一个不到十平米的小屋临时居住，就是这样，儿子也不放过母亲，他用刀架在母亲的脖子上，逼她去单位给他借钱上网<span lang="EN-US" xml:lang="EN-US">……</span>这样的孩子在网戒中心比比皆是，有的孩子恨不得父母马上从这个世界上消失。因为在他们心中只有网游，挡我者死――谁阻挡我上网谁就是我最大的敌人！有的孩子为了上网一天吃不上一顿饭，半年不洗一次头。一个中国科技大学的学生离家出走<span lang="EN-US" xml:lang="EN-US">14</span>天，母亲为了找他，背着行李沿街乞讨。有个父亲为了找孩子遭遇车祸，为了唤醒孩子那一颗颗迷失的心。杨叔付出了超乎常人难以想象的艰辛，每天苦口婆心，晓之以情动之以理。每当这些孩子一个个康复出院的时候；每当浪子回头涕泪双流，大喊一声：“妈妈，儿子错了！”的时候，每当《感恩的心》唱起的时候，二、三百个人哭成一团。杨叔，是您给了我们家庭第二次生命，是您给了孩子第二次生命，您恩同再造。网游把花季少年变成了魔兽，是杨叔让魔兽变成了天使。<br/><p align="center"><b>一</b>?<b>个</b>?<b>都</b>?<b>不</b>?<b>能</b>?<b>少</b></p><br/>“只要你在网戒中心挂了号，杨叔就管你一辈子！”“你一天不改变，杨叔就一天跟你没完！”杨叔是这样说的，也是这样做的。杜<span lang="EN-US" xml:lang="EN-US">40</span>天就出院了。回去不久网瘾再一次发作，杨叔又一次把他带到了医院。因为家境贫寒，父母以卖咸菜为生，根本支付不起儿子的住院费用，杨叔一方面申请医院给他减免部分住院费用，一方面发动家长爱心捐款，这样一共反复了五次，杨叔这种不放弃的爱终于感动了这个邪性的孩子，他在<span lang="EN-US" xml:lang="EN-US">2007</span>年以优异的成绩考上了大学，并在大学校园里脱颖而出，成为校学生会干部，公益活动的积极分子。<br/><br/>小邈，<span lang="EN-US" xml:lang="EN-US">24</span>岁，父母都是部队干部。为了达到长期上网的目的，从<span lang="EN-US" xml:lang="EN-US">16</span>岁起，整天在家上网，父母让她上学或工作，她是一哭、二闹、三上吊、四装疯、五跳楼、六吃药。<span lang="EN-US" xml:lang="EN-US">8</span>年来，为了给她治病，父母走了大半个中国，过着生不如死的日子，来到网戒中心以后。她故伎重演，闹的整个楼的人都无法入睡，她的做法让大家实在忍无可忍了，纷纷上报杨叔让她转院。在这种情况下，杨叔向大家鞠了躬：“她的父母千辛万苦的找到这里，咱们能撒手不管吗？求求大家了，给孩子一个治疗的机会<span lang="EN-US" xml:lang="EN-US">……</span>”杨叔的不抛弃、不放弃的精神终于感染了大家，目前，这个孩子已经康复出院，并且已经工作。<br/><br/>胡<span lang="EN-US" xml:lang="EN-US">X</span>，家住胶南，已经出院一年多了，结婚后由于思想偏颇，与妻子发生矛盾，妻子回了娘家。当网戒和谐行走到胶南的时候，杨叔听说了这件事，就连忙赶往胡<span
