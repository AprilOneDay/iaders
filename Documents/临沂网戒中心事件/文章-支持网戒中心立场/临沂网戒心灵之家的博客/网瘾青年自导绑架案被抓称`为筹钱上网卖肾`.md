align="center">　　4月8日中午，杨晓兵的父亲从派出所接儿子回家。看到憔悴的父亲，杨晓兵哭了。</p><br/>　　尽管上大学时曾为了筹集上网费抢劫他人财物而获刑，但杨晓兵似乎根本没有吸取教训，仍然每天坚持上网。儿子在网上的“胡作非为”，让气愤不已的老杨夫妇痛下决心，不再给他零用钱，但这一切都似乎没能让这个沉迷网络的青年迷途知返。“2010年底，我到河南卖了一颗肾。”杨晓兵说，为了筹钱上网，他通过一个黑中介，到外地将自己的左肾卖掉。<br/><br/>真吓人<br/><br/>腹部伤口长达20余厘米<br/><br/>4月8日，因导演“绑架案”被抓后，警方要对杨晓兵进行处理时，他的一个朋友说，杨晓兵身体不好，他的左肾被卖掉了。当时办案民警也感到非常不解，难道迷恋网络这样的事也能发生？民警根本不相信这是真的。随后，他们撩起杨晓兵的衣服发现，他的腹部左侧有一道20余厘米长的口子，伤口处仍然黑乎乎的，看上去还没有完全康复。据杨晓兵称，这就是他卖肾留下的痕迹。<br/><br/>经询问，杨晓兵说 ，2010年12月，因为他常年在外面上网，而且也没有经济来源，无钱玩网络游戏的杨晓兵，在网上找到一家黑中介，然后在河南郑州一黑中介的介绍下，将一颗肾卖出。<br/><br/>“当时中介说给8万元，可是自己只拿到6万，中介可能赚了两万吧。”据杨晓兵回忆，那家买肾的小医院只有几间低矮的平房，具体位置他已经记不清了，当时把左肾取出来后，他就躺在医院治疗了一段时间，这段时间他又花掉了3万多元的治疗费，最后只剩下2万多元钱。他从医院出来后，拿着自己用身体器官换来的钱又开始上网。最终，感到身体不适的他突然越发想家，就再次回到父母身边。<br/><br/>真气人<br/><br/>在哪儿卖的肾都忘了<br/><br/>“我们当时得知儿子因筹上网费，卖掉了左肾一事后，感觉天都要塌下来了。”老杨说，2010年底，新年的钟声就要敲响之际，杨晓兵这才将自己卖肾的事情告诉了他们，听到这事后夫妻俩惊呆了。从此，他们开始对这个让人疼又让人恨的儿子感到失望甚至绝望。<br/><br/>除夕夜里，别人家的鞭炮声响起时，这个家里没有温暖。“肾卖了8万，中介拿走2万，自己手术后花了至少3万，最终剩下的，应该是2万多元。”2万元，让儿子少了一颗肾，老杨应该找谁问罪？<br/><br/>“我也不知道为什么，但我不是为了上网才去卖肾的。”对于卖肾的决定，杨晓兵表示“至今想不通”。他告诉记者，自己拿到钱之后，并没有想好要怎么花、花在哪儿，然而到最后，钱大部分还是花在网络游戏上了。<br/><br/>老杨也曾问过儿子，是通过什么途径、具体在哪家医院卖的肾，儿子却“淡定”地回答：“忘了，算了，我自愿的。”<br/><br/>真伤人<br/><br/>为会女网友再次离家<br/><br/>4月23日上午，记者再次联系采访时，老杨称儿子杨晓兵已经离家一周了，现在去了四川乐山，他在那里有一个女网友。现在他也联系不上儿子。<br/><br/>“他现在去了四川乐山，他说是出去透透气。”老杨说儿子杨晓兵已于一周前离家出走去了四川。“走的时候他告诉我，想出去转转，我知道留住他的人，也留不住他的心，就给他一些路费，让他走了。”老杨无奈地告诉记者，儿子称是去了四川，他在乐山有一个女网友，从走了之后这段时间，他只是给家里来了个电话，说他在四川一切都很好，其他的什么都没有说。<br/><br/>在老杨看来，儿子去了哪儿并不重要，他关心的是，儿子何时才能彻底离开网络，回到正常生活中。老杨说，杨晓兵是一个非常聪明的人，而且也很有思想，就是让网络给害了，现在一家人在周围的邻居面前都抬不起头，儿子出了这样的事让他们觉得“非常丢人”，真不知道哪天他又会做出什么样的事来，他现在真的不知道孩子将来能成什么样，他只希望孩子在四川待一段时间后能够回来，然后他们一家人坐在一起好好沟通。<br/><br/>老杨说，他妻子认为，孩子现在出了这样的事，让他们一家人抬不起头，不想让此事公之于众。但他认为，现在不是面子的事，而是如何能拯救儿子，不管用什么手段，只要有一线希望，也希望儿子能够重新振作起来，回到正常的生活当中。?<wbr /><br/><br/>■老杨<br/><br/>为了筹钱上网使尽骗术希望专家帮儿子戒网瘾<br/><br/>“我现在真的不知道该怎么面对这件事。”4月23日上午，记者在胶南铁山街道找到了老杨，得知记者的来意后，他非常高兴，他希望通过媒体的帮助，使他的儿子重新振作起来。老杨说，这些年，他儿子为了上网，用尽了各种手段筹集上网费。<br/><br/>曾杜撰车祸骗上网费<br/><br/>“叔，你儿子出车祸了。”今年3月，杨晓兵告诉父母，自己要和在济南某职业学校工作的同学王某，到四川招生。当时王某用杨晓兵手机的打电话过来，说他儿子在四川乐山出车祸，让赶紧打3万元到儿子卡里做手术，现在他儿子就住在乐山人民医院。<br/><br/>听这个消息，老杨当时就吓坏了，但是转念一想，他有个叔叔在成都，然后他就打电话过去，让那位亲戚到乐山人民医院去查找。听闻此事，他叔叔立即带上钱开车赶往乐山。然而，找遍乐山所有医院，都未发现杨晓兵的入院登记。老杨有点明白了，随即给杨晓兵的同学打电话：“我已经安排人在医院门口，你出来拿钱吧。”不出所料，对方挂了电话。三天后，儿子给老杨发来一条短信：出车祸的事是骗你们的，对不起，自己想做点事，但我在外面借了高利贷，整天被压着无法翻身。<br/><br/>本来宽裕的家庭，因杨晓兵迷恋网络而负债几万。当父母为了家庭生计而发愁时，杨晓兵却因为练到“最高级”而感到无聊。<br/><br/>是否卖肾尚未证实<br/><br/>针对杨晓兵是否把左肾卖掉一事，记者在派出所采访时，他一直坚称将自己的左肾给卖掉了，而且他父亲也证实了这一点。杨晓兵真的把左肾卖掉了？因杨晓兵本人已经离家去了四川，也无法将其带到医院去检查确定，而他父亲也提供不出已将一颗肾卖掉的证据。因此事始终都是杨晓兵自己说的，现在只能说明他的腰部位置确实有一道20余厘米长的伤口，他的左肾是否被取走，现在无法得到证实。<br/><br/>临近采访结束，当记者问及老杨，是否有可以帮助的地方时，这位身材魁梧的汉子哭了。“我希望找个专家帮忙看看，我感觉似乎已经走投无路了。”老杨说这话时，他妻子正在胶南市做钟点工，他们的儿子杨晓兵，却不知身在何处。<br/><br/>■心声<br/><br/>只要有希望绝不放弃<br/><br/>4月23日上午，记者在采访老杨时，他向记者介绍孩子的人生轨迹时，始终在自责，他认为自己的教育方式可能不对，但只要有一丝挽救孩子的希望，他都不会放弃。<br/><br/>记者：这次儿子出走，你感觉放心吗？<br/><br/>老杨：或许他也有心理压力，关住他的人，关不住他的心，不如让他出去走走。<br/><br/>记者：儿子走到今天，自己是不是也找过原因？<br/><br/>老杨：可能主要因为我和孩子他妈教育方式不当吧，至今我们也很后悔。<br/><br/>记者：下一步有什么打算帮助儿子？<br/><br/>老杨：我们现在毫无头绪，也希望孩子回来后变好，只有这样整个家庭才能回到正常状态。<br/><br/>记者：如果孩子依然迷恋网络呢？<br/><br/>老 杨：我也想过，也很担心，但是只要孩子还有一丝转好的希望，我们绝对不会放弃。<br/><br/>■对话杨晓兵<br/><br/>对不起爸妈不觉得有网瘾<br/><br/>4月8日上午，记者在派出所内，与杨晓兵对话中，他表示知道自己整天在做什么，知道人生的方向已经迷失，就是回不到正确的道路上来。他说知道父母伤心，很对不起父母，他也很想回家。<br/><br/>记者：如果现在一天不接触网络能受得了吗？<br/><br/>杨晓兵：可以，我觉得我没有网瘾，只是没什么事做才去玩游戏。<br/><br/>记者：家里因为你玩游戏，已经欠了不少钱，你知道这事吗？<br/><br/>杨晓兵：知道一些。<br/><br/>记者：从你上大学出事，到你所说最近卖肾，你父母对你非常担心，你有没有考虑过他们的感受？<br/><br/>杨晓兵：我知道，我对不起爸妈，但跟他们沟通少，很多时候我更愿意一个人去上网。<br/><br/>记者：现在身体状况怎么样？<br/><br/>杨晓兵：感觉浑身没劲，有时候头晕，不想到处跑了。<br/><br/>记者：你很长时间没见你家人了，你母亲还在家等你。<br/><br/>杨晓兵：上次离家时，我不经意回头，看到母亲在村口看我(哭泣)……我对不起她……从2010年很少回家，我其实很多次想回家……<br/><br/>记者：回到家后，今后有什么打算吗？还会继续迷恋上网吗？<br/><br/>杨晓兵：我也不知道，我也想努力，但现在很乱很迷茫，很想先安静一段时间。<br/><br/>■专家说法<br/><br/>“没有网瘾”是个危险信号<br/><br/>“造成孩子这个情况的原因有很多，首先应该是家庭因素。”对于杨晓兵的问题，记者咨询了青岛市裕荣心理诊所的心理专家。<br/><br/>该诊所负责人马森医生表示，在孩子成长过程中，父母的教育方式以及家庭环境是否融洽极为重要。<br/><br/>“如果父母与孩子沟通少，孩子很容易有失落感，遇到问题时，孩子会更愿意通过接触家庭以外的事物来获得慰藉。”马森医生认为，从杨晓兵成长经历来看，他性格中有追求完美的一面，当学习成绩落后或者工作不顺心时，没能够及时获得指导奋起直追，而是自暴自弃，并通过网络游戏，在虚拟世界中获得在现实中难以找到的成就感。<br/><br/>对于杨晓兵认为自己“没有网瘾”，从业经验丰富的马森医生表示，这可能是一个比较危险的信号。<br/><br/>“越是不承认，往往问题越严重。”马森医生建议，杨晓兵父母应对其多加关爱，首先帮助其检查、恢复身体，在抚平其生理的伤痛后，通过正规网瘾治疗机构让杨晓兵彻底摆脱网瘾的困扰。文/图记者
