align="left"><b>杀人<span lang="EN-US" xml:lang="EN-US">“</span>像网上游戏打死人一样<span lang="EN-US" xml:lang="EN-US">”</span></b></p><br/>本报记者 蔡艳荣 通讯员郄世民 蔡艳荣<span lang="EN-US" xml:lang="EN-US">/</span>图<br/><br/><b><span lang="EN-US" xml:lang="EN-US">■</span></b><b>新闻提纯：</b><br/><br/><span lang="EN-US" xml:lang="EN-US">12</span>月<span lang="EN-US" xml:lang="EN-US">11</span>日凌晨，鹿泉发生一起凶杀案：<br/><br/>一名出租车司机被杀死在高速桥下，出租车不知去向。鹿泉警方经过<span lang="EN-US" xml:lang="EN-US">18</span>小时的缜密侦查，在一网吧门前将杀人劫车的犯罪嫌疑人抓获，而凶手居然是一名刚满<span lang="EN-US" xml:lang="EN-US">18</span>岁的网瘾青年。<br/><br/><b><span lang="EN-US" xml:lang="EN-US">■11</span></b><b>日<span lang="EN-US" xml:lang="EN-US">1</span>时：高速桥下惊现死尸</b><br/><br/><span lang="EN-US" xml:lang="EN-US">12</span>月<span lang="EN-US" xml:lang="EN-US">11</span>日深夜<span lang="EN-US" xml:lang="EN-US">1</span>时许，鹿泉市公安局刑警一中队的巡逻民警在巡逻时，发现鹿泉市高新技术开发区南新城村西的高速公路桥下躺着一个人，头北脚南。民警一下子提起了警惕，走近一看，发现竟是一具血淋淋尸体。<br/><br/>巡逻民警立即将情况报告给了中队长张国志，<span lang="EN-US" xml:lang="EN-US">10</span>分钟后，鹿泉警方迅速抽调警力成立专案组。当日<span lang="EN-US" xml:lang="EN-US">7</span>时，民警分别对现场展开认真细致的勘查。据勘查：死者为男性，<span lang="EN-US" xml:lang="EN-US">40</span>多岁，尸长<span lang="EN-US" xml:lang="EN-US">172</span>厘米，胸部和背部<span lang="EN-US" xml:lang="EN-US">4</span>处刀伤，系刺穿肺动脉导致失血过多死亡。<br/><br/><b><span lang="EN-US" xml:lang="EN-US">■11</span></b><b>日<span lang="EN-US" xml:lang="EN-US">9</span>时：死者为出租车司机</b><br/><br/>为尽快破获这起命案，警方成立了<span lang="EN-US" xml:lang="EN-US">10</span>个小组，对尸源展开调查，当日<span lang="EN-US" xml:lang="EN-US">9</span>时左右，指挥中心接到一名家住鹿泉市城东小区的妇女樊某的报警称：丈夫胡某昨晚开出租车一夜未归，打他的手机和小灵通都是关机，不知是不是出了什么事。<br/><br/><span lang="EN-US" xml:lang="EN-US">“</span>死者是否是樊某的丈夫呢？<span lang="EN-US" xml:lang="EN-US">”</span>樊某很快被接到现场，经辨认，正是她的丈夫胡某，今年<span lang="EN-US" xml:lang="EN-US">43</span>岁，鹿泉市铜冶镇北铜冶村人，现住鹿泉市，是城区的一名出租车司机，而他所驾驶的银灰色两厢夏利轿车不知去向。<br/><br/><b><span lang="EN-US" xml:lang="EN-US">■11</span></b><b>日<span lang="EN-US" xml:lang="EN-US">19</span>时：守株待兔抓获嫌犯</b><br/><br/>专案组当即确定此案为<span lang="EN-US" xml:lang="EN-US">“12・11</span>抢劫出租车杀人案<span lang="EN-US" xml:lang="EN-US">”</span>。<span lang="EN-US" xml:lang="EN-US">11</span>日下午<span lang="EN-US" xml:lang="EN-US">3</span>时许，民警韩哲勇在联盟小区排查时，发现该车停放在某网吧北边的路口处，迅速与专案组取得了联系。<br/><br/>车就停放在闹市的路边，牌照都完好无损，难道嫌疑人是弃车潜逃？办案民警决定在此蹲坑守候，等待嫌疑人员出现。时间在一分一秒地过去，眼看就到晚<span lang="EN-US" xml:lang="EN-US">19</span>时的时候，忽然，一名青年从网吧里走出来，下意识地左右望了望，急步来到车前，掏出钥匙顺利打开了车门。就在他坐进驾坐的一刹那，中队长张国志第一个冲到车前，拉开车门，将其拽趴在车下，冰冷的手铐锁在了该青年的手腕上。该青年正是杀人劫车的凶手。<br/><br/><b><span lang="EN-US" xml:lang="EN-US">■11</span></b><b>日零时：杀人劫车只为上网</b><br/><br/>据这名青年交代，他叫王勇志，鹿泉市大河镇大河村人，<span lang="EN-US" xml:lang="EN-US">18</span>岁。初中时他就辍了学，整天上网，没钱了就出去偷。<span lang="EN-US" xml:lang="EN-US">2004</span>年<span lang="EN-US" xml:lang="EN-US">9</span>月，王勇志因盗窃罪被鹿泉市人民法院判刑一年，去年<span lang="EN-US" xml:lang="EN-US">9</span>月刑满释放后，在平山一家企业开铲车，后来网瘾越来越大，铲车也不开了，还是整天泡在网吧里。<br/><br/><span lang="EN-US" xml:lang="EN-US">12</span>月<span lang="EN-US" xml:lang="EN-US">11</span>日零时<span lang="EN-US" xml:lang="EN-US">20</span>分，他从鹿泉市某网吧出来，上了网吧门口停着一辆出租车，<span lang="EN-US" xml:lang="EN-US">20</span>分钟后经过南高庄村西的十字路口，又一直往南走到一个高速桥下时，想以撒尿为名寻找机会逃跑。谁知司机也停下车跟着他来到桥下，他只好摊牌说自己没钱，司机一边对他进行了讽刺挖苦，一边继续要钱。最后，他推辞到植物园附近的姑姑家借钱，然后乘司机开车门不注意时，从上衣口袋掏出一把单刃刀向司机猛刺过去，随后拔出刀就向南逃走。大约跑了<span lang="EN-US" xml:lang="EN-US">20</span>多米，发现没有动静，又返回去，见司机还会动，便又向其胸口刺了三四刀，直到看着人不动了，才从司机的衣袋里翻出了一部手机和一部小灵通，拿了车钥匙。<br/><br/>王勇志开上抢来的汽车回家换了件上衣，驾车到石家庄?市联盟路某网吧继续上网打游戏，随后又开车去某洗浴中心洗澡睡觉。当日上午<span lang="EN-US" xml:lang="EN-US">9</span>时许，以<span lang="EN-US" xml:lang="EN-US">130</span>元的价格将手机和小灵通卖掉，开车回到鹿泉县城转了一圈后，又重返到该网吧上网，直到当日晚上<span lang="EN-US" xml:lang="EN-US">7</span>时左右才从网吧出来。<br/><br/><b><span lang="EN-US" xml:lang="EN-US">■13</span></b><b>日上午：网上杀人更厉害</b><br/><br/><span lang="EN-US" xml:lang="EN-US">“</span>你知道用刀刺心脏和背部会致人死亡吗？<span lang="EN-US" xml:lang="EN-US">”</span><br/><br/><span lang="EN-US" xml:lang="EN-US">“</span>知道。不然我没钱上网。我刺死他才能抢他的钱。<span lang="EN-US" xml:lang="EN-US">”</span><br/><br/><span lang="EN-US" xml:lang="EN-US">“</span>你刺死他不怕吗？<span lang="EN-US" xml:lang="EN-US">”</span><br/><br/><span lang="EN-US" xml:lang="EN-US">“</span>就好像网上游戏打死人一样，没什么可怕的，无非一个真一个假。<span lang="EN-US" xml:lang="EN-US">”</span>王勇志的回答让办案民警大吃一惊。<br/><br/><span lang="EN-US" xml:lang="EN-US">12</span>月<span lang="EN-US" xml:lang="EN-US">13</span>日上午，记者来到了看守所，见到了王勇志，与他对话过程中，王勇志态度非常冷漠，很不耐烦。<br/><br/>记者：你多大了？<br/><br/>王勇志：<span lang="EN-US" xml:lang="EN-US">5</span>月<span lang="EN-US" xml:lang="EN-US">21</span>日刚过<span lang="EN-US" xml:lang="EN-US">18</span>岁生日。<br/><br/>记者：知道这么做的后果吗？<br/><br/>王勇志：死刑呗。<br/><br/>记者：你经常玩什么游戏？<br/><br/>王勇志：《大话西游》<br/><br/>记者：游戏中有杀人吗？<br/><br/>王勇志：有，杀人比现实中厉害多了。<br/><br/>记者：你不上学，平时都去什么地方？<br/><br/>王勇志：网吧、浴池。<br/><br/>记者：对你的行踪，父母平时不管吗？<br/><br/>王勇志：想管他们也找不着我呀！<br/><br/>记者：想对受害者家属说点什么？<br/><br/>王勇志：叫他们别急，过两天我就下去给他（被害人）做伴去。<br/>首次转载时间:
