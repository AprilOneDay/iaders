10:50</span></p><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><br/><br/>据记者了解，南康一网瘾青年曾某为上网用农药毒害生父，<span lang="EN-US" xml:lang="EN-US">10</span>月<span lang="EN-US" xml:lang="EN-US">20</span>日，南康市人民法院对该案作出一审判决，曾某因故意杀人（未遂）被判处<span lang="EN-US" xml:lang="EN-US">5</span>年有期徒刑。<br/><br/><b>缺少父母亲情的童年</b><br/><br/>被告人曾某生于<span lang="EN-US" xml:lang="EN-US">1991</span>年<span lang="EN-US" xml:lang="EN-US">1</span>月<span lang="EN-US" xml:lang="EN-US">9</span>日，其父是南康市龙岭镇一农民，平时在家干农活，农闲时外出务工。曾某是他在年逾<span lang="EN-US" xml:lang="EN-US">40</span>岁时所生的独子，因妻子有残障，无奈之下，将曾某送到在龙南县的岳父家中抚养。<br/><br/>幼小的曾某因父母不在身边，缺乏管教，<span lang="EN-US" xml:lang="EN-US">14</span>岁时开始上网玩游戏，从此一发不可收拾。<br/><br/><b>畸形发展的人格</b><br/><br/>因上网玩游戏，曾某想方设法偷亲戚朋友的钱。其父无奈，在曾某<span lang="EN-US" xml:lang="EN-US">15</span>岁时，将其接回，带他一起到钨矿务工。曾某受不了苦，加上父亲又常责骂他，为此，曾某心里一直怨恨父亲。<span lang="EN-US" xml:lang="EN-US">2008</span>年<span lang="EN-US" xml:lang="EN-US">6</span>月<span lang="EN-US" xml:lang="EN-US">16</span>日，曾某偷了父亲的存折，将里面的<span lang="EN-US" xml:lang="EN-US">1.2</span>万元钱全部取出，在网吧玩了<span lang="EN-US" xml:lang="EN-US">8</span>个月。钱花光后，曾某回到家里，老实的父亲狠狠地责骂了曾某，并逼他下地干活。无网可上的曾某在日记里写下：<span lang="EN-US" xml:lang="EN-US">“</span>我已经不能再维持自己，我必须尽快去上网，但是没钱，我整天度日如年。<span lang="EN-US" xml:lang="EN-US">”</span><br/><br/>灵魂出窍的曾某疯狂了，在日记里说：<span lang="EN-US" xml:lang="EN-US">“</span>尽管我再怎么努力干活，还是被父亲痛骂。我曾经想过自己一死了知，但也不值得我枉出此生，我决定用农药毒死我父亲，如果成功我再去上几天网，后事就听天由命了！<span lang="EN-US" xml:lang="EN-US">”</span><br/><br/><b>痛下杀手毒父亲</b><br/><br/><span lang="EN-US" xml:lang="EN-US">2009</span>年<span lang="EN-US" xml:lang="EN-US">6</span>月<span lang="EN-US" xml:lang="EN-US">4</span>日，曾某趁家中无人，从邻居无人居住的老屋处弄了些农药，倒在厨房装有饭汤的盆内。其父亲回来后，准备喝时发现颜色不对，曾某心虚了，便将饭汤倒在门口菜园里便走了。其父闻到浓浓的农药味，出来后发现蚯蚓都死了。对儿子的举动，父亲倒吸一口冷气。随后，父亲决定向公安机关报案。接到报案后，警方很快将曾某抓获，并提取了曾某书写的日记，对犯罪事实，曾某供认不讳。（赵晓林?黄学鸿）<br/>首次转载时间:
