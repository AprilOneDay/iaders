<br/><p style="text-align: center;">一位网瘾孩子的家长</p><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><br/><br/><wbr />?<wbr />?<wbr />?2002年11月7日，我国一知名网站推出一则神秘离奇的新闻《千年木乃伊出土后怀孕》。原文是这样写的：“一埃及考古小组今年3月在开罗发掘出一具已逝世超过3000年的经防腐处理女木乃伊。但该学院负责人塞尔德最近获得一个惊人发现，证实这具木乃伊竟在出土后怀孕，至今其腹中胎儿看来已有8个月，经超声波检查后得出胎儿正常成长。该学院看守人西塔尔被指是胎儿的父亲，而他也承认自己难以抗拒女木乃伊的美色，不禁对她表达了爱意。”<br/><br/><wbr />?<wbr />?<wbr />?稍有基本常识的人都知道，怀孕必须具备这样几个最基本的条件：女性卵巢功能正常、正常体温，有维持胚胎生命的液体、激素等，而木乃伊经过了古代人为的防腐处理，其大脑、内脏统统都已拿掉，只是一个在特定干燥环境中才能存在的“标本”，已经没有生命了，根本不能提供怀孕的基本条件，怎么可能怀孕呢？但是这则内容荒诞不经的报道，还是接着被一些传媒一阵风地疯狂转载报道。南京博物院的院长徐胡平看了这则报道，评价是12个字：惊天之作，违背常理，不可思议。其实这则报道来自美国《世界新闻周刊》，而此周刊是一份专门编造荒诞故事博读者一笑的“超级市场小报”。<br/><br/><wbr />?<wbr />?<wbr />?可悲的是，这家网站的编辑将这个滑稽的故事从科技新闻转移到社会新闻，又从社会新闻归纳到搞笑新闻。他们也不考虑、判断一下消息是否准确、真实，只要能“吸引眼球”，哪怕是宣扬一些低级趣味、迷信、伪科学、恐怖暴力、耸人听闻，甚至是别有用心、夸大其词的都照用不误。可见其采编人员的品味、素质和道德水准如何。这则新闻被评为当年的十大假新闻之一，被录入新闻学教材作为一个“臭名昭著”的造假的案例。<br/><br/><wbr />?<wbr />?<wbr />?如果说这则新闻不是这家网站主动造假的话，尚情有可原，因为毕竟是引用的报道，可是你看现在整个社会流行炒作，某些无良媒体，为了生存，不惜代价，极尽挖掘奇闻之能事，甚至一些党报、党刊，作为党的耳目喉舌，也来效仿那些无聊小报的卑劣行径，同流合污，推波助澜，哗众取宠，信口开河，极不负责。2003年10月，中共中央宣传部、国家广电总局、国家新闻出版总署、中华全国新闻工作者协会联合开始在新闻战线深入开展“三个代表”重要思想、马克思主义新闻观、职业精神职业道德“三项学习教育活动”，要求作品务必要贴近实际、贴近生活、贴近群众。可是教育活动年年开展，年年还有一些失实的报道，年年都有假新闻产生，那种故意造假、完全捏造、拼凑抄袭、恶意炒作的不实报道，仍然像苍蝇一样令读者恶心，也使主流媒体为此蒙羞。去年5月以来一段时间内，围绕着临沂网戒中心发生的一系列报道，纵观中国青年报白雪、中央电视台新闻调查柴静、经济半小时胡长春、南方周末杨潇、都市时报李国豪、21世纪经济报道郭建龙、香港凤凰卫视闾丘露薇的的报道，其实是对临沂网戒中心的肆意侮辱和妖魔化虚假报道，其故意捏造，哗众取宠，以偏概全，偏听偏信，玩弄读者的雕虫小技，背后隐藏着险恶的用心，和不可告人的秘密，正可谓司马昭之心路人皆知。其妖魔化的伎俩不外乎以下几个手段：<br/><br/><wbr />?<wbr />?<wbr />?<strong>一、以煽情的文字抓眼，找到共鸣的切合点。</strong>新闻报道首先讲究的是有没有新闻价值，行业里最简洁的叫法是：“新闻点”，通俗讲叫能不能抓住眼球。负面新闻表现新闻点最常见的途径就是运用浅层次感官刺激的手法，集中渲染、夸大事件，特别是在标题、评论中不惜使用煽情的文字。比如西方媒体丑化中国常常使用这样的语言：践踏人权、压榨和拐卖妇女、强迫流产、屠杀孤儿院的婴儿、出售犯人器官、走私军火、出售导弹、扩散核武器、武力威胁邻国、到处是便衣警察、运动员服用兴奋剂等等，大量散布丑陋和恐怖信息。借助一些耸人听闻的色情、暴力、犯罪、性、名人丑闻、流言蜚语、离婚、天灾人祸等素材对一些社会阴暗面大加渲染而赢得关注。<br/><br/><wbr />?<wbr />?<wbr />?现在我们看以上人物对网戒中心报道所用的题目和文字：电击，捆绑，暴力，虐待，出走，跳楼，暗访，多数被骗，强行治疗，强行押制，把人弄来就走不掉，谁都想在网瘾治疗市场分杯羹，数十亿元！网瘾少年变摇钱树。这和西方媒体丑化中国有什么区别吗？<br/><br/><wbr />?<wbr />?<wbr />?闾丘露薇是凤凰卫视著名的战地记者，仅仅是看了中国青年报的报道，则更直接地叫嚣：谁给了杨永信电击网瘾少女的权力？她在博客中这样想象并十分“生动”地描述：女孩被五花大绑在病床上，一个孩子被家长骗吃了十二颗安眠药。最后一句说：“杨永信在临沂是楷模人物，这点不出奇，临沂的名声，就是以强硬出了名的了。”临了，还把临沂踏上去猛踩一脚。这丫不亚于重磅炸弹。临沂上辈子没欠她家什么债务吧？<br/><br/><wbr />?<wbr />?<wbr />?<strong>二、傲慢歧视，无异于在伤口上撒盐。</strong>闾丘露薇说：“这样的网瘾中心和行军学校之所以存在，说到底，还是家长们缺乏教育孩子的耐心和爱心。依靠强制手段遏止网瘾，就更是本末倒置、简单粗暴。”如果上面对临沂的名声不怎么认同的话，是为地域歧视、政治歧视，那么分析家长缺乏耐心和爱心，简单粗暴，则体现出鲜明的群体歧视。这个貌似“客观”的叙述是在转移新闻焦点，强烈地传达着一个明确的暗示，引导读者联想这是一个简单粗暴、缺乏爱心、愚昧无知的家长群体。孰不知，这一种歧视，误导了受众的思维，导致人们认为，孩子沉迷网游完全是由于家长“无知”造成的，把成因简单地归结在家长身上，事实上，巨大的经济利益诱惑着不良网游开发商的头脑，正是他们这些罪魁祸首才使一个个聪明可爱的孩子成为迷途羔羊，一个个家庭为此面临着家破人亡。他们把矛头指向网戒机构的同时也结结实实地刺伤了无数个受害的家庭，这才是名副其实地受到二次伤害，这些不实报道实际上在往受害家庭尚未愈合的创口上一次次的撒盐，这是对他们精神的摧残，一把更锋利的刀。白雪用了这样的小标题：“令人难以想象的家长们”，写到：<br/><br/><wbr />?<wbr />??<wbr />临沂四院的前身是一家精神病医院，然而记者接触到的家长，大多都不认为自己的孩子是精神病患者，但他们却通过签协议，认可了杨永信给孩子吃药、通电等行为。<br/><br/><wbr />?<wbr />??<wbr />一些家长并不清楚孩子吃的是什么药，几名接受采访的家长和孩子表示：“大概是补脑的药吧，具体我们也不懂。”但家长每天督促孩子吃药。一名家委成员告诉记者，有时孩子没记圈或犯错，一些家长主动要求加圈，让孩子经受电击。“电完就乖了”。<br/><br/><wbr />?<wbr />?<wbr />?这不是往家长的心上捅刀子是什么？！对人性的漠视，极不道德的行为，是一个年纪轻轻的女记者所为吗？藏着这样的用心，怎不令人心惊胆寒？假如她得了抑郁，她精神失常了，还不知道她这样的拿把大砍刀冲进哪家幼儿园，砍死砍伤多少无辜的生命呢。<br/><br/><wbr />?<wbr />?<wbr />?<strong>三、贬低成就，放大问题，鸡蛋里面挑骨头。</strong>在丑化中国的西方媒体中，尤以美国媒体最令我们“叹服”，他们固执而敬业，因为在任何有关中国的大事发生时，除了报道事件本身，他们会立刻联想到中国所谓的人权问题，即使是在奥运会在北京成功举办这类体育报道、神州飞天这些科技报道中，也从不忘记大费笔墨捎带对中国的批评。只是这种批评采用了很高的写作技巧，被隐藏了，受众是被暗示的，而且正因如此，其效果才更见显著。新闻有倾向性，新闻在报道或评述新闻事实时表现出的特定的立场和思想倾向。即在政治上拥护什么，反对什么；思想上赞成什么，排斥什么。这种倾向有时表现得直接而明显，有时则表现得含蓄而隐晦，但在新闻传播过程中，新闻倾向性总是客观存在的。新闻倾向性是通过选择事实和编写新闻表现出来的，它表达着作者或媒体的意见。随着网戒中心的快速发展以及在挽救盟友模式的进一步探索和逐步完善，网戒中心在社会上的影响越来越大越引人注目，引起许多关注，特别是中央传媒的正面介绍，对解决近年来出现的这一社会问题探索了一些经验。然而，一些负面报道也显示出很高的写作技巧，他们联合起来通过正规采访也好、暗访也好，信誓旦旦打着正面报道的旗号也好，事实上都在最后露出了“有意渲染，耸人听闻”的狐狸尾巴，他们塑造了一个网戒中心自己都觉得陌生的形象，而这些报道的杀伤力远比正面报道的影响力大的多，因为这些报道的出发点不是为了宣传网戒中心的工作业绩，而是借着“百分百治愈，3000名患者”，巧妙地算计网戒中心赚了多少钞票，进而表现网戒中心?<wbr />?<wbr />?“敛财”的贪婪面目。是啊，中国强大，美国人感到不安而且不舒服，一旦发现中国有灾难或什么负面事件，就如同看见了可以肆意宣泄的拳击沙袋。临沂网戒中心一强大，一些网游设计运营商感到不安甚至不舒服，一旦发现有人比如腾飞想报复网戒中心，无数个“网络水军”群起而攻之，甚至有几十万个帖子在网上展开谩骂，还有个什么《网瘾战争》搞得“轰轰烈烈”，有鼻子有眼，打的热火朝天，白雪还手舞足蹈地庆祝“全胜而归”！<br/><br/><wbr />?<wbr />?<wbr />?到此，想起《论语》里记载了这样一个故事，就是孔子家的马厩着火了，家人赶来报告，说：夫子，你家的马厩着火了。孔子说“伤人乎”，这是孔子说的唯一一句话，《论语》中讲“伤人乎，不问马”，马死了，无所谓。马厩着火了，房子着火了都没有关系，但是有没有人受伤了？<br/><br/><wbr />?<wbr />?<wbr />?这个故事送给我们以上谈到的某些新闻工作者。请他们反思，在报道中是人重要还是新闻重要，还是所谓的趣味，所谓的新闻价值重要？<br/><br/><wbr />?<wbr />?<wbr />?如果一时想不明白，我把这个捉猴子的故事再讲给他们听，请他们自重。<br/><br/>浙江龙泉、庆元一带猎人捕捉猴子，先制作一个较沉的木箱，木箱上凿一个可伸进手的洞,再在木箱里面放上苹果，猴子将手伸进木箱里取苹果，苹果却怎么也取不出来,而猴子又死活不肯放弃，于是，多数贪婪的猴子最终被捉。<br/><br/><wbr />?<wbr />?<wbr
