simon22的故事叹了一口气。「如果不是被电威胁，多少人可以自由的享受生活。多少人不会被逼着说些违心的话。多少人会有享受自己的人生。人生本来就有对错，可是被父母送到这里，给了杨永信敛财的机会。他打着治疗的旗号，用骇人听闻的电击威胁我们一不能说出电击的真相，二不能表达出自己意图回家的念头。用传统封建的一些思想对我们进行道德绑架，这里做的不合他意，加圈，那里做的不合家长意，专场。让杨永信盯上了就进程序。谁不表面上老老实实的？」simon22的故事有些激动的回答记者。<br/><br/>「在这里受委屈要忍着」一个家长对simon22的故事这样说过。说完这句话，家长摸了摸他的头。<br/><br/>「他孩子只有14岁，被杨永信都折磨的幻听了。我跟她妈说了，快走」simon22的故事知道他这样说的后果可能是什么，但他还是说了。<br/><br/>「那个家长对我很好，经常给我送好吃。在我刚进去的时候，只有她会对我微笑。那里有一种等级制度。自上而下依次是杨永信&gt;家委&gt;班委&gt;老家长&gt;住院时间长的家长&gt;新入院的家长。我刚进去一个星期内，被当麻瓜一样，几乎所有家长都对我指指点点，品头论足。这里该怎么干才行，那里该怎么干才行。我忍了。后来时间稍长一点，有个盟友犯了事，被人举报了。他家长来找到我，问我很多问题。我听出他的意思是怀疑我打的报告。最后他还威胁我，我想让你进去（十三号）就让你进去。所有家长都有这个权利。」simon22的故事向我回忆起这件事时，脸上露出了一种孩子的童真的笑。「他以为他这样能吓唬住我，没想到这给了我出院一次推进剂，我妈知道后，我小室的一个老家长知道后，都跟那个家长理论，说他这样怎么怎么不对。人家可是住了一年的老家长诶，这样被两个家长批评，脸红的不得了。这事在网戒中心特别罕见」simon22的故事笑的更甜了。<br/><br/>当天下午，我想继续守株待个兔纸。我和simon22的故事在另一家饭馆等。可是这次没有成功。记者发现到，现在的家长比上午的家长更精。不知道是不是透露了消息。「一般一天早中晚三次晚饭时间，可是今晚上的家长是分三次出来买饭的。这很不正常，明显是有人下了命令不让我们采访到在院的家长。」simon22的故事喝了一口羊肉汤说。在一口酒一口肉的快意人生里，我们结束了第二天的采访行程。<br/><br/>2016年12月30日，我被一阵噪音吵醒。我第一时间想起的是晨跑。我翻开手机，手机显示的是6点35.我来到网戒中心门口，此时有许多家长出来买饭，有粥，菜煎饼，馄炖和汉堡。记者试图询问每一个遇到的家长，家长都不予回复，而且看记者的眼神也特别奇怪，像是在看特务。最后，就在记者打算放弃采访的时候，有个家长主动上前和记者交谈，并且往人少的地方啦。对方年龄大概有50多岁，女家长。「这里的军训已经停了一段时间。你们不要只听那些没改变好的孩子说的。这里以前是有“治疗”可现在连低频脉冲都停了。盟友们一起做磕操。确实累。但这是为了孩子好。他这次犯错受到了一点教训，下次他肯定不敢犯了。」女家长表情坚定的说。「你们也要注意安全啊，现在一些家长开始拿警棍巡逻，以前没这回事。杨叔多次让这些家长巡逻的时候注意，不能让陌生人进入网戒中心。却还是有家长因为马虎领了不少圈」。「谁愿意交那么多钱让孩子受罪啊。但是很多孩子从这出去后都变得很精神，改工作的工作，该上学的上学。网上那几个没改变好的孩子也很精神，就是有点精神过了头。」<br/><br/>当记者打算结束今天的采访时，记者龙徒听到楼上传来的咆哮。「你听，杨叔又在批评个别家长了，就因为把你们放进来了。」女家长叹气说。<br/><br/>「****，你***，你怎么当的巡逻组组长。你好歹也是个家委，我以前怎么跟你们说的
