class=" wrap external" href="https://link.zhihu.com/?target=http%3A//weibo.com/u/5872795520%3Fprofile_ftype%3D1%26is_all%3D1%23_0" target="_blank" rel="nofollow noopener noreferrer" data-za-detail-view-id="1043">未消逝的青春2015</a>”上刚刚发出那篇名为《我在临沂网戒中心的真实经历》文章的时候，我就联系到了他，并通过微信进行了一次简短的采访。但是没想到，那周的周末，因为恐惧，也因为临沂四院网戒中心家委的出动，他连夜独自一个人离开了家乡，开始了一场前途未卜的大逃亡。<br/><br/>我见到小辰的时候，已经是他离开临沂的很多天之后了。他看起来安静、消瘦、疲倦，和我见过的那些“山东汉子”完全不同。我邀请他一起吃饭，他很客气地说自己已经吃过了，就只是在我对面坐下，并且很细心地帮我刷洗餐具，并倒上了一杯茶水。<br/><br/>他看起来有些紧张，说话声音也很小。我开始闲聊、询问的时候，他也会很专注地看着我，想一想，然后再回答。不知道是有意还是无意，他在交谈中也很少会把话说死，只是很平静地重复着“应该”、“很可能”、“我估计”这样的词汇。<br/><br/>他告诉我，自己还是会回去的，他也不想给任何人添麻烦。<br/><br/><strong>触乐（以下简称触）：你跑出来的时候的具体的情形是什么样的？你刚才说，听到你妈跟家委在打电话，当时立马就想到了要逃出来吗？</strong><br/><br/><strong>小辰</strong>：嗯，接到电话以后，听说他们要过来时候，第一感觉，就是觉得不太对劲。我在临沂是和我爸妈是分开住的，我就去了我那边那个家，在那边等着，等他的口风，然后我想下一步怎么走。然后家委他们12点左右来到我家，找我妈妈出去谈话。我妈到5点左右才回来，有一位家长还跟着一块去我家了。然后我妈给我打电话，说，“我觉得你又不正常了，你赶紧把那个东西给删了，现在人家都找到家里来了”。<br/><br/>我当时其实怎么说呢，也挺生气，我是挺悲哀的吧。我就问，“那如果不删的话会怎么样？你会把我再送进去吗？”我妈就说，“你赶紧把它删了，就什么事都好说，你要是不删，那好不好就不一定了。”但其实微博上面很多都已经截图、扩散开了，你说那时候删或不删，意义不是很大。但我觉得她有那种想把我送进去的意图了。她当时一直在问我在哪儿？在哪儿？当时我就很恐慌，联想到别动队，就准备想要走了。<br/><br/><strong>触：你妈妈不知道你在哪儿吗？</strong><br/><br/><strong>小辰</strong>：因为当时我一直在微博上面放烟雾弹嘛，就说我也在外面玩，我也没说我在哪，他们就一直问我在哪。<br/><figure><img class="origin_image zh-lightbox-thumb lazy" src="https://pic3.zhimg.com/80/f587f75b3811796e7fd285c2f6d1f1fd_hd.jpg" width="662" data-rawwidth="662" data-rawheight="385" data-original="https://pic3.zhimg.com/f587f75b3811796e7fd285c2f6d1f1fd_r.jpg" data-actualsrc="https://pic3.zhimg.com/f587f75b3811796e7fd285c2f6d1f1fd_b.jpg"></figure><br/><i>小辰当天的微博</i><br/><strong>触：他们怎么知道是你呢？</strong><br/><br/><strong>小辰</strong>：因为里面我曝光的东西太多了，可以联想到我。你知道吗？他们立刻就可以看出来。<br/><br/><strong>触：那你大概什么时候决定要走呢？</strong><br/><br/><strong>小辰</strong>：在我妈打完电话以后我立即就出门了，大约是5点，然后就先去朋友那边躲起来。当时我一直刷了一天的微博，手机快没电了，就先去充电，然后和朋友商量了一下，毕竟多一个人多一个想法对不对？我找了两个我挺信得过的朋友，觉得还是走。因为没办法，如果你再留下来，就很可能会被抓回去，或者遭黑手。<br/><br/>但临沂毕竟是一个小城市，我不能连累到我的朋友，之后事情会发展成什么也是未知的。所以我也没有请求他们帮助。<br/><br/>在决定了要走之后吧，我就联系到了那个我比较信任的网友。然后我充电到10点钟左右，就打了一辆车去附近的县城。我当时面对这样的情况之后很慌，会不自觉的往最坏的方向去想，就觉得他们会不会展开全城抓捕那种，毕竟你不往坏地方想，如果等它发生了在去的话，估计就完了。我当天晚上离开我家那边，第二天一早就坐了长途大巴离开了临沂。<br/><br/><strong>触：我听包先生（帮助小辰逃跑的人）说，当时你手上没有足够现金买车票，就找了他在当地认识的人给您买了票，对吗？当时是怎么确定他是朋友，不是坏人的呢？</strong><br/><br/><strong>小辰</strong>：我没办法呀，因为我可以相信的只有这些人。当时到了车站，然后在那等他朋友过来。包先生的朋友离车站也挺远的。到车站之后，我等那个朋友等了大概一个多小时。<br/><br/>当时我随身带了一些可以伤害人的东西，就是想着要是实在没办法，他们要来抓我了，或者我被（我爸妈）背叛了，就可以……<br/><br/><strong>触：你现在还带着它吗？</strong><br/><br/><strong>小辰</strong>：没有。因为坐车需要安检。我当时走之前就把它扔到厕所里的工具箱那里，因为在那个地方不会被人捡到，就不会被其他人拿来干坏事，而且因为是工具箱，所以也不容易被发现。我当时真的是想到说如果他们过来的话，就得下手（伤害自己）。不然怎么办？没有办法，求别人救不了你。<br/><br/><strong>触：你没有想到找爸爸帮忙吗？因为当时也是他执意带你出来的，他应该其实不相信杨永信那一套。</strong><br/><br/><strong>小辰</strong>：当时那种情况之下，没办法去找家人，你不确定家里会不会被说动了，而且我也不想给家里人添麻烦。如果要连累他们的话，就还是尽量避免，因为（在微博上的）那篇文章是我自己写的，自己做的事情自己承担。<br/><br/><strong>触：但是送你进去的也是他们呀。并不能说是连累或者添麻烦。</strong><br/><br/><strong>小辰</strong>：……毕竟是你的家人，他毕竟是家人，真没有办法。<br/><br/><strong>触：当时你上车，要离开临沂的时候，是什么感觉？</strong><br/><br/><strong>小辰</strong>：当时肯定是一路恐慌呀。嗯，就是每一次突然堵车了啊，停下来的时候，就会回头四处看。总觉得那个地方（网戒中心）能量那么大，如果出动警察了，走漏风声了，就会在半路上抓住我。<br/><br/>还有就是我晕车，身体太难受了。之后离开临沂很远的时候，就只顾着自己难受了。是那种天昏地暗、天旋地转的感觉，车上还把胃里的所有能吐的东西都吐了出来，几乎只能趴在那，这让自己好受一点。<br/><br/>我是第二天早晨坐的早班车离开临沂的。<br/><br/><strong>触：你是一直都在外面呆着吗？一个人吗？就等在车站门口吗？</strong><br/><br/><strong>小辰</strong>：没有没有，就找了一家很小很破的小旅馆先住了一晚上。当时是因为去那个车站已经关门了，而且毕竟这路上还会有很多事情，你必须要休息，你还要有精神才行。当时是用包先生那个朋友给我的钱住下的。他除了给我买车票的钱，还另外给了我一些钱。虽然不是很多，但至少够我住一晚。<br/><br/>唉，他真是好人，当时我说不用了，不用了，但他坚持要给我。他说，你到那边会有情况，还会用到的，就先拿着，到那边再说之后。我觉得这个世界上还是好人居多。<br/><br/><strong>触：离开了以后是什么感觉呢？就放心了吗？</strong><br/><br/><strong>小辰</strong>：其实还是有点怕，另外就是很累。其实网上网友们留言会说得比较可怕，你知道吗？什么定位啊，公安系统啊，政府啊之类乱七八糟的东西，我就会担心。但是实在是太劳累，而且晕车的感觉确实不太舒服了。等我下了车，到了地方之后，没有吃饭就直接睡了。<br/><br/><strong>触：所以你一路上也是跟网友或者是跟包先生有联系的是吧。</strong><br/><br/><strong>小辰</strong>：对，主要就是和网友们，还有就是联系媒体这方面的朋友。最早我发文章的时候，也有一些和我联系的媒体，还有就是当时回复我的那些网友，我会比较信任他们。<br/><br/>当初我发文章的时候也没有多想，因为没有多少人看，很少人看。后来就是传播开来了，他们（网戒中心）看到就觉得很气愤，之后才联系到我。<br/><br/><strong>触：那你现在的想法是什么？</strong><br/><br/><strong>小辰</strong>：就是想要找媒体朋友把这事情弄大一点。把事情弄大了之后，我就至少算是半个公众人物嘛，他要如果对我下黑手什么的，就毕竟要考虑一下。<br/><br/><strong>触：当时您还和包先生一起发了一个长微博，说要众筹对吗？</strong><br/><br/><strong>小辰</strong>：我跑出来的时候也没有带银行卡，包先生的经济状况也不是很好，再说我也不想去麻烦他。<br/><br/>欠人太多了，没办法，就是说只能先接受一部分网友的这种援助，然后就是有了那个微博。其实我发文章开始的时候，也有一些打赏，但是数额很小，不过，这对当时的我来说也是一种鼓励。不过包先生发了那篇微博之后，愿意出财力的非常多。很短时间之内就收到了很多网友的帮助。后来钱越来越多，我就发了微博说，不需要再给我钱了。<br/><br/>但还是不断陆续有人发红包过来。我都没有收。我目前所有收到的钱也都记了账目。<br/><br/><strong>触：那你爸妈知道这个事情吗？</strong><br/><br/><strong>小辰</strong>：我爸妈他们平时不上网的，他们了解我这个事情，大部分可能就是听别人说。我觉得很有可能，那个一直待我在我家里的家委，会给我爸妈讲我的事情，因为从他们嘴里讲出去可能就是另外一个味道了。所以我父母现在就一直很担心。我的想法是，赶紧把这个事情解决，然后赶紧回家。<br/><br/><strong>触：你不怕吗？家委一直在你家，会给你爸妈“洗脑”？</strong><br/><br/><strong>小辰</strong>：没办法，他们毕竟现在还是为了孩子好。<br/><br/><strong>触：那你想过诉讼吗？</strong><br/><br/><strong>小辰</strong>：现在暂时没有吧。因为我必须要保证我自己安全和家人的安全。如果真到了起诉那个阶段的话，就完全是你死我活了。到那时候我会担心说我个人的身份隐私和安全，我的家人也会受到牵连。<br/><br/>我还是相信我们政府和相关部门会出面解决这个事情。<br/><br/><strong>触：我们也采访了另外一些人，关于是否要起诉，大家的态度都挺分散的。</strong><br/><br/><strong>小辰</strong>：因为毕竟他们出去之后也要有自己新的生活，为了一件已经过去的事情，再去把现在的生活毁掉的话，不值得。<br/><br/><strong>触：但不会觉得自己的青春年少就这么被毁了吗？</strong><br/><br/><strong>小辰</strong>：只能这么想啊，不然怎么办。<br/><br/><strong>触：你觉得杨叔是那种会报复别人的人吗？</strong><br/><br/><strong>小辰</strong>：我觉得会，否则他就不会煽动那些家长到我家去。<br/><br/>之前不是有一个节目叫网瘾战争吗，那些人都被钉在了他们网戒中心的耻辱柱上，杨叔也一直会提到，比如，嗯，有一个很出名，叫鹏飞的，是里面的标志性人物，杨叔就会从各方面去批评他，说他坏话，说他从这边出去之后，就忘记了杨叔的好之类的。<br/><br/><strong>触：他会要求你们一起批评吗？</strong><br/><br/><strong>小辰</strong>：不会，因为上课的时候只能他一个人说话。<br/><figure><img class="origin_image zh-lightbox-thumb lazy" src="https://pic3.zhimg.com/80/970d12b08bded8860042ca3abf6c9fc5_hd.jpg" width="780" data-rawwidth="780" data-rawheight="486" data-actualsrc="https://pic3.zhimg.com/970d12b08bded8860042ca3abf6c9fc5_b.jpg"></figure><br/><i>杨永信（图片来自其新浪博客）</i><br/>在里面，所有东西必须要公开透明，我知道你的底细，你也知道我的底细，所有个人信息都是公开的。点评课的时候，家长们会说自己家孩子以前犯了什么事，做了什么见不得人的勾当，大部分都是家长在说。盟友那边的话，反正是他会尽量的去讨好杨永信，就是说让杨永信高兴他听得舒服，那样的话那个盟友就会免于被治疗，需要把自己说得很坏，说以前很危险，最后你还要衬托说，来到这里、在里面被关着，但是里面的管理都那么好，然后称赞一番杨永信，这样的话，他心里就会很高兴，这会叫他会觉得他这种教育方式是对了。<br/><br/>在里面，不能大声说话，不能大声笑，不能剧烈运动，你就老老实实的，走路别走得太快啦，说话别很大声，表情不能很丰富，你呆呆傻傻了，就会呆得比较“舒服”。<br/><br/><strong>触：这个对你现在还有影响吗？</strong><br/><br/><strong>小辰</strong>：你养成习惯了，你现在怎么笑也不会露出牙齿那样大笑了；看到什么特别好笑的，也很少很少会大笑，也没有那种很强烈的情绪了。<br/><br/>里面真的是很恐怖。在里面要时刻注意周围的人的视线，而且你没有信得过的人，就需要不断提防别人，然后小心做自己，每说一句话做一件事，你先想一想能不能做，会不会就是造成不好的结果。<br/><br/>我觉得那种精神压力是最恐怖的，在里面的电击治疗只是其中很小的一环，其实所有的都是围绕这个来的。他必须要用那种方式去控制他们（盟友），让他们变得就像一个木头人，行尸走肉一样去听从他。你反抗、反抗，时间长了也总会被潜移默化了，他会被里面的东西给渗透进去，到最后，你可能就会接受他，这是很恐怖的一件事情。<br/><br/>除了家长以外，每个人表面上都是支持的，必须要装出来支持，或者是赞同这方面的，所以你看不到别人的真心，这真的非常可怕。而这些，都是因为他不想去做那个电击，他不想被“治疗”。而除了“治疗”之外，它还会有一些日常上的一些惩罚，比如说限制你的伙食啊，只能让你吃青菜豆腐，或者干脆就不让吃饭；另一种就是抱沙袋，那种十几二十公斤的那种沙袋，它会让你一直站在那点评课堂上抱着，一直到下课，不能动；还有就是跪拜操，西藏朝拜的那种，会有家长看，一直盯着。你必须要从站着开始，再跪一下、趴在地上，然后再恢复跪着的姿势，再站起来，然后再趴下，就是这样，循环往复。<br/><br/>你逃不出去的，也自杀不了。我刚进去的时候，有一个盟友他吞了一个风油精瓶子，但是没有什么用处，吃了之后就被很多男家长一起押着出去做了手术，取出来立即就回来了。<br/><figure><img
