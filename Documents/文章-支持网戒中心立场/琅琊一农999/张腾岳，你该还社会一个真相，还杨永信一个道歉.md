1=2”的智商就能解答的真相，何以炒作至此？<br/><br/>8月7日，莉莉丝游戏人雷斯林（蔡江舟）出于职场利益和团体的炒作需求，以《杨永信，一个恶魔还在逍遥法外》为题，在2009年的第一次炒作寂静多年之后，重新把相关材料片面剪辑，对杨永信发起了新一轮攻击。其贴文如一纸巫符，让众多急躁的媒体如霎时唤醒的恶灵纷繁跟风炒作。<br/><br/>现实上，对一些被炒作的题材，假如你稍有感性，用“1 1=2”的智商就能解答：<br/><br/>杨永信，这位国务院政府特殊津贴的享用者，被国度授予一系列荣誉的面前，假如没有付出的艰苦与奉献，你信吗？<br/><br/>一位资深的肉体科医生，在使用一种医疗手段时，他能胆小妄为到突破法律的底线拿孩子当小白鼠吗？全程陪护的家长们都干什么去了？他们能眼睁睁地看着杨永信像魔鬼一样摧残本人的亲生骨肉？<br/><br/>临沂网戒中心――这家国度公立医疗机构的存在，假如没有合理而根本的存在价值，没有社会的根本一定做根底，它能设立吗？<br/><br/>09年的炒作不可谓不甚嚣尘上，哪一家政府职能部门胆敢冒亵法尽职不作为之天下大不违而无视？<br/><br/>假如杨永信誉的是骗术，他骗得了一时，骗得了一世？十年戒网瘾，有一例医疗平安事故，杨永信还能走到明天？<br/><br/>但是，没有谁去关注这些根本的客观存在，没有谁去思考这些复杂到不能再复杂的基本不是成绩的成绩，更没有谁糜费一点珍贵时间做一番零间隔的深度调查，这些媒体人似乎就这么急躁地、零智商地一窝蜂地炒起来了，包括CCTV2，包括那个因掌管科普解密爆料节目著称而构成本人一惊一乍奥秘夸大作风的张腾岳！<br/><br/>在他们的想象中，网络普及意味着网瘾消逝，杨永信就是逆历史潮流而动顺从互联网摧残青少年的魔鬼，送孩子戒网瘾的家长几乎愚不可及，外地政府涉嫌不作为。总之，在这些媒体人的眼里，临沂，这个反动老区这片白色土地仍然贫穷蒙昧，就是一个远离古代文明、相似洪荒野蛮的法外之地。<br/><br/>但我所知道的现实并非如此。<br/><br/>作为一名基层教育任务者，我知道互联网的普及并未令网瘾消逝，我知道玩网络游戏给目前的根底教育秩序带来了多大的费事，我知道有很多学校甚至高校公布了相似于制止先生带手机进课堂进校园的规则，我知道这些痴迷网游和黄色信息的先生是如何顺从教育，冲突关爱。我也知道在停学、劝退、开除的学龄孩子里大多是由于病态痴迷网络，这些孩子旷费学业、自大自闭、与世隔绝敌视父母，甚至年岁悄悄就染上毒瘾赌瘾乃至贩毒卖淫、嗜血成性。试想，这些孩子被赶出校园流落街头给社会波动带来多少隐患？学校不要，监狱不收，他们将流向何处？谁为他们的青春买单？！<br/><br/>而杨永信的网瘾戒治形式也绝非像外界所传只有“电击”，杨永信更不是逆历史潮流而动的互联网的冲突者。戒网十年，杨永信不可谓不呕心沥血。他创建了108个环节，包括军训、外务整理、演讲、才艺展现、生活拓展、意志训练，等等。真正的中心环节，就是以社会主义中心价值观为根本内核、以优秀的国学传统为根本依托，以理想信心培育为目的指向的心思点评。我有数次走进心思与品德讲堂，那种充溢正气的浩然、寓方于圆的处世之道、撼人心魄的品德评价、指点迷津的感化引领、迂回迂回的心思攻坚，让有数浪子回头，也让许多分裂家庭言归于好。杨永信常常讲的就是：网络是工具而不是单纯的玩具，人要做愿望的主人而不是愿望的奴隶。<br/><br/>所谓“电击”，中华肉体迷信会副主任委员陈彦方教授承受《小康》杂志记者采访时说道：“电击和电疗基本就不是一回事。杨是使用了我们国度的DX-II型治疗仪，这个机器我90年代初期在英国作为拜访学者回国后，展开专项研讨的时候也用过，经过我们课题组的专项科研实际使用证明我们使用的这种仪器是平安无效的。”在杨氏戒网瘾形式中，这是让激动狂躁嗜用暴力危及平安的孩子进入冷静形态承受心思疏导的根底性辅佐手段。<br/><br/>这些孩子的家长，更不是蒙昧混沌对古代教育理念一无所知的“愚氓”，他们中有农民，也有教员、心思征询师、律师，也不乏政界商界军界学界的豪杰精英，很多家庭本来十分幸福，事业十分成功。孩子的走偏，缘由是复杂多元的，但网络游戏，一定是重要的爪牙。<br/><br/>朗朗乾坤，清平世界，杨永信戒网瘾十年，阅历2009年的炒作风雨，媒体不可谓无视，民众不可谓不知，政府不可谓不晓，中国没有法外之地，难道真不成是政府职能机构的亵法尽职不作为？<br/><br/>看到节目中掌管人张腾岳那种傲然于家长医生之上，潇洒自若正义凛然自信满满有如万能导师般的点化凡界众生，我心里真的十辨别扭！<br/><br/>央视名嘴，掌管一哥，急躁轻率如此，呜呼哀哉！<br/><br/>媒体急躁乱象，职业担当缺失，令人悲痛<br/><br/>互联网时代，谁都想过一把媒体瘾，正儿八经的标题似乎少有人看，所以标题党盛行，对一般不良媒体或记者来说，就仿佛不炒作就吃不上饭，不雷人就博不得眼球。<br/><br/>微机一开，各资讯网站纷繁弹出，删不去关不掉，惊天密闻刺瞎双眼：<br/><br/>惊天揭秘，中国正在南海某地建超级航母――30万吨级！<br/><b
