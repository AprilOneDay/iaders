class="preface">继续！“小作文”又来了：为正义执言，文章均为亲历者所写；用事实说话，让“经营事件”的”恶媒“见光死！――</div><br/><div class="preface">――面对恶媒谣记的谎言，谁才是真相的见证者？</div><br/><div class="preface"></div><br/><!--more--><br/><div class="WB_editor_iframe"><br/><br/>??近日看了触乐网杨中依写的一篇名为《和杨永信战斗的人们》的文章，给我的第一感受就是在看玄幻小说。此文不仅文笔拙劣、漏洞百出，而且句句胡说八道，简直就是是非不分，颠倒黑白！为什么这样评价？因为我就是杨医生的学生，现在正在中心进行改变，而我眼前的中心与你们口中的完全南辕北辙！<br/><br/>我是2016年12月份入院的，如今已有三个月了，我在这里学习、生活，杨医生不仅纠正了我的错误认知，更帮助我找到了人生的方向！至于你们这些胡说八道满嘴跑飞机的喷子，劝你们还是乖乖回家洗洗睡吧，你们有什么资格来侮辱网戒中心和杨医生？<br/><br/>针对这篇让我很无语的文章，我有以下疑问：<br/><br/>一、题目。从题目上看，和杨永信“战斗”的人们，我就呵呵一笑。你们安的是什么心？还“战斗”，要战斗也得看敌人是谁，是不是出于正义。难道你们战斗的敌人就是挽救问题孩子，拯救患难家庭的杨永信医生？你们觉得自己挺伟大，是英雄？还是很卑劣？觉得自己挺能，与正义为敌，连战斗的目的都不对，说轻了你们是炮灰，实际上你们是在作恶，你们甚至不能让父母舒心地活着，什么东西也不是。<br/><br/>想用这个题目来引起读者兴趣，想出名？你们可真是煞费苦心了。<br/><br/>二、小辰。文中的小辰，你们和他狼狈为奸，为了达到你们不可告人的目的而不择手段，无中生有，胡编乱造！看似正义的语句背后实则是你们巨大的阴谋！你们说小辰是从中心走出去的，有什么证据？为了达到你们的目的，虚构一个小辰出来也说不定。暂且就算小辰是个离院盟友，但他仍然是个问题孩子，是父母眼里的叛逆，世人眼里的另类而已，否则怎么解释父母把他送往网戒中心，怎么理解在法治社会的光天化日之下，不寻求法律保护而被害妄想般的逃亡？随随便便离家出走不管父母死活的人你们就这样袒护？<br/><br/>当然，也不否认这是为出名而自导自演的闹剧。<br/><br/>三、《中国恐怖故事：疯诗人》。这篇文章是“赵蕾”写的，副标题是：“由真人真事改编的短篇小说”，原型是一位盟友，可赵蕾在文章结尾竟编造这位盟友跳楼自杀来蛊惑不明真相的网友，是想以此来抹黑临沂网戒中心，还是对他人人格的不尊重？或者是赤裸裸的诅咒与侮辱？实在可气、可恨、可笑！<br/><br/>四、家委会的“探底”与小辰的逃亡。感觉这就像“猫抓老鼠”的故事一样，太扯淡了！首先家委们是由我们这群问题孩子的家长推选组成的，他们只是为了更好的照顾我们在中心的生活而提供无私的服务，没有去“抓人”的权力与义务，更没有去“探底”。小辰的“逃亡”，我说你们的想象力可真够丰富的，竟然用“逃亡”来形容你们的阴谋，煞费苦心啊！我们来网戒中心，不是杨医生的指使，更不是家委的活儿，是我们这些孩子自己的家长把我们送来的，比如我自己。我就是被我爸爸带亲朋好友将我送来中心的，因为我在家已经完全迷失了方向，整天像个“活死人”一样不上学，在家玩游戏。那我能不能也把这件事怪在中心头上？如果杨医生派家委抓盟友，岂不犯法？我们都是被家长送到中心改变的，和杨医生与家委没半毛钱关系！小辰，你可以不理解父母，那是你们的亲子矛盾，但是你不能因此为借口抹黑他人，而做一个是非不分、颠倒黑白的人。<br/><br/>五、陈默的走访。文中说陈默从08年末到09年初只身来临沂4次，2次进网戒中心，但事实上，并没有这样一个人，还2次进入中心。陈默说他通过《战网魔》一书描写的23个人物，走访到了15位，书里写盟友对杨叔感恩戴德，事实上恨之入骨。我不得不佩服作者的才华，太会断章取义了。走访15位，不说有几位是改变好了的，只说个别没改变好的盟友如何对中心恨之入骨。那如果是14位盟友改变好，只有一位没改变好的呢？请问作者如何解释？<br/><br/>再说了，陈默采访的真实性在哪里？<br/><br/>六、陈默的痛苦。文中提到有一位记者把陈默提供的资料告诉杨永信，从而可以在中心畅通无阻的采访。但中心从未有过这样一个记者！这作者又在瞎编乱造了。陈默说他自己十分痛苦，可又是谁造成的呢？纯属自己没事找事吧！<br/><br/>七、关于赵蕾。赵蕾实际上就是无明之痴。说实话，我这次来中心前在网上是和她有接触的，她为了得到中心的爆料，在与盟友们的聊天中不择手段，甚至说各种暧昧的话。那句“某大学，女学生。十九岁，去陪睡......”我敢证明绝对是有根据的！赵蕾也说她自己痛苦，甚至文中提到在寝室砸东西，放声大哭，下跪，用手打墙，割腕，甚至要拿刀杀她自己养的乌龟，这不就是典型的心理变态嘛！我看她这个状态已经不适合继续上学了，应该送进精神病院治疗，这样一个丧心病狂的人让网友们怎么相信？<br/><br/>八、盟友打电话。文中提到，如果盟友打电话被发现，就会被认为是在向外界透露消息，就会被治疗。简直可笑至极，你编也得编的靠谱点儿吧！在中心，盟友是碰不到手机的，盟友们连见都见不到，拿什么打电话？<br/><br/>九、赵蕾面见盟友。有男朋友的赵蕾说与一位盟友见面，那位盟友要求牵手，赵蕾拒绝后和对方保持仅仅10公分距离一起走了七、八个小时！那这是什么意思？目的何在？真是搞不懂，我越来越觉得这个赵蕾的个人作风很有问题。<br/><br/>十、“磕头操”。“课操”二字故意加一个“头”字，让人望文生义，这会让不明真相的网友怎么想？简直不要脸，什么损招儿都使得出来！课操是中心多年来一直存在的一个重要环节，它可以强身健体，磨练意志，调养身心，但因人而异。文中说盟友每天要完成500―1000次，简直瞎扯。还说课操在3月份改回了“电击”，并且陈默说“非常确定，但没有证据―可是谁又有证据呢？”最后这句就意味深长了，这分明就是在误导网友，况且我来中心三个月了，我还没有你陈默熟悉情况？<br/><br/>十一、文中的措辞。不了解网戒中心的人看到这篇文章可能会感到很正义，但毕竟是不了解。在我的眼中，这是篇伪正义的文章，因为临沂网戒中心是让人变好的地方，他挽救了那么多问题孩子，拯救了那么多患难家庭，这是有目共睹的！而这篇文章歪曲事实，肆意抹黑，言语中透露着阴暗心理，妄图破坏搞垮中心！<br/><br/>这种心术不正的文章怎么可以发布在网上？<br/><br/>多的话我也不说了，人在做天在看，心术不正枉为“媒”，请触乐网好自为之吧！<br/><p
