/><br/>无意为谁洗白，只为还原真相；无需哗众取宠，不惧围攻谩骂<br/><!--more--><br/><br/>此文只为关注杨永信事件真相的理性思考者所写，情绪极端盲目追风或居心叵测的“倒杨派”请慎重阅读。<br/><br/><strong>纠结概念无义，客观存在是真</strong><br/><br/>一个社会问题的产生，必然伴生相应的担责机制。没有天生的游戏规则，规则只能在探索中产生。<br/><br/>譬如“网瘾”与“戒网瘾”。<br/><br/>针对杨永信的口水战，首先集中于“网瘾”的概念之争。<br/><br/>我觉得，先放下概念之争，看看现实的存在，十分必要：自上世纪末开始，伴随着互联网的普及，学生痴迷网络，一直是困扰基础教育的严重问题。随着近年来手机的智能化，这个问题更加普遍和突出。学生玩手机游戏，浏览不良信息，严重干扰正常的教育教学秩序，早已是不争的现实。带手机是公民的自由，但你仔细调查下，有多少学校甚至高校在甘冒风险颁布手机进课堂进校园的规定？<br/><br/>这还不是最严重的。更加让人焦虑的，是造就了一个对网络产生病态依赖的特殊群体：多少人成了社会功能严重丧失无法融入正常社会生活的宅男宅女？多少学子为此被学校劝退开除，与家庭关系破裂，有多少校园惨案家庭悲剧社会治安问题与此相关？<br/><br/>而杨永信事件的幕后黑手和跟风媒体所谓的“互联网普及――网瘾已经消失――网瘾戒治失去存在的法理依据”，则纯属不负责任居心不良的荒谬逻辑。<br/><br/>互联网的伟大意义毋容置疑，可与此伴生的社会问题谁来承担？这显然不是传统意义上单纯的家庭、学校、医院、监狱功能所能解决的。<br/><br/><strong>真实的杨永信网瘾戒治模式究竟如何</strong><br/><br/>从2006年开始，杨永信介入网瘾戒治领域。在临床中他发现，真正病态痴迷网络而导致心理精神病状的，只是网民中的极少数。这除了人们公认的家庭、学校或社会环境影响外，生物遗传因素也不容忽视。真正意义上的网瘾现象只产生于网民中的意志薄弱、心理脆弱、易干扰易诱发的性格缺陷者。所以，在介入网瘾治疗的很短时间内，杨永信就把网络成瘾、性格缺陷及其影响因素进行关联研究。<br/><br/>在此期间，他以大量的临床案例为据，以传统精神医学为基础，广泛关注国内外网瘾问题的研究动态，形成了大量的理论资料，发表和出版了许多学术论文和专著，并在实践中构建起集医院、家庭、学校、军营特色于一体，集网络成瘾戒治、性格缺陷矫治、问题家庭诊疗功能于一身的综合性救治系统。同时，大量成功改变的案例也为他赢得了无数家长的认可和各级政府的嘉奖。<br/><br/>这个模式包含了百余环节，其中最核心的环节是心理点评：杨永信认为，戒治网瘾从矫治性格缺陷开始，矫治性格缺陷从改变认知开始，改变认知从重塑心灵开始，重塑心灵从树立正确的理想信念和培养阳光心态开始，改变孩子从改变家长开始。同时包含了大量的辅助性环节：如意志训练，习惯养成，感恩教育，社会功能康复训练（如阅读、写作、演讲、才艺展示、亲子交流、内务整理等）等。<br/><br/>这种模式，可以说是集精神医学、心理学、教育学、社会学、伦理学于一体，是克服了传统精神医学单一性功能弊端的综合性创新研究，是解决互联网伴生社会问题的大势所趋，甚至从某种意义上说是引领了精神医学领域的一场技术性革命。<br/><br/><strong>备受诟病与质疑的“电击”是什么</strong><br/><br/>这首先从杨永信收治的特殊患者群体开始谈起。<br/><br/>因为杨永信模式（以下简称“杨氏模式”）在性格缺陷矫治和社会功能康复方面的有效性，来此诊疗的患者成分复杂，真正与网络沉迷有关的不足六成，就这六成也几乎无一例外地伴有焦虑、抑郁、强迫、幻觉、妄想、冲动等心理精神病状，普通精神病患者、性格缺陷者、社会功能缺陷者不乏其人，甚至包括一些涉毒涉黄涉赌涉黑的“亚犯罪者”。为了稳定情绪有效沟通，顺利配合治疗，该中心在医疗许可范围内和监管机构的有效监督下，使用了目前大中小型医院和家庭普遍使用的康复理疗产品――低频脉冲治疗仪，作为一个辅助环节。<br/><br/>需要说明的是，杨永信是人不是神，十年戒网瘾，他在理论建树和临床上的巨大成就不容否定，但作为一种新生事物，其模式环节并不完美，成功率也绝不可能是百分之百。在教育家辈出的教育界，每个学校每个班级尚有一些不成功的教育失败者，何况在杨永信这里正是那些被学校家庭社会基本“淘汰出局”的问题青少年集群？甚至有那么一些从心底拒绝改变，自始至终仇视父母、仇视社会的孩子也不足为奇。<br/><br/>杨永信自己也说，我也特别希望能有更好的方法代替低频脉冲治疗！<br/><br/>家长为什么把孩子送到这里<br/><br/>这里的家长，并不是如北大“专家”王岳所说“无一例外有两大特征，第一，父权思想非常强；第二，文化水平偏低”，而是工农商学军教各界精英不乏其人，很多具有良好的家庭教育背景。其孩子之所以走偏，原因是多方面的，正如杨永信所说：具有同样家庭教养背景的孩子，在相同的互联网环境下，罹患网瘾的只是个别，在很大程度上，取决于与生俱来的性格因素和抵御诱惑的意志力。<br/><br/>决定家长把孩子交给杨永信的原因有两个：<br/><br/>一是被送到这里的孩子，不是普通意义上单纯的上网成瘾，而是其社会功能严重丧失、人格偏执扭曲和极端叛逆冲动的行为，已经让学校无法接受，已经严重叛离家庭，无法融入社会主流生活，混迹于市井游走在自我毁灭与犯罪的边缘。孩子已经拒绝任何形式的教育或心理救助，在不被社会认同的困境中，他们借助网络游戏进行自我解脱，实现虚幻的价值认同。他们的青春已经被毁，人生看不到任何出路。<br/><br/>二是孩子被送到这里之前，家长从信得过的受益家长那里得到口碑认证并做过实地考察，是慎重之举。他们认可这种模式并相信杨永信的医德人品。杨永信是他们重拾家庭希望的最后寄托。<br/><br/><strong>杨永信究竟惹恼了谁</strong><br/><br/>这次杨永信风波的幕后推手：蔡江舟，上海莉莉丝网络科技有限公司员工，该公司代表作品：手游《刀塔传奇》；林琪雪，新媒体影像制作人、导演，著名游戏人，手游《刀塔传奇》熊猫酒仙配音；祝佳音，触乐网编辑，资深游戏业者――三个拥有浩荡粉丝的大V。他们搅动舆情狂潮的阵地――触乐网，是国内最具影响力的手机游戏推广媒体之一，也是国内顶端的手游会所和众多手游玩家的集结地。<br/><br/>而杨永信恰恰代表了一个符号――“网瘾”和“戒网瘾”，代表了社会对网瘾的忧患、对激进发展的游戏业负面效应的担当和所有的网戒机构。而作为以网络游戏为核心利益的人来说，让游戏利益最大化是他们必然的职业追求，“网瘾”和“戒网瘾”正是他们潜意识里最大的隐患。<br/><br/><strong>杨永信网瘾戒治模式的历史意义不容抹杀</strong><br/><br/>笔者最后强调：<br/><br/>一 网络成瘾是互联网时代伴生的现实存在的社会问题，但成因与家庭环境、学校环境、社会环境和生物遗传因素有直接关联，解决这一问题，不是单纯传统意义上的医院、家庭、学校甚至监狱功能所能独立担当的，它必须是医疗、教育、家庭、军营等众多功能的系统集合，必须具备精神医学、心理学、教育学、伦理学、社会学等关联学科的集成式理论支撑，必须是医疗、教育、文化、司法、网信等各部门跨领域的联动合作。而杨永信和他的团队在这方面的探索，具有毋容置疑的创新意义和历史意义。<br/><br/>二 杨永信的探索无论在理论建树还是临床实践上，都具有大量成功的事实依据，临沂市科技局对相关研究成果组织的鉴定结论――其模式可广泛应用于网瘾、酒瘾、毒瘾、赌瘾、药物成瘾等各种瘾症的戒治，用于其他各种心理精神疾病的治疗和社会功能康复，挽救问题青少年，有效干预其危险行为及预防违法犯罪，开辟了多学科、多领域综合应用的成功典范――足以说明其科学性、创新性与前瞻性。<br/><br/>三 据我所知，到目前为止，杨永信创建的临沂网戒中心从未发生过一起医疗伤害性事件，相关监管机构和政府机关也早已就其依法依规的医疗行为发表声明。但这场风波急剧升级恶化，远非正常，其始作俑者制造舆情动乱的行为动机值得警惕。而至今依然有人假借杨永信之名发表动机不纯的声明或文字，依然有媒体把恶性新闻案例与杨永信进行捆绑式关联评论以博眼球，造成极坏的社会影响和严重的人身伤害。政府应该有所作为，坚决终止舆论乱象，彻底铲除社会意识形态的新毒瘤，坚决维护国家主流意识形态的主导地位，引领维护社会心态的健康稳定。<br/><br/>四 对个别爆料的问题青少年的扭曲心理、极端行为和社会危害进行调查追踪和有效控制，对这种特殊人群的治疗或教育探寻有效的解决之道。<br/><br/>五 对有使命担当精神、具有前瞻视野、积极探索社会问题解决之道的个人或机构，政府应该以发展的眼光给予足够的资源支持与精神鼓励，在法理范围内给予足够的探索空间。<br/><br/>谎言难掩真相，自古邪不压正。相信政府不会被舆论绑架，杨永信事件的幕后黑手和造谣中伤者理应受到法律的审判，而盲目跟风发声导致舆情恶化的不良记者和媒体，也必将受到道义的谴责。<br/>首次转载时间: 2017-02-04
