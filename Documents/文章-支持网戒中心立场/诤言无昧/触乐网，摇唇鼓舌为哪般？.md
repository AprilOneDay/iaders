/><br/><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？<br/><br/><!--more--><br/><br/>近期关于临沂网戒中心的炒作又时有出现，我也是一位曾在网戒中心住了近九个月的盟友，如今早已出院。暂且抛开网络上的纷纷舆论不说，我也想谈一谈网络游戏的影响。<br/><br/>车尔尼雪夫斯基曾说过：“只有抗拒诱惑，你才有更多的机会做出高尚的行为来。”<br/><br/>人的一生中要面临太多的诱惑，往往因一时的诱惑，错失时机，遗憾终生。特别是处在青春期的少年们，心智发育还不成熟，自制力较差，面对这个精彩的世界，他们还不能认清自己最需要做些什么。轻者：游戏机，网吧，打架斗殴；重者：赌博，毒品，堕落。它会让人光阴虚度，浪费青春年华，毁坏大好前程，后果非常可怕。<br/><br/>网络游戏是随着互联网才普及的，属于新鲜事物，对于青少年的诱惑力也很大。说起网络游戏，许多家长、老师都深恶痛绝。当年我沉迷游戏时，我的老师也是经常发火，甚至有不管我的想法。正因为网络游戏对于青少年有太大的诱惑，使得一些人沉迷其中，无法自拔，乃至学业荒废，误入歧途。生活中这样的例子屡见不鲜。<br/><br/>但如今，网络游戏摇身一变，披上了电子竞技的时尚外衣，成了一种名正言顺的竞技运动，这无疑会让人们进一步审视这样一个问题：如何看待学生倾心于网络游戏。如果你跟他说：“玩游戏会浪费时间，影响学习。”他此时或许会振振有词的说：“网络游戏现在是一种竞技运动，国家都承认了，我为何不能玩？”他才不会管你网络游戏与电子竞技到底有没有区别，我在劝告沉迷游戏的同学时也会遇到这种情况。<br/><br/>其实，说起游戏，人人都爱玩，或许这是一种天性，我小时候在乡下玩的是跳皮筋，弹玻璃球，踢毽子。到了近几年，网络游戏席卷了无数学生，众多学生被网罗到电子游戏的营盘。无论游戏形式如何变化，游戏的本质也是一样的。只不过网络游戏的诞生使成百上千，甚至成千上万的人一同卷入游戏之中而已。<br/><br/>在老师和家长的眼中看来，玩网络游戏的同学，几乎每一位学生的成绩与原来相比都有不同程度的下降，有的甚至下降到不可救药的地步。当初我在班中排名第八，后来随着一步步沉迷网络，成绩滑到20名开外，后来甚至放弃了学业。一个连成年人都难以把握和控制的游戏，怎么能希冀一个未成年人的自律与控制呢？<br/><br/>玩物丧志要解决，无非两条路。不玩物，要立志。孔子曾说：“吾十有五而志于学。”十五六岁的人不学习，能知道什么呢？当我们坚定了崇高的目标，并为之付出不懈的努力，我们还是会把时间浪费在无聊的游戏之中吗？很多人走向堕落都是因为胸大无志，生活十分无聊。于是就去寻找刺激，最终慢慢步入歧途。<br/><br/>不玩物，要立志，怎么做呢？劝导？家长和老师一定不止一次的劝导过，也无济于事。难道就这样放任青少年沉迷网络而置之不理吗？<br/><br/>社会不断发展，网瘾问题已经成为了社会的一大问题，老师、家长束手无策，总要有人管。于是，临沂网戒中心便顺应社会潮流诞生了。但是，近半年来，对于临沂网戒中心的炒作却一波又一波，原因很简单。<b>杨永信院长做的事情，是为了让众多网瘾孩子回归正道，这就无形之中触及了游戏开发商的利益，就好像林则徐的虎门销烟，触及了英国殖民主义的利益一样，于是游戏开发商一直在打着“维护青少年利益”的幌子，来为自己牟取暴利，有更多人玩游戏，他们才有钱赚</b>啊，利用手中的资本，让媒体为他们说话，这次交易他们自认为天衣无缝，但却幼稚得令人可笑。他们没想到的是，网瘾孩子不去中心，又能去哪？他们以后的生活会怎么样？这些他们是不会想的，因为他们只管赚钱，只是欲望的奴隶罢了。<br/><br/>最近，我看到触乐网上的一篇文章《和杨永信战斗的人们》，不禁暗暗称奇，故事情节趣味横生，甚是有趣，不过了解真相的我可以为其添加些许“注释”，暂未征求作者同意，甚为抱歉。“三月份‘磕头操’又被改回了电击，陈默说他非常确定，但没有证据――可谁又有证据呢？”看到这里，我实在是忍俊不禁，截至3月25号，整个三月我四次到中心分享，也没听说什么电击。况且，低频脉冲治疗与电击有实质的区别，作者混淆视听。哦，抱歉我忘了，你本来是个写小说的，虚构故事乃是强项，但你妖言惑众，这种行为则就是很不妥了。另外，小辰的“逃亡”故事也颇为离奇，倘若杨永信真的抓人到网戒中心，那么可以直接向公安寻求帮助，也可去法院诉讼，又何须大做文章，摇唇鼓舌妖言惑众呢？小辰的逃亡为哪般？真是不得而知，或许玄幻小说就是这样。<br/><br/>如果中心真的是摧残人的地方，哪一个家长会送自己的孩子去受“折磨”？事实证明，绝大部分的孩子和家长是受益于中心的，当然这一群体估计是受不到记者的采访了，因为他们所谓的采访本就是为了造谣，就要找坏孩子说中心的坏话！实践是检验真理的唯一标准，网戒中心到底叫人做好还是叫人变坏，自有事实说话，大点评课堂的那些谆谆教导，估计一个字也不会被他们写进去，因为断章取义这一招，各大媒体是屡试不爽。<br/><br/>1840年的鸦片战争，英国胜利了，这是政府的悲哀。而现在的这场电子鸦片战争，定不会让屈辱的历史重演，因为党和政府是有正确的判断的，不会听信一面之词。<br/><br/>作为网戒中心的受益者，我只希望政府和人民正视网瘾危害。少年强，则国强，戒治网瘾，拯救网瘾少年，为了祖国长远考虑！<br/><p
