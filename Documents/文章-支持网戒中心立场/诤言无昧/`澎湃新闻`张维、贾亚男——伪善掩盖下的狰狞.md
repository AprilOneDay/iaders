/><br/><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？（之三）<br/><br/><!--more--><br/><br/>2017年2月8日，澎湃新闻发表了一篇题为《无法告别的临沂网戒中心：杨叔， 盟友和家委会》的文章，文章看似站在人权的制高点上，对临沂网戒中心的种种“暴行”进行揭露和炮轰，命运多舛的临沂网戒中心再一次被拖进了舆论的漩涡，不明真相者认为临沂网戒中心就是一个活脱脱的人间地狱，是残害孩子的集中营。<br/><br/>可是，事实真是那样的吗？答案是否定的。我儿子是典型的网瘾孩子，在临沂网戒中心经过10个月的矫治，早已脱胎换骨，改掉了所有的恶习，成为了一个优秀的孩子。作为全程陪护的家长，我对临沂网戒中心了如指掌。那里氛围温馨和谐，大家互帮互助，心理点评师循循善诱，到处充满正能量，根本不存在虐待孩子的情况。那里是网瘾孩子的重生之地，是患难家庭的幸福之源！<br/><br/>澎湃新闻的文章纯属子虚乌有、胡编乱造，存在两方面的严重问题：<br/><br/>一是刻意忽略数千例成功改变好的网瘾孩子，片面采信几个未改变好的网瘾孩子的一面之词，就断章取义，歪曲事实真相，并大量粘贴使用网上的虚假炒作内容，严重违背新闻职业操守；<br/><br/>二是该文章看似关心保护未成年人，实则是毁了这些问题孩子和家庭！<br/><br/>一、作者未进行实地调查，文章内容或粘贴或虚构，完全失实并带有明显的主观倾向性。<br/><br/>2017年2月6日，澎湃新闻的记者张维、贾亚男虽然到过临沂市第四人民医院，但是并未走进网戒中心进行调查了解，也未采访过医护人员、在院家长和孩子。只是在医院门口采访了一个卖饭的老太太，就公然违背新闻职业操守和社会公德，从网上的虚假炒作中大量复制粘贴，有些竟然是2009年的炒作内容，加上他们丰富的想象和按照预设观点选取了16年8月份的采访片段，以移花接木、扭曲夸张的方式，别有用心地炮制出了《无法告别的临沂网戒中心：杨叔， 盟友和家委会》这篇文章，企图再次掀起风浪，进一步打压网瘾戒治机构的生存空间。该文章内容严重失实、漏洞百出，我仅从中列举几点：<br/><br/><b>1</b><b>、关于“电击”。</b><br/><br/>文章的着力点还是“电击”，所谓“电击”，其实是国家允许使用的低频脉冲治疗，是在医学背景和监管部门的许可下针对特殊患者采用的专业治疗手段（2016年8月18日临沂市卫生计生委就此发布官方回应，低频脉冲治疗合规合法），而作者却大肆渲染临沂网戒中心如何变着花样使用该治疗摧残孩子。作者明知低频脉冲治疗就是一种简单的医学治疗手段，为什么偏偏要使用“电击”这样的字眼并极尽夸张歪曲之能事？其真正用意是什么？难道仅仅是为了博人眼球、赚取点击率来提高记者本人的新闻效应吗？若只为这点，岂不太丧尽天良了吗？视职业操守如弃履，为一己小利，毁掉千千万万的问题孩子和家庭，就不怕遭天谴吗？<br/><br/><b>2</b><b>、关于点评课中间不许去厕所。</b><br/><br/>作者在文中说，点评课从上午九点上到下午三、四点钟，中间不许吃饭，不许去厕所。佩服作者的想象力，假设真如作者所云，一堂课足足上了六、七个小时，中间不许上厕所，那点评课堂上还不得尿流成河，臭气熏天！全程陪护的家长能否愿意？当你渲染六、七个小时不让孩子如厕、不许大家吃饭来攻击杨永信的非人道时，那杨永信在同样长的时间里有没有如厕？有没有吃饭？你为什么不报道杨永信的敬业精神呢？<br/><br/><b>3</b><b>、关于鼓掌超时也会被电，做一次治疗，都要单独收费。</b><br/><br/>作者说，除了睡觉，任何一个小事情做不好都会加圈，鼓掌超时，也会被电，照这样被电的标准，100多个盟友，每个人每天会被电多次，网戒中心就几个医生，光这项就忙不过来了。再说如果真是这样，那盟友们岂不是人人自危，孩子一天被电多次，全程陪护的家长会愿意吗？临沂四院是公立医院，网戒中心是其下属科室，每次治疗都要单独收费的话，公立医院的收费就那么随意吗？物价、审计、财政部门的监管是虚设的吗？<br/><br/><b>4</b><b>、关于点评课上盟友被要求挺直腰杆，一动不动，否则会被加圈或者电疗。</b><br/><br/>作者一点常识不懂，有谁能做到几个小时挺直腰杆，一动不动？国有国法，校有校规，没有规矩不成方圆。这些网瘾孩子的行为正常吗？他们多数是偏执叛逆、焦虑暴躁的。这些孩子聚在一起，如果管理松散，岂不要乱成一锅粥了！对这些自控力差，缺乏明辨是非能力的问题孩子，管理严一点既是安全的需要也有利于培养良好的习惯。作者拿这点说事，可以说是实在找不到什么可以攻击的了！就把这当做一条罪状了。<br/><br/>二、作者打着保护未成年人的幌子，却在做对未成年人釜底抽薪的勾当。<br/><br/>作者抓住子虚乌有的“电击”为噱头，大肆渲染网戒中心对盟友的残害程度，看起来作者好像是拯救青少年的爱心大使，可是稍加分析，就不难看出，作者是在为网游开发商扫除障碍，是在充当网游开发商的马前卒。因为网戒机构是挽救网瘾少年的最后希望，网戒机构没有了，网瘾少年就可以继续沉迷网游，成为网游开发商的摇钱树，为他们提供不竭的财源。<br/><br/><b>1</b><b>、作者刻意绕开未成年人网络保护的核心内容，去攻击挽救成瘾孩子的网戒机构。</b><br/><br/>《未成年人网络保护条例(送审稿)》明确指出，开展未成年人网络保护主要内容有两条：一是规范净化网络市场，二是开展宣传教育和干预矫治。作者如果是真的想保护未成年人的话，应该去为净化网络市场摇旗呐喊，去曝光那些不良网络游戏产品和黑心网吧以及网络市场监管机构；或者是曝光网络游戏对未成年人的种种危害，为宣传教育助力啊！净化市场和宣传教育是事前预防，对待预防不了的成瘾少年，就要开展矫治了，如果不去矫治，这些网瘾孩子将是社会的不安定因素。而网戒机构是挽救网瘾孩子的最后阵地，临沂网戒中心是目前国内最规范最科学有效的网戒机构。成立11年来成功挽救6000多名网瘾孩子，为家庭、社会、国家做出了突出的贡献，是网戒领域的领头雁。作者你一不去为预防青少年沉迷网瘾尽心尽力，二不去为挽救成瘾人群献计献策，却在抹黑、攻击挽救网瘾孩子的医疗机构。你这是在断网瘾孩子的后路啊！网戒机构没有了，网瘾孩子就无处可医，就会成为网络游戏的铁杆粉丝。事情到此已真相大白，作者打着保护的幌子，实则在干釜底抽薪的勾当。<br/><br/><b>2</b><b>、作者为达到否定临沂网戒中心的目的，倾力塑造网瘾孩子“正常化”形象，刻意忽略网瘾孩子的恶劣表现和家长备受煎熬生不如死的痛苦。</b><br/><br/>作者对网瘾孩子在家的表现只字未提。送孩子来网戒中心是父母的决定，哪个做父母的不到万不得已、走投无路之际，会痛下决心、抛家舍业地把孩子送过来？以我儿子为例，简单介绍一下儿子沉迷网瘾的表现及父母的痛苦：<br/><br/>小学毕业，我儿子以全校13名的成绩升入初中。初一上学期，儿子开始沉迷网游，成绩急剧下降，勉强上了高中，高一上学期就逃课去网吧。后来就不再去教室，泡在网吧十多天不回家，偶尔回家一次也是洗澡、换衣服、要钱后接着走人，浑身上下散发着烟臭味。我去网吧找他，晚上他坐在电脑前，戴着耳麦，夹着烟，抖着腿，满嘴脏话；早上多数趴在电脑桌上睡着了。这时的儿子，网吧就是他的家，网游就是他的全部。玩游戏要想升级快，就要购买装备。三年来，儿子玩游戏花了6万多元，没钱就问我要，因担心他铤而走险去偷去抢，我每次都会不情愿的把钱给了他，其中有3次是4000元的，其他像2000元、1000元、几百元的不计其数。一次夜间偷刷我的透支卡3600元，半月后我才发现。苹果手机先后买了4部，只要是新款就得买。儿子走偏这几年，我在他身上花了10多万元。没有钱就用透支卡提现，由于不能如期归还，我的个人征信有7次不良记录。<br/><br/>在网吧，儿子结交损友，抽烟喝酒打牌，有时打群架。和损友去人家里打架，晚上往人院子里扔烟花。更为可怕的是，他的朋友圈里有两个吸毒的，幸好他还没有沾染毒品。2014年5月的一天晚上，儿子酒后跑到一中教学楼上闹事，打了4个老师，造成了恶劣影响。见到他时，他正趴在派出所的院子里不省人事，吐得满身都是。这一闹，我又是登门给老师赔礼又是往返于派出所协调关系，折腾了几天，才平息下去，搞得我颜面扫地。眼看着儿子这样颓废下去，我心急如焚又毫无办法，和老婆是三天一小吵，五天一大吵，那日子过得是生不如死。<br/><br/>沉迷网瘾的儿子逃学厌学、结交损友、贪图享乐、抽烟喝酒、打架斗殴、早恋同居、消极颓废、淡漠亲情，活脱脱一个行尸走肉，时刻游走在犯罪的边缘。照此下去，等待他的不是监狱就是地狱。万般无奈之下，我和老婆把他送到了临沂网戒中心。<br/><br/><b>?3</b><b>、作者刻意忽略临沂网戒中心11年来的巨大成就，却只拿2个未改变好的网瘾孩子说事。</b><br/><br/>临沂网戒中心是公办机构，隶属于临沂市第四人民医院。11年来，中心创始人杨永信院长呕心沥血，不断探索，总结出一整套网瘾戒治模式，科学有效，国际领先，先后有6000多名网瘾孩子在这里成功改变，回到了正确的人生之路。为家庭和谐、社会稳定、民族复兴做出了巨大贡献！<b>这里是集医院、学校、军营、家庭于一体，集网络成瘾戒治、性格缺陷矫治、问题家庭诊疗等功能于一身的综合性救治机构。家长全程陪护。实践中总结出的</b><b>108</b><b>个治疗环节环环相扣，相辅相成。</b>每天上午的大点评课是治疗的中心环节，主要对网瘾孩子进行传统文化教育、社会主义核心价值观教育、孝悌教育等。对孩子身上存在的问题进行深层次剖析，帮助他们树立正确的三观，建立规则意识和后果意识，强大他们的内心，变得阳光自信、担当进取、亲情回归，同时对家长身上存在的问题进行点评，纠正家长在认知和教养上的偏差。<br/><br/><b>――可以说，临沂网戒中心是我知道的世界上最系统、最全面的教育孩子如何做一个好人的地方，从这里走出去的孩子大多比常人还要优秀。</b><br/><br/>我儿子经过10个月的改变，变得阳光自信，亲情回归，规则意识和后果意识都很强，树立了正确的三观，彻底摆脱了网瘾的诱惑。现在的他，亲情好有礼貌；内务卫生做的好；学习动力十足；和父母沟通交流好，做事沉着冷静；注重体育锻炼；和同学友好相处；能够明辨是非，远离了过去的损友。<br/><br/>临沂网戒中心不断演绎着魔鬼变天使的奇迹，一个个骨瘦如柴的网瘾少年在这里脱胎换骨、破茧成蝶，成为学校的佼佼者；一个个支离破碎的家庭在这里找回亲情、重拾幸福。澎湃新闻的记者不去报道这些，却选择性的采访两个因父母不坚持而没改变好的问题孩子。什么事情都没有百分之百，医生也不能包治百病，6000多名网瘾孩子有那么几个不成功的也属正常。<br/><br/><b>记者张维、贾亚男拿着放大镜找问题，带有明显的观点预设和主观臆断痕迹，心态和出发点都有问题。</b><br/><br/>滴水之恩，当涌泉相报。杨永信院长拯救了我的儿子，改变了我家庭的命运，我们永远感谢他，感谢临沂网戒中心这方热土，这里是我们命运的转折点，是孩子的重生地，是我们的命运之根、幸福之源！<br/><br/><b>澎湃新闻公开发表不良记者张维、贾亚男的虚假文章，打着保护未成年人的旗号，对临沂网戒中心进行抹黑、诋毁，妄图葬送孩子的前程，断掉我们的活路，毁灭我们的最后一线希望，貌似善良，实则狰狞，此无耻行径引起了受益于网戒中心的全体家长和孩子的公愤，以“澎湃新闻”张维贾亚男为代表的恶媒谣记终将被送上道德的审判台。</b><br/>首次转载时间:
