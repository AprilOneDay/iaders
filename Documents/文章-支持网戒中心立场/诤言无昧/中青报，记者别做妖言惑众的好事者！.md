class="WB_editor_iframe"><br/><br/>??大家好！我是山东临沂网戒中心的一位盟友，我想跟大家谈一下对于中国青年报发表的题为《不称职的家长比电击治网瘾的杨永信更可怕》这篇文章的看法。<br/><br/>此文乍看角度客观，说的像真的一样，但实际上通篇都是对中心的恶意诋毁和对杨医生及中心家长的人身攻击。首先我先说一下文章中的可笑之处，那就是文章中多次提到“孩子们在中心受尽虐待”，可是如果中心真如文中写得那么可怕，那么惨无人道的话，那我只想问问中心为什么没有关门歇业？在社会的关注和媒体的聚焦中，为什么杨医生依然在兢兢业业地做着拯救问题少年和患难家庭的事业？并得到上级主管机构的认可，且卫计委也已给出公正说法，难道说卫计委的检查和声明还不如一个一心想当网红的“未消逝的青春2015”？<br/><br/>正是因为临沂网戒中心的正义与道义，所以杨医生才能继续开展他治病救人的事业，而且中心几经炒作依旧长存，就足以证明杨医生所做的事业是一件利国利民的好事。<br/><br/>我本人正在中心，中心都是靠着积极向上的心理教育让我们逐渐从网梦中醒来，并不依靠文章中多次提起的“虐待”。更令我感到生气的是，这些炒作临沂网戒中心的人竟然把杨医生比作商人，说杨医生是为了“敛财”才收治的问题孩子，我真的为杨医生感到气愤！如果杨医生仅仅为了赚钱而去帮助我们这些问题孩子的话，那么现在面临炒作，杨医生完全可以全身而退，选择退休或者转行。但为何杨医生依旧坚持？就算被外面很多人误解依旧不放弃，这一切都是因为杨医生高度的责任心以及他对我们这些问题孩子和家长的同情心！<br/><br/>我对于这些恶意炒作中心的人恨之入骨，无法用言语形容他们的无耻。<br/><br/>之后文章中还提到了“残害”我们这些孩子的人除了杨医生以外，还有我们的家长。我觉得十分可笑，用脚指头想想都知道，这个世界上哪里会有父母残害自己的孩子的？只有那些恶意炒作中心的人才能编出这样的鬼话来。文中还说孩子们在中心过的生活如同噩梦一般，并且说我们在中心改变好之后虽然孝顺、听话、老实，但是这全是被中心逼出来的心理病变。我真的无法理解这些人为何如此会编，如此无耻。就拿我自己来说，我在来中心之前就是一个活生生的魔鬼，那时候的我早已辍学在家，整天游手好闲，在家中除了玩手机就是玩电脑，并且还经常出去上网吧通宵，每次父母阻止自己，自己要不就是骂父母、打父母，要么就是砸家里东西，那时的自己仅仅十七岁，但是抽烟、酗酒、早恋我都干过，已经算得上学校里的渣滓了。最严重的一次我因一点小事持刀伤人，差一点将人捅死，直到现在我依旧十分庆幸我能够被父母送来中心，才救回了我的这条小命。我深深的明白自己为什么能活到现在，因为杨医生，因为中心，是他们将我从死亡边缘拉了回来，如果父母不及时将我送到中心，我早就已经在监狱里了。<br/><br/>也正是多亏了父母对我的不放弃，我才能来到临沂网戒中心改变，有了一次重生的机会。<br/><br/>因为有了杨医生和点评师们的教诲，我逐渐懂得了是与非。现在的自己逐渐从一个无知、偏执、幼稚、自私冷漠的极端问题少年变成了一个懂事、听话、诚实、孝顺的阳光男孩，我没有受人逼迫，因为是非曲直我自能判断！在来中心之前，我每天都过得十分空虚，那种感觉如同被掏空了灵魂一样，整日沉浸在虚拟的世界里，那时我甚至想过自杀，因为自己对于未来真的很迷茫。但是来到中心之后，我有了生活下去的勇气和信心，我有了中心赋予我的正能量的支撑，在中心每天都过得十分充实，每一天都能收获到很多不同的东西，每一天我都过得十分开心。<br/><br/>至于文章中提到的“噩梦”一说，我只能用四个字表示：纯属胡扯！<br/><br/>对于现在的我而言，孰好孰坏我清楚得很，所以说我对中青报的这篇文章要作此评判，不为别的，只是希望不明真相的人千万别被所谓的“媒体人”迷惑！<br/><p
