class="Custom_UnionStyle"><br/><br/>　　大家好，我是高宁，戒网中心的第一位盟友，不知不觉，网戒中心已经成立了十年了，而我，也从一个曾经迷途的羔羊，变成了现在家里的男子汉，顶梁柱。<br/><br/>而以前的我，在小区里，见到了我的父母，我不会和他们说话，在家里，他们吃饭我睡觉，他们睡觉我吃饭，就像陌生人一样。<br/><br/>而在外面，我结交了社会上一些人员，打架斗殴，抽烟酗酒，还经常的夜不归宿。打架打厉害的时候甚至都差点出了人命，当时家里真的就像是天塌了一样，爸爸整天唉声叹气，妈妈以泪洗面，整晚整晚的睡不着觉，甚至听到我一上楼的声音，妈妈就会跑肚子！而最疼爱我的爷爷也差不多都快放弃了，说我这样的，已经没人能管我了，除了进监狱，没有别的方法了，而我自己本人也是抱着破罐子破摔的想法，学也不上了，天天在网吧里待着，没事就出去欺负欺负别人！<br/><br/>记得2005年的11月，我和我的爷爷和妈妈一起来到了网戒中心，第一次见到了杨爸爸，当时我其实只是抱着一种去玩的心理来到了这里，家里人也是死马当活马医，因为家里为了我已经想尽了所有的办法，而我和家里的关系也是非常的僵。杨爸爸在向我和我的家人了解了一些情况以后，说需要住院治疗，当时我不以为然，认为不就是不想上学了，喜欢上网吗？至于住院吗？医生没有一个好东西，肯定是为了奖金，才让我住院的，无所谓，住就住，有什么大不了的！就这样我在网戒中心住了下来，当时杨爸爸为了照顾爷爷和妈妈不用天天来回跑，让霞姐帮忙照顾我！第一天就这也过去了，我也在想这个庸医，让我住下什么都不敢就算了吗？结果到了第二天，杨爸爸让我把头发染回来，并且耳钉拿下来，当时我一听可不乐意了，这可是哥以后闯荡江湖的标志，你说给我换掉就换掉，那以后我可怎么有脸混啊，不换，坚决不换，当时杨爸爸没有说什么，一直到了第三天，又要求我换，我就火了，这个院我不住了，我来是让你看病的，又不是听你指挥我干什么的，我不住了。杨爸爸和医护人员拦住我的时候，我很霸气的说，杨永信，有本事你今天把我制服，不然等老子出去以后，你看我不找人弄死你。然后对医护人员说，今天你们谁拦着我了，我都一个一个记住了，别让我出去，出去了有你们好看！后来的情况，我就不说了，说说都是眼泪啊！哥被制服了，服服帖帖的，当然，这只是表面上的服，心里还是想着其他的办法呢！<br/><br/>但是慢慢的，和杨爸爸每天接触，杨爸爸用他那父爱般的温暖，在一点点的感化着我，我开始转变了，我知道了家人的不容易，知道了自己以前是有多么的可恶，也知道了以后自己的正确的人生价值观，我在那时候就决定，一定要走回来，远离网魔！<br/><br/>在06年元旦的这一天，我们网戒中心终于挂牌成立了，也有越来越多的盟友加入到了我们的这个大家庭里面，杨爸爸也越来越忙了，有时候半夜起来去厕所，都会看到杨爸爸在办公室忙碌的身影，那时候自己就想，杨爸爸这么辛苦，而我作为组长，是不是应该帮他做一些什么事情，当我把想法告诉杨爸爸的时候，杨爸爸只是笑了笑，然后告诉我，只要你们能不再走偏，能在以后好好的在社会上做人，那么我再辛苦也是值得的，当时听到这句话，我真的哭了，为以前的我哭，哭自己为什么那么不懂事，为现在的我哭，哭我终于又回来了，做回阳光的我，可以和家人好好的沟通，可以在社会上做一个好人了！<br/><br/>在网戒中心的那一段时间，杨爸爸并不只是让我戒掉网瘾，还让我真正的明白了自己应该怎样做人做事，在这里，真的感谢杨爸爸让我获得了第二次生命！<br/><br/>我和杨爸爸一起学习普通话，一起到外面宣传，做公益活动，而这些事情无时无刻不在警醒着我要学会做人，并且还要学会帮助别人！亲爱的杨爸爸，如果没有你，我的家庭可能会因为我而破裂，也会有更多的家庭会和我们家一样！所以，杨爸爸，谢谢您，谢谢你舍小家，为了这么多的家庭，谢谢您对我们的宽容，谢谢您没有用有色眼镜看我们，才让我们这些已经出院的盟友可以在社会上作一个对社会上有用的人，让现在的盟友家庭可以看到希望！<br/><br/>在出院以后，我又重新回到了学校，因为以前拉下的太多了，所以成绩一只跟不上去，但是我一想到不能让杨爸爸失望，便咬牙努力追赶，最后虽然中考成绩不理想，但是我没有气馁，上来一所中专院校学习民航服务专业，凭着自己的努力，在一次招聘考试中考了全班前三，民航英语满分，日语韩语98分，航空法，运输规则满分，于07年被南京禄口国际机场录取，成为了一名安检人员，当得到这个消息的时候，我的家人很高兴，爸爸和爷爷，两个很倔强的男人，在那一刻都留下了眼泪，我知道，这不是失望的眼泪，是高兴激动的眼泪，而这些，都是杨爸爸给予的！第一时间，爷爷和妈妈带着我回到了中心，把这个好消息和大家分享，爷爷的话我到现在还记得，如果我的孙子一年治不好，就住一年，一辈子治不好，就治一辈子！杨爸爸也很高兴我能取得这样的成绩，杨爸爸说，我终于可以放心了，因为看到高宁这样，我相信以后会有更多的盟友也像我一样，重新获得新生！<br/><br/>12年，我在南京机场辞职回到了家乡，因为爷爷的身体不好，在大年初四这天爷爷走了。在弥留之际爷爷告诉我，有空了，一定要去中心看看杨医生，给他说一声谢谢，因为他救了我的孙子！<br/><br/>在去年的3月2号，我结婚了，因为很多原因，所以没有办法来到中心报喜，而我现在是一名物业人员，工作还可以，今年不出意外的话，我就要做爸爸了，在我的肩膀上，有了更大的责任！那就是养家！<br/><br/>在这中心成立十周年之际，真的很高兴！衷心的祝愿杨爸爸，医护人员，兰姐，霞姐身体健康，工作顺利，也祝愿亲爱的盟友们能早日的康复，不让家人再为我们受累了！<br/><br/></div><br/></div><br/><div
