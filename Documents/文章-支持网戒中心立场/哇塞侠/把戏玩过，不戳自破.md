/><br/>――谈谈“无明之痴”笔下的“小天使”<br/><!--more--><br/><br/>最近经常被网友“无明之痴”@，看了他的几篇文章，感觉他像一位正义的记者，要为那些被杨永信“摧残”的孩子们伸张正义，也颇有些文采，所以默默关注了他，期待他能带给我们不一样的“真相”。可是读完了下面的这篇文章，心中产生了些许疑惑，在此提出，希望“无明之痴”指点一二。<br/><br/>“以下内容是来自于一个真真正正被杨永信送到临沂四院精神科折磨的小天使，一个很善良很单纯的小天使。他因为服用了大量精神类药物引起了身体一系列不良反应，出院后的现在疾病缠身，而他，不是唯一一个被送去精神科折磨的‘盟友’，他只是其中一个。虽然他的身体状况仍然不太好，但意识还是很坚定。”<br/><br/>开头第一句话就让我心生疑惑。“被杨永信送到临沂四院精神科折磨的小天使”，这句话实在难以理解，在我们常人的认知里，病人往往都是由自己的家属送往医院的，而这位作者口中的“小天使”竟然是被杨永信，一个外人送到精神科的，这符合逻辑么？<br/><br/>第一个疑问：杨永信与这位“小天使”有亲属关系吗？他可以行使属于父母的权利吗？很显然不是，他没有这个权利，医院的规章制度也不会允许这样的无厘头事情发生。<br/><br/>第二个疑问：就算杨永信无视医院制度的存在，无视他父母的存在，自作主张把“小天使”送进了精神科，但住院治疗是要花钱的，杨永信会脑残到为了折磨这个“小天使”，不惜为他交住院费？<br/><br/>第三个疑问：如果杨永信真像作者所说，要“折磨”这个“小天使”的话，那放在杨永信自己的科室里不是更能为所欲为，为何要把自己的“折磨”对象送去别的科室？于情于理，这种逻辑站得住脚么？<br/><br/>“他委托我发出他的经历，并寻求广大媒体的帮助，他十分渴望接受采访揭露杨永信的行径和透露更多细节，这是他目前，唯一活下去的希望！希望善良富有爱心的广大媒体，请帮帮这个孩子！”<br/><br/>这一段更加有趣。这位“小天使”在杨永信那遭受了如此大的“委屈”，出院之后首先想到的竟然是发微博？实在难以理解这位“小天使”的动机，按照正常人的逻辑，既然已经遭受了如此“非人的待遇”，重获自由后的第一件事难道不应该是报警走司法程序吗？他的父母或监护人究竟在哪里？更耐人寻味的是，一个已经摆脱杨永信“魔爪”的受害者不走司法程序，获得自由了还在微博上喊冤求救，这个所谓的“小天使”究竟有何居心？看来在“小天使”的心目中，这个社会已经没有司法正义，只有“无明之痴”可以伸张正义，简直是21世纪的包公再世。<br/><br/>“09年我已经去过一次网戒中心，那次出院以后我‘改好’了。16年，我又再次被父母送到网戒中心。我自认为，我不是一个问题孩子。只是在成长过程中，遇到一些问题，导致自己很自卑。然后慢慢地，不愿意接触人群甚至恐惧人群。也就是所谓的社交恐惧症吧，可我并不认为这是我的过错，而是我的成长环境和家庭教育因素导致的，而且我本来性格就内向。”<br/><br/>“小天使”自认为“我不是一个问题孩子”。但在医院里，能以患者说自己没有问题或没有病来判断是否有病？有些精神病人说自己是总统，他就是总统？说他是上帝，他就是上帝？说你害了他，你会承认自己真的害了他？况且，根据“小天使”自己所描述的内容：“不愿意接触人群甚至恐惧人群”来看，“无明之痴”你认为他一点问题没有么？无论原因是什么，问题的存在是客观事实。如果问题不严重到一定的程度，父母会反复把他送给杨永信？杨永信有什么能耐把一个没有问题的孩子，在父母不同意的情况下，强行“抓”到自己的科室里？<br/><br/>“也许是父母不理解我导致我心理扭曲吧。第一次把我送到网戒中心之后，我被迫‘改好’上学了，但还是不能够和人正常相处，很孤僻。后来因为家庭变故，我又辍学了，走向社会，找的工作也不顺心，我一如既往也不合群。后来我辞职在家里呆着，不想出门。可我也很想摆脱现状，不想过这样浑浑噩噩的生活。所以我变得很急躁，还在家里砸了些东西。父母很生气，又在今年2016年再次把我送到了网戒中心。”<br/><br/>“小天使”说：“在网戒中心‘被迫’改好上学了”。自己能回去上学是因为被网戒中心逼迫的？在这里我很想跟大家探讨一个问题：在一个孩子不想完成学业的时候，有人帮助他完成学业到底是一件好事还是坏事？我们见过太多的孩子，十几岁的年纪因为种种原因或者种种诱惑放弃了学业，在需要学习汲取知识的年纪匆忙踏入社会，非主流、杀马特成了这些孩子的标签，网管、保安、服务生成了这些孩子的职业，甚至还有些打架斗殴、偷摸抢骗、吃喝嫖赌混社会的呢？也许第一次见到我们可能还会感到痛心，但是现在呢？你也可以说生死由命富贵在天，每个人有每个人的活法，这话不假，但是当初如果有人愿意帮助他，他的父母愿意阻拦他，让他年轻浮躁的心多一些冷静，让他不切实际的梦想多一些思考，这难道不是一件好事么？可悲的是，这位“小天使”时至今日尚不认为上学是件好事，而是“被迫”的。<br/><br/>一个“小天使”，天天宅在家里，浑浑噩噩，摔摔砸砸，你见过这样的天使吗？如果“无明之痴”喜欢这样的天使，你何不把他领回家让他天天给你耍宝玩可爱呢？“天使”高兴，你也开心，顺便彰显你的博爱，岂不美哉？<br/><br/>“杨永信一看我这么冲动不利于保持里面的氛围，就自作主张把我转进了十楼的精神科（因为他们说了把我送精神科是因为他们没有约束的办法）。在里面，医生没有和我交谈关于治疗的相关事宜，就给我用很多狂躁症的药物。我记得护士给我打针的针管很粗，每天也要吃很多药，唯一的自由活动时间，也就是和一堆真正得了精神病的人一起，聚在一个房间看电视，但我不喜欢看电视，很无聊，我只想见我的妈妈，可精神科的医生护士都跟我说：治疗阶段不可以见妈妈。所以我一直一个人呆在十楼，而妈妈却在楼下网戒中心里……我感到和妈妈很近，可我却不能和她在一起，我很孤独，很难过。可我不能有任何情绪激动的表现，比如哭，因为我害怕被‘治疗’……很害怕，我只能夜深人静的时候悄悄哭，我想妈妈，我想回家。”<br/><br/>“我的身体受到了很严重的伤害。在精神科里的十三天，就像一场噩梦一样……我出现了焦虑症症状，坐着躺着站着都很难受，不知道怎么样才好。十三天后出来，都需要人扶着，且头重脚轻。”<br/><br/>“从出院之后起，我的身体也发生了很大变化：开始是失眠、眼神木讷、体力不足、记忆力很差、还有忘记了很多人的名字，后来又开始持续发烧，还得了肺炎，肠道也很不好吃得很多也不排便。”<br/><br/>“我出院后有半个月一直住院，现在高烧是退了，但是肺炎一直不见好。我这段时间一直想安乐死，因为我不想因为我身上的病而死去。几乎每天都想死的事情，我甚至以为这就是对我痛苦的解脱……而这些，都是杨永信给我带来的。”<br/><br/>以上文字很长，作者的描述中有很多矛盾且令人疑惑的地方――<br/><br/>1.是杨永信把作者转进了精神科？<br/><br/>这个问题上面已经说过了，不合逻辑。如果属实，作者完全可以走司法程序。<br/><br/>2.作者转入精神科后，作者的妈妈没有陪护而是留在了网戒中心？<br/><br/>病人在哪个科室家属不是应该跟到哪个科室陪护吗（有些科室规定不需要陪护的例外）？杨永信都把你转走了，就像你说的，他已经拿你没办法了，还有什么理由继续把你妈妈留在网戒中心？<br/><br/>3.护士给你打针的针管很粗？<br/><br/>针管的大小或粗细是出于治疗的需要，难道这根针管是为了折磨你本人而私人定制的吗？<br/><br/>4、“小天使”把自己说得那么可怜，真的挺令人同情，但仔细想来，你若和同龄人一样好好上学做事，快乐的生活工作，你父母会花钱花时间把你送进精神病院让你受罪？<br/><br/>5、如果说，到了精神科，也依然遭受了非人的待遇或折磨，为什么独独是你“小天使”？其他的患者会是怎样的体验？你若认识不到自己的问题，你可以走遍全国所有的精神病院，找你所能找到的所有精神科医生，看他们能给你做出什么样的诊断，采取什么样的治疗。<br/><br/>6、你又是失眠、眼神木讷、体力不足、记忆力很差，又是发烧、肺炎、肠胃病、吃得很多也排不出便来，你还有多少事能赖上杨永信？难道你将来遇到的所有倒霉事都是杨永信的罪过吗？<br/><br/><br/>同情心难能可贵。在这位“小天使”的文章下面，我看到了诸多富有同情心的网友，同情“小天使”的同时，还对杨永信的“恶魔”行径进行了猛烈的抨击。不过我很想提醒大家，同情心要有，但是不要被同情心蒙蔽了辨别是非的双眼。真可怜需要真同情，装可怜可就要遭人白眼了。而装出来的同情究竟是何居心？就更加耐人寻味了。<br/>首次转载时间:
