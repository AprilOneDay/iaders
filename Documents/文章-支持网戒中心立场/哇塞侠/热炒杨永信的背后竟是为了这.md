熙熙攘攘的声讨之中，到底有没有“托”？<br/><br/>这家医院因为一件没有责任的事情差点担上了责任，如果没有后期的深入报导揭露真相，我们可以想象这家医院的下场。最终真相大白后才知道：原来还有莆田系作乱，原来还有“大姑姐”这种高大上的新兴职业，原来这背后的关系错综复杂。不禁引人深思：热炒杨永信的背后是否也有类似的“大姑姐”存在呢？这个话题我们以后专门谈。<br/><br/>总而言之，有人想让网戒中心关门大吉。<br/><br/>上面已经分析过了，对于整个中国游戏市场而言，网戒中心的存在不足为惧，可以说是蚍蜉撼树，但是仍然要热炒杨永信，甚至达到让网戒中心关门的目的究竟为何？<br/><br/>两个字――网瘾。<br/><br/>小小的网戒中心不足为惧，但是其所倡导的“网瘾论”却是能引起游戏市场足够重视的大敌。目前政府相关部门已经准备介入此事，征求意见稿也已出台，但其中牵扯面太多，教育部门、卫生部门等单位，还会牵扯市场经济，不能仓促，所以至今悬而未定。但是如果网戒中心关门了，支持“网瘾论”的声音没了，你们猜谁会受益？<br/><br/>这么多年来，与“网瘾”沾边的机构在媒体们一次次的“曝光”下纷纷倒下了。对于网瘾少年，跑步锻炼可以被说成过度体罚，上课说教可以被说成强行洗脑，保护约束可以被说成无视人权，就连统一着装都可以被说成束缚个性。时至今日，尚能被人叫出名字的网瘾机构还有几家？在不知不觉中，媒体们利用人们的同情和怜悯已经悄无声息地攻陷了几乎所有网瘾机构的阵地，而媒体是掌握在网戒中心手中，还是掌握在谁的手中呢？况且，同样的照片，配上不同的文字，就可以写出两篇大相径庭的报导，这样的趣事，我们见过的还少么？<br/><br/>排除异己，独吃自屙，这是第二个目的。<br/><br/>吸粉捞金也好，排除异己也罢，我们左右不了旁人。但如此热炒杨永信最终到底会成就谁呢？我们拭目以待。<br/><br/><br/>下期小作文：哇塞侠与你共同探讨新兴神秘职业――“大姑姐”<br/>首次转载时间:
