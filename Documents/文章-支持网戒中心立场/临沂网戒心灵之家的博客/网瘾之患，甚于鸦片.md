/>?首先声明，我和此期相关节目的当事人均无任何关系，纯粹站在个人观点：当前年、去年两年中央电视台节目曾经长篇累牍的播出临沂市网戒中心和杨永信的故事的时候，我就知道会有今天的结果，因为杨永信触动了太多人的利益，他早已成为了别人的眼中钉肉中刺，中央电视台只不过捅开了这层纸而已。<br/><br/><wbr />?<wbr />?<wbr />?8月15日晚，看到了我经常收看的中央电视台的《新闻调查》，更看到了我最喜欢的节目主持人柴静在主持这一期，我想一定又是有大案要案发生，由这样一个睿智、严肃而柔弱有力的知名主持人出面。但看完这一期节目之后，我颇有些失望。是的，这是一期没有充分准备而且观点片面、导向不明的节目，或者说做这期节目的目的只是为了跟风前段时间已经告一段落的“电击治疗网瘾”的炒作，甚至略带点讨好网游玩家包括游戏网络运营商和有偏见的人的意味，也许这并不是柴静的本意，但经过强大的世界第一大舆论机器的剪辑和断章取义，至少在我看来是一期令亲者痛仇者快的节目。<br/><br/><wbr />?<wbr />?<wbr />?为一名法律工作者，我也一直在关注网瘾的事情，因为从前几年我就一直在注意，每天打开电视和电脑，大部分青少年犯罪包括出现各种问题的法律方面的新闻都是和上网成瘾有关，只要一有绑架、强奸等案件发生，不等看介绍我就开玩笑说，看！犯罪嫌疑人肯定在网吧被擒获！事实也几乎都是如此……<br/><br/><wbr />?<wbr />?<wbr />?是网络被家长乃至学校、社会开始妖魔化，网瘾治疗也应运而生，也同样是中央电视台的关于杨永信教授的网瘾治疗的故事不断被搬上银幕。然而风云突变，随着被“魔兽化”了的“魔兽”让数以千万计的玩家无法游戏，那些蠢蠢欲动的“反戒网瘾”势力开始疯狂的反扑，甚至制造了很多新闻，包括陶宏开教授的“网游毒品说”，包括杨永信教授的“电击治疗说”，包括“国外无网瘾说”，大有为网瘾正名之倾向，于是也产生了很多的争议，我所不明白的就是作为一个国家的舆论机器的中央电视台，为什么不能客观真实可信的去想办法澄清一些东西，而也会歪曲事实的盲目跟风？<br/><br/><wbr />?<wbr />?<wbr />?我可以肯定的说，为了达到证明自己所持观点的目的，你们在采访中所表现出的咄咄逼人和设圈诱导、断章取义等现象或手段，就不怕给人以强势和随意操纵民意之嫌吗？被采访对象的话语权在哪里？<br/><br/><wbr />?<wbr />?<wbr />?对于一个社会热点，关系千家万户的事情，是有关注的必要，但有没有同时考虑我们中国的国情呢？柴静你是正义的化身吗？你想要表达的观点就一定是对的吗？你是什么样的权威？<br/><br/><wbr />?<wbr />?<wbr />?首先我个人认为，要真正知道电击的目的和效果才有发言权，而不是任由被一叶障目的个别人“忽悠”。质疑者要知道，你们不是医生，对治疗的情况可以说是“管中窥豹”，只知“电击”，而不知为什么“电击”。如果真的不是出于治疗的目的，对网瘾患者进行“电击”是不能容忍的，甚至是违法犯罪，甚至不应该由中央电视台来采访，而是应该由公安局来抓捕，采访的地点不应该在临沂网戒中心，而是看守所。但是我们看到的事实并非如此，同样是中央电视台对此的报道此前已经有过很多，这种小剂量电流“电击”只是一种治疗手段，使用是有严格限定的，就像服用戒烟或者戒毒的药物，是教育治疗的一部分辅助措施。不然我想大家都看到了那些上网成瘾的孩子都是什么情况，如果任其发展的话，我们的国家，我们的未来，我们的后代……呜呼哀哉，不可想象的可怕！而且在孩子冲动暴躁、伤人毁物时，你难道坐而论道、置之不理？怎么样才能保护别人也保护他自己？单纯的说服教育也许柴静能够做到，但是中国只有一个柴静！无数人不明就里的跟着骂，却忘记了原来的目的，无限被放大的“电击”不能不说是痛恨网瘾治疗的网络既得利益者的胜利！不能不说是外面这个时代的悲哀！<br/><br/><wbr />?<wbr />?<wbr />?再者，网戒中心的制度问题，所谓没有规矩不成方圆，国有国法家有家规，没有制度的约束，一切就会形同虚设，一盘散沙。网戒中心之所以能成功救治这么多孩子，也得益于良好的制度、良好的氛围，就像和平年代却一定要拥有军队一样，这是我们今天幸福生活的前提。网戒中心所有环节均是孩子家长互动，难道三千多家长集体癔症发作来花钱花时间摧残孩子？！<br/><br/><wbr />?<wbr />?<wbr />?其次我要说的是，任何一种新鲜事物的产生都是有一个让民众接受的过程，探索过程甚至会失败和挫折，民众要多一些宽容和理解，作为新闻媒体机构应该发动社会的力量使他尽快成熟和成长，才对我们的社会我们的国家我们的下一代有利，而不是落井下石、恶意攻击。戒网瘾也是如此，要经历很多挫折，戒毒在历史上更是如此。就像鸦片之于中国，林则徐的戒烟运动经受了巨大的阻力，想当年也是全民皆“烟”，连军队都是“双枪”。虎门销烟后甚至当时的满清政府都要对林则徐撤职查办，发配边疆，现在全世界对毒品又是什么态度呢？也许有很多人在质疑拿网瘾和毒品在比较，这个话题在陶宏开教授和游戏玩家的争论中已经有结论，这里不说也罢，而且每天在媒体上出现的各种刑事案例、家庭悲剧已经充分证明了这一点。<br/><br/><wbr
