<br/><p style="text-align: center;">――关于在临沂网戒中心八个月治病的感悟</p><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><br/><br/><wbr />?<wbr />?<wbr />?我的儿子王XX，经过在临沂网戒中心救治8个月后，现已痊愈出院48天，回到了学校，坐稳了学习的板凳，这让我全家无比高兴，这才有心来回顾和总结整个为儿子治病的艰难历程和主要感悟。<br/><br/><wbr />?<wbr />?<wbr />?作为一个家庭，最让人不愿看到的是孩子有病，只要孩子有了病，作为对子女最无私的人――父母来说，那真是不顾一切，不遗余力了，可以说孩子是最大的了。<br/><br/><wbr />?<wbr />?<wbr />?我们家就是这其中的一个家庭，下面把我们为儿子寻找能人、名人和专家教授而亲历过的主要事件和情节进行回顾和总结，其目的在于永远记住，终生感恩！祝儿子和我们全家人今后的人生更加美好！<br/><br/><wbr />?<wbr />?<wbr />?<strong>一：八年苦寻，不堪回首</strong><br/><br/><wbr />?<wbr />?<wbr />?儿子八年前被社会坏青年伤了颈椎和下颌关节后，由于没能及时诊治，虽经多处寻医问药，最后还是留下后遗症，由于常常头脑昏沉，学习效果不佳，生活质量也大大下降。原本学习曾考过全校第一名，平常多在10名以内，老师评语甚至也难以提出缺点的好孩子、好学生，自那以来便常常不尽人意，多少次，多少年是在昏睡、半迷糊状态中度过的，听课、做作业和考试等都是如此，不愉快的生活，可想而知。<br/><br/><wbr />?<wbr />?<wbr />?为解决这种状况，先后三上北京、南下广州、到过邓州、去过莱州。青岛本地能看能治的不用说，全找了，钱花了几十万，心累的老了不少，鞋底磨透了数双・・・・・・全家人总是常常处在无奈无助之中。记得2003年，正是国内闹“非典”，我们和儿子在京住院看病，两次，近半年。北京的有关医院，看了近二十家。但没有从根本上消除病症。后来我们就先在青岛后上北京和济南、南方几个地方去看心理，搞心理咨询。2008年我们又去了北京，在北京曾去拜访咨询过中国社科院、中国科学院、北大、清华、北师大有关咨询机构，还去了几家北京社会上的心理咨询机构，没有达到目的，于是在网上广泛查寻名人、名机构，也未解决，最后便信迷信，找巫婆……，这一切的一切都没解决问题，虽然我一直没有放弃，还在不断地寻找着救命能人，可儿子却受不了了，他不想活了，他说我就还有三天活头，七天必走！爸妈，治我病的人还没生下来呢！别再忙活了，您看，这真的绝望了！<br/><br/><wbr />?<wbr />?<wbr />?可是，俗话说得好：对于坏人，不是不报，是时候未到。而对于好人确是：天无绝人之路！希望，很可能还会有！<br/><br/><wbr />?<wbr />?<wbr />?<strong>二、认识“永信”，希望变现实</strong><br/><br/><wbr />?<wbr />?<wbr />?正在最难过之时，一个偶然机会，让我们认识了杨永信和临沂网戒中心，记得那是2008年的春夏，在青岛听到了“网戒和谐行”来作报告的消息，于是我带领全家三口去现场亲自感受。但当时还不敢太相信，特别是我儿子，他认为他的病不对他们的“口”，他们治不了。我们只好暂时放下，又过了大约3个多月，我这个永不愿放弃儿子的人，又一次想起了杨永信和临沂网戒中心，这正是儿子病情最坏的一段时间，那时，好好的大学他不上了，在家卧床三个多月，不要好，不想活。连请他吃饭、理发和洗澡都常挨冷脸，遭骂声。有好多次推打，闹得家里玻璃破了，称杆断了，手指伤了，身上的伤口刚好，脸上又抓破了……，我跟儿子说，咱去临沂，他不认可，他不干。没法子，我必须亲自到临沂去看个究竟，我于是在2009年2月29日偷偷来到临沂网戒中心，暗访家长和孩子，并来到点评课堂仔细观察，既旁敲又侧击。经过几天考察了解，我下定了决心，在这里打电话，让他妈带儿子来，可由于没有备好资金等，我又回到青岛，直到2009年4月8号才踏上征程，归了正路。而且，还是骗他来的。<br/><br/><wbr />?<wbr />?<wbr />?来到这里，开始一段时间，儿子还是不能真正认可和接受，反正也走不了，只是无奈，儿子做了一段时间的“顺从治疗的表面文章”。但儿子是大学生，经过治疗实践，他由不认识到认识，从不认可模式到完全接受模式。刚来到中心，体力不支，体质差，好长一段时间早操和军姿训练都是护士、家长、班委和盟友等架着、拉着手做的。几个月后，他认识提高了，体力增强了，他对中心、对杨叔、对大氛围坚信不疑，他好起来了，他自信了，他阳光了，他被推选为小室长（还是最佳小室长之一呢），节庆晚会上他自编自演上台演节目了……<br/><br/><wbr />?<wbr />?<wbr />?我们和儿子，从认识杨叔和中心，到离院回去重新上大学，经历了一年之多，光住院治疗调理恢复就用了八个整月。孩子已离院四十八天，状况良好。家人欢喜，家庭幸福，重新开始。<br/><br/><wbr />?<wbr />?<wbr />?<strong>三、现身说法、感悟极深</strong><br/><br/><wbr />?<wbr />?<wbr />?从我们认识、了解，到孩子接受治疗，到孩子离院后能够恢复到病前的好状态，现能坐稳上课，帮做家务，打工济家，充满自信，正常交往……我最深的感受是：找到并认识杨永信（杨叔）这样一个能人已经不容易（八年多时间），但，真正了解、接受“永信模式”，并主动地做到“相信、配合、永不放弃”，就更加更加地不容易啊！为什么这样说？因为第一：首先是用了那么多年（八年多），走了那么多路（有的属于弯路，数千公里），第二：是因为“永信模式”的治疗效果之快、之好是其他人所无法与之比拟的。第三：我们也是经历了一个从不了解、不认可，到完全真心地接受。自觉相信、配合、永不放弃，这是一个逐渐变化的过程，是一件用心想、用心做的“全身心”投入的艰难历程。这很是件“不易之事”。<br/><br/><wbr />?<wbr />?<wbr />?讲良心是中华名族之美德，无论在什么情况下，无论你有什么想不通，有什么不理解和认为有不尽人意之处，“当初，你自己在活不下去，而又没有任何办法的时候，是杨永信网戒中心救了你，给了第二次生命”，这样一个事实，千万不能忘记！这是做人之根本的根本！<br/><br/><wbr />?<wbr />?<wbr
