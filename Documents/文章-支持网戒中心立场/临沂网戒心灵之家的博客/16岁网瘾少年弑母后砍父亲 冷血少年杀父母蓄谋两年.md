rel="noopener">北京娱乐信报</a></p><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><wbr /><br/><br/>13日零时30分，因沉溺于网络与父母发生口角，在杀害母亲、砍伤父亲后潜逃的重大犯罪嫌疑人“涛涛”（化名）被广州越秀警方抓获。<br/><br/>从发案到破案，警方仅用了８个小时。而这个不满１６周岁的少年竟冷漠地表示“我两年前就已经有杀他们的念头了”。他作案的惟一目的就是为了杀死双亲，使自己得到“解脱”。记者第一时间与涛涛“面对面”，并再度走近他的亲人和朋友，希望能从中找出可以避免这起罕见人伦悲剧的点滴。目前，此案仍在进一步审理中。<br/><p align="center"><a href="http://photo.blog.sina.com.cn/showpic.html#blogid=621bbada0100g9nd&amp;url=http://s9.sinaimg.cn/orignal/621bbadat79a9e8e16c58&amp;690" target="_blank" rel="noopener"><img style="margin: 0px; padding: 0px; border: 0px; list-style: none;" title="16岁网瘾少年弑母后砍父亲?&lt;wbr&gt;冷血少年杀父母蓄谋两年" src="http://s9.sinaimg.cn/bmiddle/621bbadat79a9e8e16c58&amp;690" alt="16岁网瘾少年弑母后砍父亲?&lt;wbr&gt;冷血少年杀父母蓄谋两年" /></a></p><br/>　　　　等待涛涛的未来是什么呢，除了法律的惩处，还有就是内心的煎熬，虽然他自己不承认。《羊城晚报》／供图<br/><br/>&nbsp;<br/><br/><wbr />?<wbr />?<wbr />?供述<br/><br/>计划多时<br/><br/>“不杀死他们我就无法发展”<br/><br/>“我两年前就已经有杀他们的念头了”，涛涛面无表情地斜靠在派出所的靠椅上，漫不经心地说出这句话时，在场的人无比的震惊。据警方资料，涛涛是１９９１年１１月出生，今年不满１６周岁。对于这起骇人听闻的命案，他表现得异常冷静。<br/><br/>至于为何要杀父母，他告诉记者，原因有两条：一条是因为父母常年呵斥他，“他们很烦，经常骂我，用很恶毒的词语”，常常让进入青春期的涛涛觉得很没有面子和自尊；而另一条理由是，“父母每天都会管束干涉我的行为，让我觉得被人束缚着，如果不杀死他们，我就无法向前发展”。<br/><br/>&nbsp;<br/><br/>丧心病狂<br/><br/>木棍敲昏母亲再用刀夺命<br/><br/>“那天（１２日）下午起来后，我一直告诉自己不能想太多，一定要等他们两个之中的一个不在的时候才能动手。”涛涛平静地叙说。<br/><br/>于是下午２时５０分许，其趁母亲张某不备，先是用木棍将其打晕在地，随后抓起家里的菜刀将母亲连砍数刀，在确认母亲已经死了之后，又平静地用拖把将家里满屋子的鲜血拖干净，把已经停止呼吸的母亲拖到洗手间内。下午３时许，父亲回来在洗手间发现妻子时，涛涛又将手中的刀砍向了父亲，其父亲躲闪不及，脸上挨了两刀后逃出房门。<br/><br/>行凶后，涛涛躲到了附近楼房楼顶一直到被抓获。<br/><br/>&nbsp;<br/><br/>对话少年<br/><br/>父母从来没有跟我谈过心<br/><br/>“从我被妈妈带到广州那天起，我仿佛就与世界隔离了。每天，我所做的就是出摊卖烧烤、收摊、吃饭、睡觉、干活、再出摊。我感觉到了从未有过的孤独，有时还在被子里偷偷地流泪，想念远在家乡的同学……”在矿泉派出所，落网的涛涛向记者道出了心路历程。<br/><br/>&nbsp;<br/><br/>动完手后啥也不记得了<br/><br/>记者：还记得当时是怎样动手的吗？<br/><br/>涛：动完手后，根本记不起了。我怎么也记不清了，感觉是不是一场梦呢？<br/><br/>记者：这个计划是什么时候开始的，为什么会选择在那天动手呢？<br/><br/>涛：以前有过一两次机会下手。有一次也是已经在妈妈背后举起了木棍，但当时还有一些感情，毕竟是自己的亲生父母。<br/><br/>记者：那这次为什么就下得了手呢？<br/><br/>涛：这次没有想那么多，举起木棍时什么也没有想，整个过程中大脑一片空白。<br/><br/>记者：你那么恨他们吗？以至于非杀不可吗？<br/><br/>涛：不杀了他们，心里的枷锁解不开。我的自由被“监禁”了。<br/><br/>记者：你还会想起你的妈妈吗，如果她在这里，有什么要对她说的吗？<br/><br/>涛：到了这里，还会想到我妈妈，想到她以前的样子，想到她的表情，但是该说的已经都说完了，再没有什么可以说的了。<br/><br/>记者：为什么你这么平静？<br/><br/>涛：真正的我早就死了，在我最大梦想破灭的那天早就死了。<br/><br/>记者：那你最大的梦想是什么？<br/><br/>涛：将我写的小说搬上银幕。<br/><br/>（据了解，涛涛已经写了三部科幻小说）<br/><br/>上网是惟一的娱乐方式<br/><br/>记者：你上网一般做些什么？<br/><br/>涛：玩游戏、聊天。<br/><br/>记者：跟网友聊天吗，跟谁聊得比较多？<br/><br/>涛：其实，一般跟不认识的网友也不怎么聊，主要跟一些认识的朋友（以前在老家的一些朋友）或者堂哥等亲戚。<br/><br/>记者：你在广州有认识的朋友吗？<br/><br/>涛：没有朋友，也没有一个玩得很好的，只能上网。<br/><br/>记者：那每次上网的钱从哪里来的呢，一般一个月要花掉多少钱？<br/><br/>涛：每次从烧烤档放钱的地方拿５元、１０元的，随便拿的，爸妈也知道的。一般一个月大概２００元－３００元。<br/><br/>记者：有在网吧里认识什么朋友吗？<br/><br/>涛：没有，都是自己玩自己的。<br/><br/>来广州后亲情死掉了<br/><br/>记者：你父母是怎样管你的？<br/><br/>涛：主要是限制了我的自由，不让我出去玩。比如一次白天干完活后想出去上网，但是不敢，就对父母说去打个电话，然后爸爸说５分钟回来。<br/><br/>记者：你和父母之间的关系是怎样发展到这种地步的？<br/><br/>涛：来广州之后，亲情关系开始死掉，他们从来都没有跟我谈过心，爸爸只是命令我去干什么，从来不问我自己的意愿。<br/><br/>记：你父亲在你一出生就离开了你，你对他的印象是什么？<br/><br/>涛：我对他没有印象，只是记得他每次回来会给我买东西。<br/><br/>记者：你盼望他回来看你吗？<br/><br/>涛：不希望，和他一点感情都没有，我小时候主要是奶奶和妈妈把我带大的。长大后，和他们根本没有话说，他们只是每天管着我。<br/><br/>&nbsp;<br/><br/>对话父亲<br/><br/>我的教育是完全失败的<br/><br/>１３日下午，记者在某医院重症监护室，见到了涛涛的父亲王建军。<br/><br/>记者：儿子一直和你们生活在一起吗？<br/><br/>王：不是，他才来广州两年，是奶奶把他带大的。我１０年前来广州打工，他妈妈也是前几年就出来了，他当时还很小，在老家和奶奶一起生活。<br/><br/>记者：与孩子怎么沟通？<br/><br/>王：我每天只想着赚钱，很少与孩子聊天。现在想，是我和他之间缺少沟通。平时看他很听话，实际上根本不知道他在想什么。<br/><br/>记者：出事后你觉得自己有责任吗？<br/><br/>王：儿子走到今天这一步，我要负很大的责任。我对他的教育可以说是完全失败的。做父母的，不是说把他（她）生出来、养活了就够了，还是要多关心，多交流。<br/><p
