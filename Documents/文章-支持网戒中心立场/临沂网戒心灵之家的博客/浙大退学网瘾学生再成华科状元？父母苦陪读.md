align="center">2010年09月10日07:55</p><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><wbr /><br/><br/>本报记者陈博雷<br/><br/>“弟弟今年考上了华科，他曾因沉迷游戏而退学重考，所以父母都过来陪读。父亲姓黄，今年57，身体健康，曾是一个老实的农民，吃苦耐劳，愿意从事任何正当工作，如搬运，看厂等……”昨日，一个求助帖出现在华中科大白云黄鹤bbs上，引起众多学子关注。<br/><br/>据了解，这名“网瘾”学生正是该校2010届录取新生中的状元黄敏（化名），他考了655分。发帖者是他的哥哥，去年从该校毕业后被保送中科院硕士研究生。<br/><br/><b>补鞋匠家庭走出两个名牌大学生</b><br/><br/>昨日下午，记者在华中科大校园里见到了兄弟俩的父母。黄师傅及妻子都已头发花白，瘦弱的身躯让这对夫妇显得更加苍老。<br/><br/>黄师傅告诉记者，他来自湖北孝感市孝南区闽集乡，本在农村务农，为了让两个儿子能有更好的学习条件，于1999年将家迁到了孝感，“我整日游街寻找补鞋的活来养家糊口，孩子妈妈在家中照料。”<br/><br/>生活虽然艰辛，但两个儿子成绩优异，让夫妇俩尤为欣慰：从小学到高中，哥哥勤奋踏实，弟弟黄敏虽然贪玩但极聪明。黄师傅告诉记者：“小儿子学得很轻松，经常打游戏，可他看看课本就能考得很好，基本上在年级前10名。”就这样，黄敏边玩边学，2008年仍以644分的高分被浙江大学录取，分到生物学专业。此时，他的哥哥已在华中科大读大四。<br/><br/><b>父亲四次赴浙最终带回</b><b>“</b><b>网瘾儿子</b><b>”</b><br/><br/>令黄师傅没想到的是，小儿子远赴浙江读书成了这个家“噩梦”的开始。<br/><br/>大一上学期，黄敏因为连续数次通宵打网游不上课，期末考试有数门不及格。大一下学期，他仍没有好好复习补考，黄师傅只有远赴浙大，“前后一共去了四次，让他回家休学了半年，这半年里，不给他钱用，只关家里看书，他倒是还听话。看着有转好的迹象，大二时我把他送回浙大，没想到我刚走几天，他又在网吧里几天不上课了，还把住宿费取出来去打网游，用完了后又把自行车卖了继续玩……”黄师傅回忆说，自己在浙江“守儿子”的日子里，没有钱住旅社，就在学校办公室里“窝”了几晚，因没睡好还误了回家的火车。<br/><br/>最终，黄敏还是在大二时办了自动退学手续。临走时，学院里的辅导老师建议黄师傅：这孩子很聪明，但缺乏自控能力，最好在武汉读大学，家长去陪读。<br/><br/><b>父母校内租房苦心陪读</b><b>“</b><b>华科状元</b><b>”</b><br/><br/>退学后的黄敏被父亲送到一家装公司打杂，“想让他知道生活的艰辛。”黄师傅说，可只干了20多天，心疼孩子的黄师傅夫妇还是让黄敏回家了。去年12月，黄敏回到孝感一中补习班复读，可他仍需要父亲“监督”着才能远离网吧，“我在网吧里去守过他几次，抓到他就让他回家跪着反思……”黄师傅说，小儿子自控能力较差，一次看到他在日记本上写着逃课半天后的反思：本来只想去玩一会儿，可一玩就不想走了。<br/><br/>今年高考，黄敏考出了655分的高分，他毫不犹豫地填报了华中科大计算机系，而他的父亲也将家中的房子租了出去，在华科内租房陪读。“我们9月1号搬的家，本想在这里找份工作，苦点累点都行，可找了几天也没有找到，一般都要55岁以下的，我年龄大了点。”黄师傅有些怅然地说，之前儿子读大学已经花光了家中所有的积蓄。<br/><br/><b>自信黄敏表示想要一个新的开始</b><br/><br/>昨日，记者在军训现场找到黄敏。1.74米左右的个头，瘦瘦的身板，有时候，他不经意间的一句话，又透着比同龄人多一分的成熟。<br/><br/>知道自己是“华科状元”吗？“不知道，但我能猜到。”他自信地笑着回答。<br/><br/>黄敏告诉记者，初中时自己学习还很用功，高中接触网游后就有些分心。“只有玩过网游的人才能理解那种痴迷。”他说，平日里，自己还喜欢打篮球、看小说，杂志上连载的小说，他一点不落地每个月守着看完。说到曾经从浙大退学的经历，黄敏不愿多谈，他只说：当时不是很喜欢自己的专业，想学的是计算机。<br/><br/>对于父母的苦心，黄敏很沉默，但他说：“家人总拿我和哥哥比，真的很烦！其实每个人都不一样，都是不同的个体。”<br/><br/>怎么看待曾经走过的弯路，黄敏的目光投向远方，“想要一个新的开始，不用背负过去的种种，有时也会想，或许走过的弯路对于我来说真的是一件好事情。”如今，他基本上不跟同学谈起过去的经历，当有人问他为什么快20岁了才读大一，他只简单地说，因为复读了一年。<br/><br/>还能管住自己吗？新的开始有什么目标？“自控，比较难，但我会管住自己的。”黄敏咬咬嘴唇，坚定地说，“因为我现在有目标了，我要和哥哥成绩一样好。”<br/>首次转载时间:
