<br/><p align="center">------央视今日说法《网住的花季》观后感</p><br/><p align="center">一位网瘾孩子的家长 小宣爸</p><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><br/><br/><wbr />?<wbr />?<wbr />?<strong>字字辛酸泪，句句母子情。漫漫十五载，凄凄至此景。呼天天不应，叫地地不灵。子在天上行，母在地狱生。谁能悟此意？唯有地狱人。</strong>这是一位母亲在儿子因上网死去，她在监狱中写下的一段话。看完节目之后，语句在耳畔回响，画面在眼前萦绕。<br/><br/><wbr />?<wbr />?<wbr />?央视今日说法2010年9月15日播放《网住的花季》，报道了河南省信阳市光山县十五岁少年张棋，连续七天七夜在网吧上网，8月9日母亲张宽英找回孩子，气愤之极的母亲将他捆锁在一楼的小仓库中，从下午5点直到第二天早上6点，结果孩子惨死在小仓库中，母亲入狱。节目中的嘉宾――北京市青少年法律心理咨询中心主任宗春山，在节目最后却把这样悲剧的责任推给了家长和学校，实在令人不解。像张棋那样沉迷网络游戏的孩子，还有许多许多，谁来救救我们的孩子？<b>?<wbr /></b><br/><br/><b>?<wbr />?<wbr />?<wbr />?<wbr /></b><br/><br/><strong>?<wbr />?<wbr />?<wbr /></strong>?科学是一把双刃剑，这种说法在信息化时代表现得最为明显。网络一方面给人带来了便捷，没有任何一个时代比现在更容易了解外面的世界，“一网打尽天下事”；另一方面，也没有哪一个时代比现在更让人对生命的意义产生困惑：有网络，我还需要干什么呢？<br/><br/><wbr />?<wbr />?<wbr />?我国现在有2400万沉迷网络的青少年，而且每年还以一定的比例上升，网瘾带来了一系列的犯罪案件和悲剧性的事件。据相关部门统计，青少年犯罪有百分之七十与网络成瘾有关。有人把网络游戏比做“电子海洛因”，可以说形象、准确。网络游戏走进国门以后，被青少年们当作了宝贝，一时间趋之若鹜，蜂拥进网吧。巨大的利润让国内外许多网络游戏商不遗余力开发和炒作，从而迅速形成了一个产业，更因为是新生事物，在游戏引进和开发的过程中，其内容或级别的限制、审批和监管就相对混乱和滞后，其结果是，大量有着暴力或色情内容的网络游戏给青少年的健康成长带来了一场灾难，庞大的网瘾患者群体以及由网瘾引发的众多恶性案件或悲剧性事例，也引起了党和政府的高度关注，从中央到地方虽然在这一问题上也做了大量的工作，但网瘾的发生率却居高不下，给人的感觉是，会议开了，文件发了，事情做了，问题却未得到根本的解决。<br/><br/><wbr />?<wbr />?<wbr />?政府也一再强调要加强未成年人思想道德建设，搞了许多未成年人思想道德建设教育基地，但在强大的网络游戏面前，却收效甚微，有的孩子因为上网已经徘徊在犯罪边缘，更甚者，为获得上网资费实施偷摸抢骗，或为寻求刺激模仿游戏的暴力色情情节犯下重罪，而被关进监狱，实乃个人之不幸，家庭之悲剧，社会之灾难。<br/><br/><wbr />?<wbr />?<wbr />?很多家长说政府为什么不把所有的网吧都关了？这是不切实际的气话，心情可以理解，而事实上做不到。也有家长说政府为什么不加强对网吧的监管？其实，政府对未成年人进入网吧有限制，可是全国有多少家黑网吧，谁能数的清。《网住的花季》中，张棋生前连续上了七天七夜的网吧，就是个黑网吧，无相关部门的备案和审批。在张棋的悲剧发生后，该网吧接到停业通知。可是张棋去过的光山县的另外36家网吧，其中也有一些黑网吧，由于没有出现类似的悲剧，不是还在营业吗？黑网吧打而不绝，从繁华的闹市搬到了更为隐蔽的小巷，只能是网吧老板和工商管理部门的游击战。<br/><br/><wbr />?<wbr />?<wbr />?看来，单纯靠政府加强对青少年的思想引导和网吧监管，目前也很难遏制网络成瘾这一社会现象，政府并没有不作为，只是总有政府一时难以顾及到的角落。?<wbr /><br/><br/><b>?<wbr />?<wbr />?<wbr /></b><br/><br/><wbr />?<wbr />?<wbr />?可怜天下父母心。对于上网成瘾的孩子，我相信绝大多数家长都使出了浑身的解数。从苦口婆心的劝说，声泪俱下的哀求，求神拜佛的无奈，到恨铁不成钢的打骂，甚至不惜给孩子下跪，当所有这一切都无济于事的情况下，家长还能怎么办？《网住的花季》中，张棋从小学三年级迷上电脑，到一味沉迷网吧玩游戏，母亲张宽英为了把孩子从网络的泥潭中拉出来，可以说是掏心掏肺、费尽唇舌，在屡教不改的情况下，只好忍痛施以棍棒，在依然无效之后，只好把其关禁家中，但张棋还是从三层高的楼上顺着下水管道跑掉了，当把张棋关在小仓库里时，他却用螺丝刀把门撬开溜之乎也，被逼无奈的母亲，只好采取最下策，把孩子绑锁在小仓库里，于是就出现了孩子在狭小空间里窒息死亡的悲剧。河南信阳光山县的37家网吧的老板没有不认识张棋妈妈的，为了找孩子，她不知道鞋被磨破了多少双，脚被磨出了多少泡，脚趾甲脱落了多少次，走路摔了多少跤。作为一个每天靠打两份零工度日的家长，还能做到什么？难道说没尽到一个家长应尽的责任吗？但结果又怎样呢？<br/><br/><wbr />?<wbr />?<wbr />?有人说，学校在控制孩子上网这一点上没有尽责。但大家知道，虽然全国都在推行素质教育，实际上在应试教育、高考制度的框架下，文化成绩和升学率高低仍然是家长和社会衡量学校教学水平的标准。作为学校，也反复强调学生要以学习为主要任务，反对甚至禁止学生进入网吧玩游戏，一旦发现还会对其采取批评、通报或警告处分，可当孩子真正成瘾后，又有几个会把学校的这些措施放在眼里？<br/><br/><wbr />?<wbr />?<wbr />?就学校而言，对这样的学生，出于各种原因，既不能体罚，更不能打骂，但又怕放任放纵会影响整个学校或班级氛围，或者带来大小不等的安全问题，学校在万般无奈的情况下，只好劝退或开除，除此之外，学校还能怎么办？《网住的花季》中，张棋所在学校的老师并不是不尽职，试想，一个深陷网游的孩子，岂是老师的几句规劝能改变的了的？我们除了对其表示痛心和惋惜，还能做什么？?<wbr /><br/><br/><b>?<wbr />?<wbr />?<wbr /></b><br/><br/><wbr />?<wbr />?<wbr />?还记得“藏羚羊穿越青藏铁路”照片获奖的丑闻吗？还记得“华南虎事件”的闹剧吗？现在许多媒体，可信度和公信力还有多少？有些记者更是违背新闻工作者的职业道德，不惜杜撰或编造假新闻，成为某些利益群体的喉舌和工具，混淆是非，颠倒黑白，所作所为令人痛心疾首。一些靠编造假新闻来引起轰动的记者，不妨把自己的所作所为，跟普利策新闻获奖记者的公正言行对照一下，同样是记者，报道水平高低不论，职业道德差距怎么会如此之大？<br/><br/><wbr />?<wbr />?<wbr />?作为有些媒体，是否报道一则新闻，并不把社会价值放在首位，而更注重这则新闻能否吸引读者的眼球，是否能够耸人听闻，是否能给自己带来更大的经济效益。媒体的另一个特点就是跟风意识强，一窝蜂：捧你，可以把你夸的难为情；骂你，可以让你为之跳楼，从媒体对网戒行业的炒作就很好地体现了这一点。2009年4月以前，从央视到地方，媒体对网戒是一片赞美和追捧。从2009年4月开始，一些记者不知出于何种动机，受到什么驱使，一轮一轮的负面报道开始了，反复炒作，轮番轰炸，真是你方唱罢我登场，红脸唱完唱黑脸。许多记者根本就没有走进网戒机构，对网络成瘾究竟是怎么回事，根本就毫无了解，对沉迷网络游戏的社会危害没有深入的认识，完全是道听途说，人云亦云，胡编乱造，以期达到骇人的效果，制造眼球风暴。也有的记者出于不可告人的目的，混进一些网戒机构，采访之后，对内容断章取义，移花接木，混淆视听，肆意歪曲事实，放大问题，迷惑的是观众和读者，苦的是我们这些网瘾孩子的家长。信还是不信？信谁的？到底能不能到网戒机构去救治孩子？到哪一个网戒机构去？媒体的权威性和公信力在哪里？谁还敢轻易相信媒体和记者？<b>?<wbr /></b><b>?<wbr />?<wbr />?<wbr /></b><br/><br/><wbr /><br/><br/><wbr />?<wbr />?<wbr />?《网住的花季》的嘉宾，是北京市青少年法律心理咨询中心主任宗春山。在节目中，当主持人问他，连续上网7天7夜的张棋算不算网络成瘾时，他的回答是“疑似”。多么幽默的回答，难道要连续上网70天70夜才能确认是网络成瘾？其实可以理解，专家嘛，说话当然要考虑全面，不能犯错。<br/><br/><wbr />?<wbr />?<wbr />?宗春山主任还说，家长对网络依赖的孩子，不要怕，不要恐慌，要让孩子适应社会环境，还打比方说，你想健康吗？你每天早晨起来到街摊最脏的地方去吃碗馄饨，保持你体内菌群平衡。多么荒谬的理论！孩子已经网络依赖了，你还让他到网吧去适应吗？真是站着说话不腰疼，要是他的孩子沉迷网络游戏，他还会这样说吗？<br/><br/><wbr />?<wbr />?<wbr />?在节目的结尾部分，宗春山主任认为，家长要了解网络，进入网络，与孩子共同在网上冲浪，家长应该在孩子和网络之间，起到过渡、引导、桥梁的作用。专家的建议就是好，可是这位主任，像节目中每天要打两份零工的张棋的母亲张宽英，让她陪孩子上网冲浪，这现实吗？实际上，能有时间和有能力陪孩子共同上网的家长有多少？<br/><br/><wbr />?<wbr />?<wbr />?宗春山主任还认为，学校应该改变对学生的考核标准，扮演更重要的角色，开展更多活动，如体育或者社团，学校老师要给孩子更多的自信空间。说的不错，如果这位主任能够进入校园，了解学校教学工作的重点，真正明白中国现行的教育体制和高考制度，做一名普通的老师或班主任，他还会这样说吗？如果让宗春山主任做一名中学校长，他还会这样说吗？<br/><br/><wbr />?<wbr />?<wbr />?宗春山主任是网戒行业以外的人，他提出的这些意见，虽然有点不着边际，倒也可以理解，毕竟是希望像《网住的花季》中张棋那样的悲剧不再重演。<br/><br/><wbr />?<wbr />?<wbr />?可是在网戒业内，也并非声音一致，有些自诩为网瘾戒治专家的人，譬如号称“中国网戒第一人”的陶宏开教授，就公然说“目前的网瘾戒除行业，90%都是骗钱的，临沂的杨永信、北京的陶然、广州的白云医院和江苏的徐向阳，我愿意和他们现场辩论。很多心理学家是伪心理学家。只有中科院心理所的高文斌博士是老实的，他承认他们对网瘾的研究只是取得了阶段性成效。” 正是这番言论让许多网友愤愤不平：“你说网游是毒品，家长是帮凶，在你眼里，所有的网戒机构都是以暴治暴，所有的机构都是非法的，所有的机构都是骗钱的，所有的机构都应该被取缔，哪家机构做的好，哪家机构就是你攻击的对象，就成了你的眼中钉肉中刺。前两年，陶然的名气盖过了你，你天天骂陶然，诽谤陶然，造陶然的谣，今天，杨永信的名气盖过了你，曾求过你的那么多的家长在对你彻底失望后，纷纷带着孩子去了杨永信那里，你就对杨永信恨的咬牙切齿，变着法的诋毁杨永信，攻击杨永信，要是真的把这些积极的探索者都封杀了了，你可给无德的网游公司立了大功了。”“陶宏开，你扛着许多头衔，走了许多城市，凭借媒体当喉舌，在各地招摇撞骗，这么多年下来，你到底成功救治了多少网瘾孩子？谁来评价和核实？你能把真相公之于众吗？”“按照陶宏开的逻辑，全国无数的收费行业或机构都是骗钱的，所有医院或医生都不允许有治疗失败的案例出现，所有的治疗方法或手段都不允许有任何的痛苦或毒副作用，更不允许动刀或流血，按照治病自愿原则，孩子不想吃药或打针就绝对不允许强迫，精神病人否认有病拒绝治疗，就得任其杀人放火，否则就是残无人道、暴力暴利，只有你自己的谈话可治百病，一谈就好，其他人都是伪专家，只有你是‘救世主’。”?<wbr /><br/><br/><wbr />?<wbr />?<wbr />?政府有很多问题有待落实，网络成瘾应该属于教育部门来管还是应该由卫生部门来管？有些专家尽说些无关痛痒的建议，或者对真正有所作为，敢于探索的同行大肆攻击，新闻媒体只是找炒作的热点，家长和学校无能为力，作为网络成瘾孩子的家长正生活在水深火热之中。?<wbr /><br/><br/><wbr />?<wbr />?<wbr
