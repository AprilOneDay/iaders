/>?<wbr />?<wbr />?新华网湖南频道长沙2月11日电（记者 帅才）“离开学只有一周了，孩子寒假作业都没怎么动笔，整天猫在网吧聊天，害得我天天跑到网吧寻女儿。” 家住湖南省汉寿县的宋先生谈起染上“网瘾”的女儿愁眉不展。“有些家长跟我反映，整个寒假，孩子不是在用电脑打游戏，就在用手机聊QQ、发微博。”长沙一位中学老师告诉记者。<br/><br/><wbr />?<wbr />?<wbr />?许多学生家长、老师以及社会工作者反映，青少年因沉迷网络而引发的社会问题日益突出，特别是由网络游戏引发的青少年犯罪问题日益严重。专家建议政府、家庭、学校多方联动，引导青少年健康上网，让“网瘾”少年们走出网络、走入户外。 ?<wbr />?<wbr />?<wbr />?<wbr /><br/><br/><b>　?<wbr />?孩子沉迷</b><b>“</b><b>网游</b><b>”</b><b>，家长多烦忧</b><br/><br/><wbr />?<wbr />?<wbr />?比起宋先生的女儿，15岁的“资深网民”史鉴对互联网的依赖有过之而无不及。家住汉寿县的史鉴在父母离异后跟着奶奶，染上网瘾后，他整天泡在网吧玩网游，有时干脆睡在网吧。史奶奶谈起这个孙子气得直跺脚，但她表示是一点办法都没有。史鉴的邻居告诉记者，孩子以前成绩很好，但自从染上网瘾，就无心学习了。<br/><br/>烦心的不止宋先生，家住长沙的任先生着急得很，刚上初二的儿子自从收到新年礼物――一台可以上网的手机后，就迷上了手机上网，整天按手机，一个小时可以发几十条微博，成为了一名小小的“微博控”。<br/><br/>网络展现方便青少年搜集信息、互动交流等美好一面时，也给他们的成长带来了巨大的干扰。 据中国青少年网络协会2010年发布的《中国青少年网瘾报告（２００９）》调查结果显示，目前我国城市青少年网瘾群体约占青少年网民的１４．１％，人数约为2404.2万。网瘾青少年主要是“网络游戏成瘾”，其次是“网络关系成瘾”。<br/><br/><wbr />?<wbr />?<wbr />?共青团湖南省委2010年提供的《青少年网络游戏使用调查报告》显示，这个省10岁以下开始使用网络游戏的青少年群体占到49.7%，近九成的青少年14岁以下开始使用网络游戏。10.2%的青少年每月玩网络游戏花费500元以上，更有4%的青少年，每月花费在千元以上。<br/><br/><wbr />?<wbr />?<wbr />?有九成青少年认为当前网络游戏里带有脏话、暴力、欺诈、赌博和色情等负面内容；四分之一的青少年表示经常在黑网吧上网，另外有12%的青少年表示周围网吧没有禁止未成年人进入标志；调查中有一半青少年以上认为网络游戏防沉迷系统效果一般，另外有41.6%的青少年认为网络游戏应该加强分级管理。 ?<wbr />?<wbr />?<wbr />?<wbr /><br/><br/><b>?<wbr />?<wbr />?<wbr />?青少年染</b><b>“</b><b>网瘾</b><b>”</b><b>危害大</b>?<b>专家解析</b><b>“</b><b>网瘾</b><b>”</b><b>成因</b><br/><br/><wbr />?<wbr />?<wbr />?专家指出，沉迷网络影响了青少年的人生观、世界观、价值观的形成以及健康成长。染上“网瘾”的青少年面临着视力下降、学习成绩退步、人际交往能力下降的问题。<br/><br/>近年来，网瘾少年被殴致死事件、网瘾少年抢劫事件、网瘾少年自杀事件、网瘾青年弑父伤母事件等网瘾引发青少年犯罪事件频发。<br/><br/>《青少年网络游戏使用调查报告》显示，青少年因上网而引起的网瘾、伤害、强奸、恶搞、婚恋甚至暴力犯罪在不断增加。<br/><br/><wbr />?<wbr />?<wbr />?长沙市心理学会秘书长、周南中学心理中心主任刘正华告诉记者：“在我这进行心理咨询的青少年中，有30%沉迷网络。曾有位名叫石军（化名）的15岁网瘾少年告诉我‘自杀没什么，游戏里都是这样，人死了，然后充几滴血可以立马活过来……’”<br/><br/><wbr />?<wbr />?<wbr />?刘正华认为，部分网络游戏不加节制地渲染、刻画一些暴力、恐怖场面，并将暴力视为解决人们之间矛盾的有效办法，造成青少年暴力崇拜，影响了青少年的价值观。这也体现了我们教育的错位，而对孩子的“生命教育”还处于“真空状态”。<br/><br/><wbr />?<wbr />?<wbr />?武警总医院心理学专家史宇接触过不少染上“网瘾”的青少年，她告诉记者：“染上网瘾的青少年会出现情感反应迟钝、情感淡漠和焦虑等反应。大部分染上网瘾的孩子都是由于家庭型结构不合理、家长教育方式产生偏差、与父母关系不好等原因在现实生活中找不到寄托，网络成为这些孩子们逃避现实世界，重新塑造自我认同的空间场域，他们在网络这个虚实交织的世界寻找慰藉。”<br/><br/><wbr />?<wbr />?<wbr />?史宇说：“孩子们通过上网消除寂寞、放松心情、释放现实压力、重塑自我形象、拓展社会交往，当孩子觉得在网络世界获益后，他会重复这种行为，把寄托从现实生活中转移到网上，从而对网络产生依赖。”<br/><br/><b>?<wbr />?<wbr />?<wbr />?专家建议让</b><b>“</b><b>网瘾少年</b><b>”</b><b>走出网络，走入户外</b><br/><br/><wbr />?<wbr />?<wbr />?周南中学心理中心主任、长沙市心理学会秘书长刘正华告诉记者：“青少年沉迷网络的原因之一是业余活动单一。要让网瘾少年走出网络，学校除了开设有关健康上网方面的课程与社团活动，帮助青少年培养健康的上网习惯，还可以开展形式多样、寓教于乐的文体活动，使青少年的课余生活丰富多彩，帮助青少年把注意力从网络游戏转移到其他有益、健康的休闲娱乐活动。”<br/><br/><wbr />?<wbr />?<wbr />?刘正华介绍，家长对孩子关心、与孩子交流不够、简单粗暴的“家长制”教育方式也是导致青少年转向网络寻找慰藉的原因之一。据共青团湖南省委开展的青少年上网调查，有六成家长对待孩子的网络游戏使用问题多是简单的批评训斥，缺少耐心的说服引导和沟通交流。<br/><br/><wbr />?<wbr />?<wbr />?“在网络中，大家都是平等的，这也是青少年喜欢在网上聊天的原因之一。”刘正华建议家长跟孩子零距离接触，与孩子平等交流。<br/><br/><wbr />?<wbr />?<wbr />?武警总医院心理学专家史宇建议：“家长不要一味责怪染上网瘾的孩子，想想是不是家庭出现了什么问题，自己对孩子是不是关爱不够、苛责过多。”<br/><br/><wbr />?<wbr />?<wbr />?刘正华说：“希望青少年走出网络，走入户外，多参加其他有益、健康的文体活动，多和老师、同学、朋友面对面的交流。希望政府提供和开放更多健康的课外休闲娱乐和学习场所，建立青少年能够就近参与的社区青少年活动阵地。”<br/><br/><wbr />?<wbr />?<wbr />?“听说现在大城市美术馆、博物馆都将免费对青少年开放了，这就是大好事。”家住湖南省汉寿县的宋先生告诉记者，“我也想带孩子参加一些有意义的文体活动，但是我们县里，一没有美术馆，而没有博物馆，连电影院都没有，也不知能去哪。”<br/><br/><wbr />?<wbr />?<wbr />?刘正华建议：“对于没有好的文体活动参加的孩子，家长们可以赠送书本让孩子增长知识，现在孩子们的文史知识很贫乏，都在使用网络语言。开学前一周，家长可以帮助孩子们调整作息时间，以好的精神状态迎接新学期。” (本文来源：<a href="http://media.163.com/special/007625CB/xhw.html">新华网</a>?)<br/>首次转载时间: 2018-02-28
