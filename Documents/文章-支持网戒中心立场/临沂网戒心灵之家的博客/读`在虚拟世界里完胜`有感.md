/>?昨天早晨上网，看到网易首页有篇所谓的“在虚拟世界里完胜”的文章，用黑体加大特殊处理的一文，即点击浏览，实际上是一位叫“白雪”的记者发表在《中国青年报》的文章，说的是什么“《网瘾战争》在虚拟世界里完胜。”<br/><br/><wbr />?<wbr />?<wbr />?读后不禁失笑，呵呵，原来是凑个热闹或者搅搅水让水浑起来的主吧，看到有帖子也弄了几个，从网上发了点议论，想这些逻辑不通、是非不分、出口成脏的人，真是不能与其较真的货。不过看起来有点热闹，她说――“《网瘾战争》的胜利来得迅速，消失得似乎也快。在现实世界里，有关网瘾的战争仍在继续。现在打开搜索引擎，你会发现，持续火爆的，是不断增多的一大片网瘾戒治机构。”这是她――中国青年报的一个作者白雪愿意还是不愿意看到的呢？<br/><br/><wbr />?<wbr />?<wbr />?今早一起床，妻子从临沂打来电话，我问儿子这些天表现怎样？她告诉我儿子昨天做了50个俯卧撑，我说那好啊，这个锻炼他的毅力。妻子问我啥叫“毅力”？我说，毅力就是一个人说到做到、不管遇到多大的困难都能咬牙坚持，儿子在网戒中心是不是真改变，就看看他有没有提高认识、有没有培养毅力。他走偏这一年，对学习失去兴趣，办事缺乏恒心，遇到困难不能迎刃而上，这些问题解决不了，即便是再回到学校仍然可能打退堂鼓的啊！<br/><br/><wbr />?<wbr />?<wbr />?到今天，是我们送儿子到网戒中心的第61天，回头看，两个月真是一晃而过，时光飞逝啊。<br/><br/><wbr />?<wbr />?<wbr />?去年的12月25日，圣诞节那天，我们把儿子刘XX“骗”到了网戒中心。而这个行动，我策划了不短时间，却快刀斩乱麻，非常利索。因为我之前从网上搜索看了《网瘾之戒》，也把这个第一时间从QQ上发给了儿子，那时他再次辍学，回到日照的家里。他有没有看过此节目，我不晓得，但是，送他去网戒中心我们态度坚决。12月24日，我从河北邢台工地上直奔日照，第二天早晨8点半把他从家里骗上了车。那天，他一头长发，也没有洗过，身材瘦弱、目光迷离，现在戴在他胸卡上的照片可以留作永久的纪念，日后让他回忆吧。<br/><br/><wbr />?<wbr />?<wbr />?临沂有个网戒中心，我媳妇在电视上看过报道，我原来一直没有留意过，只是我在几年前买过一本陶宏开关于网瘾少年的书，一口气利用双休日看完，就扔进了书柜里。我对儿子的教育起初是有信心的，从电脑进入我们的生活开始我就教他摆弄电脑。休息时间一些儿童益智游戏也是准许他玩玩的。上网看看新闻、掌握基本的电脑操作技术也是必须的，要不学校还开设电脑教育课呢？但是没承想，我离开机关一年，我的儿子发生了天翻地覆的变化，撒谎、逃学、弃学、仇视父母，简直是变成了恶魔。我电话请求做过教师的老父亲从老家、当过兵转业参加工作的弟弟从九江，先后专程去日照，跟他沟通、劝解，均告失败。这个时候，临沂网戒中心成为我们唯一的选择！<br/><br/><wbr />?<wbr />?<wbr />?其实，我在送儿子的时候就坚信：他在这里应该得到很好的教育！当天上午，我办好所有手续，赶紧听了一会杨叔的点评课，我第一次亲眼目睹到杨叔的风采，聆听到杨叔的教诲，感觉这个电视上的名人就在我们眼前，多么的亲切和蔼啊，真的，仿佛回到我的中学时代，虽然杨叔穿的是白大褂，但我从眼神、语言、那种收放自如的神态里面断定：这个人就是能够让我们、我们的孩子受到真、善、美最基础教育的一个恩师，他循循善诱、他苦口婆心、他引经据典、他高屋建瓴，他的知识和才华、他的坚韧和博爱，是我们在无情的竞争、社会的磨砺、人情的冷暖历练后而难得一遇的、不得不崇拜的一个人，他有感情，有血肉，就在我们身边！<br/><br/><wbr />?<wbr />?<wbr />?也巧，那个晚上，我还参加了盟友组织的一场联欢晚会，感觉整个大家庭充满了温馨。第二天，我回到工地，老父亲从老家打来电话，说电视上看过有个军训的地方治网瘾，把XX（我儿子的乳名）送去吧，我说真巧，我已经送过去啦，您老放心吧。<br/><br/><wbr />?<wbr />?<wbr />?春节前几天，我回了网戒中心，专心地听杨叔讲点评课，认真地从第一篇儿子的日记看起，用心地体会网戒中心的点点滴滴，更多的是深深地受到感动。第一次正襟危坐在大课堂上，我还不知道有唱歌的环节，看着屏幕上那熟悉的歌词，我眼泪唰唰地情不自禁地往下流。唱《父亲》的时候，我想起我的老父亲。12岁，我考上了县城最好的中学，星期六我从县城回到家，晚上一家人在冰冷的水里面洗地瓜，然后整齐地码在地板车上。凌晨三点，拉着几百斤的地瓜到城西去卖，那里有专门的粉丝加工商铺。父亲套上车袢、弓起身子在前面拉，我在后边迷迷瞪瞪地帮着推，25里路，从家里一路往西走，全是用脚丫子一步步地量过，但是奇怪，走了那么远的路，月亮一直追着我们走，星星好像一颗也没有被甩下，眼睛眨呀眨地看着我们爷俩。天色微明，我回到教室上自习，父亲一个人拉车去排队，卖完地瓜再跑到学校给我送学费。是父亲，打小亲手教会我写字，亲手把我送进中学，亲手给我铺好床铺，毕业考试那两天亲手给我送饭，看着我吃个底朝天。至今都感觉再没有吃过那么香的西红柿炒鸡蛋！父亲把爱传给了我，我把爱传给了我的儿子，我把他也亲自送到学校，从小学、到中学、到技校，可是他没有了解这份爱，他当了逃兵，他背弃了一个父亲、一个家庭的希望，他就坐在这个大讲堂里，在这个四周写满亲情、孝道的标语，充满着感恩的氛围里，他应该受到心灵的触动，得到洗心革面的教育！<br/><br/><wbr />?<wbr />?<wbr />?在这个熙熙攘攘的世界上，有人忙忙碌碌，有人寂寞难耐，有人惹是生非。只有杨叔，在挖空心思地挽救我们这些沉迷网游的孩子们，殚精竭虑地在呵护着一个个被折磨得死去活来的家长们。我以一个普通家长的身份默默地接受着教育，用心地观察着孩子的每一个细微变化。虽然短短几天，工作关系不得不回来上班，但是我听到的是由衷的赞美、看到的是真切的氛围。我参加了临时的值班，接受了24个小时的生活养成，也参与了盟友们的座谈讨论，还担当了盟友演讲选拔的评委，和大家一块看了精彩的文艺演出，吃了一锅煮出来的过年饺子。我们这些本地的东道主也好，天南海北的远道朋友也好，大家相亲相爱一家人，结下了难得的值得终生珍惜的友谊。座谈会上，我曾给一个小室的盟友提了三点希望：一是要有思想，在这里把自己培养成能够分辨良莠、懂得是非、知道感恩的人，正如把《弟子规》背熟理解一样就善莫大焉；二是要有好的心态，配合治疗，修身养性，无论现在还是以后遇事不急不燥、分清轻重缓急，牢牢抓住需要突出解决的问题，各个击破；三是要有好的习惯，从生活养成开始，有板有眼，井井有条，说到做到，锲而不舍。这样的孩子出来哪个家庭不欢迎？哪个单位不需要啊？！<br/><br/><wbr />?<wbr />?<wbr />?我留恋在网戒中心的这个春节，想念网戒中心的心理大课堂。有机会我一定回去听课，提前准备调剂时间回去接受教育。我曾经告诉我的儿子和盟友们，杨叔的大课是一个丰富的宝藏，发生在每个孩子身上的问题，都要用心体会，有则改之无则加勉，这是社会上花钱买不来的教育！在这里大家要<span style="color: #da0000;"><strong>“学会面对现实，学会承担责任，学会自我克制，学会承受压力，学会承受挫折，学会承受痛苦，学会抵抗诱惑。”</strong></span>这是多么神圣、多么高尚的境界啊！我们家长也要学会这些，不是修来的福分吗？<br/><br/><wbr />?<wbr />?<wbr />?今天，我用了一个下午的时间，来打开研读网戒中心去年的风风雨雨，特别是看到某些所谓记者的报道原文、跟帖、电影，感到义愤填膺、怒火中烧，心头有万言不解之恨。但是反过来，最后再打开《网瘾之戒》从头看到尾，我反而心情突然平静下来，没有了激动和冲动，异常的冷静，想：乌云遮不住太阳，注意那些采访的细节，镜头都进行了处理，哈哈，悖人心、求刁钻，脸都遮起来啦，不要脸啊，心理阴暗着呢。走我们的路，让他们折腾去吧！<br/><br/><wbr />?<wbr />?<wbr />?历史是人写的，不是谎言编的。我们这些感恩戴德的家长将永远带着我们的孩子感激您，支持您，永不言弃地向网瘾作彻底地斗争！<br/><br/><wbr />?<wbr />?<wbr />?夜深人静，万籁俱寂，心潮难平。遥祝杨叔身体健康，元宵节快乐！<br/><br/><wbr />?<wbr />?<wbr />?<wbr />?<wbr />?<wbr
