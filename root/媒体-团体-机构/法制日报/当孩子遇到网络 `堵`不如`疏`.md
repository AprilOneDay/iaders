<br/>□法制网记者　余瀛波<br/><br/>50天内完成各种各样的自残任务,挑战自己的极限,最终诱导青少年参与者自杀……“蓝鲸”死亡游戏开始在中国社交网络中出现。5月12日是互联网社会责任日,首都互联网协会新闻评议专业委员会2017年度第四次会议就抵制“蓝鲸”游戏,保护青少年健康成长进行探讨,全力拦截这条死亡的“蓝鲸”。北京市文化行政执法总队网络执法队干部石铁表示,“蓝鲸”游戏本质上不是传统的网络游戏,是通过建立社交,引诱、教唆青少年犯罪。<!--more--><br/><br/>早在10年前,由管虎导演的电视剧《紧急链接》,就讲述了网络少年如何一步步走上沉迷网络的道路,并最终跳楼自杀的故事。不幸的是,10年过去了,类似的悲剧仍时有发生。随着智能手机、网络游戏的快速发展,网络对儿童青少年包括对整个社会的影响,都在不断加大,由青少年网瘾引发的社会新闻近年来屡见报端,引起全社会的关注。<br/><br/>近日,中共中央、国务院印发了《中长期青年发展规划(2016―2025年)》,其中指出,要强化网上思想引领,在青年群体中广泛开展网络素养教育,引导青年科学、依法、文明、理性用网。<br/><br/>那么,一个现实而严峻的问题正摆在人们眼前:应如何指引孩子们健康上网、戒掉网瘾?<br/><br/>&nbsp;<strong>每年新增千万儿童网民</strong><br/><br/>今年1月,中国互联网络信息中心(CNNIC)发布的最新数据显示:截至2016年12月,中国网民规模达7.31亿,其中19岁以下网民达1.71亿,约占全体网民的23.4%。<br/><br/>据长期专门研究治疗儿童青少年网瘾问题的北京师范大学心理学部副教授张锦涛介绍,我国在过去五六年时间内,每年大概新增儿童青年网民有1000万以上。张锦涛团队通过对比2014年和2015年的数据发现,在新增的1000多万网民当中,大约有70%到80%都是6到11岁的低龄化网民。也就是说,近几年来低龄化儿童网民正呈现出持续快速增多的趋势。<br/><br/>实际上,这一问题自上世纪八九十年代起,在欧美一些发达国家就已经受到关注。2013年,美国精神疾病协会出版的《精神疾病诊断与统计手册》第五版中,已经将这一问题列入“亟待研究的问题”。同时,最近世界卫生组织正在进行修订的国际疾病分类系统当中,也首次提出“危险性的网络游戏使用”。<br/><br/>正是在这种背景下,我国不断加大互联网时代青少年权益的保护。十三五规划纲要明确提出,完善网络空间治理,营造安全文明的网络环境；今年2月,国务院法制办公布了《未成年人网络保护条例(送审稿)》,这也是我国第一部专门针对未成年人网络使用和权益保障的行政法规。<br/><br/>今年全国两会期间,全国人大代表、腾讯公司董事会主席兼首席执行官马化腾还专门提交了《关于加强未成年人健康上网保护体系建设的建议》,建议加强未成年人健康上网保护体系建设,大力倡导在互联网行业内形成规范、统一、完善的未成年人保护行业标准。<br/><br/>&nbsp;<strong>韩国“游戏宵禁”效果不理想</strong><br/><br/>据报道,为了解决青少年网瘾问题,有的学校甚至采取断网、禁网等比较极端的形式。其间,更有被称为“全国戒网瘾专家”杨永信提出“电击疗法”戒网瘾,受到舆论广泛关注和巨大争议。<br/><br/>但是,今天的人们毕竟已经生活在一个互联网几乎无处不在的时代,某种意义上,互联网就如同人们赖以生存的空气和水一样,仅靠强制性的断网或者禁网,已很难从根本上解决问题。<br/><br/>这方面最典型的例子,就是同为青少年网瘾重灾区的东亚近邻――韩国。为防止青少年因过度沉迷网络游戏而影响到正常的睡眠和学习,2011年5月,韩国国会通过《未成年人保护法》修正案,责令网络游戏提供商在规定时间段禁止向16岁以下的未成年人提供网络游戏服务的法律制度,该法律也被称作“灰姑娘法”。<br/><br/>韩国也由此成为世界上第一个、也是唯一一个强制推行“游戏宵禁”制度的国家。但是,这项制度出台后的效果却并不理想。据统计,为避开“游戏宵禁”制度的影响,40%的青少年通过盗用身份证号码的方式继续玩游戏,盗用父母的身份证注册游戏的青少年游戏用户旋即增长了5%。<br/><br/>正如“头痛医头,脚痛医脚”的道理一样,问题的症结,其实并不在游戏本身,所以“只堵不疏”根本解决不了问题。<br/><br/>对此,游戏研究学者、中华游戏研究协会副主席刘梦霏认为,很多时候,我们在看待网瘾的时候,把游戏当作一个替罪羊,而并没有看到游戏的本质。实际上,只要摘掉有色眼镜,你会看到,游戏不仅只是一种娱乐,而且也为社会带来许多积极影响。<br/><br/>比如,将游戏应用于医疗,可以帮助哮喘病的小朋友更好地治疗疾病；将游戏应用于教育,能够大大提升学生的学习兴趣；将游戏应用于管理,能让整个团队充分自我实现,高效率地完成工作等。这些在国外已经有很多实践的领域了,只是在国内还没有引起很多人重视而已。<br/><br/>&nbsp;<strong>“300对父母,没有一个是满分”</strong><br/><br/>那么,问题的症结在哪儿呢?专家指出,在诸多出现游戏沉迷问题的家庭中,其较为糟糕的亲子关系、抚养方式都依然没有得到纠正和改变,而这正是真正影响儿童青少年是否能健康使用网络产品的关键所在。<br/><br/>浙江师范大学心理与脑科学研究院副院长董光恒,针对网络游戏成瘾也进行过多年研究。截至目前,他的研究团队搜集的相关群体已经有300人。<br/><br/>“在这300人的青少年群体中,可以毫不客气地讲,他们的父母没有一个是满分的,甚至几乎没有一个是合格的。”他说,人们往往是在出了问题以后,父母抱怨称,你们这些游戏设计者,不顾我们的孩子,让他们每天沉迷于游戏中。<br/><br/>“但是他们有没有思考过一个问题?我尽到一个做父母的责任了吗?未成年人正处在需要爱、社会关注、别人给他温暖、社会黏性的阶段,这些东西父母有没有及时提供给他?”董光恒说。<br/><br/>既然问题的症结找到了,那么该如何解决呢?<br/><br/>近日,北京师范大学联合腾讯研究院共同发布《儿童青少年网络健康使用指导手册》(以下简称《指导手册》),提出了一种以“疏”为主的解决方案。<br/><br/>《法制日报》记者了解到,《指导手册》结合儿童青少年心理发展特点,从儿童、家长、学校三方出发,提出防止儿童青少年过度使用网络的“四大原则”。该方案基于动机访谈及行为改变理论帮助儿童青少年做出改变决定,并通过具体的时间管理、情绪调节、放松与冥想等压力应对方法帮助儿童学会有效地积极应对压力并进行自我提升,最终通过综合使用这些技巧和方法达到抵制网络诱惑、减少过度使用网络的目的。<br/><br/>“研究机构及社会公众的关注点,常常落在已经出现网络成瘾症状的儿童青少年群体,但实际上这部分儿童青少年仅占很小的比例,大约只占未成年网民的10%左右。我们认为,如何指导和引导绝大多数儿童青少年网民更有效、健康地使用网络,才是当前更为迫切需要关注的。”张锦涛说。<br/>首次转载时间:
