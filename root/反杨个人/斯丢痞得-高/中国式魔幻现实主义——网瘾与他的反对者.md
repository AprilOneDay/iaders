align="right">――网瘾和他的反对者</h2><br/>不少90后都听过父母说过这样一句话：你这么喜欢玩电脑，这是网瘾，要不咱们去戒掉吧。简单一句话，孩子们也都心惊胆战。《三国演义》也有个能“止小儿啼”的将军，叫做张辽。如《魏略》中所讲：（张辽）由是威震江东。儿啼不肯止者，其父母以辽恐之。同书又引曰：江东小儿啼，恐之曰：‘辽来，辽来！’无不止矣。”<br/><br/>09年一期新闻调查中，柴静深入临沂网戒中心，揭开了杨永信治疗网瘾100%成功率的真相。杨永信的“低频电子脉冲治疗”被人诟病已久，甚至有不少网友将杨永信称为“磁暴步兵”。09年也紧急叫停了杨永信的电休克疗法，不过杨永信不仅没有收敛，而且变本加厉，将刺激性更强的电针灸仪器用于他的臭名昭著的“13室”。甚至将没有临床试验或者严禁青少年服用的精神类药物大量用于盟友（网戒中心对“病友”的称呼），这些盟友如《1984》中的温斯顿，从“13室”出来时流着泪说：“我爱杨叔，我要留着治疗”。可怕的是，这类事件早已曝光，却屡禁不止，甚至明目张胆利用此事件为自己宣传，这大概就是中国式的魔幻现实主义吧。<br/><p class="picbox"><img src="https://ww3.sinaimg.cn/large/abd05170gw1fb3yuop5q8j20j809lmz0.jpg" alt="网络图片"><span class="picinfo">网络图片</span></p><br/>?<br/><h2 align="center">受访者的故事</h2><br/>@未消逝的青春是一位曾进入临沂网戒中心进行治疗的“盟友”，他于8月24日在家中发布了一篇名为《我在临沂网戒中心的真实经历》的揭露贴。文中详细介绍了他在网戒中心3个月的经历和网戒中心的机构设置以及收费标准。三天后，他收到了临沂网戒中心的“家委会”的威胁，因此离家出走寻求帮助，至今已4个月过去了。<br/><br/>采访中，“消逝”表示将孩子送去戒除网瘾的家长家里条件多是不错。但是，这些家长中也有条件十分有限，却宁愿砸锅卖铁、东拼西凑将孩子送去的。不论家长初心是什么，我们可以看看后续的情况。据网友提供的信息，09年前后的能够联系上的家长40人中仅仅只有7人表示仍旧支持杨永信；而大部分的家长则是表示孩子在回家后就离家出走，很久没有见到了。“消逝”也表示，最近杨永信正在给所有出院的盟友的家长们依次打电话，颇有一番警告的味道。<br/><p class="picbox"><img src="https://ww1.sinaimg.cn/large/abd05170gw1fb3yutiqhpj20j80c176d.jpg" alt="此文已难从微博搜索查看"><span class="picinfo">此文已难从微博搜索查看</span></p><br/>?<br/><br/>@无明之痴 可以说是一名热心网友，看到了在网戒中心遭受如此遭遇的盟友们，感觉于心难忍，于是站出来为他们发声。她也是最早开始提高岩（化名）的人，曾经还去重庆大学向学校请求发布寻人启事，全程跟踪高岩的事件。甚至还为他写了一篇小说：《中国恐怖故事：疯诗人》。<br/><p class="picbox"><img src="https://ww4.sinaimg.cn/large/abd05170gw1fb3yuwtitmj20j80dggn9.jpg" alt="有录音收录于喜马拉雅FM"><span class="picinfo">有录音收录于喜马拉雅FM</span></p><br/>?<br/><br/>她是一个年轻乐观的小姑娘，在聊天之中不乏表情包的参与，整个采访过程还是十分轻松愉悦的。采访的内容却实在让人揪心不下：她在采访中告诉笔者，虽然网戒中心目前不再使用“低频电子脉冲治疗仪”了，但替代方法依然没有人性――将孩子绑在床上，禁止他参与任何活动。据了解，根据最近出院的一位盟友的说法，有一位盟友已经在床上绑了至少10来天了。作为自发帮助盟友的博主，“吴明芷”（无明之痴化名）已经得到了20位左右盟友的自发联系。<br/><h2 align="center">杨叔和他的疗法</h2><br/>提到杨永信的疗法，电击是多数人联想到的第一个词。不过根据消逝和“吴明芷”提供的材料，我们可以发现，除了电击这一项早已为众人所知的非法勾当，还有着“磕操”和精神类药物的滥用。<br/><br/>09年，国家卫生部紧急叫停了电刺激疗法用于治疗网瘾，不知之前为何没有调查杨永信这样使用电刺激疗法的合法性；不过关于电针灸仪器的使用资格也是疑点颇多。据悉，此类电针灸仪器的使用必须要在行业内考取资格证。不过据了解，该院内所有的点评师和杨永信自己并没有考取相关资格证，也就是无证行医。这个词对于读者来说并不陌生，而最近被曝光的莆田系医院就是一种无证行医，不过杨永信的性质可能远比这个更恶劣。<br/><br/>所谓“磕操”，就是一种所谓“行为矫正操”。其动作基本就是多见于藏族朝圣的动作，在央视《战网魔》中有具体的动作拍摄。笔者试着自己做了一套，大概做到20个的时候就已经气喘吁吁了。“吴明芷”向记者透露，现在在网戒中心“磕操”仍旧存在，并且一次性需要做500个。<br/><p class="picbox"><img src="https://ww3.sinaimg.cn/large/abd05170gw1fb3yv3yu4vj20g60bctat.jpg" alt="磕操"><span class="picinfo">磕操</span></p><br/>?<br/><br/>两位网友为我们列举了杨永信给盟友们提供的药物清单，其中为大多人印象深刻的是一款叫做“盐酸帕罗西汀”（即乐友）的抗抑郁类药物。通过查看该药物的说明书，笔者发现该药物是严禁用于小于18岁的青少年的，否则极易产生自我伤害，甚至自杀的念头。并且，药物中明确标明，该药物尚未有过与电休克疗法联合治疗的临床经验。而不那么出名的如“丁螺环酮”、“茴拉西坦”也明确标注了有强烈副作用，尚未有给未成年人使用的临床经验。“丁螺环酮”也明确标注严禁给儿童使用。<br/><p align="center">“中国恐怖故事”</h2><br/>“吴明芷”曾写到过以一位名为高岩（化名）的重庆大学研究生的真实故事为蓝本的小说，目前在喜马拉雅上也有一位主播将其以音频的方式表达了出来――《中国恐怖故事：疯诗人》。<br/><br/>小说虽然是杜撰而出的，但是其中故事多是由真实的故事改编而来。故事的主人公叫做高岩，研究生，毕业于重庆大学。据“吴明芷”为记者提供的材料，高岩的家里住在乡下。高岩通过自身的努力，考取了重大的研究生，毕业后希望继续攻读博士学位。但是家中的父母持续的逼婚和反对，导致了他产生了抵触和抑郁的心理。将委屈和抑郁发泄在网吧的高岩，终日在网吧中挥洒自己的青春。也导致了他最终被父母认为有网瘾，并送去了临沂网戒中心接受治疗。由于多重的压力与杨永信的一系列的非法治疗，导致了精神疾病的加重。而见势不对的杨永信，也将高岩送去了真正的临沂四院接受正规的治疗。<br/><br/>“吴明芷”告诉记者，高岩现在已经出院，并住在他其他的亲戚家，远离了他的父母。不过为了尊重他不愿意被打扰的意愿，记者没有要求采访他，只希望他能早日走出这件事的阴影，去做他真正想要做的事情。<br/><br/>家长应该理解对专业的热爱推动自己攻读博士的想法。“催婚”现象更是目前社会面对的一个巨大的问题，假如为了考取博士而不立刻婚娶也是一种错误，那就没什么是人身自由了。高岩的遭遇的罪魁祸首固然是其父母，却不能不看到杨永信的误诊和治疗是压倒骆驼的最后的一根稻草，作为一名心理医生，他非但没有正确指导孩子与家长间的关系，反而用更加暴力的方式促使父子反目，母子成仇，实可谓可怖。<br/><h2 align="center">早已打响的网瘾战争</h2><br/>早在09年，一部叫做《网瘾战争》的视频就广泛流传于网络之中，视频以游戏魔兽世界为底，表达了当时网友对于杨永信、陶然等人对于网瘾的言论和治疗，并痛斥了电击的治疗方式。可以说，那时在网上声势浩大的“讨伐”行动就是第一次网友集结起来为了权利奋争的行动。<br/><br/>关于网瘾是不是精神病，国内一直争论一直不断，有人为网瘾就是一种精神病；有人认为网瘾如果是病，那打麻将、看书学习、社交，都是有瘾的，都是精神病。根据最近提出的未成年人网络保护条例（修改稿）中，国家倾向于将“网瘾”定性为一种“网络使用过度”的行为。早在09年国家的定性，认为网瘾是其他精神类疾病所导致的并发症，并不能单独作为诊断的标准。<br/><br/>认识为什么网瘾不是病很简单，有这么一群人：在十来年前，这是一群16-25岁的人，他们为了网络游戏聚在一起，租住在简单的出租屋里，一个月工资不超过500。有一个爱抽烟的人，他所赚取的工资只够买几包烟，几乎每个月底都会偷偷捡烟屁股抽两口。但是他们即使再怎么不堪，每天仍保持玩游戏10小时（远超过国内“网瘾标准”）。读到这里，应该很多人都认为他们就是传统意义上的网瘾少年无疑了，可是他们其实是我国近十年来电竞选手的真实写照。他们中的绝大多数，收入十分低下，仅仅能够靠微薄的收入保证生活的需要。还有一群人，由于天生或者后来的遭遇导致了心理产生了疾病，于是将全部的精力都放在了电子游戏上，并以一种病态的表现方式表达自己的不满。<br/><p
