<br/><img src="http://wx4.sinaimg.cn/wap720/006nOlTXly1fbmq78i1wxj310e0kujwp.jpg" alt="" /><br/>哇塞侠提出了一个神奇的论点，低频脉冲不是电击，不是电休克，所以不违反相关规定，笔者又一次登录了原卫生部，现国家卫计委网站找出了当年的文件的原文。<br/><!--more--><br/><br/><img src="http://wx1.sinaimg.cn/large/006nOlTXly1fbmkhkiy9hj30hi0hnjus.jpg" alt="" /><br/><br/>在二零零九年七月八日的这份文件之中，我们可以看到的这样的一段话“停止辖区内有关医疗机构电刺激（或电休克）治疗网瘾技术的临床应用，若开展科学研究应按照规定申报，经过批准后须充分尊重受试者知情权选择权不得收取相关费用。”这是原卫生部现在国家卫计委的公文。其中明确提到了电刺激（或者电休克），而低频脉冲，从其本身的原理来看就是电刺激疗法一种，按照相关规定属于被禁止的范围内，同时这个文件也没有看到说废除，所以其不属于电击这个论点是十分荒谬的。<br/><br/>同时低频脉冲戒网瘾疗法是不是有效安全，并且经过规范了呢？笔者查阅了一个重要的文件就是2009年卫生部发布文件之中引用的医疗技术临床应用管理办法，该办法是在2009年3月2日发布，卫医政发〔2009〕18号，并于同年5月1日起正式实施，其中第二章第七条规定：<br/><br/>第一类医疗技术是指安全性、有效性确切，医疗机构通过常规管理在临床应用中能确保其安全性、有效性的技术。<br/><br/>第二类医疗技术是指安全性、有效性确切，涉及一定伦理问题或者风险较高，卫生行政部门应当加以控制管理的医疗技术。<br/><br/>第三类医疗技术是指具有下列情形之一，需要卫生行政部门加以严格控制管理的医疗技术：<br/><br/>（一）涉及重大伦理问题；<br/><br/>（二）高风险；<br/><br/>（三）安全性、有效性尚需经规范的临床试验研究进一步验证；<br/><br/>（四）需要使用稀缺资源；<br/><br/>（五）卫生部规定的其他需要特殊管理的医疗技术。<br/><br/>第八条<br/><br/>卫生部负责第三类医疗技术的临床应用管理工作。<br/><br/>第三类医疗技术目录由卫生部制定公布，并根据临床应用实际情况，予以调整。<br/><br/>第十二条<br/><br/>医疗机构开展的临床检验项目必须是卫生部公布的准予开展的临床检验项目。<br/><br/>第十三条<br/><br/>医疗机构不得在临床应用卫生部废除或者禁止使用的医疗技术。<br/><br/>第三章 医疗技术临床应用能力审核<br/><br/>第十四条<br/><br/>属于第三类的医疗技术首次应用于临床前，必须经过卫生部组织的安全性、有效性临床试验研究、论证及伦理审查。<br/><br/>第十六条<br/><br/>卫生部指定或者组建的机构、组织（以下简称技术审核机构）负责第三类医疗技术临床应用能力技术审核工作。<br/><br/>而在简政放权以后，笔者也没有找到低频脉冲疗法在临床的许可，也就是说2009年的禁止电刺激治疗网瘾的技术仍在被禁之列，换而言之，杨永信的这个低频脉冲疗法仍然不符合相关规定，仍属违法之列。<br/><br/>哇塞侠，请你好好看看规章制度，别学缩头乌龟，装无辜。<br/>首次转载时间: 2017-02-08 23:05:50"
