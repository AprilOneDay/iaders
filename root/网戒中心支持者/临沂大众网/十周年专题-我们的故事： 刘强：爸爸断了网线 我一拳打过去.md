src="http://linyi.dzwww.com/2015/jwzx/wmdgs/201601/W020160111327280599360.jpg" alt="" /></p><br/><p align="center">刘强获得“优秀学员”荣誉称号</p><br/><p align="center"></p><br/>　　在四院十周年庆典之际，首先祝各位身体健康，阖家欢乐。<br/><br/>做个自我介绍吧，我叫刘强，来自河南邓州，我在2007年5月份入院进行改变，同年9月份出院，是一位老盟友了，2013年7月毕业于郑州大学，目前在北汽（镇江）汽车有限公司工作，出院已经有八年有余了。<br/><br/>下面大概讲一讲自己的转变和出院以后的一些经历吧，也希望能对各位盟友们有一定的作用。<br/><br/>在2004年中招考试的时候，我以高出重点高中录取分数线31分的好成绩，进入我市最好的邓州市第一高级中学。可我没有和其他高中生一样，为自己能考入自己梦想的大学而忙碌，而是一度的跟班主任闹矛盾，由于自身的认知偏差，不能适应她那种管理班级的方法以及单纯追求的分数的理念；也由于自己缺乏吃苦精神，面临着高中课程的复杂和困难面前，自己选择了退缩和逃避。在学校找不到自尊；对身边事务的厌倦；对父母唠叨的错误理解，我被网络游戏的这只手紧紧地抓住了。<br/><br/>那时候，我经常以各种理由欺骗父母，不是同学过生日，就是学校搞活动。想方设法要钱、骗钱、甚至偷钱，来满足玩游戏地欲望。一个电话打回家：“今晚我住同学家”，便一夜不归。妈妈曾经叫上表姐挨个网吧地去找，明里暗里拉网式的进行寻找，当好不容易的找到我的时候，我却撒腿就跑。还说：“谁让你们找我？”也曾说过：“我不姓刘，也不姓姚，没名没姓，不用他们管。”记得有一次妈妈得知我就要放假到学校找我时，我说已跟同学商量好一起去玩，不想回家时，妈妈心脏病当即发作，要我陪她回家，我却没良心的说：“你不会打120啊？”有一次家里的烟不见了，问我拿了没有？自己死不认帐。当听到爸爸说：“家贼难防，偷断屋梁。”我讲出了丧尽天良的的话：“你想死啊？”还有一次玩游戏正在兴头上，爸爸骂着砍断了网线。自己一拳打过去，父亲左眼框上顿时起个大包。<br/><br/>父母不惜花重金将我转入省城一所较好的高中，让我重读高一，希望我能够重整旗鼓，从头再来。自己却没有珍惜这次难得的机会，不但没有悔改，相反越发严重。有一次我模仿班主任签字外出上网被劝退，再次转学后又一次夜间翻墙出去上网被开除。虚荣攀比、撒谎欺骗成了家常便饭。学校一转再转，游戏一玩再玩。磨灭了应有的人性、丧失了自己的良知、荒废了学业、浪费了时间玩掉了肩上的责任。冲动、任性、焦虑、烦躁。那个时候真的是儿子不是儿子，父母不是父母，家无宁日，整天过着非人的生活。<br/><br/>父母为了唤醒自己，带我去深圳开过眼界；求助过同学、朋友、亲戚进行劝说；听过戒网瘾专题报告会；跟中国戒网瘾第一人陶宏开进行过沟通和交流：2006年5月21日，跟陶宏开在一起交谈的大副照片刊登在河南省发行量最大的《大河报》的头版头条上；在郑州市电视台作过周末面对面节目；找心理专家谈过话；让我打工体验过蓝领和白领的差别；还放开让他彻底玩过；妈妈也求过神、拜过佛、算过命，均不凑效。相反，自己在网络的泥潭里越陷越深，不能自拔。<br/><br/>心酸的回忆，实在数不胜数…<br/><br/>即便是这样，我让父母伤心欲绝，痛不欲生，他们仍然不肯放弃我，终于找到了四院这个神奇的地方，07年5月27日父母又把我“骗”到四院，自此我开始了自己打改变之旅。<br/><br/>在网戒中心的日子里，一幕又一幕亲情重现动人场景的感召，这良好氛围的感染，我开始转变自己的思想，开始改变。<br/><br/>在这里，经过3个多月的学习，我与父母重新找回失落已久的亲情，我的家庭重拾温馨和欢笑，我学会了孝和韧，学会了感恩和理解，懂得了宽容和责任，明白了珍惜和诚信，调整了自己的心态，还有很多、很多，在住院后期，我甚至在以一个家长的身份去听杨叔讲话，觉得杨叔是在教自己怎样去维持一个家庭，怎样去教育自己的孩子…<br/><br/>网戒中心，是医院，有真诚服务的医生和护士；是军营，大家身披迷彩装实行军事化的严格管理；是学校，能让人学到一生受益无穷的东西。这里更是一个大家庭，人人互帮互助，相亲相爱。这里有独特的模式，有各种不同的环节，有世间难寻的氛围，这里是一方没有污染的净土。<br/><br/>在这里，我脱胎换骨，可以说是从新活了一次，我也深深地理解到父母的良苦用心。通过自己的努力，我逐渐得到了家长和盟友的一致认可，从入院起的一名普通盟友，到成为放心班的一员，再到班委的一分子，最终我成为了同盟班会的班长。同样是在这里，我明确了自己的目标，许下了自己的承诺：永不走偏，直至成功。同时我也明白，住院的表现并不能代表什么，出院以后才是真正的开始。<br/><br/>07年9月14日，我走出了网戒中心，开始了自己的新生活。<br/><br/>介绍一下自己出院以后的情况吧。<br/><br/>出院之后我的学习情况和思想活动可以分为几个阶段。<br/><br/>一．初回课堂，从零开始。<br/><br/>由于沉迷网络，我的前两个高一没能学到任何东西，之后休学一年，并来到四院进行改变，这三年的时间都荒废了过去，而高二新学期也已经开学一个半月了。出院后，我没有同意妈妈重读高一的建议，而是强烈要求上高二，因为经过在四院的改变，我对于很多事都有了新的认识，我深知道我不能再耽误时间，还要把失去的时间都补回来，于是我告诉母亲：“我的高中不能上六年！”而在刚进班的时候，由于我是一个插班生，老师自然有看法，我自然少不了老师的冷眼。至今我仍然清晰的记得，那时班里已经没有多余的位置了，老师就找了个凳子，往倒数第三排的过道里一扔，告诉我说：“你就坐这吧！”这与之前四院相亲相爱的氛围形成了巨大的反差，心里不知道是什么味。<br/><br/>仅仅几堂课上完，我就发现我真的什么都听不懂。出院时信心满满，现在连受打击，心里十分着急，也伴随着一种失望。我告诉母亲：“妈，我什么都听不懂。”父母听到后，没有责备，而是对我进行鼓励，告诉我能学多少是多少，哪怕是倒数第一，他们也不埋怨我。我知道，自己不能再荒废时光，不能再让父母失望，不管怎么样一定要坚持下去。也许就是这样一种信仰，我在教室里坐稳了板凳。<br/><br/>二．努力学习，有所收获<br/><br/>我还记得在我刚上初一的时候，由于年纪小，比较淘气，给班主任留下了很不好的印象，刚开学不到一个月就叫家长。但是，在第一次考试时，我考了年级的并列第三，一下子改变了老师对我的看法。我也因此明白，老师对于一个学生的评价和看法，在很大程度上源于学生的成绩。但是现在什么都听不懂，很难改变老师的看法。我想到了与老师进行沟通。<br/><br/>有一天，我把自己的这一想法告诉了母亲。之后妈妈找到了班主任，把我的网戒经历告诉了老师，希望老师给予帮助。班主任知道后，也抱着一种同情，对我也自此改观，并把我的位置调到了前列，而且周围都是学习比较好的同学，平时有不懂的就问旁边的同学。在初中的时候，自己学习非常好，带着一种骄傲，根本不屑于向其他同学问问题，但是我知道，自己是四院走出的一员，不应该有这样的心态，况且自己现在什么都不会，一切从零开始，更应该虚心向同学请教，哪怕是学习上去了，遇到不足之处仍应当如此。妈妈也告诉我，不要贪多，自己能听一句是一句，能懂一点是一点，一点一点来，只求学有所得。<br/><br/>就这样，在父母，老师，同学的帮助下，抱着这样一种心态，一段时间后，终于有些内容能听懂了，逐渐有了收获。在期中考试时，名次由倒数第一提升到了班级第六十八名，遗憾的是有几门课程没有及格。父母对此非常满意，而我知道这是父母的鼓励，不想给自己太大压力，自己决不能这么想，不能因此而满足，不能有任何的松懈，也更明确了下一步该如何学习。<br/><br/>三．有所突破，找到兴趣<br/><br/>大家都知道，做什么事只要找到了兴趣，哪怕过程再艰难，自己也会想办法克服，最终得到自己想要的结果，学习自然也不例外。<br/><br/>在经历了一段异常艰难的学习过程后，自己终于在学习上有了突破，由于自己初中时物理的成绩比较好，也很喜欢物理的内容，这种突破出现在了物理这一门课程上。渐渐地，物理这一门已经基本上能完全听的懂了，能跟上老师讲课的进度和思路，有了成绩，自然非常喜悦，就更喜欢这一门课程，产生了兴趣，就更用心的去学习这一门，就这样形成了一个良性循环。在高二上期的期末考试中，成绩有了极大地提高，考到了班里的第二十四名。虽然在物理上有了这样显著地突破，成绩更是直线上升，但是我明白仅仅是这样还不够，自己在四院留下了承诺：毕业时拿着大学的录取通知书来见杨叔。仅仅是这样根本难以实现自己的诺言，难以自己的目标。<br/><br/>四．带动热情，全面提升<br/><br/>有了前面学习物理的成功经验和那一份坚定的信念以及明确的目标，这一阶段的学习已经由最开始的急躁，艰难，慢慢的变成了积极主动，掌握了属于自己的学习方法，已经由物理这一门，延伸到了所有的课程。就这样，物理赶上来了，数学赶上来了，生物，化学……这时，我知道自己虽然在学习上已经有了非常大的突破，但是与其他同学之间还有相当大的差距，而且，对于理科的学习，根本不能偷懒，只有依靠多练习，多做题才能熟练掌握。我给自己定下了一个计划：在每天按照老师的计划学习，完成各项任务之外，每一门都要多做三十道题。那时候，真的除了课间出操，吃饭和上厕所之外，自己就没有离开过座位，晚上，也开着床头的台灯，把每一天老师布置的，自己布置的任务都完成后才睡觉。<br/><br/>就这样，持续到了高二结束的那个期末考试，一下子考到了班里的第十二名，从倒数第一，到六十八，现在到第十二，一步一步都挥洒着自己的汗水，这种成绩也是对自己付出的最大肯定，最大的收获。学习现在已经不是问题了，每一门的成绩都在全面提升。现在甚至能帮助其他同学解决他们不懂的问题，从听不懂，到求助别人，到现在能反过来帮助别人，真的那种喜悦溢于言表，心里异常的骄傲，异常的自信。<br/><br/>五．充满自信，成绩突出<br/><br/>俗话说自信是成功的一半，确实是这样。当时脑子里就有这么一种声音：我一定没问题，碰到难题我一定能够解决，我一定能顺利兑现自己的诺言。<br/><br/>高三的学习主要是复习，备战高考。平常有了更多自由支配的时间，我知道这时更需要严以自律，不能因为学习的全面提升产生任何的松懈。而且，高三的复习有好多遍，一边过后，还要从新再进行复习。此时决不能产生厌烦或者自满的情绪，否则只会是前功尽弃。<br/><br/>自信，自觉，有目标，有动力，高三一年，自己的学习有了一个质的飞跃。不谦虚的说，自己的成绩有了很突出的地方。月考时，我一跃成为年级的第一名。但是我知道，成绩是对自己的肯定和鼓励，而不能成为自己放松要求的一个借口。<br/><br/>就这样，按照老师的计划，配合自己的自主学习。在最后几次模拟考试中，自己的成绩一直很稳定。<br/><br/>然而就在这个时候，一次偶然的身体检查，我被告知身患一种治愈率只有不足四成的大病，医生令我立即住院治疗，杨叔也叮嘱我说健康第一。当时眼前一片黑暗，心里根本不知道该怎么办，但是直觉告诉自己要履行自己的承诺，我再次告诉妈妈：“妈，我的高中不能上六年”。父母在我的祈求下答应我在不住院的情况下进行治疗。在承受病痛的折磨和强烈的药物反应，在发热、口干、头晕、肌肉酸痛、失去味觉等的情况下，自己依然坚持着学习，我甚至自己学会了给自己打肌肉注射。整日没有胃口，吃不下，睡不着……在这期间，父母天天晚上给我送饭，想让我多补充一点营养，风雨无阻，整整一年多的时间，没有少过一次。也许是父母的不放弃，也许是自己的坚持，非常幸运的，在经过两年的持续治疗后，我已经与一个正常人没有任何区别了，战胜了病痛。<br/><br/>在2009年6月份，我带着杨叔，父母，老师的深切期许，带着自己许下的诺言，我走进了高考的考场。最终以班级第二，阶段第八的成绩顺利拿到了郑州大学的录取通知书。也以此回报四院，报答父母。<br/><br/>上了大学之后才发现，大学的生活与高中截然不同。高中老师的严格管理没有了，相反环境格外宽松，需要自己高度的自觉、自律。周围的同学来自全国各地，也不似高中时期只埋头学习，玩游戏，逃课也大有人在，这也更需要自己对自身的严格要求，辨别是非。<br/><br/>在大学期间，说没有放松那是骗人的，这时，四院的长效机制就发挥了巨大的作用。大学时期我依然坚持每年至少回四院一次，充电加油，解决问题。有问题不可怕，可怕的是不敢面对问题，听之任之，这样就会酿成严重的后果。同样在父母，杨叔的帮助下，在自己对自己的严格要求下，我得以同时在班级和院学生会担任学生干部，同时辅修商学院金融专业的双学位，在学习之余还发展自己的爱好特长，担任院篮球队副队长以及其他一些社团的干部。在2013年7月，以院优秀毕业设计的成绩顺利毕业，拿到了两个专业的毕业证以及学位证。<br/><br/>大学四年，我学习到的不仅仅是更深一层的知识，更重要的是把在四院学到的东西运用在生活中，这四年我逐渐成熟，健全了自己的心智，学会了怎样与人相处，学会了处理事情的先后顺序，学会了怎样更好的去融入这个社会。在2013年春季校园招聘时，经过笔试，面试等层层筛选，我依靠自己的能力应聘成功，成为比亚迪股份有限公司的一员。<br/><br/>2015年4月，我再次通过面试跳槽到北汽（镇江）汽车有限公司，这是一个刚刚起步的企业，是一个崭新的环境，公司建设与生产准备同步进行，每天很忙碌但是也很充实，充满了机遇和挑战。在第一批员工培训中，我也获得了“优秀学员”的称号，入职几个月也已经得到了部门领导的认可。12月28日第一台整车下线，标志着公司正式向生产运营阶段迈进，工作的重点也会随着转移到我们身上。我相信在今后的工作中按照四院的标准继续严格要求自己，一定能够取得更优异的成绩，待到那时，一定像以前一样，像杨叔汇报，与大家分享。<br/><br/>各位盟友，人生在世，就应当履行好自己的责任，而何为责任？我们每个人在这个世界上都会随着时间环境的改变需要去扮演各种各样的角色，而怎样扮演好这不同的角色，就是我们的责任。譬如医生的责任就是救死扶伤，士兵的责任就是保家卫国，而我们现在的责任就是努力改变！四院有一句话叫做既来之，则安之。我仅以一个过来人的见识告诉各位：既然我们现在来到了四院，那么请安心下来，为了家庭，为了父母，更重要的是为了自己，去做一些改变，去重塑自己的人生。<br/><br/>最后感谢杨叔，感谢各位才让我有了新生，有了今天的生活。愿每一个盟友都能成为精品，每一个家庭都能重拾欢笑，愿四院越办越好，愿我们的家永远温暖！<br/>首次转载时间:
