align="center"></p><br/>　　说起我与网戒中心，得从2010年高考结束说起，那年我第一次高考结束，考上了青岛大学，在暑假我第一次真正接触网络游戏，开始时也只是消磨时间，娱乐罢了，很快到了开学，进入大学后，一切都与高中不一样了，自由的时间多了，由于自己的自控力弱，加上高中老师给灌输的到了大学就轻松的思想，我有开始上网玩游戏了，开始时也只是在周末偶尔去一下，但慢慢的我上网的时间和频率都在不断增强，对于一般人在工作学习累了，来说上网玩一下，是正常的，但对于我来说，上网玩游戏，确是精神鸦片。<br/><br/>在大一下学期时，我几乎每天晚上都要玩三个小时左右，而这时由于白天还是学一点的，我升到了大二，这是我也发现，上网已经严重影响我的正常生活学习，我也尝试减少上网频率和时间，但实验多次，都均告失败，而且我的上网时间越来越长，我开始翘一些不重要的课去上网，慢慢的我很少去上课了，就这样我由于挂科太多，在大二留级了，留级后，开始时我也非常后悔，也决定一定要戒了网，一定要好好学习，一定要把失去的补回来，但是，在坚持了几天后，我又开始上网了，由于留级只上没过的课，我在这个学期的课很少，而且恰恰在得到留级消息前几天，我买了电脑，就这样，只是在开学前几周上过课，之后一直在宿舍玩游戏，就这样度过了一个学期，而这时我的导员，父母，同学，也都在劝我，要少上网，好好学习，我也明白，我也想少上网，但是却控制不了自己。到了下学期我爸妈专门从临沂跑到青岛看过我几次，就是想让我能好好学习，但是我依旧如此，由于没学习我再次留级，而这次再上学期，我选择了休学打工，希望这样能自己慢慢戒了上网，我爸妈也同意了，在打工的开始几天，我的确没有上网，但是从第一次发工资，我又开始上网了，每次下班，都得上几个小时，就这样一个学期过去了，在我辞职时，我不想先回家，因为回家后就不能上网了，于是我在网吧待了两天，这是在家等着我的爸妈着急了，来到了青岛在网吧找到了我，把我带回了家。由于一个假期在家没怎么上网玩，一开学，我直接钻进网吧待了一个星期，由于过了休学的时间，导员又找到了我的父母，妈妈又从临沂来到了青岛，又在网吧把我找到了，在导员办公室，导员开始想让我退学去戒网，但是我不想，最后，我没有退学，导员让我写了保证书，保证能在下个学期好好学习，但是一张纸又能有什么用？我开始更加疯狂的上网，在这一个学期，我几乎没有上课，前两个月，白天在宿舍睡觉玩手机，晚上到网吧通宵上网，到了后两个月多月，我开始一上上好几天的网，实在不撑了才会学校休息，到了期末考试，一部分科目我根本没去考，考的也全都挂科了，而我心里也清楚，如果补考不过，我就得退学，即使这样，我回到家里也没怎么学习，反而因为一个月没上网，渴望早点开学去上网，于是我在开学日期前一个星期来到了学校，一头扎进了网吧，连续上了一个星期的网，补考开始了，由于一点没学习，我全部没过，又过了一个星期，我收到了被勒令退学的通知，当时我不仅没有伤心，反而有些高兴，因为当时想到终于可以自由自在的上网了，办完了退学手续，我就进了网吧，我爸妈让我回家，我不想回去，因为回家就不能玩游戏了，所以我就告诉我爸妈我要在青岛打工，就这样我在网吧又待了一个星期，正当我花没钱正打算去找工作时，我爸妈来到了青岛，找到了我，把我带回了家，在回来第二天，便把我带到了一个改变我命运的地方――临沂网戒中心。<br/><br/>从青岛回来那天是4月4日，第二天就是4月5日清明节，早上我和爸爸到爷爷家一起上了坟，紧接着爸妈就以给我看眼睛的理由把我带到了网戒中心，刚进入我就发现有些不对，有人看着的铁门，墙上贴满了，各种戒网瘾的宣言，我开始明白自己来到了什么地方，可能因为自己心里也觉得自己该戒网瘾就没有冲动。由于那天是清明节，是集中返院分享的日子，我来到大点评课堂时，当时正有盟友在分享，我当时带着看热闹的心里在听着，而且当时我也不认同中心的模式，更多的是抵触，到了晚上我告诉妈妈，这里是改变不了我的，到了星期一，杨叔就我的问题给我了点评，我当时是一点也听不进去，在大点评课堂和杨叔叫板，当时杨叔也没和我过多的浪费口舌，因为当时的我根本听不进去，到了中午，我做了专场，我表面上顺从了，但在心里就是不服气，也痛恨这里，别时我知道要在这里至少待六个月的时候，我不仅痛恨中心，更痛恨我的父母，但是没有办法，我只能暂时的呆在这，这时周围的盟友和叔叔阿姨开始帮扶我，更重要的是我在杨叔身上看到了一种高尚的敬业精神，中心相亲相爱的氛围在不断影响着我，我开始慢慢的有些转变对这里的态度了，我开始能听进去点评师的话了，我开始能认识到我以前的问题了，开始理解父母的良苦用心了，游戏毁了我好多年的光阴，我后悔自己以前虚度光阴，自毁前程，但是时间不能重来，我明白了改变不了过去就改变现在和未来，我誓与网瘾血战到底，但江山易改，本性难移，多年的诟病岂是几天能治好的？我还需要长时间的治疗。来到中心一个月，我成为了一名班委，紧接着我们班委因为工作不力，杨叔给了个集体专场，那时我懂得了责任，开始时我还以为当上了班委就能偷点懒，但那是我明白班委是盟友们的领头羊，必须做出表率。这时我还是被动改变的，虽然想改变，但不知该如何改变，只能跟着点评师的指导，和盟友们一起经历中心得流程，但随着住院时间的增加，和见到许多精品盟友的分享，在一节节点评课的洗礼下我，内心的污垢被逐渐冲刷，我看到了榜样，当时和我一个小室的盟友左德祥，他在院内的表现非常好，我在他身上看到了那种强烈的主动改变意识，我开始以他的表现来要求自己，而我也逐渐进入了主动改变期，随着住院时间的增长，距离离院的日子越来越近，我有些离院恐慌，我开始犹豫，将来离院后自己干什么，是打工还是继续上学，当时是想上学的，但是因为自己走偏了几年，耽误了很多时间，自己一直有心里压力，后来求助于点评师，兰姐告诉我，我自己要走什么路，要看我心里自己想走什么路，经过一番斗争，在点评师和父母，叔叔阿姨的帮助下，我决定在哪跌倒就在哪里爬起来，即使荒废了几年，只要自己努力也一定能追回来，就这样，我在学习组待了一个月，10月8日，我正式离院了，开始奔赴自己的人生，真正对我的挑战开始了，离院才是改变的真正开始。<br/><br/>10月8日离院，10月9日我就来到了一个复习学校兰补，开始了我的复习高考道路，在学校内，由于年龄的原因，我还是有一定的心里压力的，由于大家都是来复习的，开始时都有一股卧薪尝胆的劲，大家都在努力学习，没多少人关注我，所以在学校学习状态还是很不错的，所以我只能尽量的努力学习，由于我在上高中时底子比较扎实，我学起来还不是很吃力，在11月份的期中考试，我考了560份，成绩应该算还不错，但离考一所不错的学校还差的太远，我必须更努力的学习。刚离院时都还是豪情壮志，但随着时间的流逝，这股豪情壮志也会随着一起流走。所以我一直是每两个星期到中心分享一次，用来充电，给自己加加劲。所以我的状态一直很好，学习也非常的努力，周围的同学开始时也是非常的努力，但是随着时间的流逝，他们身上那股卧薪尝胆的劲慢慢的没了，很多同学开始浮躁，他们开始影响着我，我开始明白了离院才是改变的真正开始，就是在外界变化，自己一直能坚守住自己，杨叔，点评师们，中心，把我们从斜路拉回正途，但自己的人生终究要自己走的，遇到的困难不是都能求助别人的，有些是自己必须面对的。由于经常到中心分享，状态保持的很好，我的学习在不断进步，到了一轮考试，我已经能考到600分了，已经达到了四年前我上高中时的最高水平了，到了下学期，开始时我还是两周分享一次，但是由于周六模拟考试，我开始一个月分享一次，我开始自己慢慢发电，虽然大多数同学在越来越浮躁，但是我的学习动力一直没有减少，但我也好像遇到了学习上的瓶颈，在一段时间内，学习止步不前，但努力是不会辜负人的，虽然最后进步比较少，最后在高考正常发挥的情况下，我考了615分，并被燕山大学录取，但随之也带来了一些问题，一个目标的实现，而没有下一个目标，我的状态也是下滑了不少，鉴于我出问题就是从高考结束开始的，为了能为大学时光打下一个好基础，在暑假我又回到了中心深造，巩固。深造完时间不长，我就又一次走进了大学校园，开学就是军训，由于在中心的锻炼，当然也是轻松就度过了，为了能减轻心理压力，我没有隐藏我的年龄，和同学们相处的也很好，虽然没能当上班委，但这并不影响我的积极性，大学并不是高中老师所说的可以放松的地方，而是一个比高中更需要努力地方，因为没有了老师和父母的约束，一切得靠自己，与自己的惰性做斗争。现在我虽然只是一名普通的大学生，但我会努力成为一名优秀的大学生，将来成为社会上一个优秀的人。<br/><br/></div><br/></div><br/><div
