href="http://www.iaders.com/lywjzx/enemies/4560/attachment/sd4/#main" rel="attachment wp-att-4564"><img class="aligncenter size-full wp-image-4564" src="http://www.iaders.com/wp-content/uploads/2018/05/sd4.jpg" alt="" width="600" height="450"></a><br/><br/>一名16岁男“盟友”的健康教育处方，对服药情况、不良反应和注意事项等进行了说明。大众网记者 马俊骥 摄<br/><br/>大众网记者看到，在一名诊断为“IAD”的16岁男孩的“健康教育处方”中，注明他服用的药物有丁螺环酮、茴拉西坦、帕罗西汀三种，医护人员介绍说，这些药物分别用于缓解焦虑情绪、帮助改善记忆、帮助改善或活跃情绪。医护人员介绍说，IAD指的是网络成瘾综合症，“健康教育处方”中特意使用英文缩写，是防止孩子们讨论“病情”隐私，对心理造成不良影响。<br/><br/>这里的“病人”被称为“盟友”。从病床床头的信息卡中，大众网记者看到，很多“盟友”都是十几岁的未成年人。据院方人士介绍说，现在有100多名“盟友”在此接受戒治，他们不全是未成年人，从十二三岁到近40岁。每张病床床尾都靠墙放着一个可折叠陪护椅，展开后，就变成了一张一人宽的小床。医护人员说，其实病床是给陪护的家长睡的，“盟友”睡的则是这张陪护床。网戒中心要求，每个“盟友”都要有家人24小时陪护，把相对宽敞舒适的病床给陪护家人，是要让“盟友”懂得孝顺感恩。<br/><br/>在一些未成年“盟友”的床边，可以看到英语语法、初中教材、字典、《三字经》、《弟子规》等书籍。医护人员说，这些书是“盟友”自学的，家长中有的人是老师，也会给“盟友”们上课。<br/><br/>杨永信介绍说，在收治的盟友中，不全是因为网瘾而进入网戒中心，有的是因为酒瘾、赌瘾甚至有的沾染上了毒品。<br/><br/><strong>心理点评课</strong><br/><br/>大众网记者采访时，二楼的病房区域一个“盟友”也没有。医护人员介绍说，“盟友”们现在正在上心理点评课。<br/><br/>在网戒中心，会给“盟友”安排心理点评课、辩论赛、团队拓展活动、爱心服务等活动，其中，心理点评课被杨永信和家长们视为“改造盟友”的核心和关键。在媒体报道和网戒中心的宣传展板上，不少“盟友”和家长在心理点评课上受到触动、泪流满面，有的“盟友”显示出幡然悔悟的样子，向家长或杨永信跪倒感谢。<br/><br/>在杨永信的带领下，大众网记者沿着缠着防护网的楼梯走上三楼，进入了一间大会议室。会议室里，100多名身穿绿色迷彩服的“盟友”和身穿便装的家长分成两拨儿，相对而坐。两个群体中间空出一片，是分享点评的地方。大众网记者和杨永信等一众人一进门，随着一声“起立”，“盟友”和家长们齐刷刷站起，“盟友”们大声喊出：“重塑自我，打造完美”，声音雄壮。坐下后，这些“盟友”也保持着挺拔的军人坐姿。<br/><br/>大众网记者看到，“盟友”们看上去年龄不一，以男性居多，其中很多脸上还带着稚气，每个人面前都摆着笔记本。<br/><br/>大众网记者随手简单翻看了一位小“盟友”面前的笔记本，笔记本上工工整整记录着来到网戒中心后的日记和感想，厚厚的笔记本已经写了一大半，其中“父母”“杨叔”这样的字眼频繁出现，表示悔恨、改变的语句出现多次。<br/><br/>这位小“盟友”告诉大众网记者，他今年14岁。他左手小臂内侧，一个扑克牌大小的六角星纹身分外扎眼。<br/><br/><strong>“挺杨”的“盟友</strong>”<br/><br/>大众网记者发现，众多网友组成了“倒杨派”，认为杨永信是“恶魔”，被送来的孩子是受害者，家长被蒙骗。但同时，如果按照上述说法而应是“受害者”的很多家长和孩子，却是杨永信坚定的支持者。于是，这些“挺杨”的家长们也在网上受到了攻击，被认为是遭到洗脑、太愚昧。<br/><br/>网戒中心的医护人员介绍说，“盟友”的家长中有很多是老师、公务员、成功商人、知识分子，网友对于家长“愚昧”的批评很难成立，“有个家长是心理学硕士，他都把孩子送到这里来了”。<br/><br/>8月18日下午的采访中，杨永信找来5位“盟友”和家人，这些人已经出院了，是利用暑假和闲暇时间回来参与“盟友”分享活动的。对于目前网上对网戒中心和杨永信的攻击，他们当中有几位家长义愤填膺，几度哽咽落泪。<br/><br/>蓬莱栾先生的儿子小晨（化名）今年15岁半。栾先生是一名初中教师，以前对孩子简单粗暴，小晨后来迷恋网络游戏，家人将他“骗”到了网戒中心。小晨形容说，“在车上，爸爸带着6个人按不住我”，他已经做好了来这挨打的准备，结果发现这里不是自己想象的那样，很明亮阳光，自己也渐渐发生了变化，如今他已经回归校园了。<br/><br/>另一位前来分享的小辉（化名），他父母在他很小时就离婚了，爸爸酗酒家暴留下了阴影。小辉从三年级开始自闭、打架，后来迷上了网络游戏，常常一连五六天不出门，小辉妈妈甚至曾用绝食自杀的方式感化小辉，结果绝食近5天，小辉回家后却根本没有感觉，最后还是邻居把她送到了医院。在网戒中心呆了7个月，小辉发生了改变，今年中考考了全临沂市200多名的好成绩。<br/><b
