class="TRS_Editor"><br/><div class="Custom_UnionStyle"><br/><p align="center"><img src="http://linyi.dzwww.com/2015/jwzx/wmdgs/201601/W020160111322168565781.jpg" alt="" /></p><br/><p align="center"></p><br/>　　我是盟友雷天翔，时光飞逝离开中心已经15个月了，在中心时有痛苦也有快乐，痛苦并不是因为约束，痛苦是看到了父母的痛苦，快乐的是每改变一点父母得痛苦就减少一点。幸好有杨叔和中心所有点评师我们才有改变的机会，改变的最大受益人是自己，是杨叔帮我们脱离苦海，找到方向，感受到幸福。改变了我们错误的认知和扭曲的心灵，给予我们阳光的心态，让我们学会自我反省懂得感恩，变成正常的人，优秀的人。<br/><br/>简要的回顾走偏的过去，如同地狱一般，自己是无力解救自己、不但自己痛苦，家人更是痛苦，走偏是从初中就开始的，那时是逃学旷课厌学，打游戏上网，整日浑浑噩噩。到了高中我也意识到需要改变，可就是不爱学习，宁愿每天打6个小时篮球也不愿意学一个小看一个小时书，勉强考了2本，上了没一年就因为严重违纪，加旷课被开除，父亲更是为了操碎了心，为了不让我成为废品，赶着我再去考医学院上，勉强上了一个，勉强毕了业，可在那里我也没有认真学过习。毕业后基础的事情也做不好，就连字都写得扭扭巴巴像个小学生，现在想来那时父亲一定着急坏了，我每日过着猪一样的生活，等待的也会是猪一样的命运。要怎么改变呢，自己能改变吗，幸运的是父亲带我来了网戒中心，刚来的时候我是很抵触的，觉得自己就是爱玩的小毛病，没什么大问题，无非每天上网2个小时，慢慢的发现自己的心态有巨大问题，这里不光是单纯戒网，这里教会你如何做自己，如何去面对困难，不逃避，不退缩，树立正确的是非观价值观，这使得自己终身受益，慢慢的也就安下心来，没有几日中心开了一趟下跪课，父母给孩子跪，让我们知道父母为了我们改变为了我们幸福，他们可以连面子都不要，可以为了孩子连生命也不要，对于这些付出盟友们改变就那么难吗？改变最大的收益者是自己，人要有羞耻心，当父母要给我下跪的那一刻我的心要被撕碎了，我拼命的拉住父亲，大声的喊着我改，我改还不行吗，在那一刻我发誓我决不要再把父母逼到这一步，我要改变，必须改变，要怎样改变呢，我当晚失眠了，相信杨叔，跟着杨叔模式走，就这么简单，不知不觉中自己一点点的改变了，首先杨叔教诲我们要有羞耻心，羞耻心是改造命运的开端和关键，也是改造命运的动力。比如有些人偷偷做出不义之事，还以为别人不知道，面无愧色，一天天沦为禽兽却不自知，缺乏耻心，中国的古代圣贤也十分重视“知耻”，孔子曾说：“行已有耻，使于四方，不辱君名可谓士矣”又说：“好学近乎知，力行近乎仁，知耻近乎勇。”孟子说：“人不可以无耻，无耻之耻，无耻矣”又说：“耻之于人大矣。”人活在世界上积极方面要“立志”，消极方面要“知耻”从伦理学意义上来看，耻，是对人的道德行为的一种社会评价，使人们对那些不履行自己义务，损害他人利益，违背社会公德和违反国家法律，有损国格等行为的批评与谴责，知耻是人们对这种行为的羞耻之心，羞耻之感，是人们基于一定社会认可的是非观，善恶观，荣辱观而产生的求荣免辱之心，是人们一种为了维护自身尊严的道德上的反省和自律。人们以这种羞耻感来鞭笞自己，克服缺点，修正错误。羞耻心是人类情绪之精华，正是因为羞耻心存在，才阻止了人类免于堕落，我们在学习工作中一旦落后要能知耻，倘若做昧良心背仁义的事，也要知耻，自己集体国家若受到侮辱更要知耻，这样知耻就能给人上进的力量。能让人更清楚的看待自身和周遭的世界，我们应当把无耻之心看成人生最大的耻辱，这样就能落实于行动，知过必改，受辱必雪，也就不会有自取其辱的事了。<br/><br/>我们要有畏惧之心，即使在很隐秘的地方，没有人看到的地方，做一点小小的过失，天地鬼神也能看的清清楚楚，并给予惩罚，其因果缘起的思想来看起心动念及所为，她们产生的后果“如影随行”不会因外人看不到或个人意愿而改变或消失。《论语》中有云：君子有三畏；畏天命：畏大人：畏圣人之言。“而小人不知天命而而不畏也，狎大人，侮辱圣人之言.”而君子敬畏天命，敬畏处于高位的人，更敬畏圣人的言语：过去我们盟友是没有畏惧心的，不尊重师长，不尊重父母，没有什么人能放在眼里。“畏”的情绪是对个体自身良知的呼唤的一种表现，良知是人类所固有的判断是非善恶的本能，同时也是一切高尚行为如：‘人道，博爱，奉献“等伦理学基石。知道畏惧，就是能够感应良知，明白什么该做，什么不该做，这样才能产生敬畏之心。在现实中每一个人都应对老师，父母，尊长，皆应有敬畏之心，既敬爱又害怕。正因为有“畏”，才会言行举止三思而后行，使之符合于“应当”。<br/><br/>第三点是要有勇心，杨叔常常告诉我们要“知耻而后勇”。人们不能改掉自身过错，多数是由于拖沓和畏难退缩的缘故，因此必须发奋振作，当机立断，不可优柔寡断，不可消极等待。罪过这种东西，小的像钻进肉中的芒刺，应该尽快剔除，大的像手指被毒蛇咬噘，为防止毒液扩散，应赶紧斩断手指，不能有丝毫迟疑。<br/><br/>知耻心，敬畏心，勇敢心。才能知错改错。还有杨叔常说：“要从根挖起，要除根。”下面我们谈谈事理心，就是说人们对于所犯过失的改正，遵循的路径不同，效果不同，有从事情本身进行纠正的，有从事理上加以纠正的，有从心灵上加以纠正的，不同的方式所需要下的功夫不一样，所取得的效果也不大不相同。<br/><br/>首先，从事相上改，比如有个人前天杀害了生灵，现在不再杀生，前天生气骂人，现在不再发怒，这就是从事情本身进行纠正，这种改过方式主要是从外部采取强制干预，所以困难比较大，而且导致错误的根源始终存在，这一方面的过失没有了，那一方面的问题又会产生，因此从事相上改，不是彻底根除过失的好方法，即所谓的“治标不治本”。杨叔常常说要挖根，不要去管那些枝叶，根不去，枝叶会再生。<br/><br/>第2种从事理上改就是在没有从行为上改正以前，就先弄清楚其中道理。在未动之前就主动地去思考自己行为可能产生的后果，从而遏制自己的行为。追根究底，探寻做错事的深层原因，从而找到从根本上避免再次犯错的方法，也是中心常对我讲的从根源抓起，要有后果意识。并学会自省，每天的日记就是自省，自悟，自纠的过程。<br/><br/>而最后一种从心理上改过，什么是从内心纠正呢？过失随是多种多样的，却都源于人的内心，中国有句古话：“相由心生，相随心改”。《般若经》说：“一切法，心为善导若能知心，悉知众法，种种世法皆由心”。对于当今文人学士对待自身存在的徒好虚名，贪幕财力，动辄发怒等问题，我们不必煞费苦心一项项地去探求克服戒除的办法，只要一心一意的培养善良之心地，只想着多做善事，造福他人，那么，这种光明正大的念头就会主导人的身心，邪僻的念头自然就无法乘虚而入，这就像杨叔曾说的：“好比炽热的太阳，高挂晴空，一时间鬼魅魍魉自然就瞬间消失了，而这正是扶正祛邪的关键所在，所以杨叔常对我们讲要有阳光心态。<br/><br/>综上所述，改正过错最佳方法是调治内心，可以立即收效，刚一萌生邪念就有所察觉，刚一察觉就立刻扑灭，如此就不会犯下过错了。毕竟每个人的道德水平和觉悟程度不同，理解有快有慢，即使现在做不到从心灵上立刻改正，也要做到从事理上改正，不可以逃避过错，要勇敢面对。不停的向心灵改进靠近，提升自我觉悟力。人最可贵的就是时时反省自己，时时自我教育。那些伟人都是做到了这些。<br/><br/>在中心得到的是灵魂的升华，道德的提升。拥有了执行力，自省的能力，是一生受用的，是宝贵的人生财富。<br/><br/>离开中心以后，我始终没有忘记杨叔的教诲，从院内改变到院外改变，我继续读一些正能量的书籍，用来巩固自己的心态。15年5月我开始备考。7月份操作考试顺利通过。我开始理论课复习。9月理论课考试通过了。我顺利通过了医师执业考试，一个月后我又参加了药师考试，12月我又参加了研究生入学考试，我感觉学习是快乐的，按照杨叔说的将大目标分成若干小目标，一个个去完成。<br/><br/>杨叔教育我们要迎难而上，面对困难难一阵子，逃避困难难一辈子，并提高了我们的执行力，记得当时杨叔发给每个宿舍一本执行力的书，并让我们写了读后感。那本书对我的触动很大，让我在工作与学习中收益匪浅。<br/><br/>离开了中心这么久，仿佛自己还在中心一样，人在哪都要行为一致，始终严格要求自己，有一个积极阳光的心态，始终不要忘了杨叔说过的话，要明白改变最大的受益人是自己，感谢杨叔，感谢所有点评师，让我做回一个正常人，日后有机会成为一个优秀的人。<br/><br/></div><br/></div><br/><div
