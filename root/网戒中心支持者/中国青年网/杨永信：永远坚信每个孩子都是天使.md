周围围）采访开始前，杨永信一再表示，已经淡出媒体视线好多年了，有些媒体或个人不知出于何意，总是用外行人的视角来评价专业性很强的东西，把治病救人的医学手段渲染成令人恐怖的“摧残”或“酷刑”，使得戒治网瘾的探索一度受到阻碍，甚至差点夭折。为了确保网瘾戒治的科学探索不再受到干扰，杨永信和家长们尽可能地婉拒了各种媒体的采访。<br/><br/>在一番交流之后，杨永信才打开了话闸子。谈及十年的网瘾之争，杨永信说：“任何一种新兴事物的发展都不可能一帆风顺，总要经历一个过程，只有经得起时间的检验和沉淀，才能留下来。”他还提醒，当前互联网已经深入人们生活的方方面面，网瘾问题非但没有减少，而是在增加，只是表现形式不一样了。<br/><br/><b>　　正确面对：网瘾不是猛虎</b><br/><br/>对于网瘾，杨永信仍坚持十年前的观点，网瘾属于心理精神疾病的范畴，是性格缺陷的外在表现形式。<br/><br/>杨永信说：“简单地说，网瘾就是因对网络的不正确使用而不同程度地影响了自己的工作、学习和生活。”他解释道，正常情况下，互联网应该作为一个工具，不会影响到人的身体健康，但是有些人，不把网络当工具，而过度地沉迷，成了网络的奴隶，毁掉了自己，影响了家庭，甚至给社会也带来危害。”<br/><br/>杨永信告诉记者，网瘾的定义其实很简单，但网瘾的成因却不简单，更不能小看。他分析说：“网瘾的成因，主要取决于如下几个方面：一是先后天因素形成的不良性格，如自私、懒惰、贪图享乐、爱慕虚荣、抗挫抗压抗委屈抗诱惑能力差等；二是不恰当的父母养育方式或不良家庭环境，如过分娇宠溺爱、过分简单粗暴和夫妻关系不睦等；三是当今教育体制的弊端和急需改良的学校环境和社会环境。”<br/><br/>“从网戒中心十年来的探索实践中，我们在网瘾戒治的问题上已经形成了一套安全有效的治疗方法，除了以矫治性格缺陷为切入点，更把父母纳入同步诊疗对象。”杨永信说，“其实很多孩子上网成瘾的原因来自父母，多数上网成瘾的孩子家庭氛围不和谐，父母离异或者经常吵架，这样家庭背景的孩子多数情愿呆在网吧，沉溺在网络的世界不能自拔。对于这种情况，我们建立了一套父母和孩子一起戒网的治疗方法，让父母一起参与戒网的学习和治疗。”<br/><br/>“还有一些原因是来自学校教育和社会环境。”杨永信呼吁，“关注青少年上网安全，是每个家庭和社会共同的责任，为青少年营造安全、健康的互联网环境是互联网企业的社会责任。”<br/><br/><b>　　如何救治：孝心召唤人性</b><br/><br/>如今的杨永信，已经不愿过多地谈及过去网瘾之争的问题。他说：“我的职业是医生，医生的天职就是治病救人。网戒中心成立十年，至今还存在，就说明其有存在的价值。而且，在网瘾戒治的基础上，我们还在进行着系列的探索，如其他成瘾行为戒治、青少年性格缺陷矫治、青少年危险行为干预、问题家庭诊疗等。”<br/><br/>关于治病救人，杨永信也有了新的认识，百善孝为先，教育人们有孝心，才能唤回网瘾孩子的人性。“用传统文化的精髓，帮助青少年建立健全的人格，才能保证青少年的健康成长。”<br/><br/>杨永信坚信每个孩子都是天使。他说，“只有不会教育孩子的父母，现在我的大部分时间在教育父母、感化父母。只有父母先做到孝敬老人，孩子才会有孝心，家庭才会和睦幸福。在一个和睦家庭中成长的孩子，自然心智健全。”<br/><br/>杨永信介绍到，很多“走偏”的孩子从网戒中心走出去后，通过自己的努力，已经走出了新的人生道路。他一一细数着，“来自泰安的走偏长达十余年的一位网友，已经当了物流公司的小老板，在婚姻已经几近崩溃的情况下，夫妻又和好如初，而且又生了个大胖小子，前些天还带着父母及老婆孩子来中心分享。来自威海的小霄、小臣、小晓三兄弟，分别于2007、2010、2011年在中心接受治疗，目前他们分别在中南财经政法大学、美国UMHB大学、山东农业大学就读。”<br/><br/>最让杨永信自豪的是，曾经在网戒中心接受治疗的邱扬，从网戒中心出去之后考取了大学，还报考了心理学专业。大学毕业后，竟成了杨永信的同事，传承着这份大爱的事业。<br/><br/>采访快结束时，杨永信说：“最近发布的一份未成年人网络保护研究报告，提出了‘网络素养’的概念，我非常赞同。培养青少年的网络素养，让青少年了解基础的网络知识、学习判断网络信息的价值，就能让青少年远离不良网络信息的毒害，做一个展翅飞翔的天使。”<br/><br/></div><br/></div><br/><footer>中国青年网版权所有?2017</footer></div><br/><footer></footer><footer></footer><br/>首次转
