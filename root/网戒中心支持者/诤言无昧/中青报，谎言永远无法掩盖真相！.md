青少年网络成瘾触目惊心<br/><br/>随着互联网的飞速发展，网络游戏空前泛滥。学校附近网吧密布，未成年人成为网吧的主要客源。无数青少年沉迷网络游戏不能自拔，严重影响身心健康。随之而来的是厌学辍学、贪图享乐、结交损友、淡漠亲情、消极颓废、打架斗殴、偷骗抢劫，有的泯灭人性、仇视社会、丧心病狂。众多的网瘾少年辍学退学，混迹社会，成为社会的不安定因素，他们不仅伤己，而且害人，由网络成瘾引发的恶性事件频频发生，给人民的生命财产带来巨大危害。<br/><br/>青少年是国家发展的后备力量，少年强则国强。但由于受沉迷网络影响，很多青少年不学无术、自私懒惰，罔顾责任与法纪，导致国民素质严重下降，国家发展后继乏人，由此产生的影响是灾难性的……<br/><br/>二、???? 关于临沂网戒中心<br/><br/>沉迷网瘾的孩子家长管不了，学校不要，监狱不收，成了社会的害群之马，严重影响社会安定和经济建设。统计结果显示，这个群体相当庞大。<br/><br/>如何去挽救这些迷途的羔羊？<br/><br/>临沂网戒中心隶属于临沂市第四人民医院，是公办医疗机构，中心成立于2006年，近12年来，网戒中心创始人、副院长杨永信呕心沥血，不断探索，总结出一整套网瘾戒治模式，科学有效，国际领先，6000多网瘾孩子在这里成功改变。这里是融医院、学校、军营、家庭于一体，集网络成瘾戒治、性格缺陷矫治、问题家庭诊疗等功能于一身的综合性救治机构，家长全程陪护。实践中总结出的108个治疗环节环环相扣，相辅相成。每天上午的大点评课是治疗的中心环节，主要对网瘾孩子进行传统文化教育、社会主义核心价值观教育、孝悌教育等。对孩子身上存在的问题进行深层次剖析，帮助他们树立正确的三观，建立规则意识和后果意识，强大他们的内心，变得阳光自信、担当进取、亲情回归，同时对家长身上存在的问题进行点评，纠正家长在认知和管理上的偏差。可以说，临沂网戒中心是国内最系统、最全面的教育孩子如何做一个好人的地方，从这里走出去的孩子比常人还要优秀。<br/><br/>这里不断演绎着魔鬼变天使的奇迹，一个个骨瘦如柴的网瘾少年在这里脱胎换骨，成为学校的佼佼者，一个个支离破碎的家庭在这里重拾幸福。<br/><br/>不良媒体受网游开发商唆使，拿临沂网戒中心使用的低频脉冲治疗仪说事，把普通的理疗说成“电击”。而令人愤慨的是，作为共青团媒的中国青年报，也在未经实际调查的情况下跟风炒作，操守尽失！实际上，这种仪器普遍应用于医疗机构和家庭，是国家相关部门允许使用的，对人体无害。个别网瘾孩子长期在社会游荡，性格桀骜不驯，根本无法进行心理疏导，在这种情况下，使用低频脉冲治疗让其冷静下来，才能进行心理诊疗。就像有病要打针吃药做手术一样，吃药苦打针疼，做手术更要动刀，但都是为了治病啊！<br/><br/>作为全程陪护的家长，我对临沂网戒中心了如指掌。那里氛围温馨和谐，心理点评师循循善诱，到处充满正能量，根本不存在虐待孩子的情况，也从未发生一起医疗安全事故。恶媒谣记恶意忽略了数千例成功改变好的网瘾孩子，片面采信几个未改变好的问题孩子的一面之词，断章取义，歪曲真相，极尽歪曲诋毁之能事，严重违背新闻职业操守，公然为网络游戏鸣锣开道。<br/><br/>三、我儿子的重生经历<br/><br/>我儿子从9岁开始接触电脑，刚开始玩单机游戏，不久就开始玩网络游戏，当时也没怎么影响学习，我也就没控制他，有时我还陪他一起玩，帮他往游戏账号里充钱。初中时，儿子的游戏越玩越大，花钱也越来越多，每周花在游戏里的钱就100多元，学习成绩直线下降，在学校就是混日子。高中第一学期，儿子还能勉强呆在教室，因为基础差跟不上，他对学习彻底失去了兴趣，感觉坐在教室里是一种煎熬，有时逃课去网吧。从高一下学期，儿子就不再去教室，明目张胆地泡在网吧里。现在想起来，高中三年对我家庭来说是噩梦的三年。儿子呆在网吧十多天不回家，偶尔回家一次也是洗澡、换衣服、要钱后接着走人，浑身上下散发着烟味和臭味。我去网吧找他，晚上他坐在电脑前，戴着耳麦，夹着烟，抖着腿，满嘴脏话；早上多数趴在电脑桌上睡着了。这时的儿子，网吧就是他的家，网游就是他的全部。玩游戏要想升级快，就要购买装备。三年来，儿子玩游戏花了6万多元，没钱就问我要，我不给，他就一遍遍打电话发短信，经不住他的软磨硬泡，又担心他铤而走险去偷去抢，每次都不情愿的把钱给了他，其中有3次是4000元的，其他像2000元、1000元、几百元的不计其数。还有一次夜间偷刷我的透支卡3600元，半月之后我才发现。苹果手机先后买了4部，先是买了苹果5，摔坏后买了苹果5S，又摔坏了，就又买了一部5S，苹果6刚上市，又软磨硬泡的买了一部，不给他买，他总能捏着我的软肋，一次次的承诺以后好好上学，拿完钱接着就原形毕露。其他的冤枉钱也花了很多：把同学的相机丢了，赔了5000元，把人家的平板电脑摔坏了，赔了3600元。儿子走偏这几年，我花在他身上的钱算起来有10多万元。没有钱就用透支卡提现，由于不能如期归还，我的个人征信有7次不良记录，后来买车连车贷都不能办理。<br/><br/>在网吧，儿子结交损友，抽烟喝酒打牌，有时还打架斗殴。到临沂网戒中心之后，经过教育改变，他开始自我晾晒，我才知道他原来做了那么多令人后怕的事情。比如：晚上往人家里扔烟花，和损友去人家里打架。更为可怕的是，他的朋友圈里有两个吸毒的，幸好他还没有沾染毒品。眼看着儿子这样颓废下去，我心急如焚又毫无办法，和老婆是三天一小吵，五天一大吵，那日子过得是黯淡无光、生不如死。<br/><br/>天无绝人之路，经人介绍，2015年11月，我把儿子带到临沂网戒中心，向杨永信院长求助。<br/><br/>儿子在临沂网戒中心经过10个月的改变，变得阳光自信，亲情回归，规则意识和后果意识都很强，树立了正确的三观，彻底摆脱了网瘾的诱惑。现在的他，亲情好有礼貌，内务卫生做的好，学习动力十足，和父母沟通交流好，做事沉着冷静，注重体育锻炼，和同学友好相处，能够明辨是非，早已远离过去的损友。<br/><br/>是杨永信院长拯救了我的儿子，改变了我家庭的命运，如果没有临沂网戒中心，等待我儿子的不是监狱就是地狱。<br/><br/>乌云遮不住太阳，正义终将战胜邪恶。作为临沂网戒中心的见证者和受益人，在此奉劝中青报，作为共青团媒，应坚持操守，伸张正义，切莫背弃宗旨，沦丧操守，往国媒脸上抹黑！<br/><p
