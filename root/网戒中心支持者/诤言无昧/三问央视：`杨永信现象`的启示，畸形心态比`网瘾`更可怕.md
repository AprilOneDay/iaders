/><br/>作为一名教育工作者，作为临沂网戒中心的审视者，有一些话不得不说――<br/><!--more--><br/><br/>有时候，我觉得今天的人们似乎很难耐烦去完整地读一篇文章，譬如我刚写的这些。<br/><br/>任何事物都是不断完善而不会绝对完善的<br/><br/>事物的发展都是一个不断完善的过程，但无论这个事物已经有了多么悠久的历史，在它发展的每一个阶段每一个时期，似乎都不会绝对完美。<br/><br/>譬如教育。经过几千年发展的现代教育，每个学校甚至每个班级都会存在一种失败，都有一些无论怎么教育都无法阻止他们破坏教学秩序的学生：扰乱课堂，欺负同学，甚至报复老师――这种现象过去有，将来似乎也不会消失。<br/><br/>应试教育体制普遍受到诟病，尽管今天我们在努力于教育公平，但是在举国关注的高考面前，似乎还没有一种更好的体制取而代之：没有了应试这把尺子，你更难保证教育公平。教育公平不会一蹴而就，也不会有绝对的教育公平。<br/><br/>目前，很多学校对学生玩手机扰乱教学秩序很无奈，而不得不在取得家长同意并签字的情况下，担风险颁布一些诸如手机进课堂进教室的禁令，尽管遭到很多学生的抵制――但是，社会认可了这种现象，没有哪几个家长愿意让手机毁了孩子的学业。<br/><br/>再譬如医院。没有哪个医院可以绝对杜绝医疗事故，越是权威医院，就诊者越多，出现医疗事故的几率就越高。<br/><br/>再再譬如监狱。无论社会的道德体制、教育体制多么健全，但是教育不是万能的，道德约束也不是万能的。无论哪一个多么民主发达先进的国家，都没有否定监狱这种国家暴力机关和警察这种国家暴力力量的合法性而取缔。<br/><br/>凡此种种，社会早已认可并习以为常。<br/><br/>而多年来从事问题孩子、问题家庭救治与探索，并已经作出重要突破和巨大成就的杨永信，就没有这么幸运。<br/><br/>简单还原真实的杨永信<br/><br/>谁都不会否定互联网的伟大意义，但他的确是一把双刃剑。因病态痴迷网络而导致大量青少年人格扭曲、心理变态、丧失社会功能甚至走向犯罪的案例早已屡见不鲜，而且并未随着互联网的常态化而消失或降低，杨永信的出现也是历史的必然。<br/><br/>其实杨永信也并不完全是 “戒网瘾”，由于杨永信发现网瘾的出现缘于孩子的性格缺陷，所以他在10年的探索中构建起了108个环节，这里面大量的内容就是科学的精神引领、正确的价值观重树、不良的习惯矫正和社会功能康复，并特别注重健康向上的氛围营造。所以来到这里就诊的孩子不仅仅是网瘾患者，游走于毁灭边缘或者已经涉嫌犯罪的问题少年，包括一般的精神疾病患者等都有。而杨永信改变的不仅仅是孩子，也有问题家长。<br/><br/>作为一名长期从事教育工作，曾经多次走进并从教育的视角审视过临沂网戒中心的人，我所见证的真相是：杨永信的心理点评课充满了辩证法的智慧，充盈着积极向上的力量，和刚柔相济的处世之道，让众多浪子回头，也让众多濒临毁灭的家庭重拾幸福。他是一位不是教师但绝对堪称优秀教师的教育工作者。<br/><br/>同样，杨永信的戒治模式，也不是完美的。<br/><br/>包括备受质疑诟病的“电击”（ 低频脉冲治疗仪）。但是，正常的学校都避免不了学生的暴力干扰，面对这样一个特殊的群体，你能保证每一个孩子不会情绪狂躁失控，自觉服从秩序，平静接受教育与改变？<br/><br/>改变这样一个特殊群体，有那么一部分改变不好的孩子，在不良媒体和利益者的怂恿下，用那样一种畸形的心态和视角对临沂网戒中心进行“爆料”，这不很正常嘛？<br/><br/>对杨永信的舆论绞杀，源于畸形的社会心态<br/><br/>一是利益之争。从发起对杨永信二次炒作的雷斯林（蔡江舟），到顺利接棒冲刺于央视的林熊猫，都是与网络游戏利益攸关的开发与推广者。而跟风起哄的水军，我不敢绝对肯定，但大多应该是游戏玩家。<br/><br/>二是浮躁的媒体和媒体人。媒体靠受众吃饭，爆料炒作是惯常手法。在离婚的王宝强超过国民对奥运会和G20峰会的关注，人肉马蓉的艳照已胜过仰视吴敏霞10米跳台英姿的“舆情新常态”下，对负面新闻变着花样爆炒，早就不是新鲜手段。<br/><br/>三是媒体人不具备与岗位相称的职业操守与素养。如果说浮躁仅仅存在于一些山寨市井媒体也罢，但令人惊诧的是，这种现象同样也不同程度地渗透于主流媒体，甚至身为国家喉舌和国家主流意识形态引领者的央视（个别部门或个人），这就不能不引起我们的高度警惕。CCTV2的《第一时间》和CCTV13《新闻周刊》在并未对当事者进行真相考证的情况下，片面采信不良媒体甚至是带有严重个人利益色彩的自媒体资料，其主持人更是草率予以播报，尤其是张腾岳那种与严肃新闻话题极不相称极具主观色彩的煽情与夸张，造成了严重的舆情误导。而CCTV13《新闻周刊》主持人侯丰对电竞战队新闻的植入，是否对广大青少年产生误导甚至对基础教育造成干扰，更欠考量。<br/><br/>央视个别媒体人与岗位职责严重背离的职业操守与素养，让我们深感疑惑：这里面有没有更深层的利益关联，也未可知。<br/><br/>正是因为央视的参与，导致舆情的单向倾斜：正面的澄清根本无法表达，杨永信陷入铺天盖地的站在“道义”制高点的舆论围剿。而央视随后的冷漠更令人心寒：明明已经听到真相的呼喊，为什么在事后不即时理性调查纠正？为什么眼睁睁看着正义在舆论误导下被否定被绞杀？究竟是意识到后果的严重但碍于央视的脸面在良心的谴责中徘徊不定？还是期待着更加荒谬的佐证验证悲剧的合理？难道不知道作为大国央视的使命与职业操守吗？<br/><br/>我觉得，央视的正确态度应该是：理性审视杨永信存在的社会价值，充分肯定其客观成绩，并针对社会质疑帮其不断完善治疗模式，以发挥更大的社会效益。<br/><br/>四是所谓临沂网戒中心亲历者的视角扭曲。<br/><br/>无论天气多么晴朗，无论阳光多么灿烂，总有那么一些人在少有的阴暗处，用阴冷的视角观世界，正如老鼠喜欢阴暗，正如蟑螂喜欢潮湿。这种人无论在什么国家什么体制什么社会形态下都无法融入主流。当中国共产党领导中国走出改革开放初期的困境，正在实现大国崛起的时候，当西方世界发展陷入困境凸显我社会体制优越的时候，有个别极端势力仍然与国外势力勾结，抓住一点阳光下的阴影无限放大，妄图颠覆政权分裂国家。而在临沂网戒中心，也有一部分孩子，由于人格扭曲与是非观混淆，当大部分孩子接受正确的价值引领阳光开朗积极改变的时候，他们对正确的引导总感到十分刺耳，对合理的规章制度感到极度不适，对来自同伴善意的批评监督视若黑社会的无间道。<br/><br/>说到底，这种孩子如果心态不阳光，在哪里都不会和谐融入社会，在任何环境下都会被边缘化。这种心态如果不能得到及时矫正，久而久之的边缘化，可能最终会导致他们走向社会的反面。不信？你可以去实地调查他们在学校在社会在家庭中的表现。<br/><br/>但就有那么一部分所谓的专家，还真把他们当回事，拿他们这些所谓“见证者”的爆料窥一斑而见全豹了。如孙云晓，如王岳。<br/><br/>“专家”点评和历史上的“28个半”布尔什维克<br/><br/>当我在央视或其他媒体上看到或读到这些专家对杨永信的点评，还真是“醉”得不轻！<br/><br/>他们是真正的专家，对法律法规条文和有关专业规范谙熟于心并准确表之于口，开口人权，闭嘴法律，句句确凿，条条属实。<br/><br/>当年以王明博古为首的28个半布尔什维克，从苏联留学归来，经纶满腹，背诵《共产党宣言》和《资本论》，比他们熟练多了！尤其是那个德国军事顾问李德，背诵阵地战正规战的战略战术，也是口若悬河。他们讥讽并很瞧不起毛泽东发明的“山沟里的马列主义”。<br/><br/>管用吗？还不是把毛泽东与朱老总苦心经营的中央根据地都丢了？！导致了中国革命史上惨烈悲壮的长征！彭大将军说，这叫崽卖爷田不心疼！<br/><br/>一个对问题孩子教育毫无实践经验与事实成就的人，一个待字书阁搞纯粹理论推演的理论家，借用一句时髦的话说就是：弄啥文艺范？装个什么逼？<br/><br/>干了30多年的教育，这种孩子我见得多了，我敢说：如果把杨永信收治的孩子随便挑出一个交给王岳或孙云晓，你还没开口理论，他就让你满脸开花遍地找牙了！<br/><br/>把一群这样的孩子放在一起，如何？<br/><br/>再好的理论不结合实际也是一句空话，真正解决问题的，还是来源于实践又反作用于实践的“山沟里的马列主义”。<br/><br/>说到底，最能说明问题的，还是每逢节假日携妻带子怀揣奖状胸佩军功章来对杨永信千恩万谢的昔日问题少年！<br/><br/>畸形的心态比“网瘾”更可怕！<br/><br/>良心不会泯灭，真相总会大白于天下。<br/>首次转载时间:
