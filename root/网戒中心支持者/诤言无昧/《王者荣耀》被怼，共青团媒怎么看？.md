晚上打通宵，第二天到学校上课没精力听课，课上打盹，当父母老师去劝导时，也不理会，依然我行我素，甚至是省吃省喝的拿平日的生活费、零花钱充到游戏中，更严重的则是偷了家里的钱，甚至用威胁的手段坑害父母！游戏成为了他们的主业，而学习、父母、老师、未来都显得无足轻重，这样的事例有很多，甚至说对于成年人都有相当比例的影响。公交车上，排队时，上班时，吃饭时，拿着手机打游戏的人随处可见，如果这样继续发展下去，未来的人类社会会变成不敢想象的模样。<br/><br/>最近我还发现了一个奇怪的现象，媒体们对《王者荣耀》展开批判，比如人民网连发三文批判《王者荣耀》歪曲历史，新华社更是对于《王者荣耀》给予痛批，光明日报也对《王者荣耀》发表了自己的看法，但是在国媒们痛批《王者荣耀》的时候，同为国媒的我们的共青团媒去哪了呢？为什么他们唯独对“网瘾”闭口不提？在网络游戏严重危害青少年健康成长的时候，他们何去何从？甚至在三月份，我们的团媒还在为《王者荣耀》更能激发学生对传统文化的学习兴趣大声叫好，这真的是在为青少年好吗？游戏就是游戏，它的本质是玩，但是学习却不一样，它是需要付出的，如果大量祖国的花朵都集中于游戏里，那祖国的未来谁去负责？<br/><br/>更令人不可理解的是，就在国媒集中火力猛怼《王者荣耀》的时候，我们的团媒还在为游戏开脱，说什么少年沉迷游戏的第一责任人是家长和老师，意思是家长教子无方，老师教不好学，才让孩子无所寄托沉迷游戏，看来游戏开发者是没大责任了，他们只管开发游戏赚钱而不用考虑承担什么社会责任了！而一旦游戏开发没有了良知与担当，以赚钱为根本目的，那么开发出来的游戏也一定不会给青少年带来什么好处。<br/><br/>这样一来岂不苦了老师和家长们了，他们该怎么改变教育教学方法才能把孩子从游戏里拽出来呢？<br/><br/>我希望共青团媒能理性看待“网瘾”这一社会问题，如果青少年痴迷网络荒废学习，不仅担不起历史重任，对于子孙后代更有着相当严重的危害，望共青团媒能看清现实，牢记宗旨，理性做出正确行动。<br/><p
