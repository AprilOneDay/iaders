/><br/>外孙浪子回头，重获新生，让我看到了一个真实的杨永信<br/><!--more--><br/><br/>本文的作者是一位盟友的外公。在外孙住院期间，作者亲临临沂网戒中心，感受了中心积极向上的氛围，聆听了杨永信医生的心理点评，见证了外孙浪子回头、重获新生的神奇变化。今天，他以自己的切身感受为大家再现了一个真实的杨永信。受作者之托，在此转发全文，以飨读者――<br/><br/>由于我的外孙痴迷网络，逃学厌学，被女儿女婿送到临沂网戒中心改变。在此期间，我有幸聆听了杨永信医生的一堂心理点评课，感受了相亲相爱一家人的氛围，见证了外孙可喜的改变，看到了一个真实的杨永信。<br/><br/>那是2015年夏天的一个周末，在临沂网戒中心的大课堂，能容纳几百人的课堂座无虚席，北面为盟友（对孩子的称呼）席，南面为家长席，我也像一位家长学员一样端坐在一旁，全神贯注着每一种动向，一百多名盟友个个身着迷彩服，精神抖擞，俨然一群钢铁战士，正在齐声高唱《跪羊图》、《网梦醒来》等歌曲，特别是一首《网梦醒来》不禁让我老泪纵横，唱出了多少孩子和家长的心声，沉迷网络孩子的迷茫和无助，家长那种痛心和绝望。<br/><br/>上午九点二十分，杨医生（家长和盟友对杨永信医生的尊称）身穿白大褂走进课堂，由班长总结盟友们前一天的表现，杨医生再根据具体情况进行心理疏导，尤其是表现不好的盟友，帮其找出原因、指出方法、提出目标，激励他们做好。杨医生接下来点评了一个家庭，说孩子出了问题，沉迷网络，各方面的原因都有，孩子本身的性格短板，认知偏差、自控力差、没有后果意识和规则意识是主要原因。但家长的不良的教育和教养方式，还有不良的家庭氛围对孩子的影响，也有不可推卸的责任。所以改变孩子的同时，家长必需做出改变，杨医生时而温和、时而严肃的对这个家庭进行点评，父母由开始的委屈抱怨、痛苦流涕，到后来的大彻大悟、破涕为笑，让我由衷的佩服杨永信医生精湛的医术和高尚的医德。杨医生还讲了一些励志的事例，如：陈州身残志坚，虽然失去了双腿，但以坚强的意志，用双手支撑登上泰山、九华山等多座名山，还娶妻生子，生活的幸福美满，而且还攀登了很多的名山。杨永信医生用这个实例，鼓舞在座的盟友，你们正值少年，风华正茂，是学习拼搏的大好时期，你们有什么理由不好好读书，而沉迷网络。电脑是学习的工具而不是游戏玩具，手机是通讯工具，也不是玩具，我们应该健康上网、绿色上网，而不是沉迷其中，不能自拔。<br/><br/>最后杨医生让家长说说自己孩子的转化情况，很多家长激动万分，叙述着各自的“遭遇”，有的痛哭流涕、几度哽咽说不下去，通过在中心几个月的学习教育，孩子像换了个人似的，成了一个可爱的、阳光的、有自信的孩子，几近毁灭的家庭得以重生。很多家长的困境也让感同身受，多少个家庭被网络依赖症害得无法正常生活和学习，很多家长到了崩溃的边缘，走投无路的情况下来到中心，向杨医生跪求：“救救我的孩子吧”，那种绝望和无助只有亲身经历才能感受的到。一上午杨医生从人生的意义、志向、孝道文化、家教等十几个环节讲的头头是道，盟友和家长听的认真、入神。从上午九点二十上课到下午两点三十，杨医生苦口婆心的讲了五个多小时，没有吃午饭也没有上厕所，由衷的佩服杨医生的敬业精神。在一上午的听课中，我也不时的流下泪水，这么好的孩子沉迷网络、荒废人生是多么可惜啊！遇到这样的情况家长有多心痛啊！也让我深深的认识到当今社会如何教育好孩子，使之成为栋梁之才还有很长的路要走，同时也对杨医生由衷的敬佩和敬仰。以下就自己的感受简要阐述如下：<br/><br/>一、低频脉冲治疗，我作为孩子的外公，隔代亲，对外孙的感情自然不用多说。更是对孩子深深的爱，和希望他快点好起来的深切期盼，使我非常理解这种治疗方式。网瘾孩子长期受网络的侵害和吸引，思想上会更执着那种虚拟的世界，用寻找刺激来解脱原来那种不愉快压仰的心情，长期下去思想会产生幻觉而误入岐途，用网络环境替代了现实的生活环境，患者因产生情感交流障碍，变的孤僻、自我封闭，或暴躁、焦虑，不能正常休息、生活和学习。老师的话不听，家长同学的话不听，实际他们也听不进去，更控制不住自已的情绪。在这种情况下，必须采取一种措施，用低频脉冲治疗使其头脑瞬间清醒，回归正常思维。才能有机会接受新的符合社会主流的思想。<br/><br/>二、真实的社会现象。由于当前的网吧经营乱象丛生，在网吧上网的90%是未成年人，按要求未成年人不得进入，然而这些青少年往往吃住都在网吧，从而造成了严重的网络消费，沉溺网络中的孩子不能正常的生活，个个面黄肌瘦、无精打采，严重影响身心健康。这些青少年从开始边上网边上学，逐步发展到辍学，他们便成为了网吧的常客，通过网络上的虚拟幻想，年幼的心灵扭曲，黑网吧这个大染缸成了毒害青少年的集中营，在网吧发生群殴、猝死等事件屡见不鲜；有的青少年辍学后拉帮结派、称兄道弟、打架斗殴成为社会上的小混混。学校的欺凌事件不断发生，在校生与辍学的青少年内外勾结收取保护费，他们存在吸烟、酗酒闹事、打架斗殴、盗窃抢劫、绑架他人，为获得网络消费这些青少年无所不用，而针对未成年人犯罪行为，公检法部门只能采取说服教育得方式，但导致青少年犯罪的根源与“网瘾”脱不了干系，大部分未成年人犯罪后都是在网吧被抓获。未成年人的犯罪率逐年上升，如果不能够引起社会的高度重视，后果不堪设想，而对这样一群整天游手好闲、吃喝玩乐、不思进取、惹是生非、沉溺网络的孩子，在学校老师管不了，在家家长束手无策，在社会上无人来管，任其发展下去必然走上犯罪的道路；网络的发展是大势所趋，但带来的负面影响大家有目共睹，网瘾的危害还没有被各级政府部门重视。对于整天上网的孩子不学习、网聊、六亲不认、打爹骂娘，作为家长心里是何等的滋味？面对社会现状，杨永信勇于探索，敢于创新，承担了很大的社会责任，这是所有患者家长的唯一希望，十多年来临沂网戒中心挽救了数千名走偏的孩子，对社会的贡献巨大，我们应该为杨永信的这种担当点赞，各级部门应该积极倡导杨永信的这种有效的模式。<br/><br/>三、临沂网戒中心的模式先进，效果显著。孔子是伟大的教育家，他的教育思想是中国文化的瑰宝，两千多年来对社会、对人类发挥了不可替代的作用。杨永信通过多年的潜心研究，古为今用，用孔子的教育思想教育网瘾和极端问题孩子，他从中国的孝道文化教育入手，让孩子背诵《三字经》《弟子规》，并理解运用到现实生活中，使变态的心灵得到重塑。杨永信在多年的探索中也找到了病根，不良的家庭环境、家长对孩子的过分溺爱或简单粗暴也是网瘾青少年的诱因之一。在临沂网戒中心孩子得到教育的同时，家长也受到教育，使父母与孩子之间达到畅所欲言、打开心结，彻底从根上解决孩子的问题，这也是杨永信模式的最大亮点之一；杨永信用集体的智慧、团结的力量，用正能量的氛围唤醒了如同“恶魔”般的孩子，使这些孩子变得阳光、自信，学习有了动力，生活有了信心。临沂网戒中心的模式安全有效，我们应该为杨永信的这种创新精神点赞。<br/><br/>四、网络媒体的不实评论，对杨永信极不公平。一些别有用心的人为了自己的利益而大肆攻击杨永信，让英雄流血又流泪太不公平。作为任何新生事物的诞生、发展都不是一帆风顺的，有非议在所难免，作为个别媒体用外行的眼光去审视内行的行为真是荒唐至极，试问，你们有更好的办法能拯救这些走偏的孩子吗？有的媒体和个人歪曲事实、颠倒是非、无中生有，试问，你们的良心何在？杨永信承担着巨大的压力，工作任劳任怨，每天工作十几个小时，拯救了无数的孩子和家庭，其所做的事业利国利民，相信有良知的人都会支持他，我们应该为杨永信的担当精神、执着精神、无私奉献的精神点赞！<br/><br/>经过临沂网戒中心的改变，我的外孙已经出院一年多了，现在阳光、自信，无论是学习还是亲情，各方面都发生了翻天覆地的变化，事实是最好的证明！是杨永信医生挽救了我们家族几代人的命运，我们全家永远感激、感恩、支持杨永信医生！<br/>首次转载时间:
