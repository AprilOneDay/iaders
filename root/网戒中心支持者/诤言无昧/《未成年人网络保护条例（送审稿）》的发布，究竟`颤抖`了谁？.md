<br/><img src="http://wx4.sinaimg.cn/crop.0.0.293.165.1000/0067hHJjly1fbop8922yjj30bu04lt91.jpg" alt="" /><br/>――杨永信风波问题解析之七<br/><!--more--><br/><br/>1月6日，国务院法制办针对《未成年人网络保护条例（送审稿）》（以下简称送审稿）面向社会征求意见，使持续半年高烧不退近日稍渐平寂的杨永信风波再起波澜，让“倒杨风波”的炒作者们“荷尔蒙”再度暴走，某些追风媒体如“南方都市报”之流似乎又抓住了一次“热一把”的机会。<br/><br/>甚至个别主流媒体也经不住“借热炒出风头”的诱惑。这也难怪，市场运作下的资讯丛林法则，面对“杨永信风波”炒作的“大锅烩炖”，谁愿作壁上观不分杯羹呢？<br/><br/>他们强调：“禁止‘电击治网瘾’，更要重申‘网瘾非病’”。<br/><br/>他们高呼:“颤抖吧杨永信!”<br/><br/><img src="http://wx4.sinaimg.cn/large/0067hHJjly1fbopbn3gglj30gl0cdabi.jpg" alt="" /><br/><br/><strong>?请问：网瘾究竟是不是病？《未成年人网络保护条例（送审稿）》为谁发布？谁在“电击治网瘾”？真正颤抖的应该是谁？！<br/><br/>其一，怎么看待《送审稿》用“沉迷网络”这个模糊的概念替代“网瘾”？</strong><br/><br/><img src="http://wx1.sinaimg.cn/large/0067hHJjly1fbopffctz3j30pp05naan.jpg" alt="" /><br/><br/>尽管《送审稿》未提“网瘾”，甚至用“沉迷网络”这个模糊的概念替代，但无法否认伴随互联网而出现的网瘾事实。即使在互联网高度普及的今天，因病态沉迷网游而产生的恶性案例依然屡见各种媒体而有增无减――沉迷网络到了极不正常的状态，甚至伴生了精神病状引发了恶性案件，不仅扰乱了正常的教育教学秩序，更给家庭带来了灾难，破坏了社会的和谐稳定――当这种现象已经上升到社会问题，并严重威胁青少年健康成长的时候，这难道还不能称之为病？――真有必要回避“网瘾”而掩耳盗铃吗？<br/><br/>而就在眼下，常州一男子连续上网18小时后猝死，这种司空见惯的新闻，对“网瘾非病说”是不是莫大的讽刺？<br/><br/><img src="http://wx4.sinaimg.cn/large/0067hHJjly1fbopiazia1j30j90azdgj.jpg" alt="" /><br/><br/>基于这一前提，对比三个月前的草案与此次公布的送审稿后发现：在草案基础上，原文“对未成年人网络成瘾实施干预和矫治”中的“矫治”二字被删去；“通过购买服务等方式提供预防和干预未成年人沉迷网络的教育或服务”，“推动出台网络成瘾的本土化预测和诊断测评系统，制定诊断、治疗规范”的相关内容也被删去――这是否意味着用“对未成年人沉迷网络实施干预”涵盖了以上所有内容？<br/><br/>如果用“沉迷网络”涵盖“网瘾”，这无可厚非，但若为回避炒作或来自某方面的压力，或迎合某些利益群体，刻意淡化“网瘾戒治”，这就不是向社会向国家向民族负责的态度。<br/><br/><strong>其二，《未成年人网络保护条例（送审稿）》的发布，主题是什么？</strong><br/><br/>统观《送审稿》，真正强调的核心,是规范网络行为，净化网络空间，杜绝网络暴力、不良信息和低俗游戏对青少年的侵害，防控青少年沉迷网络，对网络游戏无度开发推广设限立规，为未成年人的健康成长营造一个绿色的网络生态。而规范对沉迷网络的干预只是作为其中的一项重要措施出现。至于卫生计生部门应承担的职责，则是对防控无效的网络沉迷状态进行干预，“推动出台网络成瘾的本土化预测和诊断测评系统，制定诊断、治疗规范”应该涵盖其中。<br/><br/>针对增加的“任何组织和个人不得通过虐待、胁迫等非法手段从事预防和干预未成年人沉迷网络的活动”条款，也犹为重要，目的是为了限制不规范的“网瘾戒治”行为。<br/><br/>但是，一些媒体在对《送审稿》进行报道解读时，刻意冠以一些故作惊人的标题，硬扯上“电击”或者“杨永信”，似乎《送审稿》的发布，不是为了保护未成年人健康上网，而是为“打倒杨永信”量身定制的条例，真是滑天下之大稽！这种<strong>回避主题混淆黑白刻意误导</strong>的恶俗之举，有辱媒体人社会使命的庄重严肃。<br/><br/><strong>其三，“网瘾戒治乱象”和杨永信戒网瘾，究竟有几毛钱的关系？</strong><br/><br/>这里需要澄清的真相是：杨永信的网瘾戒治中心，是在网络立法不健全，网络监管不力，垃圾信息充斥，低俗网游盛行，黑网吧遍地，网瘾肆虐，青少年健康成长受到严重威胁的社会背景下成立的。作为一所三级甲等精神专科公立医疗机构，该中心是基于社会责任和专业医疗背景为前提成立的下设科室。成立11年来，在科学分析网瘾病理成因的基础上，构建起网瘾戒治和性格缺陷矫治的综合干预模式，成功救治了含网瘾在内的数千例问题青少年，未出现一例伤害性事件，这是无法抹杀的客观成就和社会贡献。说得更具体一点，<strong>是成功避免了数千患难家庭悲剧的产生，而不是“南方都市报”所信口雌黄的“动辄被电击的痛苦更让他们终身难忘，不少地方因此出现家庭悲剧”。</strong><br/><br/><img src="http://wx4.sinaimg.cn/large/0067hHJjly1fbopkzrau0j30hu0bj0uk.jpg" alt="" /><br/><br/>这里还需要澄清的几个概念是：网络上把“电击”“低频脉冲治疗”“虐待”混为一谈，纯属恶媒谣记刻意混淆的恶俗之举。当孩子因过度沉迷网络而把虚拟与现实混同，甚至人格扭曲、性情畸变、社会功能缺失，更甚者出现幻听、幻觉、妄想等精神病状时，杨永信根据个体不同，采取生物学治疗与心理治疗为主导的综合干预模式，并把“低频脉冲治疗”作为特殊患者的辅助性治疗手段，不能等同于“电击”，更不能等同于条例中的“虐待”,背离“低频脉冲治疗”的专业背景，混淆与“电击”的实质区别，是无意还是故意？杨永信的戒网瘾与“虐待”有半毛钱的关系吗？<br/><br/>令人不解的是，这些追风的记者或媒体把矛头指向杨永信时，为什么不去更多的关注追究究竟是谁制造了这么多的网瘾患者呢？受谴责的究竟是杨永信还是网络不良信息的传播者和低俗游戏的制造推广者？正是网络立法的滞后，监管机制不健全，和网络游戏开发商的唯利是图，才造成了黑网吧遍地，和因过度沉迷网络而出现的众多网瘾患者。<br/><br/>近期，以南方都市报“用法律终止电击治疗网瘾乱象”为代表，恶意把与临沂网戒中心毫不沾边的恶性案例硬扯在一起，并有意使用“网戒中心”作为所有网戒机构的名称混同并提，造成集天下网瘾戒治乱象之大恶于杨永信的错觉，把杨永信作噱头，别有用心误导舆论，手段之卑鄙令人不齿。<br/><br/><strong>最后，说到底，《未成年人网络保护条例（送审稿）》的发布，真正应该为之颤抖的，应该是在网络空间浑水摸鱼，钻法律的空子用不良信息和诱惑无底线的低俗游戏残害青少年的非法牟利者；而真正为之欢呼的，应该是十几年如一日担当道义保护未成年人健康成长的杨永信们。</strong><br/><br/>一项严肃的法规出台，作为肩负使命伸张正义的媒体和媒体人，岂能因博眼球牟利而刻意媚俗有辱使命呢？！<br/><br/>站稳舆论市场的法宝不是盲目跟风蹭热度，而是坚守使命的严肃与崇高。倘若不讲诚信如赚一把就跑路的奸商那样，图一时痛快而哗众取宠，最终只能砸了媒体的牌子，落到“21世纪传媒特大新闻敲诈案”那样的万劫不复的悲惨下场。<br/><br/>?//搬运工:
