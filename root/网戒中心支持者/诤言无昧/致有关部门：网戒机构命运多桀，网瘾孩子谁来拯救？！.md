class="WB_editor_iframe"><br/><br/>??网瘾戒治被妖魔化，网戒机构频遭关停，网瘾孩子无人拯救？哪些部门难辞其咎？<br/><br/>我认为，责任，并不难厘清。<br/><br/>备受瞩目的豫章书院风波终于以关门而大吉了。从杨永信二次被炒作到豫章书院事件风波再起，除了中国青年报杨鑫宇之流乘机借妖魔化网瘾戒治又火了一把，别有用心者随手捆绑了一下他人之外，一如既往的诿责于家长和学校，似乎不会有人问责于有关部门，有关部门便也就事不关己再次高高挂起。<br/><br/>然而，家长们的灾难却深重来袭。<br/><br/>11月7日，澎湃新闻记者在豫章书院大门口看见不时有家长带着孩子乘车离开。有家长见记者靠近，大声斥责称，“我们已经被你们害得这样了还想干嘛”。当晚，一位不愿透露姓名的学生家长告诉澎湃新闻：“学生、学校、家长现在受害不浅。我家孩子在里面好好的……现在没书读了”。<br/><br/>这是实话。但这个实话在网瘾戒治妖魔化已成为媒体思维定式的时代，没有人站在家长的立场上想一想。<br/><br/>作为网瘾和网瘾戒治的见证者，孩子罹患网瘾之后的表现啥样？什么是真正的网瘾患者？可谓触目惊心：连续上网，通宵达旦，废寝忘食，忘记责任，放弃学业，长发披肩，蓬头垢面，形容枯槁，目光呆滞空洞，幻听幻觉，焦虑强迫，抑郁自残，行为极端，暴躁易怒，心理变态，人格扭曲，社会功能严重退化，拒绝任何心理救助。因为网瘾，过劳猝死或走上违法犯罪的青少年屡见不鲜，而因为找不到针对性戒治机构，被生不如死的父母送到精神病院的不在少数。<br/><br/>我无意为家长和学校诿责，但再优秀的家长也有管不了的孩子，再好的学校也有教不好的学生，这就是事实。网瘾孩子大抵属于这一类，学校教不了，家长管不了，监狱收不了。没有了戒网瘾学校，面对这群心理行为严重异于常人，动辄对父母拳脚相向甚至挥刀舞棒的“魔鬼”，好多父母被迫无奈只好把他们送进精神病院，更多的是晚上睡觉时把门死死插上，生怕网瘾儿子半夜性起杀将进来，天天过着担惊受怕的日子。<br/><br/><b>戒治网瘾，要从矫正行为、重树三观、健全人格入手，当属于特殊教育或精神医疗范畴。</b>为了让这个特殊群体接受教育改变，戒网瘾学校几乎都在家长的默许下心照不宣地采取了一定的保护性约束措施。但由于缺少法律保护，如履薄冰，因为一旦出现一例安全事故或一次曝光引发的恶意炒作，就足以摧垮一所、甚至一批网戒学校，无论你此前挽救了多少问题孩子，拯救了多少患难家庭，获得了多少荣誉，赢得了多少家长的感恩。<br/><br/>这，就是当今网瘾戒治的现状――顶着污名，敏感、脆弱而不堪一击。<br/><br/>可以肯定的是：家长对豫章书院的“教育措施”是知情的，也是默许的――学校不要，监狱不收，送到网戒学校让孩子迷途知返重新做人，至少比送进精神病院或将来抓进监狱强得多。<br/><br/>而爆料者――被强制戒网瘾的孩子对“虐待内幕”的表述，显然带有强烈的感情色彩，且有夸大的成分。<br/><br/><b>豫章书院关门了，但这些孩子流向哪里？答案是肯定的：即使在政府的强力干预下分流到普通教育学校，也是暂时的。因为这是一群已经被学校赶出门（甚至是多次被赶出门），已经被证明是普通教育无法改变的孩子。随着岁月流逝，不否认其中个别孩子可能会自我省悟、浪子回头，但大多数孩子最终流向社会，或流浪街头，或走向监狱，或被强送精神病院，而成为社会潜在的不安定因素。</b><br/><br/>一个孩子就是一个家庭的全部，孩子如此，责任谁负？媒体把锅甩给家长和学校，显然是不公平的，而且是对患难家长的二次伤害！<br/><br/>游戏盛行网瘾肆虐，全国几乎所有普通教育学校（尤其是高中）都在禁止手机进校园，早已是不争的事实。网瘾作为一种社会病，让青少年荒废青春学业，诱发犯罪，早已成为社会共识。但有些媒体甚至是主流媒体，像中国青年报中青在线杨鑫宇之流，由开始跟风炒作到后来成为妖魔化网瘾戒治的旗手，他们秉承一种混账逻辑――网络普及，网瘾消失，一切戒治网瘾的机构和手段均为非法，网瘾患者皆因家长不会教养学校不会教育所致。借着共青团媒的虎威，与万千网瘾少年的呼应，形成一股妖魔化网瘾戒治的强大舆论，误导了公众，绑架了政府，严重破坏了舆论生态。<br/><br/>若如此类推，毒品危害在于吸毒者不自律，与毒品产业无关？！<br/><br/>请看中青报言论：“……不能动辄就把青年人群的文化符号妖魔化……让青年脱离主流文化控制并‘野蛮生长’，未必有我们想象的那么可怕”――《 怒怼《王者荣耀》更该反思什么》。<br/><br/>如此罔顾事实的恶意炒作，不能不让人怀疑媒体与资本的利益关联。<br/><br/>――中青报杨鑫宇之流背弃宗旨操守沦丧，为游戏利益站台，责任谁负？应该非常明确。<br/><br/>网络游戏，也被称为指尖上的印钞机。游戏产业畸形膨胀，无序开发，内容恐怖、低俗、刺激，诱惑无极限。崛起的游戏产业借助强大的资本力量，广告无处不在，各路明星纷纷代言，无数青少年纷纷沉迷，无数家庭陷入灾难，恶性案件屡见不鲜。网瘾如此为害社会多年，即使是在《王者荣耀》被全民声讨的时候，马化腾仍高调做客某大国媒网站，舆论随之倒向，媒体矛头公然指向家长学校，腾讯股价则迅速反弹。<br/><br/>随即，马化腾取代马云成为国家首富。<br/><br/>――是谁纵容游戏产业无序开发恶性膨胀？是谁为游戏利益站台？网络环境如此污浊无序，责任谁负？应该非常明确。<br/><br/>网瘾是不是病？该如何界定？<br/><br/>十几年来，网瘾早已成为家庭之灾、社会之痛、教育顽疾。在公众眼里，网瘾就在身边，就是病。然而，我们的相关部门每次炒作，都会出台草稿、草案、征求意见稿。而每次炒作一过，则束之高阁再无下文。没有统一的定义标准与诊疗规范，每次炒作最受伤害的就是网瘾戒治机构。网瘾戒治在中国已被利益集团及傀儡媒体污名化，而戒网瘾机构也从最初的四、五百家，经历次炒作关停，目前所剩无几。<br/><br/>无视网瘾少年的沉沦，无视家长的泣血跪求，无视患者的心理行为病变多么明显，对网瘾患者的一切保护性约束措施都是限制人身自由，都是虐待，一切网戒机构均属非法都要取缔。<br/><br/><b>谁去设身处地考虑过网瘾患难家庭的利益，和家长生不如死的痛苦？</b><br/><br/>――责任谁负？应该非常明确。<br/><br/>有确凿消息源证实：在明年年中将发布的ICD-11中，“游戏成瘾”将与合成毒品、酒精、烟草、咖啡因、非法药物等同样被列入物质使用及成瘾行为障碍。<br/><br/>这是否预示着网瘾戒治的名正言顺？<br/><br/>在世界卫生组织正式把“游戏成瘾”纳入精神疾病分类与诊断标准的同时，政府有关部门能否接纳并迅速推出配套的治疗规范或指南？当面对舆论恶炒时是否会被再次绑架？《未成年人网络保护条例》的出台是否依然难产，或再次被利益集团歪曲利用？<br/><br/>而游戏产业在国民经济中的分量，资本利益所拥有的强大话语权，永远不可忽视。这场看不见的“网瘾战争”究竟谁胜谁负？<br/><br/><b>习总书记指出：“我们党来自人民、植根人民、服务人民，一旦脱离群众，就会失去生命力。加强作风建设，必须紧紧围绕保持党同人民群众的血肉联系，增强群众观念和群众感情，不断厚植党执政的群众基础。凡是群众反映强烈的问题都要严肃认真对待，凡是损害群众利益的行为都要坚决纠正。”</b><br/><br/>望有关部门不忘初心，勇于担当，真正让人民利益高于一切。????<br/><br/></div><br/>首次转载时间:
