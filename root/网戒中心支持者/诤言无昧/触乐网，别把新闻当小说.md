class="preface">――面对恶媒谣记的谎言，谁才是真相的见证者？</div><br/><div class="preface"></div><br/><!--more--><br/><div class="WB_editor_iframe"><br/><br/>??近期我看了一篇文章，是触乐网写的《和杨永信战斗的人们》。看完后，觉得与事实严重不符，我作为临沂网戒中心的一位盟友，有必要给以澄清。<br/><br/>首先，有意思的是这篇文章的题目，仿佛和杨永信“战斗”的这群人是特别光荣的，整篇文章给人的感觉也是这样――杨永信是邪恶的，而和他“战斗”的人是正义的。<br/><br/>文章的开头，是从2016年8月27日开始的，这个时间正是临沂网戒中心被炒作之初，而此时我刚入院不久。我是2001年出生的，今年16岁，，我用自己的亲身经历来说说自己对这篇文章的看法。<br/><br/>有一个叫“小辰”的人在客运站，6小时后将离开临沂，小辰在客运站等得累了，但却不敢睡，身体还在一直发抖。其中传递的意思是小辰在害怕，并回忆起了一些不好的经历，因为过分的恐惧，身体才会颤抖。接着看下一段，出现了宋醒，殷雄，又追述了小辰微博上发布4篇讲述他在临沂网戒中心“真实经历”的文章，可知这个小辰是和自己一样都在网戒中心住过的盟友，但文章的题目让自己十分的惊讶――《磁爆步兵杨永信》，为什么小辰是与杨永信敌对的人呢？我不能理解，因为在我眼里，杨永信医生是我们的杨叔，是和我们这些问题孩子患难家庭休戚与共的好人。小辰与殷雄相识之后，文章中说他发给殷雄“求救短信”，再联系一下前面说小辰似乎有了危险而逃亡，究竟是何种危险呢？与小辰，宋醒，殷雄同样的是，赵蕾也是反对杨永信，与杨永信作战的。她与殷雄一起想“解救”盟友，并认为可能有危险，而这危险是因为他们反杨永信，与中心作对。<br/><br/>这令我非常纳闷，因为我所认识的中心并不这样，首先临沂网戒中心是属于临沂市第四人民医院的下属机构，是国家卫生部门，再者，杨医生也不是他们说的恶人，他们的身份和人品根本不会去做违法的事，我所看到的，是他们在教我们如何做一个有责任担当的社会人，在兢兢业业地全力挽救我们这些严重走偏的问题少年。<br/><br/>这难道说我所认识的杨医生和文章里描写的杨医生不是同一个人吗？小辰的逃亡是不是因为他患有被害妄想症呢？<br/><br/>后面写到了赵蕾与一些离院后的盟友们建立了QQ群一起“反杨”。我认为他们的做法更加无法理解，改变好的孩子多了去了，他们对网戒中心和杨医生非常感恩，这些反杨的盟友肯定是没有改变好的孩子，是不正常的人，他们并不能维持自己正常的学业和生活。而有一个盟友的父亲在被赵蕾联系后，竟然索要5万块钱，这其中就有许多漏洞。首先一个父亲的儿子如果被害，要起诉肯定是自己主动起诉，而不是别人给了自己钱自己才起诉，而赵蕾说是她让该父亲作证，“是顺便帮他自己”，这“顺便”就是次要，主要还是帮赵蕾反杨吧？为自己孩子作证还要钱，是作伪证么？看来人家并不领情，这十分不符合逻辑。<br/><br/>再往下看，赵蕾还写了《中国恐怖故事：疯诗人》，据赵蕾说是“真人真事改变的”，那么在结尾还有一句话“在文章末尾处，赵蕾虚构了主角跳楼自杀”，这不是诅咒人吗？赵蕾为了虚张声势达到反杨的目的，完全不顾事实真相，毫无底线可言。<br/><br/>再往下看，因为赵蕾写的这篇文章有很多虚构成分，而且并未采访到当事人，带来了不良影响，所以拒绝了媒体采访。文中有这样一句话“不想再捅到报纸上去了。”这就是为了逃避，因为害怕，曝光了只能让自己感到难堪。<br/><br/>而小辰在文章里被塑造成网戒中心的“受害者”，而在“反杨”后，家里受到家委会的骚扰，给那些不了解真相的人的印象就是中心的家委会因为小辰反杨，而去报复甚至抓人。这个谎言说大了吧？法治社会有这事吗？110这么方便，派出所近在咫尺，你逃亡什么呢，自己吓唬自己还是无中生有？再说了，我爸就是家委成员，为人特和善，他是家长们集体推举的，工作就是全心全意为家长和孩子们服好务，并不存在什么实质的权利。<br/><br/>文章把中心叫做“集中营”，而且有“点评课记录”，说记录了盟友被抓的过程和方法。这纯属子虚乌有。先不说这个本子的真伪，就拿自己来说吧，自己是因为出现了问题才被自己的父母送来中心，而不是像文章中所说的那样是中心将自己抓过来的，医生没有权利也没有义务去抓病人。在问题孩子的眼里，不仅中心不好，可能整个社会整个世界都不好。<br/><br/>文章还说在中心里的患者大多数是因为家庭矛盾来的，并不是网瘾。以自己来做实例，我自己就是因对手机过度依赖不上学才来的，文章里面写的根本没有实际依据。<br/><br/>殷雄为了保护小辰，搞的像是为拯救受害者要大义赴死一样，还搞了一个什么募捐，募捐当然是好事，但他们所做的募捐并不是正义的，因为你们的逃亡根本就是欺骗读者消费观众骗取钱财的拙劣把戏！<br/><br/>文章说中心因为炒作而封锁信息，又说盟友打电话会被人拉走治疗，这更扯淡了。因为自己就在中心里面住，中心不允许患者使用手机，哪来的打电话？<br/><br/>再往下看，赵蕾都有了男朋友还天天晚上跟异性聊天到半夜，不仅言语暧昧露骨，而且频繁见网友，还只隔十公分的距离与一位没改变好的孩子在一起走了六七个小时；<br/><br/>再往下看，说中心被炒作之后，盟友们每天要磕500――1000个“磕头操”；<br/><br/>再往下看，说中心三月份“磕头操”改为了“电击”；<br/><br/>读完全文，感觉就是胡扯，完全是一篇虚构的小说，再看小辰赵蕾陈默，言行非正常人所为。整篇文章实际就是作者为一己私利胡说八道。<br/><br/>杨医生及各位点评师都是我们的人生导师。自己之前认知偏差，放弃学业，沉迷虚拟，打爹骂娘，伤害父母，也伤害了自己，是特别愚蠢的，多亏临沂网戒中心的帮助，自己的身体比以前好多了，思想端正了不少，学习动力也持续提升。因为自己受益于中心，感恩中心，所以证明真相维护正义，是我的义务。<br/><br/>我并不指望所有的人都认可中心，但最起码，我有义务让大家认识一个真正的临沂网戒中心。<br/><p
