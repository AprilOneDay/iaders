/><br/><br/>套路玩得不错，但节操碎了一地<br/><br/><!--more--><br/><br/>一个以推销兜售产品为宗旨的互联网商业平台，如果采写发表所谓的“社会新闻”，你觉得这类新闻会长啥样？你会不会马上联想到当年充斥大街小巷的“莆田系”推销保健品（性用品）的那些“牛皮癣”广告，想到那些不堪入目但又十分醒目的随脚在马路上踢翻一页纸屑都会出现的诸如“七旬老翁找小姐，小姐竟累到求饶”等等如此这般的文字？<br/><br/>一个以推广网络游戏为目的，以游戏利益最大化为宗旨的网站或媒体，如果采写发表关于网瘾戒治的社会新闻，现在让你发挥想象力，大家会想到什么样的场景？<br/><br/>大家肯定会觉得这个问题的假设很搞笑。<br/><br/>但，这绝不是假设，而是很现实，而且远超想象，譬如――触乐网就能写，而且写得不赖。<br/><br/>去年8月份，戒网瘾的杨永信被游戏商再次爆炒。2016年9月7日，触乐网连续推出六篇文章：《被杨永信改造的孩子们》，《“盟友”小辰：逃离临沂》，《“盟友”曾宇：感觉自己死过一次了，活一天就赚一天吧》，《“盟友”王安：这件事我永远也没法原谅他们》，《“盟友”李林峰、王明阳：大家不要太关注锤子本身》，《触乐夜话：杨永信其实和游戏关联不大》，《律师观点：最好的结果是，异地公安能立案调查》）――一夜之间，成功救治数千例网瘾问题孩子，为青少年健康成长做出突出贡献、享受国务院政府特殊津贴的精神卫生专家杨永信，就被丑化成摧残青少年的“电击狂魔”!<br/><br/>紧接着，在2016年9月8日晚，“两年没动静”的莉莉丝在北京召开新品发布会，一口气推出了六款游戏！<br/><br/>这套组合拳你能说打得不漂亮？这也难怪，互联网时代的广告推广，哪能是莆田系时代的“牛皮癣”所能比的？！<br/><p align="left">今天，尽管杨永信风波热度渐渐平息，但有人还是看到了蹭热度的剩余价值。2017年3月20日，触乐网再次推出了以杨永信为话题的文章――“<a href="http://www.chuapp.com/article/282529.html" target="_blank">和杨永信战斗的人们</a>”，署名杨中依。</p><br/><p align="left">撇开内容采写的真伪，单从写作风格上讲，如果说去年9月7日这一天推出的六篇文章还有点像新闻的模样，这篇“<a href="http://www.chuapp.com/article/282529.html" target="_blank">和杨永信战斗的人们</a>”已经基本找不到新闻的影子，甚至连多少有点文学色彩的报告文学都不沾边。杨中依放开手脚轻装上阵，全然抛弃了新闻采写的纪实风格，全篇采用的是小说笔法，而且充满虚幻悬疑，大玩穿越。乍一读的确很吸引人：现代版的奥斯维辛集中营，1930年代上海滩帮斗的无间道，有组织有保护伞背景的秘密抓捕，三个热血满腔充满侠肝义胆的拯救者，在强大的黑恶势力面前进行绝望而顽强的抗争。</p><br/><p align="left">确实，读来起伏跌宕，悬疑紧张，抓眼球赚点击，令不明真相者义愤填膺！</p><br/><p align="left">但是，这点一捅即破一戳就穿的江湖把戏是根本瞒不住理性者的眼睛的。</p><br/><p align="left">不得不说，套路玩得不错，但节操碎了一地。真正有生命的媒体靠的是对职业操守的坚守，是社会责任的担当；但绝不是哗众取宠、混淆视听、制造矛盾扰乱社会的胡编乱造。</p><br/><p align="left">第一，请不要忘了，这是共产党领导下的人民民主的文明法治社会</p><br/><p align="left">杨永信，一名中共党员，一个地级市公立三级甲等精神专科医院的副院长，在专业背景下成立了临沂网戒中心，因为成功救治数千例网瘾问题孩子的贡献而享受国务院政府特殊津贴，曾经分别被临沂市委宣传部和山东省委宣传部作为关心青少年成长、促进未成年人思想道德建设的典型在全市全省进行集中宣传，并因此获得荣誉表彰无数。</p><br/><p align="left">然而就是这样一个人物，在触乐网的文章里，却被塑造成打着网瘾戒治的旗号，专事摧残青少年的“电击”狂魔，在光天化日下居然还经营起一座现代版的奥斯维辛集中营，豢养着专事全国抓捕的别动队（家委会）？！更令人惊掉下巴的是，这种摧残是在家长全天24小时陪护之下，而被摧残的孩子家长居然甘当帮凶，还对杨永信感恩戴德!被摧残的孩子是无病而健康的，孩子被摧残的理由仅仅是因为青春叛逆家庭矛盾！</p><br/><p align="left">在触乐网杨中依的笔下，各级党政机关不是盲聋就是渎职不作为，甚至甘当保护伞，杨永信就是十足的虐待狂，孩子家长都变态有病，而孩子是无辜的。</p><br/><p align="left">这样的弥天大谎居然也有人信？！</p><br/><p align="left">第二，杨永信到底有多大的能量，让这么多“小鬼”为之推磨？</p><br/><p align="left">在作者的笔下，杨永信成了无所不能的人，究竟“能”到什么程度？历数古今，从正史到野史，可举三例：智圣诸葛亮的智慧，巫圣公孙胜的法术，富商马云的财力。这可不是夸张，你看呐，从2009年至今，两次风波，诸多媒体，加上被舆论误导的吃瓜群众等等的口诛笔伐，居然风雨如磐屹立不倒，各级职能机构居然为之站台背书，患难家长们对他衷心拥护。杨永信不仅照样“摧残”孩子，还豢养着一支能抓善捕的“别动队”。这得需要多大的多少的“保护伞”？需要多大的权力保护和财力支撑？！</p><br/><p align="left">杨永信有权吗？一个地市级三级甲等精神专科医院的副院长，充其量也就是一名业务带头人而已，能掌握多大的权力资源？</p><br/><p align="left">杨永信有钱吗？作为体制内的专业技术人员，他没有经商的资本也没有经商的时间，更没有权钱交易的筹码，医院科室收费标准完全按照物价部门的要求，而且还略低于其他。那他的钱从哪里来？难不成是那些脑子进水的家长？那些“脑残”家长不但把没病的孩子送给杨永信“电击”摧残，还得拿自己的血汗额外“孝敬”？</p><br/><p align="left">难道杨永信真的是《水浒》中的入云龙公孙胜附体，会念念有词放咒施法？――这当然是扯淡。</p><br/><p align="left">答案只有一个：杨永信的确做出了贡献，而且是实实在在的贡献。这个贡献就是拯救了堕落沉沦的网瘾问题孩子，挽救了水深火热中的家庭。他赢得了家长的信任，社会的认可和人民政府的肯定。</p><br/><p align="left">第三，可怜可叹的“拯救者”</p><br/><p align="left">文中有三个古道热肠侠肝义胆的“拯救者”――好心人宋醒、自媒体工作者殷雄和女大学生赵蕾，事实上，宋醒在文中只不过是个受人之托的客串，真正的“拯救者”只有殷熊和赵蕾。这两个人都有一个共同特点：他们都是被来源于贴吧和QQ群里的几个未被临沂网戒中心改变好的问题孩子被摧残虐待的自述激起了义愤，都是以网络传言作为“誓言拯救”的依据，都是以问题孩子的自述为信息来源。</p><br/><p align="left">这些孩子提供的信息究竟有多大的信度？网瘾的危害自不待言，早已是公认的社会现象、社会问题。试想一下，学校里有没有因为老师的教育管理而反目成仇、恩将仇报的学生？在杨永信的网戒中心，面对这样一个群体，有些孩子拒绝接受教育而仇恨杨永信不是太正常了吗？</p><br/><p align="left">然而，这成了殷熊赵蕾誓言拯救的理由。</p><br/><p align="left">在炒作风波的持续中，10余年间受益于临沂网戒中心和杨永信而成功改变的几千例受益家庭和网瘾孩子持续发声，用事实说话。他们一概不听。</p><br/>当然，这里也不排除他们已经听到了。但是，他们无法接受被欺骗而徒劳的战斗这个现实，依然在用虚幻的正义为了自己的脸面和自尊继续做近似于偏执的坚持与支撑。<br/><br/>请看赵蕾：她把自己关在小黑屋里，反思自己的过去，开始思考自己这么做的原因，最后，她得出了结论：“我自卑，我想去保护别人。因为他们跟我很像，我是想快点走出阴影，不想再被影响了，才能去帮他们……如果真的能救出里面的人，我会很快乐。说不定就放下来了……差不多是想救自己吧。”<br/><br/>……<br/><p align="left">赵蕾开始伤害自己，有时候她在寝室里砸东西，放声大哭，跪下，用手打墙，割腕。她并不是为了求死，只是“希望能把一种痛苦换成另一种痛苦。”有一天，陷入绝望的赵蕾将一把刀带回寝室，她养的乌龟一直在边上爬。她心里想的全都是杀掉乌龟。</p><br/><p align="left">再看殷熊：殷雄甚至写了“遗书”。想过“自杀去搞一个大新闻。”殷雄已经想好了，“大不了就是死嘛。”他还找了好多朋友挨个说：“如果我出事了，我家里还有个老娘，麻烦你帮我照顾照顾，安抚一下情绪。”</p><br/>――这说明了一种可能：他们在怀疑自己，不是怀疑自己的力量，而是怀疑自己是不是做错了，是不是做出了和拯救的初衷完全背离的傻事？！<br/><br/>但是他们不敢再往下想，于是就有点歇斯底里，甚至抓狂了。<br/><br/>而且，如果没有这种怀疑，你本该平静的大学生活受到骚扰，为什么不拿起法律的武器捍卫自己的权利？究竟是你伤害了别人，还是别人伤害了你？你有这个自信吗？！<br/><p align="left">醒醒吧，可怜的孩子们，一篇扯淡文章也能激起你那点可怜的成就与自豪？还在微博里自认赵蕾。这值得你骄傲吗？</p><br/><p align="left">一个以推广网络游戏为目的的平台，能不妖魔化网瘾戒治吗？要不然怎么继续从这些网瘾孩子手里攫取无辜家长们的血汗？</p><br/><p align="left">这样的文章，尽管比不上莆田系推广春药的所谓“案例”低俗，但实际上是品位是一样的。</p><br/><p align="left">你想想，连真中医都未必能深得祖国医学文化的精髓，何况是“洋中医”，他能写出个什么东东？</p><br/>首次转载时间: 2017-03-23 17:34:19"
