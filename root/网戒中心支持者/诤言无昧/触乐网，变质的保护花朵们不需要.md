/><br/><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？<br/><br/><!--more--><br/><br/>亲爱的朋友们，大家好！<br/><br/>我是来自山东临沂网戒中心的一个孩子，今年十三岁。<br/><br/>在来到网戒中心之前，我已经有两年时间没去上学了。在此之前，我一直都是那种乖巧文静的孩子，亲戚们、老师们包括同学们都很喜欢我，我的成绩也一直保持在班级前五名。但是在最近两年的时间里，我出现了一些严重的心理问题，开始变得极端，变得封闭自我。<br/><br/>不肯去上学的这两年里，我在家闭门不出，最长的时候有两个半月没迈出房门一步。为了照顾我，妈妈直接辞掉了工作全天在家陪我，但我毫不领情，不愿和妈妈交流，在我的观念里，妈妈并不理解我；并且由于自身敏感多疑，很容易因为极平常的一句话导致情绪失控，甚至在一次情绪失控后，我偷偷地吃了一整盒安眠药。我有时会问自己：我到底怎么了？爸爸妈妈带我去了全国各地的精神卫生中心，做过几十次一对一的心理咨询，但是并没起什么作用，直到今年一月份，我来到了山东临沂网戒中心。<br/><br/>我不愿承认自己受到了网络的毒害，说出去显得很丢人，但如果要我自己来判断：到底是网瘾影响了我的心理健康，还是因为我心理出了问题才开始沉迷网络，我真的很难回答。<br/><br/>到现在为止，我来到中心已有两个半月了。<br/><br/>在这两个半月的时间里，在心理医生们的鼓励下我开始重新尝试着和妈妈沟通。从最开始每天都会有各种各样的矛盾产生，到现在我已经半个月没有和妈妈发生矛盾了，我惊喜地发现妈妈没有自己想象的那般“封建”，而且妈妈自身的变化在很多时刻感动着我。我和盟友们也能相处得很融洽了，我们经常互相帮助，互相指出对方的错误并互相监督，我很享受这样的改变氛围，比如说以前在家的时候我经常发脾气，一天要“爆炸”好多次，来到中心后开始也有一段时间还不能控制好情绪，但每次我情绪化的时候身边的人都会及时提醒我：小心，你又要闹情绪了！人的习惯确实很难改正，但是中心就是这样帮助我们克服心理难题的地方。<br/><br/>除了大氛围，中心的每一个环节也都让我受益匪浅。每天的心理点评课能让我学会很多道理，通过其他盟友出现的心理问题举一反三，对照自身，做到有则改之无则加勉；课堂上对自律的严格要求让我不再一味地想着放松自己、随心所欲，我学会了对自己高标准、严要求；日记课让我巩固消化了点评课的内容，锻炼了我的思维能力；思品工作让我学会了更合理地去安排时间，锻炼了我的语言组织能力，特别是观察问题和影响他人主动改变的能力；网上作业让我细细地反思了我的过去，知道了应以积极的心态和正确的方式去面对未来；辩论赛让我变得敢于表达自己的想法，提高了我的反应速度和表达能力；帮家长洗脚捶背让我学会了尊重父母，增进了我与父母间的感情；小室演讲让我提高了临场语言发挥能力；自我反省让我每天都能“三省吾身”，建立自我审视机制，从每一个细节开始改变；规律的作息时间让我不再顶着两个黑眼圈，现在每天我都充满了活力；打扫小室卫生提高了我的劳动能力；纠偏信念操锻炼了我的身体，历练了我的意志，改变了我的认知；忆苦思甜让我知道了现在的生活来之不易，应该好好珍惜。每一个环节都有它存在的意义，让每一个盟友不断提升自己，变得更加优秀。<br/><br/>而触乐网杨中依发表的那篇――《和杨永信战斗的人们》严重失实。文章中写“今年三月份中心又恢复了电击”，首先，中心从来不存在“电击”这回事。赵蕾称她写的《中国恐怖故事：疯诗人》是“由真人真事改编”的短篇小说，原型是一位“在网戒中心被折磨致疯”的法学硕士，但是，你有什么证据说明他是在临沂网戒中心被折磨致疯的？他的亲生父母为什么对你的“帮助”如此反感？你为什么要在文章末尾处虚构主角跳楼自杀，到底居心何在？在文中写到，赵蕾仅仅因为他人的不理解就砸东西、割腕、大哭，甚至想用刀杀死她养的小乌龟；而殷雄又想通过自杀搞大这个新闻。作者难道没有发现这两个人的心理是病态的么？请问这样的人写出的东西能相信么？并且文章一直在误导读者煽动情绪，却没有任何实质性的内容，我不知道这样的一篇文章到底有何意义。<br/><br/>我今年十三岁，应该算是作者口口声声想要保护的“祖国的花朵”，可是当这花朵生病的时候，当这花朵接近枯萎的时候，你竟然还要跳出来说：不！不能剪去它的枯枝败叶，不能修掉她的丫枝，不能在它的身上加固支架，我们应该给它自由，这是她应有的权利！<br/><br/>也许是因为我们对自由的定义不同吧。我所理解的自由，应是在规则约束之下的。<br/><p
