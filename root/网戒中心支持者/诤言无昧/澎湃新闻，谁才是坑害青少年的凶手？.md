/><br/><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？（之十五）<br/><br/><!--more--><br/><br/>??<br/><br/>大家好，我是一位问题孩子的家长，我和我的孩子都是临沂网戒中心的受益者。<br/><br/>往事不堪回首。没来临沂网戒中心之前，我的孩子染着黄毛，穿着奇装异服，作为母亲，我真的不愿意与儿子一起上街，感觉很丢人。虽然我的儿子只有14岁，可他有学不去上，天天喝酒、抽烟、打架、出入KTV等娱乐场所、拉帮结伙到了无恶不作的地步.....看着儿子整日惹是生非，有时真的想狠揍他一顿，希望能把他打醒，但又舍不得。我常常跟自己说：只要能改好我的孩子，要我付出生命代价都可以！<br/><br/>其实孩子从小非常聪明可爱，我是一名教师，孩子小学是在我所工作的学校上的，那时候儿子虽然有些调皮但是成绩一直名列前茅，也特别懂事乖巧，我们的家庭充满了欢乐幸福。但是孩子上初中后，为了让孩子上县城的重点学校，我们住到了孩子的奶奶家，早上我们夫妻俩去上班，很晚才回家，这段时期缺少了与孩子的沟通。孩子从出生到小学六年级都是我带大的，和爷爷奶奶之间的感情并不很深，而且爷爷奶奶的管教方式属于特别严肃的那种，孩子并不适应这样的生活。刚开始孩子学习很用功，成绩在年级2000多名学生中能排300名左右，但是作为家长的我们忽视了孩子的交友，他偶然接触了奶奶家小区和他同年级的一个问题孩子，这个孩子五年级就走偏了，早已经学会打架、抽烟了。他们两个长时间的一块玩，儿子的思想发生了巨大的变化。<br/><br/>2015年3月的某一天，孩子忽然问我要钱说要看灯盏，我对他说明天带他去，他就开始发脾气，然后赌气离家出走了，找到他的时候，儿子已经完全变了个样：发型怪异，染着黄毛，嘴叼着烟，穿了一身特别怪异的服装，一个十三岁的孩子已经偷偷染上了烟瘾。从此以后，儿子彷佛要破罐子破摔似的，无论父母怎么说怎么劝都没有用，在学校里上课睡觉、玩手机，老师都不敢管他，有的老师好心劝他，他就动手打老师。有一次物理老师让他看视频的声音小一些，他就拿出刀向老师捅去，幸亏老师躲避及时，没有造成伤害。<br/><br/>由于在学校多次打群架，给学校的名誉造成了极其恶劣的影响，儿子最终被学校劝退在家，这更加剧了儿子的走偏。他白天在家睡觉，想睡到什么时候就睡到什么时候，晚上去网吧玩通宵。他爸有时候会劝他不要去网吧了，他就对他爸大打出手。我已经记不清类似的家庭战争到底有多少次了，多少次把家里的家具砸碎，多少次或孩子或父母想跳楼，多少次儿子挥舞着刀要杀人，多少次深夜里儿子负气离家出走，多少次雨雪中苦苦找寻儿子的身影。<br/><br/>为了阻止他接触社会上的混混，我们甚至多次向他下跪，结果他把我们狠狠地踹开；爷爷奶奶外公外婆都用尽了办法让他不要出去鬼混，可他像冷血动物一样对老人们大吼大叫。夜不归宿，半夜离家成了他的生活常态，可钱是从哪里来的呢？最开始，在我们不知道的情况下，他偷拿了家里的银行卡，当我们发现时银行卡里1万多块钱已在不到一个月的时间内刷光了，家里的购物卡他也偷走了，当我们问他拿了吗?他说没拿，这些都是来到临沂网戒中心后才告诉我们，都是他拿的。<br/><br/>我们作为父母不能眼睁睁的看着孩子堕落。每次他一出门，我们亲自跟踪他，也请我们的亲朋好友跟踪过他，甚至以请吃饭的名义认识他的几个损友，只为了随时能知道他的下落。<br/><br/>而影响孩子最深的是网络色情。我们每次找到他时都是在网吧、KTV、酒吧，14岁的孩子彻底毁掉了。他在KTV里吃摇头丸、找小姐。谁能想象到一个14岁的孩子会频繁出入高档洗浴中心毫无遮拦，那些看都嫌脏眼的红灯区几乎成了他的第二个家！受网络色情的影响，最后发展到在家里也明目张胆的抽着烟，在卧室里看色情视频还故意把声音调得很大，边看边手淫一天有时多达七八次，然后光着身子不知羞耻地出来喊：真舒服！我不敢相信，这还是我养育了14年的儿子吗？我突然意识到再不改变现状，我的儿子真要废了，身体和心理都要废了！<br/><br/>对于网络色情的痴迷让儿子自己都接受不了自己，他也会有负罪感，于是我们以带他看手淫的问题把他带到了临沂网戒中心。<br/><br/>来到这里，我的孩子发现自己的生殖器奇痒无比，下身都被他自己抓烂了，后被医生诊断为性病。那一刻，我真的崩溃了，我哭着对医生说：再好好检查一下，是不是弄错了！一个14岁的孩子怎么会染上性病？当父母的真的生不如死，在家里我们都是死一般的活着，工作也不能安心。我家住在大运河畔，多少次我一人徘徊在大运河边，多少次想跳进去，可想到自己的父母，不能让他们白发人送黑发人啊，想到我死了孩子不更惨吗？我们的日子简直陷入了绝境！<br/><br/>万般无奈之际，当我向我的同学诉说孩子的情况时，真是像上帝想救我一样，恰巧他的外甥也是问题孩子，在临沂网戒中心经过11个月的治疗完全改变了，并且在2016年考上了大学。怀着忐忑的心情，我们带着孩子来到了临沂网戒中心。<br/><br/>“只要你在临沂网戒中心挂了号，杨医生就对你负责一辈子。”这句话给了我们很大的鼓舞，杨医生是这样说的，也是这样做的。<br/><br/>入院后，杨医生那种废寝忘食、夜以继日的工作精神以及所有医护人员对孩子无微不至的关怀、谆谆教诲，还有在院孩子和家长对我们家庭的关心和照顾，使我的儿子很快融入到了这个大家庭中。<br/><br/>早上和下午半小时的训练使得孩子们个个英姿飒爽，一堂堂撼人心魄的心理点评课使得孩子的灵魂得到了重塑，每天的小室演讲锻炼了孩子的口才，每天的辩论赛培养了孩子们团队合作的精神，每晚为父母洗脚捶背培养了孩子的亲情与感恩之心，每天的歌曲《父亲》《母亲》《跪羊图》让孩子们边唱边流泪忏悔自己过去的错误，认清前进的方向。<br/><br/>我们真的看到了儿子改变的希望。<br/><br/>在孩子心灵得以洗涤，思想得以改变的同时，作为家长的我在杨永信医生以及各位点评师的教诲下，教育观念从根源上也发生了彻底的改变：从原来过分的溺爱、简单粗暴，到现在努力做到有效地和孩子沟通，我们母子俩正在逐渐找回亲情。<br/><br/>我为儿子的点滴变化所感动，我为中心成功救治6000多名孩子而感动，我惊叹于杨永信医生精湛的医术和高尚的品格，更惊叹于杨医生的博爱和勇于担当的人格魅力。<br/><br/>我深知我的孩子是幸运的，社会上一定还有无数个问题家庭急需网戒中心的救治。可自从2016年8月到2017年2月，这半年来游戏商和不良媒体的恶意造谣诽谤，尤其是2017年2月8日上午澎湃新闻发表署名张维和贾亚楠的文章《无法告别的临沂网戒中心：杨永信，盟友，家委会》，这篇文章严重失实，断章取义、以偏概全，严重歪曲事实，由于阅读量极大，给临沂网戒中心和我们这些孩子家长造成了很大影响。<br/><br/>由于不良媒体的恶意炒作，使临沂网戒中心受到了前所未有的冲击，有些原本可以得到及时救治的孩子因错失治疗而毁其一生......<br/><br/>谁才是真正摧残孩子的凶手？是那些不良媒体，尤其是澎湃新闻的记者张维、贾亚楠！我们作为中心的受益家长万分气愤，所以我们恳求有关领导能亲自来临沂网戒中心了解孩子和家长的心声，真正的为老百姓发出正义的声音。<br/><br/>我们强烈要求澎湃新闻删除相关微博及文章，强烈要求严肃处理记者张维、贾亚男。<br/><p
