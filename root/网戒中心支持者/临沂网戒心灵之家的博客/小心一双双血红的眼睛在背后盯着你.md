xml:lang="EN-US">?</span>疯狂的杀戮<span lang="EN-US" xml:lang="EN-US">?</span>，故事是看完了，但是心情久久不能平静，就如办案警官所说的：“案子破了，却并不能轻松，没想到他们是这么小的孩子”。<br/><br/>孩子，是我们社会的未来，他们正是早上八九点钟的太阳，二十岁左右的年龄，正是读书学习或者刚踏入社会开始人生第一步的黄金花季，为什么他们却举起砍刀，疯狂砍杀，残忍的制造着一幕又一幕悲剧？是什么让他们如此疯狂、如此泯灭人性？<br/><br/>有人说是网络，但我说网络无罪，倒是借助网络的“杀人游戏”让赵继鹏这些孩子失去了理性，从这几个孩子砍人的招式上不难看出是“杀人游戏”教会了他们，网络的迅速普及和游戏的易得性，不知道有多少孩子还在步其后尘？使我感到恐惧的不单是这几个孩子的凶残行为，而是罪犯赵继鹏的一句话：我们这些孩子有很多都是犯罪的后备军！<br/><br/>危险正在四处蔓延，危险就在你我身边！我们没有理由，也不可以再漠然视之、麻木不仁，谁都不想成为悲剧中的主角！<br/><br/>说到这儿，我又想到一段时间以来，备受争议的“网瘾”与“网瘾戒治”的是与非。伴随信息化时代的到来，突然冒出来的“网瘾”让千万个家庭陷入了痛苦的万丈深渊，可怜的家长们放弃了工作，放弃了生意，放弃了事业，带着孩子天南地北，想方设法欲摆脱这倒霉的魔咒，需求带动了市场，数百家不同理念、不同模式、不同背景、不同动机的网戒机构如雨后春笋般应运而生。<br/><br/>但是好景不长，随着个别非法机构诸如“魔鬼训练营”打伤打死孩子事件的曝光，再加上某些利益群体和个别别有用心的不良记者或媒体利用被扭曲和夸张了的“电击戒网瘾”推波助澜，还有极少数没有改变好的孩子在网上呐喊助威，让刚刚起步的网戒事业一夜之间好似成了过街的老鼠。<br/><br/>利益群体阻挠戒网瘾的目的是为了守住财路，不良记者或媒体大肆炒作的目的是为了新闻卖点和借机成名的功利欲，而作为受害者的孩子是为了什么呢？网络游戏如同精神鸦片，甘愿把害人的精神鸦片当成水蜜桃的是谁呢？当然是上网成瘾的孩子们！这些孩子最为反感的就是人们把他们对网络的痴迷称之为“网瘾”，更加切齿痛恨的是通过种种方式阻碍他们徜徉于虚拟世界的人们，而像杨永信这样的网戒专家恰恰就是他们所惧怕的网瘾克星，有个别网瘾孩子在网上进行叫骂和围攻就丝毫也不奇怪。网戒机构被封杀，网戒专家被打入冷宫，并不是最可悲的，可悲的是网瘾孩子的家长重新又陷入了彷徨和无奈，救孩子的出路在哪里？走出绝境的方向标在哪里？伤心欲绝的日子哪天是尽头？<br/><br/>王刚先生在节目的最后语重心长的说：老话儿讲，这尿炕不能赖床，可你这床要是冰凉冰凉的，还一劲儿给孩子喝凉水，你想这能好吗？戒治网瘾事关孩子前途命运，事关千家万户幸福安逸，事关社会安宁和谐，事关你我千万人，不要再做无谓的纷争，不要再为了眼前利益而牺牲无数代人的利益，不要再纠缠于网戒初期方法或手段的不完美，任何事物都有它的正反两方面，青霉素把无数病人从鬼门关拉了回来，但严重的过敏反应也夺走了很多人的生命，但在利弊面前人们还是理性的接受了这样一个并不完美的事实，眼前的甲流感疫苗接种也同样出现了几个死亡病例，但并没有停止继续大面积接种的步伐，手术带来的创痛尽人皆知但无人回避，癌症化疗带来的严重脱发和胃肠反应也没有打消人们求生的欲望，同样的，也不能因一两例非法网戒机构造成的死亡案例而全盘否定了正处于探索中的网戒事业，更不能以“低剂量电刺激治疗”会让人痛苦为由让一种特效疗法擦肩而过，细细品味网瘾孩子家长说的一句话：即使送孩子受电击，也比坐牢强！为了让<span lang="EN-US" xml:lang="EN-US">?</span>疯狂的杀戮<span lang="EN-US" xml:lang="EN-US">?</span>不再无数次上演，我作为一名网瘾孩子的家长，强烈呼吁全社会共同关注孩子的健康成长，积极扶持刚刚起步尚显稚嫩和脆弱的网戒事业，更希望政府出面增加网戒机构设置，扩大网戒机构规模，“安得广厦千万间，大庇天下寒士俱欢颜”。同时还要告诫某些所谓的专家、记者、名人不要以“人权卫道士”自居，满口仁义道德和人权民生，站着说话不腰疼，不要张口马列闭口马列，不要假内行、伪真理，应该多一些理解，多一些包容，多做一些实事，少说一些外行话，多一点社会公德心，多一点对未来负责任的态度，不要为了一己私利，为了图一时的嘴上痛快，而葬送了千千万万青少年的美好前程和我们中华民族几代人的利益。<br/><br/>最后，需要特别提醒你的是，小心一双双急需抢钱上网的血红的眼睛在背后盯着你！<br/><br/><span
