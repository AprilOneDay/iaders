<br/><p align="center">[来源:华声在线][作者:李思辉]时间：2011-07-23 11:14:49</p><br/><p align="center"><b>文/李思辉</b></p><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><br/><p align="center"></p><br/>　　一对90后情侣，三年贩卖了三个亲生子女，卖钱只为上网，这是某网站爆出的一则新闻。问及卖孩子的原因，这对看似天真的小情侣，轻松地回答：“不想养，卖了可以换点钱。”<br/><br/>为了换钱上网，把自己的亲生骨肉卖掉，这种泯灭人性的做法，居然一而再再而三的被这对小情侣复制，而且他们对这种荒唐的复制手段，甚至觉得理所当然，这真是一种悲哀。这对小情侣的悲哀，引发了网友的关注，人们唏嘘、人们扼腕、人们发指，但这种理性的社会共同反射，似乎无法引起这对情侣的共鸣。在他们眼里，钱比骨肉重要，生孩子就是为了卖钱上网。这样畸形的，毫无伦理底色的心态，令人揪心，发人思考。<br/><br/>不难看出，这对小情侣身上具备90后超现实、超理性的特征。只不过他们把这种特征放大到了极端。这就像通过显微镜观察手上的细菌一样，平时不大注意，突然看到一大群大肠杆菌在手上蠕动，于是大为震惊，于是坚持勤洗手。同理，“卖子上网”的案例，让我们深切地感受到了一些90后对社会伦理(特别是家庭伦理)的淡漠，对虚拟世界的迷恋，对“等价交换”逻辑的追捧。这些消极的，甚至是极端的社会认知态度，理应引起社会的关注和思考，理应获得必要的劝导和矫正，否则真不知道还有多少90后，会就着这“天真”，做出多少匪夷所思的荒唐事。<br/><br/>我并不认为这对小情侣能代表大多数90后。只不过，他们身上暴露出的种种问题，至少可以看作是90后社会认知心态和思维方式的某一个切面。我们至少可以以此入手，找到引导部分90后树立正确人生观、价值观和世界观的途径。就拿“卖子上网”这个案例来说，假如当地的教育机构、当事者的家庭能进行合理地教育和引导，尚未成年的子女，就可能不会随意辍学、出走、同居，随后的悲剧也可能避免。同理，面对已经走错的90后情侣，我们可以“恨其不争”、“哀其不幸”，但尚需对其假以“家庭伦理”的耐心教育，以弥补此前的缺失。<br/><br/>“尊敬和赡养老人，抚养和教育子女，尊重家长，爱护弟妹”，这是家庭伦理的朴素要求，也是社会伦理的重要内容。一个社会人，从小受到来自家庭、学校、社会等方面的潜移默化的伦理教育，理应构建起坚固的朴素伦理观。问题是，现在的一些年轻人对传统的认知已经越来越“不耐烦”，对虚拟世界、对新潮事物却表现着浓厚的兴趣。更可怕的是，在功利色彩浓厚的社会环境下，他们把“自私、唯我、超现实”等人性的弱点，推到了价值观的台前，于是各种光怪陆离的现象层出不穷。于是，一些90后，对父辈(甚至是80后)的观念嗤之以“OUT”，于是出现所谓的“三岁一个代沟”的说法。这种多元的、传统与激进观念的冲突，引起了许多社会学家、伦理学家的关注，但我认为，首先应该关注和试图加以调和的，还应该是家长和教师。因为眼下的情势是，对偏颇的观念，除了就近教育外，我们似乎找不到更好的纠偏路径。<br/><br/>变卖子女，触目惊心。查阅古今典籍，虽不乏“卖女易金”、“易子而食”的案例，但那样做都是迫不得已，为人父母者走到那一步都是心如刀绞。而今的有些90后小情侣，居然乐呵呵地变卖子女，一点都不觉得伤心或是牵挂，坦然得很呢。这种境界较之古人倒是“进步不少”。只不过这种“进步”，蒙上了一层厚重的悲情色彩;只不过这种“进步”，必然受到法律的制裁;只不过，这种“进步”，糟践了那些无辜的孩子。<br/>首次转载时间: 2018-02-28 21:31:34"
