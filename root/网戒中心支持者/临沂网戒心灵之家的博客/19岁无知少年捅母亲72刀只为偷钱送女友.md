xml:lang="EN-US">……</span>作为一名<span lang="EN-US" xml:lang="EN-US">“90</span>后<span lang="EN-US" xml:lang="EN-US">”</span>，朱陵杰的生活也许不只是个案，他说：<span lang="EN-US" xml:lang="EN-US">“</span>进入社会以来，我一直没有认真过每一天，天天都是在虚拟的世界里。<span lang="EN-US" xml:lang="EN-US">”</span></p><br/><p align="left">　　<b>男孩儿捅了母亲<span lang="EN-US" xml:lang="EN-US">72</span>刀</b></p><br/><p align="left">　　<span lang="EN-US" xml:lang="EN-US">2008</span>年<span lang="EN-US" xml:lang="EN-US">12</span>月<span lang="EN-US" xml:lang="EN-US">4</span>日凌晨，一名<span lang="EN-US" xml:lang="EN-US">1.70</span>米左右的大男孩儿，抱着一个铁皮箱子，从新郑市八千乡新万惠量贩大门走出来，打电话叫了一辆出租车。他叫朱陵杰，刚刚杀了自己的亲生母亲，抱着家里的钱箱出来的。</p><br/><p align="left">　　<span lang="EN-US" xml:lang="EN-US">12</span>月<span lang="EN-US" xml:lang="EN-US">6</span>日，警方在呼和浩特市将其抓获归案。当时他待在自己的女朋友于冰冰<span lang="EN-US" xml:lang="EN-US">(</span>音<span lang="EN-US" xml:lang="EN-US">)</span>家里。</p><br/><p align="left">　　昨日上午<span lang="EN-US" xml:lang="EN-US">10</span>时许，郑州市中级人民法院开庭审理了朱陵杰杀母案件。</p><br/>对于检方所提交的犯罪证据、指控他的犯罪行为，他全部承认。他说，当晚他的目标并不是家中的钱箱，而是母亲身边放钱的保险柜，因为害怕盗窃保险柜时惊醒母亲，他才动了杀机。每当被问到事发当晚的想法和感受时，他总说<span lang="EN-US" xml:lang="EN-US">“</span>我也说不清楚<span lang="EN-US" xml:lang="EN-US">”</span>。<br/><br/>经法医鉴定，朱陵杰的母亲朱建丹系被人用单刃刺器刺破双侧肺脏及肝脏，致失血性休克而死。一组数据：她身上被捅了<span lang="EN-US" xml:lang="EN-US">72</span>刀，其中胸部<span lang="EN-US" xml:lang="EN-US">25</span>刀，腹部<span lang="EN-US" xml:lang="EN-US">25</span>刀，从头扎到脚。<br/><br/><strong>为让女友回家他抢了自家钱柜</strong><br/><br/>朱陵杰，<span lang="EN-US" xml:lang="EN-US">1990</span>年<span lang="EN-US" xml:lang="EN-US">10</span>月<span lang="EN-US" xml:lang="EN-US">6</span>日出生，祖籍浙江永嘉县，<span lang="EN-US" xml:lang="EN-US">2006</span>年随做生意的父母到新郑。父母经营着新万惠量贩，还有个弟弟成绩不错，家人都希望弟弟能考上大学。<br/><br/><span lang="EN-US" xml:lang="EN-US">2008</span>年<span lang="EN-US" xml:lang="EN-US">5</span>月，朱陵杰在上网时认识了内蒙古女孩于冰冰，两人很聊得来。<span lang="EN-US" xml:lang="EN-US">9</span>月<span lang="EN-US" xml:lang="EN-US">20</span>日，朱陵杰偷了家中<span lang="EN-US" xml:lang="EN-US">9000</span>元钱到呼和浩特找到她。两人于<span lang="EN-US" xml:lang="EN-US">9</span>月<span lang="EN-US" xml:lang="EN-US">27</span>日来到新郑，后来两人没钱了，可冰冰想回内蒙古。<br/><br/><span lang="EN-US" xml:lang="EN-US">12</span>月<span lang="EN-US" xml:lang="EN-US">4</span>日凌晨，案发后，朱陵杰回到他们的租房处，叫冰冰换衣服立刻跟他走。两人打车至郑州，坐火车，倒汽车，奔波两天赶到了冰冰呼和浩特的家。<br/><br/>在朱家超市上班的宋女士说，朱陵杰整天上网，平时整条整条烟从超市往外拿，从三楼、二楼的窗户往外跳，为此他爸朱加龙专门将窗子装上了防盗网。<br/><br/>正是因为朱陵杰在家中有了<span lang="EN-US" xml:lang="EN-US">“</span>前科<span lang="EN-US" xml:lang="EN-US">”</span>，<span lang="EN-US" xml:lang="EN-US">12</span>月<span lang="EN-US" xml:lang="EN-US">3</span>日晚<span lang="EN-US" xml:lang="EN-US">9</span>时许，当他在外拍门要回家住后，朱加龙专门晚上找儿子谈心，并和他睡在一起，防止他偷盗。但是<span lang="EN-US" xml:lang="EN-US">“</span>凌晨<span lang="EN-US" xml:lang="EN-US">3</span>点，我醒来发现身边没人了，下楼看钱箱不见了，我老婆躺在屋里包着被子，到处是血<span lang="EN-US" xml:lang="EN-US">”</span>。<br/><br/><strong>姥姥、姥爷向法院提出从轻申请</strong><br/><br/><span lang="EN-US" xml:lang="EN-US">“</span>当初听到这个案子的时候，我觉得不可思议，第一反应是怀疑案子是不是他做的。<span lang="EN-US" xml:lang="EN-US">”</span>说起为本案辩护的感受，河南豫新律师事务所律师李胜利说。来事务所找到李胜利的是朱陵杰的父亲和舅舅。<br/><br/>庭审上，李胜利为朱陵杰做的是有罪从轻辩护，他提交了一份朱陵杰的姥姥、姥爷转交的申请，两位老人在申请中表达了自己听说此事后，<span lang="EN-US" xml:lang="EN-US">“</span>震惊<span lang="EN-US" xml:lang="EN-US">”</span>、<span lang="EN-US" xml:lang="EN-US">“</span>悲痛<span lang="EN-US" xml:lang="EN-US">”</span>的心情，但随即说道，<span lang="EN-US" xml:lang="EN-US">“</span>考虑到朱建丹已不能复活，而朱陵杰也多次表示忏悔，请求法官考虑本案的实际情况和两位古稀老人的心情，能给朱陵杰一个重新做人的机会<span lang="EN-US" xml:lang="EN-US">”</span>。<br/><br/>庭审后，朱陵杰的姑姑等亲人要求见孩子。当姑姑眼含泪水告诉他，他的父亲带弟弟回浙江参加高考时，朱陵杰红着眼圈问：<span lang="EN-US" xml:lang="EN-US">“</span>我弟弟能不能好好发挥考上大学？<span lang="EN-US" xml:lang="EN-US">”</span><br/><br/>李胜利还说，朱陵杰是初犯，该案的发生也有一定的家庭和社会因素。他认为，中学生辍学至就业这一段时间，是目前社会管理的一个真空，这时候的青少年根本无所适从，没人能给予及时地引导，才导致了悲剧的发生。<br/><br/><strong>检方提出网瘾、家教等问题</strong><br/><br/>与亲友提到的网瘾少年、盗窃惯犯形象不同，与姥姥、姥爷讲的并非善恶不分的形象不同，作为本案公诉方，市检察院公诉一处检察员马玲霞说：<span lang="EN-US" xml:lang="EN-US">“</span>这孩子不善言辞，感情上可以说是很冷漠。<span lang="EN-US" xml:lang="EN-US">”</span><br/><br/>最初让马玲霞印象深刻的，是她与朱陵杰的第一次对话。<span lang="EN-US" xml:lang="EN-US">“</span>我问他：杀了母亲你什么感觉？他回答说：不知道。<span lang="EN-US" xml:lang="EN-US">”</span>从<span lang="EN-US" xml:lang="EN-US">2003</span>年至今，这是马玲霞遇到的第一个<span lang="EN-US" xml:lang="EN-US">“</span>没感觉<span lang="EN-US" xml:lang="EN-US">”</span>的孩子。<br/><br/>不善表达，如果检察员没有将问题重复多遍就不回答。没感觉，直到参加昨天庭审时才让检察员感到他有点后悔。看不出残忍，却多次在谈话中提到<span lang="EN-US" xml:lang="EN-US">“</span>想着杀了爹妈弄钱<span lang="EN-US" xml:lang="EN-US">”</span>，最终在生母身上扎了<span lang="EN-US" xml:lang="EN-US">72</span>刀。马玲霞觉得，朱陵杰就是一个对真实生活了解不深，根本意识不到自己行为后果的孩子。<br/><br/>检方就本案提出了三点意见：一、对青少年应如何正确处事的教育。朱陵杰因迷恋上网，多次回家偷钱，而家长在金钱教育方面明显缺失，过分溺爱。二、父母过早将孩子推向社会。从<span lang="EN-US" xml:lang="EN-US">16</span>岁辍学开始，家庭、学校对朱陵杰的教育基本停止，而感情沟通方面更是欠缺。三、如何正确对待网络。马玲霞认为，本案中朱陵杰的生活明显反映出网吧管理上的漏洞，以及父母对网瘾少年的教育、监管不力，这也是网络对孩子产生负面作用的典型例证。<br/><br/>由于案发时朱陵杰已满<span lang="EN-US" xml:lang="EN-US">18</span>岁，选择父母熟睡不知反抗的时候，而且事前还购买了杀人工具，在整个犯罪过程中杀人犯罪意念坚决，作案手段残忍。检方指控其犯故意杀人罪，希望法庭应认定其有罪。法院未当庭宣判。<br/>首次转载时间: 2018-02-26 17:30:53"
