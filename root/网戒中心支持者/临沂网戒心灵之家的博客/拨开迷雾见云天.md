/></p><br/>?<wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?尽管已是数九寒冬，可我却丝毫没有感觉到冬的寒意。应该说网戒中心根本就没有冬天，这里到处洋溢着春的温暖，夏的清凉，秋的收获。看着一个个阳光少年从这里走进学校，走向工作岗位,加入到中国人民解放军这个光荣的大熔炉去锤炼，去深造。我的心情也在这两个多月中一天比一天开朗起来，收获也是一天比一天的增多。<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?往事不堪回首，两个多月之前，确切的说，应该是在来网戒中心之前，我的孩子留着长发，穿着应该说是另类的衣服，走路一步三摇。如果再配上个大烟袋，去演电影里的反面角色，简直连妆都不用化，肯定会是一名星级演员。虽然只有十三、四岁的年龄，可却已发展到，有学不去上，更谈不上“认真”完成作业之类的话题。丧失人性，喝酒，抽烟，打架，拉帮结伙，简直到了无恶不作的地步，先不要说断头台，至少有一只脚已跨进高墙大院的门里……<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?就说来网戒中心之前的一段时间吧，我由于工作原因去了外地，他妈也因工作比较忙，对他管教的比较少，这样他就更加放纵自己，利用上学时间居然能外出十几天，音信皆无。满城挨个网吧找他的事，已快成家常便饭。就连我们中国人的传统节日――中秋佳节，他都不回家，留他妈妈一个人在家空对一轮明月，嚎啕大哭，别人家都在团圆，而我们家却人各一处，甚感凄凉！我不敢想象他当时在干什么？他妈妈又怎么想？借他回家的机会，他姥爷在没有任何办法的情况下，把他关在家中，全家人整天都陪着他，半步都不敢离开！可怜一位年近七旬的老人，已经退休在家，本该享受晚年的时候，却被这位不争气，自己又最疼爱的外孙，折磨得痛不欲生。而面对这一切，孩子非但没有感激之情，却变本加厉的折磨老人！甚至还想对老人动手动脚……<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?原想等我回来再处理他的事情，可巧得很，就在我回来的当天，又接到我外祖母去世的噩耗！带着亲人离去的巨大悲痛，和我在孩子教育中的失败，我失声痛哭！绝不亚于在场的所有人！到最后，大家都劝我要节哀!可他们又怎么能知道我心中的苦痛……<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?奔了一天丧，拖着疲惫的身体回到家中，可他却冷冷地说：“你回来干什么？这下该放我出去了吧？”我当时就火冒三丈,便动手狠狠地揍了他一顿，但他却连哭都没有，反到是我却又大哭一场……<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?这样的例子在最近几年中已是见多不怪了，天快塌了，家已失去昨日的模样，我不知道、更不敢想象以后的日子我该怎么面对？我该怎么活下去？在外面强装笑脸，回到家就只能以泪洗面，整日在忧愁悲伤中度过，难道我今后的生活就这样度过吗？难道我就命该如此吗？<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?真是苍天有眼,在我们就要彻底绝望之际，临沂四院治愈网瘾综合症的消息，给我们带来了天大的希望。来心理课堂考察的第一节点评课，杨叔那带着磁性，有些沙哑的声音，先进的教育理念，以及盟友们的积极表现，陪护家长对我们疑难问题的热情解答，同时看到墙上挂满一面面鲜艳的，激动人心的锦旗，这就是最好的见证，更代表了无数家庭感恩的心声。看着这一切的一切，我们立即有了希望，好像也看到了光明，同时也看到了未来！我们立刻决定将孩子送到网戒中心！<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?刚来医院的第一天，杨叔带孩子做了第一次治疗，几分钟后，儿子哭着跑过来对我说：“爸爸，我愿意住院接受治疗！”我惊呆了,同来的亲属都惊呆了！是什么力量？是什么动力？使儿子改变的如此之快。前后仅十几分钟，儿子就像变了个人似的，我不敢相信自己眼睛和耳朵，这是真的吗？这还是我的儿子吗？<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?入院后，杨叔那废寝忘食，夜以继日的工作精神及所有医护人员对孩子无微不至的关怀，谆谆的教诲，盟友和盟友家长对我们的关心和照顾，无时无刻不在我的脑海中重现。孩子的点滴进步，我们都看在眼里，喜在心头。我们的感激是无法用语言来表达的，千言万语只能汇成一句话：“谢谢杨叔，谢谢所有医护人员，谢谢所有盟友及盟友家长。”<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?一堂堂精彩的点评课更使孩子的灵魂得到重塑，杨叔哽咽着讲述着一个福利院里小女孩的故事，泪流满面的讲述着刘刚的故事，又怒不可遏的斥责一些盟友，丧失良知，失去人性，干出一些伤天害理的事。给自己，给家庭造成无法挽回的损失！给社会造成了不可弥补的伤害。并对一些即将破碎的家庭进行点评，使得他们得以重新团聚！点评课堂上，多次出现父子、母子相拥而泣的场面。记不得有多少次孩子们纷纷跪倒在父母们的面前，泪流满面，忏悔自己的过去，树立前进的方向，并暗暗下定决心：永不走偏！这可是我们早已久违了的;又有多少个家庭，多少个父母亲眼含热泪感恩杨叔，感激自己的恩人。是杨叔，一位天使；是永信模式，指引一个个迷途羔羊得以返航！拯救一个个家庭，为净化社会环境做出了自己的贡献。远有鲁讯先生发出救救孩子的呐喊，现有杨永信以其独特的模式，自创70多个环节，环环紧扣，来拯救孩子，真可谓前无古人，功德无量。<br/><br/><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr /><wbr />?<wbr /><wbr />?<wbr /><wbr />?在孩子心灵得以洗涤，思想得以改变的同时，我们做家长的也在杨叔的教诲下，从思想根源上发生了彻底的改变。发现自己以前对孩子的教育方式，要么就是过分溺爱，要么就是简单粗暴，而不是和孩子好好的沟通，从内心深处去改变孩子幼稚的想法。我要为以前对儿子粗暴的教育方式说声：对不起！同时希望儿子能谅解爸妈的心！让我们从现在开始做一对家庭中的好父子，生活中的好朋友！<br/><br/><wbr /><wbr
