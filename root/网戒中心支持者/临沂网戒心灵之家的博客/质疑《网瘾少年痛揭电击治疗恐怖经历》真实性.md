href="http://www.xhby.net/" target="_blank" rel="noopener">新华报业网－南京晨报</a>2010年9月18日登载记者王晶卉《网瘾少年痛揭电击治疗恐怖经历》（以下简称“经历”）一文，作为正在临沂网戒中心接受网瘾戒治的孩子的家长，看后十分困惑：文章的作者仅仅凭借道听途说，没有实地考察，就写出这样荒谬的文章，是不是太有悖一个新闻工作者的职业道德了？困惑之余，不免要问：<br/><br/><wbr />?<wbr />?<wbr />?一、王晶卉，你的信息来源是否真实、准确、可信？<br/><br/><wbr />?<wbr />?<wbr />?大家都知道，真实是新闻的生命，客观公正地报道社会现象是新闻工作者的使命。“经历”一文中报道说，化名“小刚”的孩子来到网戒中心的第一天做了20多次电击治疗，我想，这位“小刚’太不了解临沂网戒中心的真实情况了。临沂网戒的日程安排是非常紧凑和充实的，每天仅心理点评课就要占去4、5个小时，另外，还有军训课、日记课、网上作业、演讲辩论、内务培训、弟子规学习、专题小点评等众多环节，同期在院的就有上百名孩子，怎么可能每天做20多次电击治疗？我身为一名家长，陪孩子在临沂网戒已经快5个月了，我亲眼看到的是，至多有的孩子刚来时因不想被约束而情绪失控，会发生一些暴烈或冲动的行为，为了他的自身安全和他人的安全，为了使其尽快进入正常的心理沟通状态和融入整体氛围，在征得我们家长同意的情况下，由点评师用低频脉冲治疗仪实施心理导入治疗，即使这样，接受治疗的次数也是非常有限的，我的孩子到现在为止也不过才接受了一次治疗。一个孩子一天做20多次是记者编的还是“小刚”胡说的？众所周知，大部分网瘾孩子都存在撒谎欺骗、仇视父母、自我放纵、逃避约束和抵触逆反等现象，这样孩子说的话你能相信多少？文中还说“和教官顶嘴也要被电击”，这儿除了医护人员和家长，就是接受治疗的孩子，哪来的“教官”？<br/><br/><wbr />?<wbr />?<wbr />?二、如果真像“经历”一文中说的那么恐怖，为什么还有很多家长络绎不绝的来此求助呢？<br/><br/><wbr />?<wbr />?<wbr />?自2009年4月始，曾经一度被媒体热捧的网戒行业遭遇霜冻，许多以行走禁闭、魔鬼训练、戒尺教育等为特色的网戒机构因经不起疗效和安全的检验而纷纷关门歇业，而被媒体以“电击”和“摧残”为炒作点，使其处在风口浪尖上的临沂网戒中心却依然生机勃勃，前来求助的家长络绎不绝。是家长们“脑残”，还是什么其他的原因？俗话说：“金杯银杯不如老百姓的口碑”，“不管白猫黑猫，逮住老鼠就是好猫”，“是沙子总会被冲走，是金子总会被留下”，疗效和安全就是临沂网戒中心的成功之道，大量得到成功改变重返校园或社会的网瘾孩子就是最好的见证。<br/><br/><wbr />?<wbr />?<wbr />?三、陶教授那“点醒梦中人”的“一语”能让“小刚”经受多久的网络诱惑的考验？<br/><br/><wbr />?<wbr />?<wbr />?陶教授在作巡回演讲时，喜欢弹钢琴曲让孩子放松，然后讲道理，宣称这样三天到一个星期就可以帮孩子戒治网瘾。据我们这里曾求助过陶教授的家长说，他的孩子5天花了5800元参加“陶宏开素质夏令营”，可是并没有解决问题，只好在朋友的介绍下，来到临沂网戒中心。即使“小刚”真的在临沂网戒中心戒治过网瘾，可是由于他没有及时回来巩固治疗，一年之后再次反复也可以理解。陶教授真的能用短短几个小时或者几天就解开一个孩子几年甚至十几年的“网瘾心结”？记者同志，戒治网瘾若是如此简单的话，陶宏开介入这一行当已7年有余，怎么自己才声称治愈600例呢？何况这600例是否就真的彻底治疗好了呢？如果说临沂网戒中心能让小刚在一年的时间里不再上网玩游戏，学习状态还明显进步的话，陶教授的“一语点醒梦中人”能让小刚坚持多久不反复？ 你能跟踪报道多久？<br/><br/><wbr />?<wbr />?<wbr />?四、一个陶宏开是否能救治全国已有的2400多万网瘾青少年？哪一天才不会再有因网瘾所致的恶性案件或悲剧性案例发生？<br/><br/><wbr />?<wbr />?<wbr />?当前，青少年网瘾队伍在2400多万的基础上依然不断递增，因网瘾所致的恶性案件或悲剧性案例时常见诸报端，如何破解网瘾戒治难题已是当务之急，一些富有高度社会责任感的人士纷纷加入到拯救孩子的队列中来，当然也不排除个别人把此当作商机，以不科学、不规范的方式戒治网瘾，对青少年带来了不同程度的伤害，俗话说“城门失火，殃及池鱼”，正是这种现象使社会对整个网戒行业产生误解或错觉，加之某些利益群体的推波助澜，导致一些富有正义感的积极探索者也成为被炒作或被封杀的对象。如此以往，网瘾现象就会成为不解之惑，进而会引发毁家之祸，毁国之难。关注社会热点问题本无可厚非，关键的是记者不要以专家自居，仅凭极端个例和个人主观印象，不深入调查，不作理性思考，盲目的谈论孰是孰非，在毁掉整个网戒行业的同时，也毁掉了千千万万的孩子和家庭。本月15日，央视《今日说法》中15岁少年张棋因痴迷网络，被无奈母亲绑锁在小仓库而窒息死亡的悲剧足以给我们敲响警钟。<br/><br/><wbr />?<wbr />?<wbr />?五、“经历”一文报道的是个例还是普遍现象？<br/><br/>世上没有包治包好的神医，科学没有百分之百，任何一种医疗手段也不能做到百分之百的治愈率。由于家长配合不到位，疗程不足，长效机制未得到很好落实，出院后外界干扰因素很多等原因，网戒机构都必然要面临网瘾孩子反复的现象，临沂网戒中心也不例外。这也恰恰说明治疗网瘾的复杂性、长期性、艰巨性、系统性和差异性，这也是戒治网瘾会成为世界难题的原因。作为家长，我在临沂网戒中心几个月的时间内，看到了许多在成功改变后，少则数月，多则数年，又回网戒分享的孩子，他们带回来许多蜕变的经验和成功的喜悦，正是这些亲眼所见，坚定了我们临沂网戒全体家长战胜网瘾的信心。我们不能以偏概全、一叶障目，我们相信事实，相信科学，相信那些符合客观规律的东西，绝不相信那些子虚乌有之言。<br/><br/><wbr
