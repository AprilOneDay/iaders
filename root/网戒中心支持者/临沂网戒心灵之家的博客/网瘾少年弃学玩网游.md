摄影记者李响<br/><br/>曾经的奥赛化学湖北省一等奖得主，拥有“全国中小学生英语能力竞赛武汉赛区三等奖”、“全国初中数学联合竞赛三等奖”等多项荣誉的康康，竟然变成了网虫。更让人意外的是，满了18岁后，他竟与父亲签订协议，放弃学业，闭门专心玩网游。<br/><br/>昨日上午，记者来到康康家里采访。老式的两室一厅房子，陈设十分简陋，康康把自己关在房间里闭而不出，他的父母轻轻地将记者带到另一间屋子里谈话，言语间满是对儿子的担忧与无奈。<br/><br/>康康与父亲的协议<br/><br/>1、我自愿放弃学习，专职玩游戏，所产生的后果一切由自己承担，与父母无关；<br/><br/>2、我选择在家中使用网络，宽带费用60元/月，由我负担，每月交钱后才能使用网络；<br/><br/>3、自即日起，我的任何费用包括吃饭、穿衣、治病、通信、游戏等自理，不以任何理由向父母要钱；<br/><br/>4、家中物品未经父母同意，不拿到外面使用，更不得变卖；<br/><br/>5、每月承担家中的1/3水费、1/3电费（以交费单据为准），决不拖欠；<br/><br/>6、我自愿在外进餐，不在家中吃饭；<br/><br/>7、尊重长辈，在父母丧失劳动能力后，有义务赡养父母；<br/><br/>8、住房问题，另行协商。2011年4月9日<br/><br/>康康目前生活状态<br/><br/>不吃不喝不洗漱关在房里打游戏<br/><br/>昨日记者在康康家采访时看到，在客厅的桌子上，放着两只碗，一只碗里装着一个包子，一只碗里装着几个粽子，旁边放着一杯凉开水。<br/><br/>康康母亲介绍，虽然康康在协议中写明“自愿在外进餐，不在家中吃饭”，但他为了上网经常不吃不喝，她担心儿子的健康，每次做饭后，悄悄给儿子留些吃的在桌上，但是儿子似乎不领情，很少吃这些。<br/><br/>她说，儿子可以不洗不漱，几个月不理发，前两天她看到儿子十分邋遢，头发快半尺长了，最后在她央求下儿子洗了把脸，漱了下口。她也曾趁儿子不注意时，走进他的房间看了看，里面乱七八糟，电脑旁边放着一堆没洗的碗，十分脏乱，她正准备帮忙清理下，却被儿子吼出了房间，之后儿子迅速锁上房门。<br/><br/>母亲说，今年过年，儿子参加游戏人聚餐，领回一个杯子，一条毛巾和300元钱，儿子还特别将杯子送给她，说这是玩游戏到最高级别后的奖品。<br/><br/>她也试着劝儿子，既然已经玩到最高级别，可不可以不玩了？但儿子说，虽然自己已玩到最高级别，但还要带领一批新手玩。<br/><br/>看着儿子紧闭的房门，母亲的眼泪一滴一滴往下落，她喃喃道：“不知道儿子什么时候才能走出这间房门，和我们正常交流。”<br/><br/>昔日他曾是父母的骄傲<br/><br/>说起儿子昔日的表现，康康母亲从抽屉里拿出一堆奖状来，红红地摆满整整一张床。<br/><br/>从小学到高一，康康收获了各种荣誉，有校运会跳远比赛第三名、期末统考全年级第一名等等。最醒目的是他在初中阶段获得了全国数学联合竞赛三等奖，高一获得了奥赛化学湖北省一等奖。<br/><br/>摆完这些奖状，康康母亲的脸上又呈现出哀愁。她抹了一把泪说，儿子从小乖巧听话，特别聪明，学习成绩好，父母很少操心。长大后，儿子身高1.83米，“瘦瘦长长的，很帅”。但自从迷上网络游戏后，儿子的体重降到只有50来公斤，瘦得只剩下皮包骨，整个人“像根杆子似的”，让她伤透了心。<br/><br/>接着，康康母亲找出康康与父亲签订的协议给记者看。“没有办法！”康康父亲坐在一旁，边抽烟边摇头叹息。他说，这已是他与儿子签的第5份协议了。儿子一度是家庭和学校的骄傲，他的名字至今还挂在所读高中的校门光荣榜上，没想到他现在却变成了这个样子。<br/><br/>现在父母却拿他没办法<br/><br/>康康父母介绍，康康读初三时开始接触互联网。有一次，学校下午放半天假，康康未按时回家，父亲下班后在网吧里找到了儿子。当时，儿子虽然喜欢上网，但还没有成瘾。<br/><br/>康康读高一时，因为要参加奥赛化学比赛，家里给他配了台电脑方便他上网查资料，但康康并没有将电脑派上正当用场，而是用它来玩网络游戏。<br/><br/>看到儿子每晚都要上网玩到转钟，父亲便将路由器关掉。没想到，康康从生活费里省下钱来，买了一个无线网卡上网。父亲把无线网卡没收后，康康干脆跑到网吧玩游戏。没有上网费，他就把手机卖掉，有一次还在父母柜子里偷了200元钱。<br/><br/>眼看儿子一天天陷入网络游戏之中，父母心急如焚。2009年8月14日，康康班主任告诉他们，为了上网吧，康康居然模仿班主任签名10多次骗过门卫出校门。<br/><br/>当天，被父亲领回家的路上，康康跟父亲商量，能否让他上网玩3天，尽兴了说不定就没有兴趣了，之后他保证好好上学。他与父亲签下了协议书。三天后，康康回到学校去了。<br/><br/>然而，好景不长，2010年5月16日，因为父母阻止他玩网络游戏，康康离家出走12天。最后，父亲在网吧把儿子找回来，双方再次签订协议：康康回学校上学，家里装宽带，晚上让康康上网到零点。<br/><br/>8月，康康执意要去重庆会网友，父子俩再度签订协议：父亲陪康康一起去重庆，康康会完网友后就好好上学。从重庆回来后，康康断断续续上了学。<br/><br/>今年4月初，康康向家里要了120元生活费。一拿到钱，康康转身就进了网吧，被父亲逮了个正着。此后一段时间，为了拿到上网的费用，康康屡次威胁母亲，甚至用上了刀、螺丝刀和充电绳。这让父亲对儿子彻底失望，想到儿子今年2月已满18岁，他拿康康已没有办法，在康康多次催逼下无奈于4月9日签下最新的一份协议，也是第五份协议。<br/><br/>新闻链接<br/><br/>家庭教育支持机构昨表示愿免费救治宅家4年网瘾青年<br/><br/>（记者舒均）汉口一名20岁小伙沉溺游戏家中宅4年，家人拿他没办法（见前日本报A13版）。本报报道见报后，引起广泛关注。<br/><br/>昨日，纽特思特（中国）家庭教育支持机构表示，愿免费救治张师傅的儿子。该机构武汉区域负责人江普称，青少年网瘾与学校教育、社会环境尤其是与家庭教育有直接关系。他们将派出专家，给张师傅儿子制订救治方案。<br/><br/>多方意见<br/><br/>高中班主任<br/><br/>他很聪明但自制力太差<br/><br/>昨晚，康康的高中班主任宋老师接受采访时说，康康是个智力不错的学生，本来成绩挺好，曾获得多个奖项。如果没有网瘾，他今年考个重点大学没有问题。<br/><br/>宋老师说，由于康康自制力太差，没有养成良好的个人生活习惯和学习习惯，导致沉迷于网络游戏后不能自拔。康康还经常找同学借钱去网吧玩游戏，招致同学们的反感。作为班主任，他多次找康康谈话，想把他拉回来，但种种努力都付诸东流。<br/><br/>宋老师说，虽然康康长期旷课，他还是为康康保留了座位，希望他能回到学校，但直到今天，马上就要高考了，康康还是迷途不返。<br/><br/>心理专家<br/><br/>康康缺乏生活动力父母不能抛弃他<br/><br/>昨日，中南医院心理专家肖劲松说，康康这种情况属于“病了”不仅精神方面出了问题，而且个性发展也出了问题。<br/><br/>肖劲松说，康康比较早的获得多项荣誉，其个人已达到了某种高度，发展欲望渐渐消失，当他步入网络后，重新发现自我，发现玩网游比学习更快乐更顺心。<br/><br/>肖劲松还分析说，康康已对父母失去信任，这与父母平时只关心学习，不关心他的其它方面有关。他说，康康比较聪明，希望他的父母不要对他失去信心和耐心，否则，他会被父母和社会抛弃，那样后果更严重。<br/><br/>社会学博士<br/><br/>网瘾已成公共问题<br/><br/>（记者陈倩）从流浪网吧10年，重病缠身凄惨离世的天门网瘾青年王刚，到今天一纸令人唏嘘的“上网协议”，网络依赖症已经引起了全社会的高度关注。湖北大学政法与公共管理学院副教授、社会学博士张继涛在接受记者采访时表示，网络依赖症已经成为一个不容忽视的公共问题，需要引起政府和社会各界的高度关注，从政策、技术等方面加强应对。<br/><br/>张继涛还指出，当前城市里流行的“社会冷漠症”，造成人际关系疏离，这在一定程度上也使得部分人选择从现实生活中退却，转而向虚拟世界寻求慰藉。家长和老师需要加强与孩子的沟通和交流，及时发现他们成长过程中不好的苗头，并加以解决，而不能出现管理真空，等到问题已经凸显的时候才应对，为时已晚，也不会有好的效果。<br/><br/>作者：舒均<br/>首次转载时间:
