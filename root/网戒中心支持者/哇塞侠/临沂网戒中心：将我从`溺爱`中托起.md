/><br/><!--more--><br/><br/>十几年来，我都不知道自己将来要干什么，虽然从小到大，我的成绩一直很好，也是老师和同学们眼中的好学生，可我根本不知道自己为何而学习。<br/><br/>当我不想上学的时候，妈妈跟老师说明了我的状况，老师压根不相信。因为他们想像不到：一个品学兼优的好学生竟然可以痴迷网游到这种地步；他们引以为傲的好学生竟全然不顾自己的未来。<br/><br/>不是我不顾未来，而是当时的自己心中根本没有未来。当时的自己，没有目标和理想，只想沉迷在网络游戏里，浑浑噩噩，每天如行尸走肉般。晚上不睡觉玩通宵，玩累了抱着平板或手机就睡，下午起床接着又趴在电脑上玩。甚至，我对父母的劝说有时候还会以暴力相向，当时家里的门没有一扇是好的，电视让我打碎了，妈妈的手机我摔坏过三个。更过分的是，我曾为了做线上活动不去奶奶家过年，大年三十把妈妈一个人锁在了冰冷的门外……<br/><br/>我认为自己是绝对自由的，任何人都管不了我。可现在回想那时的自己，真的快乐吗？<br/><br/>现在想来，是不快乐的，甚至可以说是痛苦的。<br/><br/>我失去了同学、老师、朋友、亲人，我把所有关心我的人都拒之门外。同学来的电话，我不接，也不让妈妈接，一是因为我在虚拟的世界里走不出来，只想活在里面，对现实的生活的一切失去了兴趣。二是因为长期沉迷网游让我变得不敢面对他们，我害怕他们在背后偷偷地评论我。<br/><br/>为了省出更多的时间玩游戏，连吃饭我都吃最简单的。有时候过节爸妈叫我出去吃饭，我从来不去，一是怕耽误时间玩游戏，二是在家时间久了，我怕碰见熟悉的同学、朋友。想一想，那时的我已经算是与世隔绝了。<br/><br/><br/>当爸妈把我骗到临沂戒网中心的时候，我绝望了。只要玩过《魔兽世界》的人我想都曾听说过这地方的“恐怖”。我当时以为妈妈打算抛弃我，彻底不要我了。没有想到的是，她和我一起留下了。更没有想到的是，网戒中心根本网上传的那样恐怖。<br/><br/>有了妈妈的陪伴，我由原来的伤心、绝望，到慢慢适应；从被动改变到主动改变；从没有目标到终于找到了自己的人生方向，最终真正找回了自己。<br/><br/>而相比我而言，妈妈的变化更是巨大的。<br/><br/>平心而论，自小妈妈对我十分溺爱，以至养成了我在家中“衣来伸手饭来张口”的习惯，对于父母的辛勤付出更毫不在意，只在乎自己的感受。可是来中心之后，妈妈因为溺爱我的原因几次受到了杨永信医生的批评。<br/><br/>以前都是我想吃什么就让妈妈买什么，而杨医生则告诉我和妈妈：家长买什么，孩子吃什么，心怀感恩，不该挑剔。<br/><br/>以前我的衣食起居全部都是妈妈一手包办的，我连袜子都没洗过，但是在中心提倡“自己的事情自己做”，就连厕所都要我们盟友自己学着去刷洗。<br/><br/>以前我随便几点睡觉几点起床，以为那是自己的事情，但点评师告诉我们：不能因自己玩乐影响父母休息，更不能因为玩乐影响自己的健康，因为这些都是不孝的行为。<br/><br/>可以说，中心这个地方处处与众不同，每一个新盟友进来都会极不适应，因为从外到内，从里及表，中心传递给我们的都是正能量，那种难受与不适应彷佛给黑夜中的小鬼晒了个日光浴。用阳光心态来浸泡我们，让我们学会了分享，学会了感恩，学会了孝。<br/><br/><br/>现在，我离开中心三个月，现已重返校园，感慨很多。原来在我眼中毫不珍惜的同学情、师生情，现在都让我倍感珍惜；曾经落下的功课我也在努力追赶；曾经对父母的爱毫不感恩，是中心让我明白：当父母对我们有所付出时要说谢谢，而子女做了错事让父母难过时也要道歉。我想把曾经失去的东西一样一样的找回来。<br/><br/>终于发现，原来真实的生活比虚拟的世界有意思多了，我也终于有了自己的目标：那就是要以好的成绩考上高中，然后考上理想的大学，学自己喜欢的专业，才能选择自己理想的工作――工程师。<br/><br/>若不是在中心，今天的我绝不可能重返校园，更不可能取得现在的成绩。因为心中有了目标，我这次期中考试取得了全A的成绩，但自己还有很大的提升空间，我要更加努力，为了曾经给我指点迷津的杨医生，为了父母的不抛弃不放弃，更为了自己光明的未来。<br/><br/>感谢中心，让我有了今天的生活，心怀感恩，就像离院宣誓时所说的：我不会辜负爱我的人，更不会辜负自己的青春。<br/><br/><br/>盟友小斌<br/>首次转载时间:
