/><br/>――离院3年的一位盟友的亲身经历<br/><!--more--><br/><br/>亲爱的各位朋友：<br/><br/>大家好，首先做一个自我介绍。我是临沂网戒中心2013年的一位盟友，也是当届的班长，离院三年了，现在的我是一家私营企业的法人，我与妻子共同经营着一家属于自己的公司。近期我们看到网络上有关于临沂网戒中心的报道，其中有很多内容与我的亲身经历不符，我想跟大家说一说我心中的临沂网戒中心。媒体关注的话题有这么几个：“电击治疗”，家长强制带孩子来中心的方式，还有网戒中心里的“黑暗制度”。从这几个方面我也说一说我亲眼见到的，给大家提供一个参考。<br/><br/>先说下网戒中心的制度吧。在我看来，网戒中心的制度确实严，但是有没有人想过：这里的制度为会什么这么严？我认为这得从网戒中心收治的这些孩子们说起。这里面的盟友们在来网戒中心之前，包括我自己，都是极端危险的。<br/><br/>记得有一个17岁的小个子女盟友，就因为父母希望她能坚持完成学业，不愿让她通宵上网，她便离家出走找网友鬼混，甚至还在网友的怂恿下给自己的亲生父母下毒，不过幸亏没有得手，不然她就成为报纸上我们看到的新闻了。大家觉得还有比这种孩子更让人感到恐怖的么？不是夸张，网戒中心里有暴力倾向的孩子占大多数，其中有很多都做过自伤、自残甚至伤人的行为。家长们为了保护自己孩子和别人的安全，自发的成立了家委会，并由家委会监督，由全体盟友投票选拔出一些改变相对较好，状态相对较稳定的盟友组成班委会，在家委会领导下，开展工作。家委和班委再根据工作需要划分职责，并制定相关制度。而且，在网戒中心有个词叫做“传承”，每一任家委成员或班委成员，都要将自己的工作经验与新接替者交流。我作为班长，每天的工作就是要把盟友们每天的学习、生活情况以及心理波动及时反馈到医生那里，再由医生根据病情及时调整治疗方案，对一些有冲动行为的盟友及时制止，并稳定他们的情绪，而并非如网上所讲，班委成员之间勾心斗角或者官官相护。而且作为班委成员的话处处要起模范带头作用，积极传播正能量，班委只会比普通盟友的要求更高更严格！网戒中心的管理制度是自发的透明的也是适应需要的，更是充满阳光的，家长管理自己的孩子，本来就应该是和谐的温暖的，在别有用心的人眼里却变成了黑暗的见不得光的了，我只能说是你们自己的心理也太阴暗了吧！再说说大家一直关注的“电击”。柴静的《网瘾之戒》我看过，那里面卫生部叫停的电休克治疗仪反正我是从来没见过，我只见过那台低频脉冲治疗仪，之前在中医院也见过。说实话治疗时是不舒服的，有疼痛感，但绝不是不能承受，更不像网络上那些谣言说的“每一个细胞都在疼”。而且我觉得它真正的作用是在你冲动或心理问题严重的时候让你冷静下来，而且这只是网戒中心众多环节中的一个，我不理解为什么大家都把目光聚焦在这一点上。<br/><br/>还有很多人特别在意孩子是被父母骗来或者绑来的。这其实就像我们小时候生病了，被爸妈强行喂药或者打针。你可能会说那时候我们还小，心智还不成熟，需要父母来帮我们做决定。哪一个心智真正成熟的人会沉迷网络、自闭、自杀、吸毒、赌博、纵欲、甚至伤人、危害自己与父母甚至他人的生命安全吗？追着这些问题不放的人应该好好想想，网戒中心里有哪一个盟友是品学兼优或者对家庭对社会有责任而被送来的？而天天诋毁网戒中心的这些人，又有哪一个是真正能对社会对家庭有责任有担当的？我看到的网戒中心，每天教给我们的都是国学经典，是中国几千年传统文化的精髓，是教我们做人做事的道理，教人向善又有什么不对？作为曾经在网戒中心改变过的盟友，我很想劝一劝那些只会无限放大自己痛苦的盟友们，请放下你们的自私，就算为了生你养你的爸妈，承受这点痛苦又如何呢？即便你们仍然坚持自己，愿意自己待在自己阴暗的角落，也请你们不要再污蔑网戒中心，它是充满阳光的，只有你们的内心，才是阴暗恐怖的“纳粹集中营”！<br/><br/>如果有关领导能看到我写的这篇文章，也希望您们能认真地看完它。正所谓兼听则明偏信则暗，我们不能一味地去听信网络上少数几个没有改变好的孩子胡言乱语，而去冤枉一个兢兢业业、挽救无数家庭于水火的当世名医，希望大家能客观公正的对待网戒中心，因为如果没有了网戒中心，那些没有改变好的盟友们一旦流入社会，势必又演变成另一个更加严重的问题。我坚信，教人向善是绝对没错的。更何况网戒中心全程有家长陪伴，如果真像某些人说的那样煎熬，那自己的父母天天看在眼里，怎肯答应？请不要偏听偏信某些无耻之徒的中伤污蔑，请还网戒中心一个正义，还杨医生一个清白，还孩子们一个未来！<br/>首次转载时间:
