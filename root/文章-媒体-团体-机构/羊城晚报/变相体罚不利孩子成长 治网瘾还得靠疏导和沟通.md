<br/>电影导演周勇支持叫停“电击治疗网瘾”<br/><br/>本报讯 记者夏杨、通讯员邱秋菊报道：现在正处于暑假期间，有些中小学生无事可做就泡在网上，许多家长担忧孩子会上网成瘾。近日《网络妈妈》导演周勇在接受记者采访时表示，杨氏“电击疗法”被叫停，正说明变相体罚对孩子成长不利，治疗孩子网瘾还是要靠疏导和沟通。<!--more--><br/><br/>山东临沂市精神病医院医生杨永信利用电击和服用精神类药物治疗网瘾，并声称不会对孩子造成任何伤害，在社会上引起了极大的争议。日前，卫生部发出通知称，“电击治疗网瘾”技术的安全性、有效性尚不确切，暂不宜用于临床，要求停止该疗法的临床应用。<br/><br/>“杨氏疗法被叫停，正是顺应了家长的呼声！”《网络妈妈》导演周勇对社会上网瘾治疗方法一直持反对态度。他的电影《网络妈妈》是时下流行的一部电影，反映的是一个网瘾少年的救赎过程。周勇说，他在拍摄这部电影时，采访过100多个网瘾少年家庭，和百余位网瘾少年谈过心。他发现很多上网成瘾少年都不愿意让人称为“网瘾少年”，也不同意接受采访。“如果将‘精神病’的帽子扣到这些迷恋于网络的孩子身上，无疑会引起他们更大的反感。”<br/><table border="0" cellspacing="0" cellpadding="0" align="left"><br/><tbody><br/><tr><br/><td>&nbsp;</td><br/></tr><br/></tbody><br/></table><br/>周勇说，这些家长对社会上的一些治疗网瘾方式就不敢苟同，他们认为通过采用电击、棍棒等对人体进行摧残或精神上折磨的做法非常不可取，对孩子的成长会有阴影。<br/><br/>周勇也发现，一些网瘾学校根本谈不上配有教学设备和师资队伍，更谈不上有经验的心理专家。而且收费昂贵，少则每月三四千，多则过万。为此，周勇劝诫网瘾少年家长，如果非要让孩子去接受治疗，一定要对网瘾治疗机构做深入的了解，然后再做决定。<br/>首次转载时间: 2018-07-17 15:49:23"
