	<li><b>简谈“网瘾”治疗专家杨永信到底有几宗罪&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</b></li><br/></ul><br/></li><br/></ul><br/></li><br/></ul><br/></li><br/></ul><br/>1、杨侵犯了公民的基本人权.<br/><br/>所谓人权，是人之为人应该享有的权利，是公民的基本权利。我国在《宪法》第三十三条第三款明确规定了“国家尊重和保障人权。”，并且迄今为止我国已加入了25个世界人权公约，发布了十余个中国人权白皮书。<br/><br/>而杨罔顾孩子的脆弱的身体和心灵，以电击的方式对孩子的身体施加剧烈的疼痛刺激，使得孩子因恐惧而做出违背内心的意思表示。法律都只能强制人的行为，杨永信控制着孩子的思想，使其连起哭诉疼痛的言论都不准予发出，其眼中毫无人权尊严可言。<br/><br/>2、杨无权进行电击治疗.<br/><br/>2009年，卫生部在对《未成年人健康上网指导》征求意见时，否定了将"网瘾"作为临床诊断的精神病，认为目前"网络成瘾"定义不确切，不应以此界定不当使用网络对人身体健康和社会功能的损害。 同时，卫生部提出了新的概念，认为"网络成瘾"只是网络使用不当。且国内外，均没有明确将网瘾认定为精神疾病。而电击治疗（电休克）是对精神疾病的一种治疗手段，对其应用有着严格的使用标准。杨永信对着精神正常、尚处于发育期的孩子使用对待精神病人的治疗方法，俨然是不符合医疗规范的。<br/><br/>3、杨的行为显然违反了未成年人保护法.<br/><br/>未成年人保护法是我国出台的专门关爱、保障未成年人权益的法律，杨的行为显然是未成年人保护法的规定背道而驰。然而未成年人保护法的内容过于笼统，没有明确的处罚机制，杨无视该法，为所欲为，却不用受到该法的惩治，未成年人得不到未成年人保护法的保护，让人不禁唏嘘。<br/><br/>4、杨的行为涉嫌故意伤害罪.<br/><br/>我国刑法第二百三十四条第一款规定了：“故意伤害他人身体的，处三年以下有期徒刑、拘役或者管制。”个人认为，该条文所述的身体应该广义的身体，即包含了物理上人的肢体、器官、组织和人的精神健康，否则以身体未受损的行为造成人精神失常却不受刑法处罚，显然不合理。而杨自戒网瘾学院开办以来，对数千人进行了多次的电击，且该电击行为不属于正常的治疗手段，显然是一种故意伤害，对很多青少年造成了心理上的阴影，即便是未造成严重后果，但人数众多，受害者规模庞大，对社会的危害性也不容小视。<br/><br/>5、杨是恶魔，父母更是帮凶.<br/><br/>杨的行为惨无人道，但对孩子造成伤害最大的却不是杨，而是孩子的父母。法律给予了父母对孩子监护的权利，但监护权不仅仅是权利，更是一种义务。法律规定了父母照顾子女、使子女受教育、支付必要生活费、不得遗弃等众多的义务。父母身位子女的法定监护人，不代表可以对孩子进行任意的支配，没有权利可以让他人对子女进行合法的伤害，否则父母就是该犯罪行为的共犯，应与其受到相同的法律惩罚。<br/><br/>文中的父母因为自己未曾管教好子女，就将子女送入人间地狱之中，子女受到伤害后的求助被视若无睹，唯一的依靠成了帮凶，子女不再反抗，不再求助，他们不解、愤怒、恐惧。在绝望之中所有的负面情绪最终指向了自己的父母。因此才会有，某一接受治疗完毕的孩子回家后说出：“我剩下的日子就是为了让你们痛苦。从此我不做任何事情，让你们断子绝孙。”<br/><h1>知乎某个用户解读</h1><br/>&nbsp;&nbsp;&nbsp;<br/><br/>&nbsp; &nbsp; 了解了这个事件之后，我脑子里蹦出的第一个问题，就是，这种事情的发生到底应该怪谁？家长？还是杨永信这样不人道的行为？<br/><br/>&nbsp; &nbsp; &nbsp; “孩子睡觉的时候，脚露在外面。幸亏我及时看到，把脚给砍了下来，要不肯定着凉了。”这是我在知乎看到的一句评论。伊坂幸太郎说，一想到为人父母居然不用经过考试，就觉得真是太可怕了。<br/><br/>&nbsp; &nbsp;&nbsp; 事实就是中国还有千千万万这样愚昧无知的家长－原谅我的不友善，愚昧无知已经是我能想到形容他们最友善的一个词。<br/><br/>&nbsp; 这些家长只能看到孩子和他们“不亲”，而且不会去思考不亲的原因是什么，家庭教育有什么问题，而是单纯的觉得孩子走上了“歪路”，把一切责任推给孩子，推给这个“光怪陆离”的现代社会。而杨教授之流就是抓住利用了相当一部分家长的愚昧无知，成就了他的神话，其成功模式与玛咖神话，虫草传奇之类并无本质区别，只是他的危害不只是几个钱，而是伤害了无数孩子的身心健康。<br/><br/>&nbsp;&nbsp; &nbsp; 然而福柯在《疯癫与文明》中，就把定义精神疾病看作是一种权力。他认为，精神疾病能够定义谁正常、谁不正常；谁需要被改造，而谁有权力改造人；谁应该呆在人群里，而谁应该被孤立、被排除出人群外……他甚至把整个文明进程看作是“理性”通过定义精神疾病来压制“非理性”的过程。<br/><br/>&nbsp; &nbsp;&nbsp; 社会文明的进程总是伴随着对个人权力的尊重和对其他权力的限制，对个人身体支配权更是最基本的人权。几十年前，精神病学家还曾把同性恋纳入“性变态”的范畴，而今天，哪怕是在中国，如果所谓的“精神病患者”不愿就医，别人想要把他送去精神病院住院也是难上加难。从这个角度，杨永信就是一个反文明的存在。<br/><br/>&nbsp;&nbsp; &nbsp; 关于网瘾，前两年很多国内的专家确实纷纷扬扬地讨论过一段“一周上XX小时”就等于网瘾的诊断标准。其实这些专家的意思，归纳成一句话就是“你有病，而我有药啊！”所以很多人通过卖药就成了一方富豪。<br/><br/>&nbsp;&nbsp; &nbsp; 根据2015年8月29日表决通过的 《刑法修正案（九），在刑法第二百六十条后增加一条，作为第二百六十条之一：“对未成年人、老年人、患病的人、残疾人等负有监护、看护职责的人虐待被监护、看护的人，情节恶劣的，处三年以下有期徒刑或者拘役。有前款行为，同时构成其他犯罪的，依照处罚较重的规定定罪处罚。”以前虐待罪犯罪主体不包括像杨永信这样家人以外的人，现在包括了。<br/><br/>&nbsp; &nbsp;&nbsp; 虽然虐待罪案件是自诉案件，是被害人要向人民法院提出控告，人民法院才处理，不告诉不处理。但如果被虐待者受强制、威吓等而无法向人民民院起诉，人民检察院可以提起告诉。被虐待者的其他近亲属也可以控告，有关单位和组织也可以向人民检察院检举揭发干涉，由人民检察院查实后提起告诉。犯本罪，致被害人重伤、死亡的，不适用“告诉的才处理”的规定。<br/><br/>&nbsp;&nbsp;&nbsp;
