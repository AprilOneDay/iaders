<br/>6月22日，杭州一名13岁的学生因玩王者荣耀被父亲教训后从四楼跳下。此前4月底，有媒体报道称，广州一17岁少年狂打手游“王者荣耀”40小时，诱发脑梗，险些丧命。<!--more--><br/><br/>在媒体连篇累牍的报道下，王者荣耀的火爆，已经成为所谓“乱象”。虽然腾讯在舆论压力下，推出防沉迷措施，12岁以下未成年人每天限玩1小时。但质疑并未减少。<br/><br/>人红是非多，游戏红也一样。眼下的王者荣耀，已经不仅是一款手游，而是一种社交方式。年轻人因此分为两种：玩王者荣耀的和不玩王者荣耀的。从市场竞争的角度说，这是国内手游巨大的成功，而不像掌上主机、街机、个人电脑时代，游戏中清一色的日文与英文。<br/><br/>放在任何一个行业，国内厂商取得这么大成功，媒体上充斥的都应该是胡萝卜，但游戏不一样，它被视为精神毒品，一个季度60亿的收入就是“罪证”，好像这个钱，腾讯如果不赚，任天堂们也不会赚。但是不对，游戏市场一直火爆，没有国产游戏，也会有别的。<br/><br/>从电视瘾，到网瘾，再到游戏瘾，只要童年流行什么，就会有人忧虑什么，电疗网瘾的杨永信式专家，永远都有饭吃。可是，现在写文章的记者，自己就是电视瘾一代、网瘾一代啊，电视和电脑真让你后悔，让你中毒了吗？难道你们都是被杨永信电疗好的吗？显然不是。<br/><br/>类似报道都有个问题，那就是以偏概全，拿个案说话。要知道，王者荣耀注册用户超2亿，2亿人中出现小概率事件，并不能说明什么。有考生高考后自杀，而且不是一例两例，高考是否应该取缔？当然不该。小概率的个案，并不能用来代指游戏的十恶不赦。<br/><br/>如果网瘾真是病，全国人民十之八九都有病，而是已病入膏肓；如果游戏真是病，大学就不应该开什么电竞专业，全国青少年十之八九早该被毁了。这些显然是扯淡。孩子们为什么爱玩游戏？因为游戏里有探索的乐趣，有大量的社交，有清晰的即时反馈，这些都符合我们祖先采集狩猎的生活方式。<br/><br/>游戏是一种精神过程，游戏是一种平等与自由的媒介，能够最大程度地发挥人的自主性，通过即时反馈和社群认同，使人感受到自己的行为有意义。在沉重的学习压力之下，适度让孩子玩玩游戏，并没有那么可怕。<br/><br/>尤其是，当王者荣耀成为一种社交方式，不让孩子玩，其实就是剥夺了孩子正常社交的权利。和同龄人在一起，他就是一个局外人，融入不了同学的团体，没有共同的爱好和共同的话题。我们说“这就是爱”，却完全不在乎孩子的心声――“这就是哎”。死读书不可能兴趣广泛，读死书不可能阳光健康，感受不到自由，体验不到快乐，没有成功的喜悦，没有释压的通道，心理难免扭曲，青春难免迷失。<br/><br/>一个游戏能火到这个程度，必然有其理由，至少质量不会那么差。让王者荣耀背上黑锅，没有什么意义，游戏产业不会更小，无非谁来赚钱而已。防止孩子沉迷游戏，关键在于引导，可以和孩子约定一个游戏的时间，为了保护眼睛还可以约定场所，但在尊重孩子的前提下好好说话，远胜于暴君式的老子说了算。<br/><br/>湖南省常德市舒圣祥<br/>首次转载时间: 2018-07-17 14:55:50"
