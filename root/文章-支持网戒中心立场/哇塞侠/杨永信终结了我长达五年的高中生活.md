/><br/>黑格尔认为，宇宙的本原是绝对精神。它自在地具备着一切，然后外化出自然界、人类社会、精神科学，最后在更高的层次上回归自身。因此，凡是在这个发展轨迹上的就是合乎理性的，也就是必然会出现的、是现实的。反过来讲也同样成立。<br/>  就由以上所引用百度百科，关于黑格尔名言：“存在即合理”的解释说明，开始这篇东西吧。<br/>  近期关于临沂市第四人民医院网戒中心消息，负面的，积极的，各种各样接踵而至，作为2007年5月份入院进行改变，同年9月份出院的一位过来人，发表一些自己的想法。<br/><!--more--><br/><br/>   曾经的我有着太多问题，有太多不堪回首的往事，家常便饭一般的欺骗父母、逃课打架、沉迷游戏，最终磨灭了应有的人性、丧失了自己的良知、荒废了可惜的学业、玩掉了肩上的责任，甚至一度因为偷宿舍同学的钱坐在了派出所冰冷的审讯椅上...<br/><br/>在2004年中招考试的时候，我以高出重点高中录取分数线31分的好成绩，进入我市最好的高中。可我没有和其他高中生一样，为自己能考入自己梦想的大学而忙碌，而是一度的跟班主任闹矛盾，由于自身的认知偏差，不能适应她那种管理班级的方法以及单纯追求的分数的理念；也由于自己缺乏吃苦精神，班级高手林立，加之接触了网络游戏，面临着高中课程的复杂和困难面前，自己选择了退缩和逃避。在学校找不到自尊；对身边事务的厌倦；对父母唠叨的错误理解，我被网络游戏的这只手紧紧地抓住了。<br/><br/>那时候，我经常以各种理由欺骗父母，不是同学过生日，就是学校搞活动。想方设法要钱、骗钱、甚至偷钱，来满足玩游戏的欲望。一个电话打回家：“今晚我住同学家”，便一夜不归。妈妈曾经叫上表姐挨个网吧挨个网吧地去找，明地里的、暗地里的展开拉网式的进行寻找，当好不容易找到我的时候，我却撒腿就跑。还说：“谁让你们找我？”<br/><br/>记得有一次妈妈得知我就要放假到学校找我时，我说已跟同学商量好一起去玩，不想回家时，妈妈心脏病当即发作，要我陪她回家，我却没良心的说：“你不会打120？”。有一次家里的烟不见了，问我拿了没有？自己死不认帐。我爸说：“家贼难防，偷断屋梁。”竟还说出了丧尽天良的的话：“你想死啊？”还有一次玩游戏正在兴头上，爸爸骂着砍断了网线。自己一拳打过去，父亲左眼眶上顿时起个大包。<br/><br/>父母不惜花重金将我转入省城一所较好的高中，让我重读高一，希望我能够重整旗鼓，从头再来。自己却没有珍惜这次难得的机会，不但没有悔改，相反越发严重。有一次我周六未归，父母借车到网吧里找到我时，自己躲进厕所换了衣服准备从二楼跳窗而逃。被发现后，跟父母一同前去的五个人围、追、堵、截，我却仍一根筋的不回家。所有前去的人都要开车返回，只有妈妈坚持到深夜终于把自己拉回了家。终于一次模仿班主任签字外出上网被劝退，再次转学后又一次夜间翻墙出去上网被开除。虚荣、攀比、撒谎、欺骗成了家常便饭。学校一转再转，游戏一玩再玩。磨灭了应有的人性、丧失了自己的良知、荒废了学业、浪费了时间玩掉了肩上的责任。冲动、任性、焦虑、烦躁。那个时候真的是儿子不是儿子，父母不是父母，家无宁日，整天过着非人的生活。最为严重的是，2006年9月的一天，为了上网玩游戏，自己一次性偷了同宿舍同学1200元钱。被告发后，被铐上手铐用警车从家里把自己带到了当地派出所。当父母赶去接我时，自己还被锁在被审讯的铁椅子上。那时玩游戏玩迷了的我，居然不知道害怕，还很不耐烦地说：“你们来干嘛！”<br/><br/>父母为了唤醒自己，带我去深圳开过眼界；求助过同学、朋友、亲戚进行劝说；听过戒网瘾专题报告会；找心理专家谈过话；让我打工体验过蓝领和白领的差别；还放开让我彻底玩过；妈妈也求过神、拜过佛、算过命，均不凑效。相反，自己在网络的泥潭里越陷越深，不能自拔。<br/><br/><br/>  犹记得那是2007年5月27日，父母把我“骗”到临沂网戒中心，自此我开始了自己的改变之旅。<br/><br/>  在网戒中心的日子里，一幕又一幕亲情重现动人场景的感召，和受良好氛围的感染，我开始转变自己的思想，开始改变。<br/><br/>  网戒中心，是医院，有真诚服务的医生和护士；是军营，大家身披迷彩装施行军事化的严格管理；是学校，能让人学到一生受益无穷的东西。这里更是一个大家庭，人人互帮互助，相亲相爱。这里有独特的模式，有各种不同的环节，有世间难寻的氛围，这里是一方没有污染的净土。<br/><br/>  在这里，我脱胎换骨，可以说是重新活了一次，我也深深地理解到父母的良苦用心。通过自己的努力，我逐渐得到了家长和盟友的一致认可，从入院时的一名普通盟友，到成为放心班的一员，再到班委的一分子，最终我成为了同盟班会的班长。同样是在这里，我明确了自己的目标，许下了自己的承诺：永不再偏，直至成功。<br/><br/>  在这里，经过3个多月的学习，我与父母重新找回失落已久的亲情，我的家庭重拾温馨和欢笑，我学会了孝和韧，学会了感恩和理解，懂得了宽容和责任，明白了珍惜和诚信，调整了自己的心态，还有很多、很多……<br/><br/><br/>  而出院才是改变的真正开始，整个环境的改变，更需要自己做好自律坚持，以在网戒中心的标准严格要求自己，调整好自己的心态，一步一个脚印扎实的前进，至少，要守住自己的底线。出院后，我也出过问题，作为我本身也很快意识到严重性，父母也坚持长效机制不放松，遇到假期便回中心充电，解决自身问题，完善自身性格。<br/><br/>  由于沉迷网络，我的前两个高一没能学到任何东西，之后休学一年，并来到中心改变，这三年的时间都荒废了过去。刚刚出院的时候，高二新学期也已经开学一个半月了。我没有同意妈妈重读高一的建议，而是强烈要求上高二，因为经过在中心的改变，我对于很多事都有了新的认识，我深深知道我不能再耽误时间，我要把失去的时间都找回来，于是我告诉母亲：“我的高中不能上六年！”而在刚进班的时候，由于我是一个插班生，老师自然有看法，我自然少不了老师的冷眼。至今我仍然清晰的记得，那时班里已经没有多余的位置了，老师就找了个凳子，往倒数第三排的过道里一扔，告诉我说：“你就坐这吧！”这与之前中心相亲相爱的氛围形成了巨大的反差，心里不知道是什么滋味。<br/><br/>  仅仅几堂课上完，我就发现我真的什么都听不懂。出院时信心满满，现在连受打击，心里十分着急，也伴随着一种失望。我告诉母亲：“妈，我什么都听不懂。”父母听到后，没有责备，而是对我进行鼓励，告诉我能学多少是多少，哪怕是倒数第一，他们也不埋怨我。我知道，自己不能再荒废时光，不能再让父母失望，不管怎么样一定要坚持下去。也许就是这样一种信仰，我在教室里坐稳了板凳。<br/><br/>  我还记得在我刚上初一的时候，由于年纪小，比较淘气，给班主任留下了很不好的印象，刚开学不到一个月就叫家长。但是，在第一次考试时，我考了年级的并列第三，一下子改变了老师对我的看法。我也因此明白，老师对于一个学生的评价和看法，在很大程度上源于学生的成绩。但是现在什么都听不懂，很难改变老师的看法，我想到了与老师进行沟通。<br/><br/>  有一天，我把自己的这一想法告诉了母亲。之后妈妈找到了班主任，把我的网戒经历告诉了老师，希望老师给予帮助。班主任知道后，也抱着一种同情，对我也自此改观，并把我的位置调到了前列，而且周围都是学习比较好的同学，平时有不懂的就问旁边的同学。在初中的时候，自己学习非常好，带着一种骄傲，根本不屑于向其他同学问问题，但是我知道，自己是中心走出的一员，不应该有这样的心态，况且自己现在什么都不会，一切从零开始，更应该虚心向同学请教，哪怕是学习上去了，遇到不足之处仍应当如此。妈妈也告诉我，不要贪多，自己能听一句是一句，能懂一点是一点，一点一点来，只求学有所得。<br/><br/>  就这样，在父母、老师、同学的帮助下，抱着这样一种心态，一段时间后，终于有些内容能听懂了，逐渐有了收获。在期中考试时，名次由倒数第一提升到了班级第六十八名，遗憾的是有几门课程没有及格。父母对此非常满意，而我知道这是父母的鼓励，不想给自己太大压力，自己决不能这么想，不能因此而满足，不能有任何的松懈，也更明确了下一步该如何学习。<br/><br/>  大家都知道，做什么事只要找到了兴趣，哪怕过程再艰难，自己也会想办法克服，最终得到自己想要的结果，学习自然也不例外。<br/><br/>  在经历了一段异常艰难的学习过程后，自己终于在学习上有了突破，由于自己初中时物理的成绩比较好，也很喜欢物理的内容，这种突破出现在了物理这一门课程上。渐渐地，物理这一门已经基本上能完全听的懂了，能跟上老师讲课的进度和思路，有了成绩，自然非常喜悦，就更喜欢这一门课程，产生了兴趣，就更用心的去学习这一门，就这样形成了一个良性循环。在高二上学期的期末考试中，成绩有了极大地提高，考到了班里的第二十四名。虽然在物理上有了这样显著地突破，成绩更是直线上升，但是我明白仅仅是这样还不够，自己在中心留下了承诺：毕业时拿着大学的录取通知书来见杨叔。仅仅是这样根本难以实现自己的诺言，难以实现自己的目标。<br/><br/>  有了前面学习物理的成功经验和那一份坚定的信念以及明确的目标，这一阶段的学习已经由最开始的急躁，艰难，慢慢的变成了积极主动，掌握了属于自己的学习方法，已经由物理这一门，延伸到了所有的课程。就这样，物理赶上来了，数学赶上来了，生物，化学……这时，我知道自己虽然在学习上已经有了非常大的突破，但是与其他同学之间还有相当大的差距，而且，对于理科的学习，根本不能偷懒，只有依靠多练习，多做题才能熟练掌握。我给自己定下了一个计划：在每天按照老师的计划学习，完成各项任务之外，每一门都要多做三十道题。那时候，真的除了课间出操，吃饭和上厕所之外，自己就没有离开过座位，晚上，也开着床头的台灯，把每一天老师布置的，自己布置的任务都完成后才睡觉。<br/><br/>  就这样，持续到了高二结束的那个期末考试，一下子考到了班里的第十二名，从倒数第一，到六十八，现在到第十二，一步一步都挥洒着自己的汗水，这种成绩也是对自己付出的最大肯定，最大的收获。学习现在已经不是问题了，每一门的成绩都在全面提升。现在甚至能帮助其他同学解决他们不懂的问题，从听不懂，到求助别人，到现在能反过来帮助别人，真的那种喜悦溢于言表，异常的自信。<br/><br/>  俗话说自信是成功的一半，确实是这样。当时脑子里就有这么一种声音：我一定没问题，碰到难题我一定能够解决，我一定能顺利兑现自己的诺言。<br/><br/>  高三的学习主要是复习，备战高考。平常有了更多自由支配的时间，我知道这时更需要严以自律，不能因为学习的全面提升产生任何的松懈。而且，高三的复习有好多遍，一边过后，还要重新再进行复习。此时决不能产生厌烦或者自满的情绪，否则只会是前功尽弃。<br/><br/>  自信，自觉，有目标，有动力，高三一年，自己的学习有了一个质的飞跃。不谦虚的说，自己的成绩有了很突出的地方。月考时，我一跃成为年级的第一名。但是我知道，成绩是对自己的肯定和鼓励，而不能成为自己放松要求的一个借口。<br/><br/>  就这样，按照老师的计划，配合自己的自主学习。在最后几次模拟考试中，自己的成绩一直很稳定。<br/><br/>  然而就在这个时候，一次偶然的身体检查，我被告知身患一种治愈率只有不足四成的大病，医生令我立即住院治疗，杨叔也叮嘱我说健康第一。当时眼前一片黑暗，心里根本不知道该怎么办，但是直觉告诉自己要履行自己的承诺，我告诉妈妈：“妈，我的高中不能上六年”。父母在我的祈求下答应我在不住院的情况下进行治疗。在承受病痛的折磨和强烈的药物反应，在发热、口干、头晕、肌肉酸痛、失去味觉、总胆红素升高、白血球降低等的情况下，自己依然坚持着学习。整日没有胃口，吃不下，睡不着……在这期间，父母天天晚上给我送饭，想让我多补充一点营养，风雨无阻，整整一年多的时间，没有少过一次。也许是父母的不放弃，也许是自己的坚持，非常幸运的，在经过两年的持续治疗后，我已经与一个正常人没有任何区别了，战胜了病痛。<br/><br/>  在2009年6月份，我带着杨叔、父母、老师的深切期许，带着自己许下的诺言，我走进了高考的考场。最终经过不到两年时间于2009年以高出分数线30多分的成绩考入郑州大学。<br/><br/><br/>  上了大学之后才发现，大学的生活与高中截然不同。高中老师的严格管理没有了，相反环境格外宽松，需要自己高度的自觉、自律。周围的同学来自全国各地，也不似高中时期只埋头学习，玩游戏、逃课也大有人在，这也更需要自己对自身的严格要求，辨别是非。<br/><br/>  在大学期间，说没有放松那是骗人的，这时，中心的长效机制就发挥了巨大的作用。大学时期我依然坚持每年至少回中心一次，充电加油，解决问题。有问题不可怕，可怕的是不敢面对问题，听之任之，这样就会酿成严重的后果。同样在父母、杨叔的帮助下，在自己对自己的严格要求下，我得以同时在班级和院学生会担任学生干部，同时辅修双学位专业，在学习之余还发展自己的爱好特长，担任院篮球队副队长以及其他一些社团的干部。在2013年7月，以院优秀毕业设计的成绩顺利毕业，拿到了两个专业的毕业证以及学位证。同时通过校园招聘顺利找到第一份工作，进入职场。<br/><br/>  就这样一路走来，才有了今天这样的我。从班里的尖子生，到走偏甚至是磨灭人性，到回归正轨有了正常的生活，这一切都离不开中心。<br/><br/>  有人说电击，有人说洗脑，有人说传销，我想说的是：不管经历了什么，最终的结果，得到的效果是最能够说明问题的。回过头来看过去的自己，看网游、网吧，自己在那个年龄欠缺的是明辨是非的能力，缺乏的更是自制力，把电脑、网络当成了玩具而不是工具，而中心恰恰在那个时间段给了我这些。<br/><br/>  请各位在一味的抨击临沂网戒中心的时候，都能够有一颗平常心，客观的去看待这样一个机构，能够给我们带来什么，自己想要的生活又是什么。如果自己是父母，有了我们这样的孩子，又该作何感想。<br/><br/>最后，存在即合理，任凭风吹浪打，我自岿然不动。任时间去检验这一切。<br/><br/><br/>?盟友小佑<br/>首次转载时间: 2017-02-04 03:26:40"
