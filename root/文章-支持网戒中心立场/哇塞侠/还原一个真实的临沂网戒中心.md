/><br/><!--more--><br/><br/>我来自山东省滨州市的一个普通家庭，2015年12月某日，我被父母送到了临沂网戒中心。为什么要去那个地方，因为我不是个让人省心的人：叛逆、逃学、沉迷网游。我上网不像别人那样有节制，我一玩就是几天几夜，并且脾气暴躁、易冲动，和父母稍有不和我就动手打他们，他们都不敢还手；在学校里，要么和老师对着干，要么就欺负同学、打架，好像天老大我就是老二，没有任何人能管得了我。<br/><br/>“走偏”这个词是我在中心听到的，也就是在走人生道路的时候偏离了正常的轨道，当然，这不是一瞬间的事，而是相当长的一段时间，甚至可能是几年。<br/><br/><br/>我的走偏是从初中开始的。在初一我只是和几个同学去网吧上网，明知道初中生不该去网吧，但还是偷偷摸摸的去，那时候我的思想比较单纯，就是想着在游戏中找到乐趣，不顾环境的恶劣，校规的约束，为的就是一时的痛快。几个月间，每个周末必定去网吧上网，成绩自不用说，像坐滑梯一般往下掉，一向非常重视成绩的妈妈着急死了，不让我上网吧，把我的周末全都给安排上辅导班，使我上网的时间大大减少了，接下来的考试就比较理想了，那个时候我就在想：自己不上网就能有这么好的成绩，上网应该照样也可以有好成绩吧。就这样一种偏差的认知开始形成。<br/><br/>初一还算稳定，边上网边学习，成绩还能跟上，到了初二我结交了一些年级中不学习的、喜欢打架的同学，每天就跟着他们瞎混，玩手机、打架，那个时候感觉特别好玩，无忧无虑的，于是在一次考试后我以成绩为法码向爸妈要了一部苹果手机，于是天天玩，周一至周五在学校还算老实，一到周末就不是我了。骑着摩托车在路上狂奔，上网吧已经成为非常正常的一件事，我也不去什么辅导班了，天天就想着玩。这样的生活持续到了初三上学期，有一次爸爸见我从早到晚玩手机，一怒之下就把我的手机给摔了，从那之后我们就结上了仇。上了初三之后，周围的同学都在刻苦学习，为了一个好的高中而奋斗，而我却丝毫没有紧张感，依旧去网吧上网。有一天，我逃课出去上网吧，班主任和爸爸一起去网吧找到了我，这不但没有让我悔改，反而让我更恨他们，到了那个时候，上网已经成为了正大光明的事，直到离中考还有两个月，几个非常要好的同学成绩都非常好，都能考上我们当地最好的高中，而且我还差得远，于是从那时起我就下定决心要和他们去同一所高中，经过最后一个月的冲刺，终于顺利考上了，这也足以说明我不笨。<br/><br/>中考完的那个暑假就成了我放纵的最佳时机，先是买了新手机，，然后新电脑，每天除了玩还是玩，反正没有作业，我们几个初中同学经常一起出去上网打游戏，虽说家里有电脑，但要的是那种喧闹的氛围。刚刚放假的第一天，我连续上网上了30多个小时，眼睛都没有闭过，随后就是白天睡觉，晚上玩电脑，爸妈也不敢管我，那个时候我跟他们承诺的是：我假期玩够了开学就不玩了，他们信了。在高一一开学的军训中，我就开始以各种理由不参加军训，就想玩游戏，于是先后两次回到家，一回到家就像打了鸡血一样，一摸键盘就找到了感觉，爸妈以为我正式开学后就可以回归正轨了，但是他们又错了，我压根就没有想过好好学习。上课玩手机、吃零食已司空见惯，课堂对于我来说，只是个摆设罢了，就这样过了两个月，我就不想再去上学了，选择在家玩游戏，爸妈着急也是干着急，他们拿我没什么办法。记得有一天晚上十点多，我刚开始玩电脑，妈妈就走进房间，让我关掉电脑，我笑着对她说：“你觉得可能吗？”我很坚决，妈妈生气了，上来对着电脑一顿猛砸，我一下子上来火了，一把把她推到墙上，从椅子上站起来，掐着妈妈的脖子，恶狠狠的对她说：“你别没事找事，滚出去！”但她不听，我冲到厨房拿了把菜刀朝她吼：“你滚不滚！”这时爸爸赶紧过来把妈妈拉走才平息了这场战争，否则肯定会出人命。以上这些就是我在来临沂网戒中心的大体经历，概括为四个字：无法无天。<br/><br/><br/>2015年12月某日，我被爸爸骗到了临沂网戒中心，如果要问我怎样评价网戒中心，四个字：世外桃源。首先是氛围好，盟友之间互相帮助，互相关心，从来没有说脏话的，就像一家人；家长们24小时陪护，盟友们和家长们的关系都特别融洽，这里倡导“相亲相爱一家人”。<br/><br/>在我刚住进网戒中心的时候全身满满都是抵触，别人说的一切我都听不进去，并且非常冲动，为了稳定我的情绪，范医生给我做了个治疗，虽然有些酸痛，但是这让冲动的我开始冷静下来，反省我的问题，和范医生顺利沟通了很多事，我自己也觉得自己在情绪控制方面有问题，需要改善。从那时开始我走上了改变自己的道路。<br/><br/>在中心的日子过得非常快乐并充实。早上6点起床跑步，让盟友们呼吸到新鲜的空气，并通过喊口号呼出体内的废气；每晚十点半准时熄灯，保障每天的睡眠。几个月下来，所有的盟友在健康方面都得到了提升，有的盟友在刚到中心的时候由于长时间卧床上网都已经不能走路了，但经过短时间的调整，他们又重新恢复了健康，我在刚去中心时瘦的皮包骨头，头发几个星期不洗不理，衣服也不换，而如今我已经胖了20斤并且非常强壮。<br/><br/><br/>前几个月，看似平静的我心却不平静，一直不想改变，想着混上几个月就走了，直到看到很多已经出院的盟友回到中心分享他们的成绩，我才意识到改变的重要性，那是影响一生的。从那开始，我走上了主动改变的道路。<br/><br/>然而改变不是那么一帆风顺的，古人很早就说过：“江山易改，本性难移”，我的路也很曲折，大点评课堂是我改变的重要环节之一，我们早上8:00进入大点评课堂，首先是唱歌，唱一些正能量的歌，比如《真心英雄》、《父亲》、《母亲》、《天亮了》，随后是班委负责人提问一些传统文化的背诵，比如《弟子规》、《三字经》，到9:30的时候会有心理点评师来上课，杨永信医生或者各位点评师会根据昨天盟友、家长们反映的问题给予心理点评，帮助盟友们解开心中的结，不光是盟友们之间的问题，还有家长们的问题，因为中心主张的是孩子和家长共同改变，大家都是看事不看人，任何事情对就对，错就错，不会因为任何人说什么而改变，前几天有位2008年的盟友回到中心分享，他出院已经8年了，从一个叛逆的孩子到了现在，在这8年间他经历了很多，也并不是一帆风顺的，出院后的他首先考上了大学，毕业后顺利找到了工作，现在在一家银行上班，现在已经工作三年了，但他并不知足，现在一边工作一边读研究生。<br/><br/>每天12:30左右，大点评课就结束了，接下来是做手语操，有两首歌《感恩的心》和《怒放的生命》。第一首歌是让我们拥有一颗感恩的心，懂得一切来之不易，第二首歌特别有激情，能够让我们奋进。在这同时还锻炼了盟友们的身体协调能力和节奏感，很多非常态盟友刚来时手脚不协调，但经过几个月的锻炼后，他们变得非常灵活并且特别精神，就这短短的两首歌和一个简单的手语操却发挥出了这么大的作用。<br/><br/>之后就是餐前感恩词，在每顿饭之前都要念，有一个领读，短短十几句话却深深地教导着我们要珍惜食物，不要浪费，因为在很多地方还有吃不上饭的人，如果能够仔细品读每一句话，那么你会受益无穷。<br/><br/>之后是吃饭，吃完饭后一段时间就睡觉了，每天的午休时间一个小时左右，让每天的生活非常有规律，现在如果没了午休时间我就感觉下午没有精神，现在不论时间多紧我都坚持午休，这样下午的效率会倍增。<br/><br/>起床后整个楼层的盟友被分到几个小室写日记，日记能够记录一个人的成长，并且能通过课堂上的点评得到许多感悟，在我的印象中，在中心写日记是非常有用的，盟友很多，点评师们不可能面面俱到，这就要靠自己结合点评师的心理点评反思自己的错误，做到举一反三，写日记就可以来写一些感悟和自身类似的问题。到目前为止，我还在坚持写日记，记录我出院后的成长。<br/><br/>到了晚上活动就比较多了，中心共分为三个组，A组、B组、C组，不同的组有不同的活动，有网上作业，小点评，日记课。日记课就不用多介绍了，网上作业就是盟友们在电脑上晾晒出自己的过去，并且写出自己未来的打算，我觉得这个环节非常好，能够打开盟友们的心门，让自己舒服很多。当时我在打完那些作业之后感觉内心特别敞亮，整个人都换了一种状态。在小点评之前会有一个辩论赛，用来锻炼盟友们的口才，就拿我而言，我在没有去中心之前，特别的腼腆，不敢在太多人面前讲话，直到来到了中心才让我克服了这些。真的特别感谢中心的各个环节，尽管它们微小，但是它们成就了我。<br/><br/>除了这些，在周日我们还有各种各样的娱乐活动，早上八点大家会在一起下棋，象棋、军旗、跳棋，为的就是劳逸结合。既放松了身体又提升了思维能力，连玩都能玩出好处来，接下来就是令我特别喜欢的户外拓展活动，每个周末都会有点评师带领我们进行不同的活动，不光是活动本身有趣，得到的一些道理或启示也有趣，就举一个简单的例子来说，记得有一次医生带领我们做了一个叫“信任背摔”的活动，就是在一个两张桌子摞起来的很高的地方背朝大家倒下去，下面有很多人接着，可以说安全是没有任何问题的，每个盟友都有尝试的机会，当我上去的时候特别紧张，自己心里也没底，但是一咬牙，一闭眼就过来了，同样我也充当了在下面做保护的角色，为的就是让别人放心，简简单单的一个户外活动却让我懂得了许多，在以前走偏时我很孤僻，对身边的一切都充满了怀疑，全世界好像只相信我自己，但通过这个游戏，我发现值得我相信的人有很多，只要自己愿意去相信他们并付诸于行动，那他们回报给你的一定是坚强的后盾，让你觉得踏实。<br/><br/>下午是阅读课，盟友们可以选择自己喜欢的书去阅读，书架上的书都是家长捐赠的，全都是正能量的书，按照种类不同分成了几个模块，阅读两三个小时就去吃饭了。到了晚上是绝大多数盟友都喜欢的环节：看电影。每周都会放映不同的电影，记得我看过一部电影叫《阿甘正传》，主人公阿甘让人觉得傻傻的，但是他坚持不懈，有坚强的毅力，在当兵期间，他只会服从并且出色地完成任务，还救出很多伤员，到最后有了很多成就。这部电影特别激励我，尽管他天生条件很差，但是经过后天的努力一定会成功的。还有很多动画片电影，有好几部都把我感动哭了。这个环节既让我们放松了，又让我们获得了正能量，一举多得。<br/><br/><br/>现在我已经出院了，我完全可以选择像那个“未消逝的青春2015”一样编造出一些根本不存在的东西来诽谤中心，但我有一颗感恩的心，俗话说得好“吃水不忘挖井人”，什么时候都不能忘了自己的根，自己的本。离院的第二天我就进入了学校，作为插班生的我经过在中心的磨练，非常轻松地融入了班级的氛围，尽管学习成绩没有多么好，但是最起码我能正常地做一个人，我相信只要我坚持中心的理念，凡事都做到全力以赴，学习成绩自然不是问题。<br/><br/>前几天国庆放假，我没有选择外出旅游，而是选择回到中心分享，有几个目的。第一就是通过其他优秀盟友的分享找到自己的差距，从而找到提升的动力。第二就是看望杨叔和各位点评师，毕竟在院期间教了自己那么多为人处事的道理，让我走到现在。第三就是对在院盟友和家长有一个激励作用，相信他们在不久的将来能像我一样，收获一个我们这样的孩子。这一天不会太远的。<br/><br/>可以说，在这个世界上再也找不到第二个像临沂网戒中心这样充满正能量的地方，所以我特别希望大家能真正走进中心，看看中心到底是什么样的，而不要只听个别人的一面之辞。<br/><br/><br/>盟友小齐<br/>首次转载时间:
