为赚取眼球更是趁火打劫，恶意造谣，混肴视听，甚至企图置网戒中心于死地，做了游戏商们想做而做不到的事。受益的家长和盟友最清楚这里的真相，挺身而出誓死保卫临沂网戒中心这个造福千万家庭的地方。要知道，杨叔他没有必要去承担这个风险，也完全不用承担这份社会责任，他本可以安稳地当他的副院长，当他的精神科医生，可是杨叔心里装着这些无奈绝望的家长，他的同情心促使他不愿放弃我们。<br/><br/>杨叔经常教导我们改变为自己，付出为未来。我现在才深深体会到其中的味道，这其中包含了多么深的期盼和爱。杨叔把我们这些走偏的孩子改变得这么优秀，就是给社会减少了很多潜在的危害，给家庭避免了很多灾难，实属一件功德无量的好事，我们把杨叔称为再生父母一点也不为过。<br/><br/>真相只有一个。杨叔干的是正义的事情，事实就在那里，不会被谣媒改变、左右和颠倒。我永远相信杨叔，我永远感恩杨叔！<br/><br/><br/>盟友小杰<br/>首次转载时间:
