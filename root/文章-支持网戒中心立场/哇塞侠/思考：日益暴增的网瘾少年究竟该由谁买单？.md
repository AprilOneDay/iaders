src="http://ww4.sinaimg.cn/large/005Dooo9jw1f6uxn1arwdj30ln0mq45i.jpg" alt="" /><br/><br/>当你在新闻搜索页面输入“杨永信”三个字时，铺天盖地的新闻报导会像无数记者同时亮起的闪光灯一般刺痛你的双眼。“电击少年”、“枕下藏刀”、“戒瘾恶魔”、“逍遥法外”等一系列刺痛人心的词汇被用在了这个戴着眼镜的中年医生身上，不禁让广大网民心头一颤。如果这只是媒体们博人眼球的伎俩，那这次舆论轰炸势必会成为杨永信和临沂网戒中心的免费广告；如果这些报导的内容全部属实，我相信在不久的将来杨永信必将给以说法。而这些都不是我所关心的。<br/><br/><strong>关于杨永信本人</strong><br/><br/>无需我做过多的评论，因为在微博、微信、贴吧、论坛等网友集中的地方对于此人的争议随处可见。我细读了一些网友的言论，发现参与其争议的人大致分为两个阵营。<br/><br/>一方声音以游戏女主播@Miss、电竞达人@林熊猫、游戏开发程序员@雷斯林Raist为主。<br/><br/><img src="http://ww2.sinaimg.cn/large/005Dooo9jw1f6uxofgqstj30gp0ba0vx.jpg" alt="" /><br/><br/><img src="http://ww3.sinaimg.cn/large/005Dooo9jw1f6uxofhgkoj30gn09cjt6.jpg" alt="" /><br/><br/>?<img src="http://ww2.sinaimg.cn/large/005Dooo9jw1f6uxofdte2j30go0cldjs.jpg" alt="" /><br/><br/><br/>?另一方则以在临沂网戒中心给孩子戒网瘾的家长为主。<br/><img src="http://ww3.sinaimg.cn/large/005Dooo9jw1f6uxp75w4qj30n90720ts.jpg" alt="" /><br/>?<br/><img src="http://ww1.sinaimg.cn/large/005Dooo9jw1f6uxp6h8g7j30n102vgm5.jpg" alt="" /><br/><br/>?争论双方各执已见，好不热闹，但在笔者看来，这样无休止的争论终究不是办法，就像一场人数超多的辩论赛，公说公有理，婆说婆有理，但我们都知道辩论赛本身并不能解决任何实际的问题。但是这场辩论赛是值得我们思考的：<br/><br/><strong>一.为什么一个从2009年就被质疑“用电击治疗网瘾孩子”的戒网瘾机构能存活至今?<br/><br/>二.孩子到底做错了什么要被“戒网瘾”？<br/><br/>三.如此之多的网瘾少年从何而来？</strong><br/><br/>针对这几个问题，笔者站在一个非专业人士的角度想谈谈自己的看法。<br/><br/><strong>一.为什么一个从2009年就被质疑“用电击治疗网瘾孩子”的戒网瘾机构能存活至今?</strong><br/><br/>无论你看了《杨永信，一个恶魔还在逍遥法外》还是看了《戒瘾专家杨永信仍在电击网瘾少年》，我相信这个问题会是你的第一个疑问点。在笔者看来，答案很简单：市场需求够大，换言之现在社会存在足够多的网瘾少年需要戒网瘾。<br/><img src="http://ww2.sinaimg.cn/large/005Dooo9jw1f6uxpo3pv1j30nq07lgnt.jpg" alt="" /><br/><br/>?虽然在生活中我们不难发现有网瘾少年的存在，但是当我看到2400万这个数字的时候还是被吓了一跳，而且这还是6年前的数据，现在呢？我不知道（网上能搜到的而且相对权威的数据并不多）。相比如此庞大的网瘾少年群体，戒网瘾机构的存在就不足为奇了，可奇怪的是：我们所听闻过的戒网瘾机构并没有多少，听闻过的戒网瘾专家也寥寥无几，也许是我孤陋寡闻，我只知道陶宏开和杨永信。我想这场辩论最应该引起的当是政府的重视，多开几家像样的戒网瘾机构或是制定出一整套完善的相关规定，这才是解决问题的实质办法。<br/><br/><strong>二.孩子到底做错了什么要被“戒网瘾”？</strong><br/><br/>如果不深思，我可以直接回答你：因为孩子有网瘾。但是我看了很多网友的留言，发现这么回答是不负责任的，而且还很容易被骂。“只是因为我喜欢通宵玩网游就要被戒网瘾？”“我不过就是连续看了十个小时的网络视频就要被戒网瘾？”“我二十几岁了都没女朋友，上网看看美女主播都不行？”我不想回答这样的问题。第一，对和错本身是相对的，我无权审判孩子们做的事情；第二，网瘾这个词本身也颇具争议，没有人会愿意把这个标签贴在自己的脸上。那我们换一个词――沉迷网络，这样可能更容易令人接受。<br/><br/>既然谈论对错我们会陷入哲学争论的漩涡，那我们就谈一谈数据与法律。<br/><img src="http://ww3.sinaimg.cn/large/005Dooo9jw1f6uxr16kg6j30j00gin05.jpg" alt="" /><br/><br/>?此篇报导中提到：青少年犯罪与网瘾有关的占“七成”。到底占几成我想这不是讨论的重点，笔者也懒得去搜索更多的报导和数据，但我们不可否认的是：网瘾确确实实已经对青少年的成长产生了严重的影响。<br/><br/><strong>三.如此之多的网瘾少年从何而来？</strong><br/><br/>在今天刚刚爆出的关于杨永信的种种报导中，这一篇的作者算是相对用心的，为了写一篇报导，竟然读了杨永信的《网瘾的是与非》，为他点个赞。在此引用他的一段话。<br/><img src="http://ww3.sinaimg.cn/large/005Dooo9jw1f6uxrwck22j30lz06tdhj.jpg" alt="" /><br/><br/>?“其中让我比较惊讶的是，杨永信对他所“治疗”的对象有着相当清醒的认知。他没有将问题一味归罪于“网游的原罪”，而是提出了三因素理论：不良性格基础、不良家庭环境和不良社会环境。从这个角度来说，他说的完全没有错，大多数所谓的“网瘾少年”问题，其根本原因在于父母与孩子之间的沟通鸿沟，教育不当，所谓网瘾只是父母和子女之间产生冲突的表象之一，本质仍然是家庭问题。通过这个，也就不难理解为什么杨采用的父母子女一起接受治疗改造的方式成效卓著――他确实知道问题在哪里。”<br/><br/>这一段专业性较强的话帮助笔者理顺了思路，原来杨永信不只是靠“电击”戒网瘾，他还提出过三因素理论：不良性格基础、不良家庭环境、不良社会环境，我觉得挺有道理。<br/><br/>1.不良性格基础。这可能指的是先天的因素，在此不做讨论。<br/><br/>2.不良家庭环境。这应该指的是父母的教养方式了，提起这个，很多人都会有一肚子的抱怨，家庭暴力、冷暴力、娇宠溺爱、父爱缺失、母爱缺失、留守儿童等等等等，这应该是整个中国都来思考的问题。<br/><br/>3.不良社会环境。这一条我可能与一些网友有不同的看法，很多留言说：“孩子有网瘾都是父母没教育好！”诚然，父母作为孩子的监护人有着不可推卸的责任，但是现今社会的不良环境我认为比父母的责任更加重大！<br/><img src="http://ww1.sinaimg.cn/large/005Dooo9jw1f6uxsa4i2oj30d708g75c.jpg" alt="" /><br/><br/>?多么令人心痛的一张照片！可是这样的现象我们早已见怪不怪，我们甚至已经麻木到：好像哪个网吧里没有未成年人才奇怪。在二手烟的熏陶下，在充满了暴力、血腥画面的刺激下，这群祖国的花朵正在茁壮成长，网瘾少年从何而来？网吧成就了一批。<br/><img src="http://ww3.sinaimg.cn/large/005Dooo9jw1f6uxsmj8b4j30hh0ex40o.jpg" alt="" /><br/><br/>?“劝退”这个词我们也不陌生，太过稀松平常。学校为了保障教学秩序，对于那些难管的问题孩子往往采用“劝退”的方式作为惩罚，还能起到杀鸡儆猴的作用。但是在笔者看来，在某种角度上这是一种不负责任的体现。劝退了学生就能反思到自身错误了么？劝退了学生就能学好了么？图中的这位学生当然是一个个例，更多的学生不会选择这样极端的方式，他们会选择什么呢？网络就是他们一个很好的归宿。退学在家无所事事，找工作又不到法定年龄，不如玩玩游戏、聊聊天、会会网友来的逍遥自在。我会告诉你们我特别羡慕那些被劝退的学生么？因为这样就可以名正言顺地钻网吧了。网瘾少年从何而来？学校成就了一批。<br/><br/>笔者在临沂网戒中心的简介中发现这样一段话：<br/><br/>“临沂网戒中心在网络成瘾戒治、性格缺陷矫治、危险行为干预、问题家庭诊疗、特殊学生教育等方面进行了长期的系列性探索，对患者的遗传因素、家庭因素、学校因素、社会因素进行全面分析，在国内首创了集医院、军营、学校、家庭于一身的问题青少年诊疗机构，在医学的背景下，把家长纳入同步诊疗对象，从心理学的角度出发，按照“从生理、心理到病理，从医疗、教育到家庭，从个体、群体到社会”三维一体的理念，用“价值引领+爱心培养+孝道教育+道德讲堂+荣耻教育+法制教育+心理点评+感悟自省+责任承担+挫压励志+动力挖潜”等上百个环节，通过生物学治疗、人生价值取向纠偏与性格缺陷矫治，把问题孩子转变为常态孩子、健康孩子、优秀孩子。杨永信独创的问题青少年特色治疗模式成为学校教育、家庭教育、社会教育的有效补充和依托。”<br/><br/>其中最后一句“杨永信独创的问题青少年特色治疗模式成为学校教育、家庭教育、社会教育的有效补充和依托。”我想这就是临沂网戒中心最吸引人的地方吧。<br/><br/>说了这么多该回归主题了：网瘾少年究竟该由谁买单？<br/><br/>不管我们再怎么追究责任，无数网瘾少年的诞生已成为事实，这个问题需要解决，该由谁来解决？有几个选项供大家参考。<br/><br/>1.游戏开发商。按国家相关法律规定，现今市面上的游戏都必须有防沉迷系统。<br/><br/>什么是实名注册和防沉迷系统？<br/><br/>1、根据2010年8月1日实施的《网络游戏管理暂行办法》，网络游戏用户需使用有效身份证件进行实名注册。为保证流畅游戏体验，享受健康游戏生活，请广大腾讯游戏的玩家尽快实名注册。<br/><br/>2、保护未成年人身心健康，未满18岁的用户将受到防沉迷系统的限制：<br/>游戏过程，会提示您的累计在线时间。<br/>累计游戏时间超过3小时，游戏收益（经验，金钱）减半。<br/>累计游戏时间超过5小时，游戏收益为0。<br/><br/>这个系统最致命的漏洞在于：只减少游戏收益而不会强制下线。<br/><img src="http://ww4.sinaimg.cn/large/005Dooo9jw1f6uxvr6mjej30af06cq3n.jpg" alt="" /><br/><br/>?举一个简单的例子，我们都玩过《掉线城与虚弱勇士》，这个游戏看似很良心，因为它有一个疲劳值系统，疲劳刷完了就不能再刷图了，而且根据上面的防沉迷系统，超过五个小时你就没有收益了，那么对于小学生来讲该怎么办？――去PK场呗，这不需要我再另外写一篇攻略教你们吧？所以疲劳值系统也罢，防沉迷系统也罢，从根本上解决不了未成年人沉迷网游的问题。何况你从家里找个身份证有多难？你去网吧找老板借一个身份证有多难？如果我是游戏开发商，我巴不得小学生都来玩我的游戏，别问我为什么――人傻、钱多、速来！<br/><br/>2.学校。实际上在上文当中我已经讲的很清楚了，学校是拿网瘾孩子没办法的。我就喜欢逃课上网你奈我何？抓不到算我走运，抓到了顶多让我写个检讨，再不然叫家长，你还有什么办法？劝退我？来啊，这样我就可以名正言顺地回家上网了。<br/><br/>3.社会。毒瘾有戒毒所，酒瘾有戒酒科，打架斗殴有少管所，违法犯罪有拘留所和监狱，网瘾呢？对不起，是沉迷网络呢？国家并没有这样的相关机构。<br/><br/>饶了一大圈我们又回到了原点――网瘾少年到底该由谁买单？<br/><br/>                                 答案是：没人买单！<br/><br/>我用一个很搞笑的比喻来给出我的结论：<br/><br/>我们今天所抨击的戒网瘾机构，实际上正在为我们的游戏开发商、父母、学校乃至整个社会――<br/><br/>
