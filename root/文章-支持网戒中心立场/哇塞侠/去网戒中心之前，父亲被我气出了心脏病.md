/><br/>我是一位离院三个多月的盟友，看到网上个别没有改变好的盟友的所作所为，我感到十分的气愤和可笑，同时为他们的人生感到担忧，没有丝毫感恩之心，还没有过河就要拆桥的笨蛋，怎么能不让人感到担忧？作为一名中心的受益盟友，杨叔教出来的学生，我很有必要向被假象迷惑的网友们讲一讲我的过去、我在中心的真实生活、以及我眼中的网戒中心和我眼中的杨永信。<br/><!--more--><br/><br/>说起我的过去，初一开始出现问题，持续走偏6年才有幸来到中心接受治疗。初中的时候我还比较小，也就是跟一群损友去下下馆子、去去网吧、逛逛ktv、在学校欺负欺负同学、吼吼老师。虽然学习不怎么样，但还是以上学为主。我是一个很喜欢交朋友，特别讲哥们义气的很贪玩的学生，所以就有一些消极的学生、问题青年向我靠近，当然最主观的原因是我想向他们靠近。跟这群损友交往后，我的认知开始出现偏差，认为在学校里三五成群、横行霸道才是最受人瞩目的对象，帮人打架站场子、下课厕所里吸烟吹牛成了我的追求。于是乎！下课我就往厕所钻，像个苍蝇一样闻着臭味吸着香烟，然后跟别人卖弄自己有多强大的实力。慢慢地因为手里有一些零花钱跟自己一起玩的人越来越多，也有一群“小兄弟”<br/><br/>初中我就开始“混”，今天哪里有人打架自己得去站下场子，明天哪个损友要去欺负人自己得帮他找几个人一起去欺负。在学校欺负人，从来不怕别人告诉老师，因为我有“证人”，每次有人去告诉老师都会有一群人帮我做伪证，所以有人经常会被我欺负了还要给我道歉。在学校里自己也有了很多的外号，还有人喊我“痞子头”什么的，自己从来是不以为耻反以为荣，还觉得是荣耀。在学校里什么都数不到我，但要说坏，估计能名列前茅。<br/><br/> 初中的时候我叛逆的比较厉害，经常跟父母吵架。我的爸爸脾气也是非常火爆的。还记得我第一次离家出走，全家人都找疯了，晚上11点多才被找到。那一次爸爸追着我，在楼下揍了我一顿，因为全家七大姑八大姨都来了，所以很多人拦着我爸爸护着我，但那个时候的我没有道歉，没有认错，而是死不悔改，据理力争。最终我的爸爸被我气倒了，看到爸爸倒下我才清醒点。那次之后我的爸爸查出了心脏有问题，每天都要吃治疗心脏病的药。爸爸得了心脏病完全就是拜我所赐，那个时候我的爸爸才38岁。有了这个事之后我也开始略微收敛，中考超常发挥，平日100分的化学我只能考43分，但是中考50分的化学我竟然考了48分，也得以顺利考上高中，家人都说这是个奇迹，都异常的高兴，对我都是满怀希望。但他们不知道这是暴风雨的前夕，噩梦的开始。<br/><br/><br/> 高中前的暑假我近乎疯狂的玩，iPad2那个时候刚刚问世不久，爸爸就给我买了，此后，晚上有没有老老实实睡觉的时候，我已经记不清了。到了高中一个新的环境，我早就忘了上学是为了什么。开学军训班主任就没有见到我的人影，因为我经常跟他走偏的儿子一起，所以他对我更多的是容忍，所以我干坏事的时候就会拉上班主任的儿子。慢慢我又想在学校里混成“老大”，因为我以前就认识一群损友，在学校里好像没有我不敢做的事。打服了高一其他嚣张的新生，又去打打高二高三的学生，最疯狂的是考完试放假，我纠集了一群人到别的班级把班长揍了，然后迅速逃离了现场，不光在校内打架也跑去别的高校打架，甚至跟社会上帮派打架，是因为没吃过亏，所以自己一直非常的嚣张，想打谁就打谁，直到一次打群架，我纠集了三十多人，别人也纠集了三十多人，因为后来别人来了帮手，我吃了一次亏被别人揍了一顿，衣服撕破了，我的那些损友也被打伤了，眼镜手机都碎了，现在想想还心有余悸，多亏都是学生身上没带刀枪的，不然出现人命这辈子可能都要在铁窗里度过，通过这件事我消停了许多，许多社会上的哥姐听说我吃亏非要跟我去把别人剁了或敲诈一个，自己也一一拒绝了。那个时候自己已经投入到电脑游戏的世界里，在玩游戏时跟我一个包间的朋友正在吸毒，自己也像没看见一样，内心没有任何波动，一开始爸爸花钱给自己补课，但自己却在老师办公室里吸烟玩游戏，跟老师去打台球，下馆子，去网吧通宵，每天别人七点去上课自己是七点去网吧上课，自己在班上将近一年，但是上课老师却根本对我没有印象，三年的高中休学四次，总共在校四个月，在这四个月里不知道有多少时间是混日子，有时自己想上学又控制不了自己，每次从网吧出来我都发誓明天去学校，不然我的一生就完了，但第二天仍然是网吧，每天上网十几个小时，每次从网吧出来自己都感到十分的空虚，回家还是继续拿手机上网，玩《梦幻西游》不到一个月就有2万RMB投到上面，为了满足自己的虚荣心，不管父母死活，除了网络游戏自己不想别的事，感觉一切没有意义，甚至游戏玩烦了，自己都不知道活着为什么。就在这生死存亡的时刻，爸妈把我带到了网戒中心，开始了我一段堪称奇迹的改变。 <br/><br/><br/> 刚到网戒中心，自己觉得像医院又像疯人院，自己虽然对接待我的叔叔说的话非常认可，但自己心里很不情愿，还想回去完成游戏任务，自己想过自救但没有效果，自己的一些想法都是空想，我也做过中心的治疗，要说痛苦谈不上，比我爸爸用皮带抽我，比和别人打架挨拳头轻多了，的确有痛感有充血的感觉，但正是这种感觉让我冷静了下来，我第一次将心理医生的话听进去，想想以前做过的事羞愧难当，我对的住谁？我连自己都对不住，给父母家庭带来的只有伤害痛苦。我的姥姥有一次急的对我说，你别玩游戏去上学，姥姥每天给你发工资，我当时已经沉迷网络游戏不能自拔，谁的话也听不进去，这个治疗仪叫“低频脉冲治疗仪”，连杨叔自己都用这个治疗仪治疗腰病，网上炒作的“电击”就是这个，“治疗”是什么，先让人冷静下来，只有这样一些冲动的盟友才会跟心理医生沟通，因为是平等的交流沟通盟友更容易接受，说白了也只有这样盟友才能听的进去，父母揍我们为什么越打越不服，因为父母的恨铁不成钢是一种强压式的，而且也不冷静，为什么治疗那么简单，因为是平等式的交流，心理医生不会像父母一样强压我们接受什么，心理医生交流更多的字眼是你想想，你反思一下，不然想光靠强压式我们会更逆反，所有的动作都是由思想支配的，如果心门打不开一切都是徒劳的。                      <br/><br/>来到中心后，开始也跟个别没改变好的盟友一样，以为中心是传销窝点，但每天唱着《网梦醒来》《跪羊图》《父亲》《母亲》等歌曲，自己的心灵也逐渐得到了净化，就像《网梦醒来》中唱的那样，“当你把亲人的关爱，拒之门外，你是否想过为了你亲人在伤心”这完全就是自己最真实的写照，每天听着课堂上盟友们的晾晒，那些触目惊心的例子，让我每次都坚定了自己改变的决心，我们真的不能再那样活，我们必须改变自己，自己开始每天用心的写日记，每天反省自己身上的问题，在来中心之前自己拿笔都不会拿，即使写也到了提笔忘字的境地，更不用说写出来的东西简直不堪入目。在中心让我意识到学习的重要性，以前觉得学习都是为父母学的，上大学没用照样去打工，还不如混社会，来到中心之后自己第一次参加了辩论赛，才让我意识到知识的重要性，我也深刻体会到了什么叫“书到用时方恨少”，一个人的学识的确非常重要，这可以标志着你属于什么样的层次，也可以说标志着你生活在什么样的层面上，人的思想有多高，这个世界就有多大，“活到老学到老”，人一生都在学习，学是没有止境的，我知道自己不光要学习文化课，在生活中也要不断学习知识，让自己变高变大。       <br/><br/>在中心让我感受到了什么叫爱，什么叫感动。这些感动来自点评师们，他们无私的付出让我感动；那些感动来自各位家长，这些家长对孩子的不离不弃让我感动；还有盟友们想要改变命运发自内心的改变，让我感动，就是这么一群特殊的群体组成了这个可爱的大家庭，就是因为我们形成的这种爱的合力在创造一个又一个奇迹。在中心有个动力挖潜组，说白了就是背英语单词，在一个多月内背过三千多个英语单词，一千多个短语，也就是整个高中三年的词汇量，近五千个左右，不知别人怎么想，反正我是不敢想，每天我们背三百个单词，每天都有新单词，可能有人会说很简单，但每天只有两个多小时的学习时间呢！在中心我们主要不是学习，而是改变，甚至给我们的学习时间会不足一小时，这样艰难任务对我来说是难于登天，根本不可能，但事实怎样呢？一个小时四十分钟默写一千九百四十六个单词，且达到百分之九十九的正确率，还有更厉害的就是百分之百，中心的这个环节让我知道了什么叫世上无难事，只要肯登攀，没有什么做不到的，信念会让我们不断创造奇迹。<br/><br/><br/>杨永信，我眼中的杨叔不是一个普通人，而是一个圣人，他是副院长，享受国务院津贴，他原本可以过的非常轻松，但他每天工作到晚上十一点甚至到凌晨，每天的点评课一站最少四五个小时，几乎每天都少不了杨叔的点评课，一年365天，在中心很少有看不到杨叔的时候，一年四季杨叔都是在中心度过，中秋节、大年三十杨叔没有回家吃顿团圆饭，而是陪盟友和家长们在一起，杨叔用他无私的爱感染着我们。面对2009年的炒作以及这次炒作，他没有任何抱怨，更没有消极面对，他还是继续做着他该做的，笑着面对生活，他用他的行动告诉我们什么叫身正不怕影子斜，网戒中心关门了，我就解脱了，但你们怎么办？若硬要说我做的事是违法的，该蹲大牢就蹲大牢是了，这是杨叔曾对家长盟友们说过的话。看到了今天的杨叔，我想到了清朝的林则徐，林则徐是首先意识到鸦片对人类社会的危害，他是民族英雄，杨叔就是当今社会的林则徐，他对抗的不是鸦片而是电子鸦片，是他让一个个网络瘾君子重新走出黑暗面向阳光，是他让一个个濒临崩溃的家庭重拾希望，是他让一个个有着严重认知偏差的问题青年有了正确的三观，是他告诉我们要孝敬父母，是他告诉我们要回报社会，是他告诉我们人活着要有价值。在网瘾面前，他是一个勇士，一个巨人，在生活中他是我们的亲人、恩人，大到认知三观，小到行为习惯，不管盟友还是家长没有他不管的，杨叔用他的大爱拯救这些网瘾孩子和一个个濒临破碎的家庭。看到网上对杨叔、对中心的不理解甚至诋毁，我真觉得十分好笑和气愤，谁对谁错明事理的人一看便知，特别是看到有人打出保护青少年的旗号，特别像一些游戏商的虚伪言辞，让我想起了烟盒上的“吸烟有害健康”，试问这些口口声声为青少年着想的人，您真的是在保护未成年人权益吗？事实上只是披着虚假的外衣，为了自己的一己私利，不惜出卖道德良知，污蔑诋毁杨叔。无论社会认不认可杨叔，但杨叔他每天在做，谎言会不攻自破，披着羊皮的狼终归是狼，任何诋毁在正义面前都是苍白无力的，正义必胜。以上就是我作为一个离院盟友最想说的话。<br/><br/>?盟友小友<br/>首次转载时间:
