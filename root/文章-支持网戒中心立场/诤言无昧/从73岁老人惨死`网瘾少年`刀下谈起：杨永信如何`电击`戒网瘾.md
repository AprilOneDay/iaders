/><br/>――“杨永信风波”问题解析之二<br/><!--more--><br/><br/><strong>戒治网瘾――是认知纠偏、心灵重塑、性格缺陷矫治和教养环境改善的系统工程。</strong><br/><br/>继11月16日“北京时间”报道承德网瘾少年60多刀杀人灭门案之后，笔者又无意中读到了澎湃新闻10月31日的报道：73岁独居老人惨死2名“网瘾少年”刀下――<br/><br/>2015年5月，在山东鲁西南地区的一个小镇上，73岁的何东春把小卖部打扫干净，开始了一天的生意。<br/><br/>老人12岁那年跟着父母到这里定居，40岁那年丈夫发病离世，她独自一人拉扯着5个儿女长大。30多年过去了，孩子们各自成家，她将丈夫留下的房子做了修整，临街的部分改成了一个小卖部，后半部分用作起居室。<br/><br/>可谁也不会想到，这个经历了大半生困难坎坷都没低头的老人，竟惨死在两个未成年人刀下。而促使孩子举起砍刀的，是那颗被网瘾蒙蔽了的心和老人不曾了解过的――<strong>网游</strong>……<br/><br/>这让笔者震惊之余，再一次想起了CCTV2《第一时间》和CCTV13《新闻周刊》中有关类似如此的论调：在互联网普及的今天，居然还有人在谈网瘾，居然还有人在戒网瘾？！――请大家注意，这可是央视啊！<strong>互联网普及以来的20余年，有关网瘾的惨案有听闻更有亲历，不可谓不多不重，作为国家喉舌对如此铁一般的事实，更不可能置若罔闻，如此论调打脸不说，究竟代表了谁的立场？体现了谁的意志？作为国家首媒和体制内的主播名嘴，能说这是代表国家的立场吗？</strong>（笔者近日曾撰文《网瘾是病，是社会问题，法规不明，谁来担责？》诤言无昧的微博_微博http://weibo.com/ttarticle/p/show?id=2309404043265363082909#_0）<br/><br/>此话暂且不表。<br/><br/>耐人寻味的是，在<a href="http://www.thepaper.cn/newsDetail_forward_1551594">澎湃新闻</a>10月31日的报道中，作者引用了负责此案的聊城市人民检察院检察官张慧的案件评述：<strong>网络危害对青少年的成长，几乎是致命的。如果给他们这样的土壤，那么他们是甘心情愿将自己的青春拱手交出的，交给吞噬他们的激情、榨取他们的生命的网络游戏或者低俗网站。这是家长的责任、学校的责任，社会的责任，但家长的责任更重。</strong><br/><br/>更加耐人寻味的是，作者还对网瘾和网瘾戒治发表了如下看法和疑问：“杨永信网戒中心使用的低频脉冲治疗仪<strong>在为孩子戒除网瘾方面，很多家长可谓是不遗余力，甚至使用过一直以来极具争议的杨永信“电击疗法”（后改成低频脉冲治疗仪），但是，这样真的可以解决根本问题吗？”</strong><br/><br/>那么，网瘾的病根在哪里？<strong>杨永信戒网瘾真的只靠“电击”吗？</strong><br/><br/>不可否认的是，张慧在对案件的评述中，关于网瘾的危害可谓入木三分，而在对网瘾病理成因的分析中，归结为“<strong>家长的责任、学校的责任，社会的责任，但家长的责任更重</strong>”虽说中肯，但却不是全部――网络游戏开发和网络监管机制责任岂不更大更直接？再说了，没有天生合格的父母，每个孩子皆有差异，心理与生理特点各不相同。而对于已经罹患网瘾的孩子怎么办？这才是最现实的问题。<br/><br/>杨永信在对网瘾的病因研究中，除去社会、学校和家庭因素之外（因为在同样的成长环境下，罹患网瘾的毕竟是少数），发现不同的孩子对同样的网络环境刺激产生的反应和免疫力是不一样的：大部分孩子能自觉抵制诱惑，上网时间具有可控性，对日常生活学习无明显影响；而有些孩子则具有无限沉迷的不可控性，对这部分孩子而言，虚拟世界就是生活的全部。而随着对网络的深度沉迷，他们大多都伴有性格畸变、心理扭曲和社会功能退化等精神病状。<strong>他们有一个共同特点，就是心理脆弱，敏感多疑，情绪极易失控――简单说来，就是性格缺陷。</strong><br/><br/>杨永信为此找到了网络成瘾的致病根源。<strong>他认为，从根本上戒治网瘾，须首先矫治性格缺陷――纠正认知、重树三观、磨砺意志，恢复社会功能、改善教养环境。</strong>为此，杨永信把家长纳入同步治疗范围，创建了集医院、家庭、学校、军营特色于一体的网瘾戒治和性格缺陷矫治综合干预模式。这个模式以社会主流价值观引领的心理点评为核心，包含行为矫正、意志磨炼、适应训练、感悟反思、演讲辩论、才艺展示、动力挖潜等108个环节，注重精神引领、行为习惯矫正和社会功能康复，并积极营造健康向上的氛围对患者进行濡染感化――<strong>这是目前已知的对网瘾诊疗最系统也最为有效的方式。</strong><br/><br/>由于该模式在认知信念纠偏、性格缺陷矫治和行为习惯矫正方面的独特功效，来到这里就诊的孩子不仅仅是网瘾患者，还有游走于犯罪边缘的问题青少年和重性精神疾病患者，<strong>而杨永信改变的不仅仅是问题孩子，也包含问题家长。</strong>值得一提的是：相对于传统精神科仅靠药物治疗的生物治疗模式，这里的氛围熏陶和集体互动更有利于精神疾病患者的社会功能康复，很多本来在精神科治不好、或者根本无法令患者混沌开窍的病，在这里经过各种环节的“浸泡”“濡染”，都会得到很好的康复――这就是家长为什么如此推崇杨永信的原因。<br/><br/><strong>因为患者群体的复杂性（各类精神病状、极度逆反、情绪狂躁冲动与暴力倾向等），为稳定患者情绪并实现有效沟通和配合治疗，该中心在医疗许可范围内和业务监管机构的有效监管下，采用低频脉冲作为辅助治疗手段。<br/><br/>――杨永信戒网瘾，岂一“电击”以蔽之？</strong><br/><br/>十余年来，该中心成功救治数千例含网瘾在内的各类问题青少年，并成功救治了相关的问题患难家庭，未出现一例医疗安全事故，验证了杨永信网瘾戒治和性格缺陷矫治综合干预模式的安全有效，受到家长和成功改变的患者们的坚决拥护，并得到党和人民政府的高度肯定和众多奖励。<br/>首次转载时间:
