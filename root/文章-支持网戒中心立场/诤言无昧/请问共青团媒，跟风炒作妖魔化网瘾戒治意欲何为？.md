class="preface">――以保护青少年权益之名，行为害青少年权益之实？</div><br/><!--more--><br/><div class="WB_editor_iframe"><br/><br/>??网瘾到底有没有？！<br/><br/>“网瘾” 一词，流行于社会已经很久了。大概是2003年吧，因为原班主任工作变动，我半路接手一个新班。上课第一天，就发现几个空位，后来发现，这几个空位是固定的，而这几个位上的同学即使来上课，也总会迟到，即使迟到了，也老是昏睡不醒。于是，就有同学告诉我：老师，他们得了网瘾。<br/><br/>我才知道，有一种病叫网瘾；随后我又知道，其实大家早就知道有一种新病叫网瘾，很难治。<br/><br/>最早，网瘾是网吧的产物，网吧是社会的藏污纳垢之所，也是大众公认。随着互联网的普及，尤其是游戏被青少年所普遍热衷和手机的智能化，网瘾也就越来越普遍了。十几年来，网瘾一直是各路媒体的霸屏热点――辍学、自闭、猝死、暴力、赌毒黄黑，严重危害青少年健康成长，青少年违法犯罪的案例大多与网瘾有关。<br/><br/>当下，网络普及了，虽说人们对网瘾一词也渐渐脱敏，可这并不意味着网瘾的减退或者消失。<br/><br/>可以说，进入21世纪，网瘾，是社会公认的事实，更是流行持久的热词。<br/><br/>但是，对这个词，人们的感情是复杂的：有人痛苦，有人忧虑，也有人颇为忌惮。<br/><br/>痛苦的，是沉迷网络荒废青春学业的网瘾孩子家长；忧虑的，是关注青少年成长的有识之士；忌惮的，是某些网络游戏开发商――网游已经成为快速崛起的新兴产业，他们的发迹依赖于未成年网民，在某种程度上也可以说以牺牲青少年健康成长为代价――他们害怕社会关注网瘾，更害怕政府干预网瘾，唯恐由此触及利益，他们希冀社会漠视网瘾，让更多的人喜欢并加入网游，以扩张不断膨胀的资本利益。<br/><br/>有钱使得鬼推磨――于是，一方面，各路明星大腕纷纷为游戏代言，游戏广告无缝隙霸屏，正所谓游戏健心，游戏益智，游戏兴国。另一方面， 唯利是图的恶媒谣记也纷纷发表评论：网络普及，网瘾就消失了，戒网瘾是逆历史潮流而动，是没病治病，他们质疑，这世界竟然还有人戒网瘾？！<br/><br/>我们从来不否定互联网改写人类历史、推动社会进步的伟大意义，但我们也知道网瘾的客观存在对家庭、学校、社会带来的不利影响：全国有多少学校禁止手机进校园进课堂？有多少父母因孩子深度沉迷网络自毁学业青春而深感无助？有多少媒体屡屡曝光孩子因昼夜上网而猝死网吧？有多少家长把深陷网瘾而导致精神异常的孩子送进精神病院？而为了购买游戏装备，又有多少孩子榨光父母血汗甚至不惜坑蒙拐骗？！我们承认健康适度的网络游戏有益身心，但也深知低俗不设限的网游对青少年身心的戕害；我们了解电竞业兴起的事实，但也知道网瘾与电竞决不是一个概念。<br/><br/>我们质疑的是：难道青少年痴迷网游荒废学业就能担负历史重任，成就民族复兴大国崛起的梦想？！<br/><br/>然而，这样的鬼话，这样的混蛋逻辑，有人真的就信了，不但信得投入，而且推波助澜甚嚣尘上！包括中国青年报、共青团中央微博、中青在线这样的主流媒体！他们完全忘记了共青团媒的宗旨与责任，把自己混同于混迹市井靠炒作吃饭的恶媒谣记，跟风炒作，歪曲事实，诋毁攻击，矛头直指网瘾戒治及其从业者，包括患难家长。<br/><br/>杨永信戒网瘾不是社会的需要吗？面对家长的哭诉无助，面对花季少年的沉沦，面对一个个家庭希望的破灭，在社会尚没有成熟的经验以前，难道不应该有人利用自己的专业特长为社会担责，并为此做出艰辛的努力与探索吗？<br/><br/>我不会说杨永信和他的网瘾戒治模式完美无缺，因为那是不科学也是不现实的，但杨永信为此付出的努力和取得的成果却是有目共睹的：他以性格缺陷矫治为突破口，通过纠偏认知、重树三观、信念培养、行为矫正、社会功能康复等环节进行人格重塑的系统工程，历经十余年，让数千网瘾问题孩子重返健康人生路，让众多网瘾患难家庭重拾幸福，尤其是杨氏模式对性格缺陷和社会功能康复的独特疗效，让很多重症精神疾病患者回归了正常人生！<br/><br/>显然，恶媒谣记的着眼点并不在此。他们蓄意隐去网瘾患者的病理背景，恶意从杨氏模式108环节构成的完整治疗体系中抠出个别环节进行孤立解读，高举人权道义的幌子，煽动网络暴力，挑起舆论群殴！<br/><br/>他们罗列的每一条罪名，似乎都能把杨永信和临沂网戒中心置于死地：<br/><br/>1.说杨永信是逆历史潮流反互联网的守旧顽固派。<br/><br/>事实是，杨永信自戒网瘾开始，就旗帜鲜明地强调网络是工具而不是纯粹的玩具；杨永信所反对的，是游戏无度、病态痴迷网络而导致心理生理发生病变，社会功能显著退化，从而严重影响家庭、工作、学习与生活的行为。由于共青团媒的炒作，这个弥天大谎一度具有很强的欺骗性，为炒作风波兴风作浪，推波助澜。<br/><br/>2.说临沂网戒中心是现代版的奥斯维辛集中营，令人恐怖的“人间地狱”，而杨永信本人则是残虐青少年的恶魔。<br/><br/>杨永信，一名中共党员，一个地级市公立三级甲等精神专科医院的副院长，因为成功救治数千例网瘾问题孩子而享受国务院政府特殊津贴，曾经被临沂市委宣传部和山东省委宣传部作为关心青少年成长、促进未成年人思想道德建设的典型在全市全省进行集中宣传，他的每一项荣誉，都是政府在对其业绩考核评估的基础上授予的，是由业内同仁的意志决定的。<br/><br/>3.说杨永信以戒网瘾为非法敛财的渠道与手段。<br/><br/>作为三级甲等公立专科医院，网戒中心运营11年，一直是严格按照物价主管部门的规定收费，规范严格的财务制度和杨永信公职人员的身份，这种谣言经不起推敲。<br/><br/>4.说杨永信左右政府为之站台保护，雇佣具有黑社会性质的“家委会”全国范围内抓捕网瘾孩子。<br/><br/>一是杨永信没这个政治资本、资源，二是作为一名国立医院的业务副院长和优秀专家，他没这个必要，没这个动机，更没这个能力――包括财力和权力。尤其是在今天深入实践群众路线，加强党风廉政建设的新常态下，政治清明，吏心向廉，再加上风波影响，没有谁去甘冒政治风险。二是临沂网戒中心的家委会是家长们共同推举的爱心服务组织，是患难家庭的服务者，是和谐医患关系的引领者。杨永信戒治网瘾的核心，就是以社会主义核心价值观和传统孝道亲情改变问题孩子的错误认知，并改变家长错误的教养方式，其自身率先垂范，严以自律，绝不会纵容允许家长去做违法乱纪或超越道德规范的事情。三是来此就诊的孩子都是家长在被迫无奈之下经过亲自考察后自愿送过来的。<br/><br/>5.说杨永信治疗问题孩子与家长的心理点评课是邪教式“洗脑”，把专业医疗手段“低频脉冲”描绘成惨无人道的“电击”，并说成是杨永信治疗问题孩子的唯一而且是强迫的手段。<br/><br/>事实是，点评课是杨永信治疗问题孩子的核心环节，以社会主义核心价值观为核心，与传统文化中的孝道亲情仁义礼智信相结合，纠正患者的错误认知，树立正确的三观，培养健康的情操与习惯，是与国家意识形态相一致的理想信念教育，并不存在洗脑一说。?而真正的网瘾患者，因其往往伴有心理精神病状而与普通的上网爱好者有本质的区别，低频脉冲只是在监管部门许可下使用的一种配合治疗的专业手段，是杨氏模式众多环节中的一环。<br/><br/>恶媒谣记为妖魔化网瘾戒治，最常用的手段就是刻意把问题孩子描绘成正常孩子，而且隐瞒问题孩子被家长送往网戒中心的前因后果，蓄意把刑罚中的“电击”和医疗背景下的治疗手段“低频脉冲”相混淆。<br/><br/>对此，临沂市卫计委早已做出回应，临沂网戒中心执业规范欢迎监督，低频脉冲治疗与网络上热议的“网瘾电击治疗”有根本的区别。<br/><br/>6.说家长是不懂家庭教育的愚氓和盲从杨永信的信众。<br/><br/>他们几乎无一例外地把孩子出问题的责任完全归咎于家长，并把家长丑化为不懂家庭教育，被杨永信洗脑而只会盲从的信众。<br/><br/>再好的老师也有教不好的学生，再好的家庭也有失败的孩子。任何家长在教育孩子上都不是完美的，每个家长为培养孩子都会不遗余力，但孩子的成功与否，因素是多元的，问题孩子的出现有时候并不以人的意志为转移。家庭教育固然重要，但棍子岂能全部打在家长身上？<br/><br/>事实是，临沂网戒中心的家长从职业教育工作者到长江学者，从公务员到商界精英并不缺乏，工农商学兵一应俱全。孩子在医院改变错误认知，重树理想信念，皈正人生轨道，难道家长们配合治疗有错吗？<br/><br/>7.说杨永信“扼杀”孩子的个性。<br/><br/>在他们的笔下，似乎杨永信挽救问题孩子，把他们归正到健康的人生轨道，就是在扼杀孩子的个性。<br/><br/>在人的成长经历中，我们尊重并鼓励健康的个性成长固然没错，但抗挫抗压的耐受力培养更不能缺乏。试问哪个家长没有激励孩子成长，哪个家长没有对孩子的“个性”包容甚至纵容？事实证明，很多孩子正是因为家长一味的奖励和纵容才造就了孩子以自我为中心、容不得半点批评、受不得一点委屈的“恶少”狂，“公主”病。<br/><br/>“两弊权衡取其轻”。网瘾患者和问题孩子中断学业、离家出走、鬼混网吧、离经叛道、寻衅滋事，甚至涉黄涉赌涉毒涉黑等，无视规则、肆意妄为，已经严重偏离正常的人生轨道，在自我毁灭的道路上越走越远，你是用强有力的思想引领和制度约束让他们回归健康发展之路，还是让他们张扬所谓的个性，在死亡的迷途上走失？<br/><br/>可悲的是，不仅共青团媒跟风炒作，而且身为共青团中央维护青少年权益部部长的王锋，竟然在央视2017年3月10日《今日说法》的专访中，在对网瘾病理性状及卫生部门介入治疗的专业背景未做深入了解的情况下，就极不负责的对《未成年人网络保护条例(送审稿)》中“任何组织和个人不得通过虐待、胁迫等非法手段从事预防和干预未成年人沉迷网络的活动，损害未成年人身心健康，侵犯未成年人合法权益”等内容进行了极为偏狭的解读，不但否认网瘾的存在和戒治机构的合理性，而且幼稚的把专业医疗手段混同于刑罚中的电击，妖魔化网瘾戒治，给社会造成临沂网戒中心用电击虐待孩子强迫改变的错觉，更由于其解读的官方背景，对网瘾戒治事业造成了极坏的社会影响！<br/><br/>――网瘾本来是现实存在的社会问题，其对青少年健康成长的危害早已是不争的事实，而炒作媒体正是抓住并迎合了青少年网民对游戏的狂热痴迷，才成功引爆了热点。试想，正儿八经在学校潜心读书，在岗位上勤奋工作的青少年，谁有功夫整天趴在网上发帖骂人？只有热衷于游戏并沉迷于网络的人才有充分的时间与精力打口水仗，才对“网瘾说”百般戒备与排斥，才会如此一致的把矛头对准网瘾戒治！<br/><br/>媒体的操守是关注问题，实事求是，引领公平正义，而坚持操守就是媒体的生命线。可惜我们的共青团媒不但未作冷静分析判断，进而理性引导，反而盲目迎合，跟风炒作，蹭热点抓眼球，纷纷加入讨伐网瘾戒治的逆流，这是不是一种悲哀与讽刺？！<br/><br/>这是不是打着保护青少年权益的幌子，干了为害青少年权益之实？！<br/><br/></div><br/>首次转载时间:
