我国网民和网瘾者的人数和比例不断攀升的严峻局势</strong><br/><br/>中国互联网络信息中心发布的《2016年第38次中国互联网络发展状况统计报告》显示，截至2016年6月，我国网民仍以10-39岁群体为主，占整体的74.7%：其中20-29岁年龄段的网民占比最高，达30.4%，10-19岁、30-39岁群体占比分别为20.1%、24.2%。与2015年底相比，10岁以下儿童群体与40岁以上中高龄群体占比均有所增长。据不完全统计，到了2013年年底，中国现有网瘾青少年4000 万左右，四年间中国的“网瘾”少年增加了1600 万，而且这个数字是相当保守的估计。网民中以学生和个体户或自由职业者为主，分别占网民总数的24.6%（或23.8%）和22.3%（或22.3%）。这一数字令人十分堪忧！网瘾防治刻不容缓。<br/><br/><strong>三  政府应尽的义务</strong><br/><br/>网瘾侵害了青少年的肉体和心灵，使他们丧失了人的善良本性，不健康的游戏或网络文化是网瘾形成的主要因素。政府不应屈服于舆论的压力，应该秉持正义，坚持原则，维护政府的公信力。目前由于种种原因，导致临沂网戒中心被迫取消了历经10余年探索出的行之有效的网瘾戒治模式的某些关键环节。<br/><br/>某些“专家”在没有实际调查研究缺乏充分的第一手资料的情况下，不但没有出于社会责任给予必要的道义声援和智慧支持，而是盲目地跟风作秀妄加评判，施以不负责任的棒杀，对网瘾患难家庭无异于雪上加霜、创口撒盐。我们希望政府听到我们的声音，正视网络不良文化和低俗游戏的严重危害，应组织法律专家、社会学家、精神心理专家、教育专家等联手制定行之有效的网瘾预防和干预治疗措施。<br/><br/>同时，我们更要高度警惕西方价值观借网络文化对国家的渗透与侵略，严禁不良的网络游戏利益者甘当外来颠覆势力的走狗，麻醉人民思想，毒害人民灵魂，牺牲民族希望和国家利益, 践踏少年青春。政府有责任、有义务改善网络环境，改变公众的行为习惯和思维方式，提高公众的认知和道德水准，保证广大青少年在网络科技日益普及的今天健康茁壮地成长。<br/><br/>与网络不良文化制造者和暴力色情游戏商形成鲜明对比的是，一些有高度的职业道德、强烈的社会责任感和民族使命感的网瘾戒治和维护健康网络文化的探索者们一直以来不畏艰难困苦，不畏社会舆论围剿，不畏造谣者和不明真情的愤青们的诋毁、中伤、诽谤等的威胁，取得了一些阶段性的、令人瞩目的、利国利民的成果。希望政府不失时机地出台更好的配套扶持政策，引领和规范网瘾戒治队伍和戒治机构的发展壮大，让更多的网瘾患者及家庭受益。<br/><br/>对个别没改变好的的心理扭曲和行为极端的问题青少年的爆料，我个人认为这正是其网瘾所引起的人格变异的一种表现，值得社会、政府、学校及家长反思和检讨。对于不法游戏商和不良媒体的利用，我觉得其唯利是图的险恶居心令人痛恨。个别人故意发布虚假信息，恶意炒作，吸引大众眼球，甚至有些媒体，在未深入采访调查的情况下，对杨永信网瘾戒治百余环节刻意无视，唯独把“低频脉冲治疗”进行无限度的放大和扭曲，更令人感到震惊。希望政府对此保持高度的警惕，绝不能让无私忘我工作和忧国忧民的奉献者被不良媒体和跟风者无情践踏。<br/><br/><strong>四 杨永信所创建的本土化的网瘾诊疗模式应该得到保护</strong><br/><br/>临沂网戒中心在杨永信的带领下摸索出一整套行之有效的网瘾戒治办法，已被6000多名成功改变的网瘾者证实行之有效，学校和社会也急迫地需要这些。政府也应履行自身监管职能，旗帜鲜明地予以支持，并出台相应的配套扶持的政策和法规，保障其健康、有序的发展。新生事物刚刚出现必然是一个极不稳定的矛盾体，会受到各方面的质疑和非议，只要其代表民众利益、代表公平、正义、和谐、健康，经得起实践的检验，其真理性不容置疑，也只有历经这样的考验才能让其承受住历史的抉择、考验和重任。但某些“专家”现在就要求网瘾戒治机构提供网瘾戒治的确凿的循证医学依据，我作为一名资深的医学研究人员认为这既脱离实际，又为时过早，有故意刁难之嫌。他们究竟是代表民众，代表政府，还是代表黑心游戏商的利益在发声？这个只有他们内心最清楚。不过，我们深信网瘾戒治精神心理医学正如祖国传统医学一样，尽管中医许多治疗理论和手段缺乏循证医学依据，但并不影响这一民族瑰宝对人民健康发挥不可磨灭的贡献。我们是否也要要求所有的祖国医学治疗理论和手段具备充足的循证医学依据后才能用于临床诊疗实践？不管白猫黑猫，只要能抓住老鼠就是好猫。在医学实践中，我要说不管什么诊疗理论和模式，只要能为人类解除病痛，提高人们生活健康指数就是成功的诊疗理论和模式。从另一方面说，没有杨永信网戒模式在实践中的探索，何来现成的循证医学依据？杨永信的网瘾戒治模式突破了单纯“以患者为中心”的网瘾戒治理论，在国内首次实现把患者家属纳入同步心理干预的群体，并进行系统化的研究和探索，发挥家长心理健康对患者心理健康的正向引导作用，确立全方位、多渠道的患者、家属心理健康干预机制和干预模式，解决网瘾形成的根源问题，实现了标本兼治。其网瘾戒治成功率高达90%以上，复发率低，这是网瘾戒治实践方面的重大突破，也是对网瘾戒治理论与实践的创新，具有很强的应用推广价值，这将对政府制定诊断和治疗规范提供可供借鉴的经验。<br/><br/><strong>五 《未成年人网络保护条例》的作用与意义</strong><br/><br/>国家网信办正在制订并广泛征求意见的《未成年人网络保护条例》施行后，将对营造健康、文明、有序的网络环境，保障未成年人网络空间安全，保护未成年人合法网络权益，促进未成年人健康成长起到举足轻重的作用。《条例》充分体现了国家重视网络环境的管理和净化及其对青少年成长的保护作用。从立法高度上明确了网瘾的存在和网瘾戒治的必要性，给网瘾干预和矫治指明了明确的方向。建议《条例》要加大对不法网游商和不文明网络文化制造者的惩戒力度，增加他们的违法成本和提高条例的威慑力，让低级、下流、庸俗的网络文化和游戏没有存在空间，还青少年一片健康、洁净、有序的网络空间。<br/><br/>《条例》第二章第六条规定国家鼓励通过网络空间弘扬社会主义核心价值观、民族优秀文化和传统美德，倡导创作、提供和传播有利于未成年人健康成长的网络文化产品，建设未成年人精神家园。这与杨永信10余年来对网瘾者进行传统文化教育、爱国主义教育，革命传统教育、社会主义核心价值观教育、孝悌教育、感恩体验教育、文化艺术教育、德育教育、法律法规教育、拓展训练等内容和环节不谋而合。<br/><br/>《条例》第四章《预防和干预》非常重要，政府要组织相关部门和专家，制定网瘾诊断、干预治疗标准和指南及疗效评判标准，规范和引导网瘾救治行为，制止网瘾戒治乱象，保护未成年人合法权益。杨永信在网瘾发病机制、诊断标准和干预治疗措施的研究方面已探索了10余年，已形成独具特色的杨氏网瘾戒治模式，涵盖了“药物 心理 物理 工娱”四大方面，包含“心理点评、改变家长、体验感悟、军训拓展、营造氛围、长效机制”六个主要单元，共108个主要环节，成功治愈网瘾患者6000多例，无一例伤害性事件发生。政府应该予以充分的珍视和科学的借鉴。<br/><br/>总之，《条例》的制定，给网瘾戒治指明了新的方向，给患难家庭带来了更大的希望和力量，而杨永信带领的临沂网戒中心也必将迎着黎明的曙光，抓住机遇，不断探索出一条适合我国本土和符合我国社会主义核心价值观的网瘾戒治模式，为我国网瘾戒治事业做出应尽的新的义务和贡献。<br/>首次转载时间:
