/><br/>依法治理网络空间，维护公民合法权益――习近平<br/><!--more--><br/><br/>“帝都风光，千里尘封，万里灰飘。望古城内外，浓雾莽莽。高楼上下，阴霾滔滔。车舞长蛇，烟囱万象，欲与毒气试比高。双休日，看口鼻严裹，宅卧求饶。空气如此糟糕，引无数市民惧相邀。惜雾都神武，早被我踩；灰城美誉，已不风骚。一代天骄，央视裤衩，只见裤腿不见腰。俱往矣，数悲催人物，还看天朝。”<br/><br/>近年来，随着我国经济的快速发展，大大提高了广大群众的物质生活水平。但事物往往有其两面性，经济发展过程中的环境污染问题逐渐凸显，这首《沁园春 雾》虽有些诙谐、搞笑，却形象地描述了雾霾的严重程度和恶劣影响。党和政府、广大人民群众同仇敌忾，目标一致，正在掀起轰轰烈烈的节能减排、治理污染、优化产业结构的“民心”治理行动。<br/><br/>环境的污染是有形的、易感的，而经济快速发展伴生的文化思想领域的污染虽然也已逐渐显现，但却没有引起人们的足够重视，就像严重的雾霾遮蔽了善良的眼睛，其中网络污染问题就是一个典型。<br/><br/>互联网是高科技的产物，是历史发展、社会进步、人类智慧的结晶，是人类迄今所拥有的容量最大、传递速度最快的智慧载体。但网络的应用却是一把双刃剑，有利又有弊，近年来网络引发的不良事件层出不穷：猝死网吧者有之，杀父弑母者有之，坑蒙拐骗者有之，偷盗抢劫者有之，强奸淫乱者有之；有因痛恨老师没收手机刺死老师者，有为筹网资勒索群殴同学者，有沉溺网络暴力肢解同学者，有报复烧毁网吧死伤数十人者。甚至学校课堂只有老师在讲，学生却都在忙于玩手机成为“低头族”，导致学生素质下降，部分同学无法毕业；某部领导夜间突击视察军营，官兵都在忙于玩手机或电脑，无人钻研业务，导致军队战力下降。<br/><br/>网络的危害大多是无形的，作为新生事物，即使发展成为严重的疾病--网瘾，也得不到社会的深度关注，甚至网瘾作为疾病虽然被大部分心理学家、精神病学家、教育学家逐渐认可、接受，但网络营销商、游戏制造商、大量玩家或准玩家却横加阻挠、拒不接受。<br/><br/>任何疾病的发生、发现、发展、治疗、转归、预后、预防都有其自然规律，这种规律的探索、发现，特别是总结成文形成诊疗规范可能是非常复杂的过程，需要具有较高的专业素养、敏锐的感知力、强烈的责任心、吃苦耐劳精神的人不懈努力、不计回报地长期探索。<br/><br/>临沂网戒中心在杨永信主任的带领下，十几年如一日坚守在网瘾戒治第一线，共收治网瘾患者6000余例，成功率达90%以上。在大量成功案例的基础上形成的网瘾戒治综合干预模式――杨氏模式，采用“心理行为治疗+个性矫治+同步家庭治疗+药物治疗”综合网瘾戒治模式，集教育学、医学、心理学等于一体，形成了缜密、前沿、高效可行的科学系统，能够很好地解决长期困扰网瘾戒治方面的理论、技术和临床等重大问题，具有理论技术指导、操作安全可行和临床高效稳定等特性，开辟了多学科、多领域综合应用的成功典范。该模式、技术填补国内空白，在国际上具有显著创新性，居国际先进水平，已得到教育学、心理学、精神病学、医学等方面权威专家的科研论证和鉴定。<br/><br/>就是这样挽救了无数网瘾患者和家庭的临沂网戒中心，近期却深陷网络漩涡无法自拔。不但杨永信团队遭受网络攻击和现实人身威胁，而且正常诊疗过程也受到被舆论裹挟和绑架的相关人员的干扰，治疗规模、治疗效果都受到影响。损坏了中心的形象，延缓了网瘾患者的救治，甚至带来了病情的反复，严重损坏了患者和家属的利益。<br/><br/>好在党和政府早已认识到网瘾的家庭危害、社会危害，对中心进行了深入的调研，没有被雾霾蒙蔽双眼，在错综复杂的舆论乱象中站在了正义的一边，果断出台了《未成年人网络保护法（征求意见稿）》，从而为这场闹剧奠定了基调，国家相关网瘾防治的法律法规指日可待。<br/><br/>但这场舆论风波仍没有烟消云散的迹象，主要原因是临沂网戒中心动了某些人的“奶酪”。这场舆论闹剧的始作俑者也是幕后推手――一些不良网络运营商、游戏制造商对网瘾戒治如鲠在喉，倚仗雄厚的财力支撑和强大的网络资源，蛊惑人心，误导舆情，妖魔网瘾戒治事业，为游戏造势，以广开钱路，没有政府的强力干预他们不会善罢甘休；这场舆论闹剧的“倒杨”主体，是被利用的众多网络游戏玩家和心智不成熟的游戏爱好者，其中不乏深陷网瘾的患者或准患者，尤其是个别从中心走出的没有成功治愈的网瘾孩子，大量接受了“倒杨”攻击者不实信息后被蒙骗、蛊惑，正义感爆棚，却被邪恶利用，没有政府正面、客观的宣传，他们仍将错误地认为已经治愈的网瘾患者和亲属发出的声音都是在替杨永信“洗白”，难以冷静、客观地思考、站队。<br/><br/>通过以上分析发现，政府的态度和作为才是制止这场舆论闹剧、让网瘾患者尽快得到治疗、康复的关键。希望政府有关部门不要被舆论绑架，力挽狂澜，平息这场本不该出现的舆论闹剧。否则临沂网戒中心为救治网瘾患者所做的努力得不到肯定，对社会诚信体系、道德体系都是极大的破坏。更会使大量网瘾患者延误治疗时机，给患者本人、家庭和社会都会带来莫大的痛苦和伤害。<br/><br/>呼吁政府尽快通过《未成年人网络保护法》,并严格落实，预防更多网瘾患者的出现。<br/><br/>呼吁政府尽快强拳出击，制止舆情乱象，驱散雾霾，还人民以绿色健康的网络生态。<br/>首次转载时间:
