class="preface">――面对恶媒谣记的谎言，谁才是真相的见证者？</div><br/><!--more--><br/><div class="WB_editor_iframe"><br/><br/>???今天，读了触乐网的《“盟友”小辰：逃离临沂》，我又对外界的某些人是怎样诋毁我们网戒中心的多了一些了解。<br/><br/>看到这篇文章，我为作者写的内容感到可笑，什么“下黑手”、“大逃亡”、“全城抓捕”，让人感到很荒唐，给人的感觉是穿越回到了黑帮匪患横行的年代，好奇的是这种不属实的描述在吸引人们的眼球上到底能撑多久？到底为什么要把我们正能量的网戒中心描写成这么黑暗？究竟为的是什么样的大利益？<br/><br/>其实只要大家仔细看看这篇文章，然后再想一想就能知道，这里面哗众取宠的成分太多了，有些地方读起来让人有点茫然，有些内容读起来甚至让人感到无聊，这是作者的写作功底太差了吗？我想不是的，是因为有些内容是要努力暗示读者的，所以就必须进行变形的夸张，把临沂网戒中心往死里黑，无形中增加了写文章的难度，有太多信口开河的成分，所以文章看起来极不严谨，幻如隔世。<br/><br/>文章中还说到“因为没办法，如果你在留下来，就很可能会被抓回去，或者遭黑手”。网戒中心是这样的吗，会对谁随便下黑手？中心是这样子对付曾经在中心呆过的盟友的吗？这究竟是公立医院，还是暴力机构，或者是执法机关？我想问一问，“遭黑手”是什么意思？这让人感到发笑，真有“黑手”一说，你小辰和触乐网还能活跃到现在？<br/><br/>忍不住想笑。<br/><br/>文章中还写到“我不能连累我的朋友”，提到了“连累”两个字，我想说的是，网戒中心真的这么恐怖这么强大吗？会让人连累其他人的？我想知道这其中的过程，小辰是通过什么能“连累”到他的朋友的，难道他的朋友也会“遭黑手”吗？对于这个“遭黑手”前面已经提到了，这是令人感到非常可笑的，不属实的。在这里，我就好奇了，我们网戒中心是如何“盯上”小辰的朋友的，是如何让小辰的朋友受到小辰的“连累”的。所以小辰说的话完全是胡编乱造的。<br/><br/>再往下看，会看到“全城抓捕”的字眼。看来我们中心实在是太厉害了，我们的人力和权力也多到足够对小辰进行“全面抓捕”了。其实，只要来临沂网戒中心的都会知道这里的氛围是温暖和谐的，家长们的谦和与善良有目共睹，杨医生就是这样引领我们营造了一个健康向上的氛围，才影响改变了这么多问题孩子。家长们每天也都有自己的工作要做，既要看好孩子，也为中心出一份自己的力量，根本没有那样的闲功夫去对一个人进行“全城抓捕”，这也不是他们该干的事。<br/><br/>当然了，因为小辰自己以前走偏得厉害，才逼得他的家长把他送到中心戒网瘾，除了他父母还有谁能这么干？<br/><br/>文章中，小辰还提到了“那篇文章是我自己写的，自己做的事要自己承担”，这里说的小辰好像是很有正义感的样子。作者对小辰的形象进行了美化，其实，目的也是在抹黑网戒中心。你想想，如果小辰真的很争气，在国企体体面面的上班，哪个家长会傻到把一个优秀的孩子送到医院戒网瘾？其实，这样一个有问题的孩子，胡言乱语也是可以理解的。<br/><br/>下面还有更可笑的，就是小辰说的“那个地方（网戒中心）能量那么大，如果出动警察了，走漏风声了，就会在半路上抓住我”，看到这里就知道小辰的思想是有多荒唐了。网戒中心有权利出动警察么？真不知道为何敢把中心描述的这么夸张，描述成这样是什么目的？只能说这个人为抹黑网戒中心，已经到了一种黔驴技穷的地步。<br/><br/>再往下看就知道了，为什么小辰要抹黑临沂网戒中心呢？原因就是小辰说的：“想找媒体把这事情弄大一点，事情弄大了之后，自己至少算是半个公众人物。”这就是小辰为什么说出了很多荒唐话的原因，我推测这想当“网红”才是这出闹剧的主要起因。<br/><br/>文中还说到：“发了文章微博之后，有了打赏，还有不断陆续有人发红包过来，我都没有收，目前所收到的钱也记了账目。”从这里可以看到小辰是怎样为维护自己的公众形象说话的，“我都没有收”显示了自己的清高，其实就是希望更多的人打赏，因为那个样子说的自己人气越旺。还有“收到的钱也记了账目”，他真的打算还吗？他还了多少？或许他通过抹黑网戒中心成功了之后还获得更多的金钱吧！<br/><br/>这是我作为网戒中心的一位盟友在看了《“盟友”小辰：逃离临沂》后的一些想法，希望大家不要上了小辰和触乐网的当，他们的话里面漏洞太多。当然也是仁者见仁，智者见智了。<br/><p
