/><br/>一名深受网瘾之苦的患难家长如是说<br/><!--more--><br/><br/>从8月初至今，二次炒作“杨永信戒网瘾”风波，恶浪未平，暗潮涌动。就在国家网信办发布《未成年人网络保护条例（草案征求意见稿）》的当下，中国预防青少年犯罪研究会党委书记操学诚表示，防止“网戒乱象”需要卫计委等部门在之后出台的相关治疗规范中明确不能侵犯未成年人合法权益。这本无可厚非，但其“对于一些网戒机构的所谓‘电击’疗法是不能容忍的，这不仅没有充分的科学依据，也是对沉迷网络的青少年的二次伤害”的言论，却值得商榷。<br/><br/>得出一个结论，首先要看是谁作出的结论，如果是行内权威专家，则无可厚非，但操学诚是行内权威专家吗？<br/><br/>百度百科：操学诚，男，1965年2月出生，安徽桐城人，1987年7月参加工作，1986年3月加入中国共产党，哲学学士、法学硕士、高级管理人员工商管理硕士学位。现任中国预防青少年犯罪研究会党组书记。<br/><br/>1983.09――1987.07，山东大学哲学系学生；<br/><br/>1987.07――1990.09，安徽医科大学社科部助教、秘书；<br/><br/>1990.09――1993.07，南开大学社会学系研究生（其间：1992.09―1993.07 校研究生会主席）；<br/><br/>1993.07――1994.05，团中央宣传部思想教育处干部；<br/><br/>1994.05――1995.08，团中央宣传部思想教育处副主任科员；<br/><br/>1995.08――1996.05，团中央宣传部思想教育处主任科员；<br/><br/>1996.05――1999.04，团中央宣传部思想教育处副处长；<br/><br/>1999.04――2003.10，团中央宣传部理论宣传处处长；<br/><br/>2003.10――2006.04，团中央宣传部副部长；<br/><br/>2006.04――2007.07，中国青少年宫协会秘书长（正局级）；<br/><br/>2007.07――2008.08，中国青少年宫协会党组成员、秘书长（其间：2008.05―2008.07在中央党校地厅级干部进修班学习；2006.05―2008.09在长江商学院学习，获得高级管理人员工商管理硕士学位）；<br/><br/>2008.08――2012.03，中国预防青少年犯罪研究会党组书记、秘书长（其间：2008.12―2010.08
