/><br/><br/>前天，在不应当出现的地方，出现了一篇很有趣的文章――<br/><br/><!--more--><br/><br/>就是触乐网的祝佳音写的《被杨永信拯救的家长们》。之所以认为它有趣，是因为作者试图努力灌输给我们一个结论：被杨永信拯救的家长们很愚蠢。我们不妨一起来看看祝佳音是如何推理的。<br/><br/>1.<b>截取并凸显部分家长的原话以此证明：家长很愚蠢</b><br/><br/>“他走进孩子的房间，小唐正在看直播（或者是动画片，<b>采访过程中，唐学礼始终没能分清楚这两者的区别</b>）。唐学礼感觉自己‘仿佛听到了一阵哀乐。’”<br/><br/>“小康从小学起开始玩游戏，但那时候‘还能控制’。问题出现在初一，小康迷上了《英雄联盟》。<b>自从玩上了这款被赵友爱认为是由林熊猫开发的游戏以后</b>，小康就再也‘不受控制’了。”<br/><br/>“我问唐学礼，‘贩毒和卖淫的为什么归网戒中心管？’<b>他支支吾吾，觉得‘除了杀人放火，这里面什么孩子都有</b>。’”<br/><br/>“赵友爱不停向我重复当时的感觉，‘根本没法管啊，管不了啊。’但每次当我试图询问小康‘走偏’的具体细节时，<b>她都会下意识地回避</b>。”<br/><br/><b>“赵友爱突然不愿意说下去了。</b>一提到网戒中心，她的情绪就非常激动，夹杂着大段方言的语句冒了出来。”<br/><br/>“如果一家人‘<b>花10年到20年在里面住着，保证你（将来的）子孙都很正常。那这很划算</b>。’”<br/><br/>“除了‘正能量’以外，她记忆最深刻的是上点评课，‘弟子规啊，三字经啊，让你全都在那学。’每天还要自我反省，‘哪一天学的好，哪一天学的不好，都要总结。’<b>赵友爱认为，‘网戒中心里的这个教育，可以说是一个国学了。’?”</b><br/><br/>――类似的地方不胜枚举，总之通篇内容都在极力地凸显家长群体的愚蠢，这种写作手法很高明，它很真实，真实的让人难以置信，作者此刻彷佛正得意并微笑着对家长们说：你自己说你蠢不蠢？还不承认是吧，这些话可都是你们的原话啊，我可没有胡编，观众的眼睛是雪亮的。<br/><br/>可惜聪明反被聪明误。这篇文章的失败就失败在它太过于完美，完美到读者们几乎无法反驳作者的观点。祝佳音给我们呈现的家长简直就是愚蠢的代名词，甚至接近于“无知”，不禁让人觉得：太可怕了！但是你有没有那么一丁点奇怪――这么多愚蠢至极的家长，或者说他们真如祝佳音所说，接近于“无知”，那他们是怎么生存的呢？<br/><br/>看看这群家长已经无知到什么程度了。<br/><br/>唐家长，分不清直播和动画片的区别；认为动画片的声音像哀乐；认为网戒中心里除了杀人放火什么都能管；认为花几十年的时间住在网戒中心很划算。<br/><br/>赵家长，认为《英雄联盟》是林熊猫开发的；认为孩子走偏的细节不重要；莫名其妙不知何因突然不愿聊下去了；认为弟子规、三字经就是国学教育。<br/><br/>我承认，在祝佳音传递给我们的信息背景下，在他极力凸显的内容里，这群家长是无知的。但这个逻辑对不对？<br/><br/>在这我想说一段容易被我们忽略的废话。家长跟孩子的年龄差距大多在20-30年，晚婚晚育或者老年得子的年龄差距会更大。在如此大的年龄差距下，人的信息迭代、认知迭代必然有着巨大的差距。我不知道祝老板的父母有多博学，我自己肯定不会期望父母能跟上我的信息获取节奏。<br/><br/>最简单的例子，你会跟你父母聊初音未来么？会聊《火影忍者》么？会聊诸葛丞相大战王司徒么？会聊什么叫一血什么叫五杀么？也许在祝老板的眼中，父母搞不清楚直播和动画片的区别就是愚蠢，父母搞不清楚《英雄联盟》到底是谁开发的就是无知，如果您就是这么想的，我无话可说。怕就怕，您自己不这么想，但是想努力地让观众们这么想！<br/><br/>所以，我不认可您的“家长愚蠢论”。<br/><br/>当然，接下来祝老板还试图用很多方法来力证自己的观点。<br/><br/><b>2.</b><b>引用孩子的评价来辅证“家长很愚蠢”</b><br/><br/><b>“2008年</b>曾在网戒中心‘接受治疗’的<b>王涛</b>这样说。但相处也会带来一个问题，那就是猜疑。”<br/><br/><b>“王涛</b>觉得网戒中心里的孩子实际上像犯人一样，‘顺从、奉承、讨好教官，期望提前出狱。’?”<br/><br/><b>“王涛</b>说，‘家长要猜测孩子是不是真的改变好了’。他认为这让本应最亲近的家人之间没了信任。”<br/><br/><b>“王涛</b>说，‘杨永信反过来就拉走他们，你觉得孩子以后还能信任家长吗？’<br/><b>王涛</b>说，在他的印象里，网戒中心的点评课对于孩子来说就是‘打鸡血’，而对于家长而言则是‘杨是绝对的权威’。<b>‘如果你现在在大街上喊谁谁谁万岁，别人一定觉得你疯了，但是在那里说杨叔万岁就像见面说你好一样正常……家长也会这么喊，尤其是当大家跪在地上哭的时候，一直会有类似的声音，此起彼伏。’?”</b><br/><br/><b>“王涛</b>曾经看到过一个孩子偷偷对妈妈说‘我受不了，你们带我出去，我回家听话’，结果‘他妈就去找杨永信，回来就把那个孩子拉去电了……’”<br/><br/><b>“王涛</b>觉得‘我打赌她一年后不会这么说’，他觉得小康现在还处于‘打鸡血的状态’，而网戒中心对于亲情的破坏是不可逆的。‘<b>当时我接受采访的时候，就说过永远也不会原谅父母，我相信像我一样想法的人应该不在少数</b>……抛开一切不谈，孩子在家长的面前始终会有恐惧感。<b>本应是家人啊，最亲密无间的关系，竟然会有恐惧夹杂其中，这就是家长想得到的听话的孩子么</b>？’”<br/><br/>――祝老板这一次倒也实在，懒得多费功夫，用一个“2008年的王涛”解决了他所有的需求。<br/><br/>祝老板还在文章开头埋下了这样一个铺垫，“需要申明的是，为了保护受访者，我们把所有素材进行了打乱和模糊处理，你们不会从文章中发现他们是谁。”<br/><br/>所以是在下不仁，把祝老板故意打乱的信息又给仔细整理了一下，发现了这个问题：这哪是对网戒中心家长的采访？就是一篇“王涛回忆录”嘛，自称2008年去过临沂网戒中心的王涛，时隔10年，努力回忆当时的网戒中心，也实在难为孩子了，毕竟让一个人回忆10年前的事甚至要回忆出许多具体的话太难了，我为王涛的记忆力点赞，也为祝老板这种敢为天下先的采访点赞。但是靠这样的艰难回忆来描述一个现存的机构，来评价一个现存的事物，来辅证你当下的推论，靠谱么？<br/><br/><b>“如果你现在在大街上喊谁谁谁万岁，别人一定觉得你疯了，但是在那里说杨叔万岁就像见面说你好一样正常……家长也会这么喊，尤其是当大家跪在地上哭的时候，一直会有类似的声音，此起彼伏。”</b><br/><br/>――这段描述更加阴险，用没法考证的王涛的话，直接给杨永信戴上了“个人崇拜”或“邪教教主”的帽子，“杀人不见血”的手法祝佳音用的可谓登峰造极。<br/><br/>“王涛最后说：‘<b>本应是家人啊，最亲密无间的关系，竟然会有恐惧夹杂其中，这就是家长想得到的听话的孩子么</b>？’”<br/><br/>――说的无比委屈而无奈，我想反问王涛：如果家长和孩子真的还处于你所说的“最亲密无间的状态”，那为何还要去网戒中心呢？<br/><br/><b>3.</b><b>借助“看法不同”的家长辅证：家长很愚蠢</b><br/><br/><b>“2010年</b>，家在上海的庄文明亲自送孩子入院。”<br/><br/>“‘<b>杨永信是一个很聪明的人</b>，他也很有能力。’在一家销售磁疗床垫的连锁店里，年近60的庄文明对我说：‘<b>那里的氛围就是杨叔是老大</b>，他刻意打造了那么一个环境，他是绝对的权威，家长要是行的话，你怎么到这里来了？’<b>她认为杨永信利用家长的无知</b>体现自己在‘戒网瘾’方面的权威性，‘你（家长）跑到这里来就是一个失败者，你这也不行，那也不行。’”<br/><br/>“网戒中心为很多家长提供了‘解决家庭问题’的方法。这种方法简单、直接，而且对他们来说‘有效’。<b>包括点评课在内的程序设计强化了家长们对‘杨叔’和‘心理学医生’的崇拜</b>。‘你得依靠我啊，你到我这里来，我是专家。你看我这么多案例，这本书那本书……是吧，所以你就老实听话，你就全都交给我就行了。’”<br/><br/>“网戒中心曾希望庄文明能进入家委会，她觉得原因是<b>看中了她的‘大城市背景，以及就读于名牌大学的儿子</b>。’她的孩子在得知这个消息后，私下告诉她‘不许当’。她自己也从来没想过作家委会成员。”<br/><br/>――恭喜祝老板，终于找到了一个“看法不同”的家长，或者说是跟他自己“看法相同”的家长，而且这位家长是“大城市背景”，还有“名牌大学的儿子”。这些似乎可以证明：她选择离开网戒中心是无比正确的选择。当时甚至连“家委会”都看中了她，极力拉她入伙，却被她断然拒绝了。<br/><br/>这彷佛是一个“世人皆醉我独醒”的角色，彷佛是一个聪明到看透杨永信所有把戏的角色，借助这个角色的三言两语，杨永信那种“独权”、“伪权威”的邪恶印象在观众心里油然而生。<br/><br/>但我想说的是，网戒中心和杨永信如果真如你所说这般，那可是连观众都能一眼看穿的小把戏，真的只有你庄文明才能看穿？你可以用你的“大城市背景”和“名牌大学的儿子”来解释你的与众不同，但你如何解释你为什么带儿子去网戒中心呢？<br/><br/>其实说到底，“不同看法”的出现是祝老板选择性采访的必然结果，也是刻意安排采访顺序的必然导向。另外“庄文明”这个化名我给你点赞，很有深意。<br/><br/><b>4.</b><b>借助“权威专家”辅证：家长很愚蠢</b><br/><br/>“‘<b>在网戒中心，最大的权威是杨永信</b>，<b>家长和孩子需要分别向杨永信证明自己的改变和忠诚</b>。’<b>心理学从业者，一线教师夏红堂</b>告诉我，在这种巨大的压力下，家长和孩子的思维都变得狭窄，他们似乎只看到了一条路，那就是‘<b>做杨叔说对的事’，而失去了个人判断</b>。”<br/><br/>“我想强调一点，<b>家长们不是‘没有思考能力’，而是‘放弃思考’</b>。因为思考之后会带来痛苦，可能还会带来自我谴责，懊恼，失望等等负面情绪，所以他们选择还是相信在这里是对的。夏红堂说，‘面对暴行和人身伤害，不作为本身已经是恶，更何况作为帮凶呢？?’”<br/><br/>“‘<b>我怀疑</b>这些家长身上应该有一个共性，他们从小的家庭环境或社区环境可能让他们习惯了有一个绝对权威的存在。他们只要听从就可以了，乖乖的就不会有事，不乖的会被狼叼走。’夏红堂告诉我。”<br/><br/>――在文章的最后，祝老板请来了一位<b>“权威专家”</b>夏红堂（可惜我没听说过也没搜到），<b>“夏专家”</b>一共说了三段话，总结起来很简单：<br/><br/>1.家长们因为要向杨永信表忠心而“失去个人判断”。<br/><br/>2.家长们因为害怕自己产生负面情绪所以“放弃思考”。<br/><br/>3.家长们从小就习惯了“听从”绝对权威。<br/><br/>“夏专家”的这三层意思概括起来就更简单了――<b>家长很愚蠢。</b><br/><br/><b>四个不同的角度，四个煞费苦心的证明，终于论证了祝老板的观点，</b>其实您不用这么费劲，直接跟以前一样，发一个短微博就可以了：<b>“家长们太愚蠢了，轻易上了杨永信的当，坑害了祖国的花朵。让我们一起来抵制愚蠢的家长和残暴的杨永信吧！祖国的花朵们，大家携起手来，一起玩我向你们推荐的游戏――《XX联盟》火爆公测中！</b>”<br/><p
