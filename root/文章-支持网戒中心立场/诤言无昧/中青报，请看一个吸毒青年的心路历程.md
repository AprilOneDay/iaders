们开车带着我从市区跑到县城，找到一个人买了几克“冰”，这时我才知道这是毒品，开始我是拒绝的，但他们告诉我这东西不会上瘾的，在被他俩的再三相劝下，我接受了他们的邀请，记得当时我只吸了一口，就感觉到头皮发麻，也就是上头了，吸完后我们要离开出租屋的时候，我发现自己站都站不起来了，像没有腿了一样，这就是冰毒对大脑和神经系统的麻醉。之后我们来到了洗浴中心，当时我是用手拼命扶着楼梯爬上二楼的，一直到蒸完三个小时的桑拿后才好了一点，但是大腿和脚一直在抽搐，导致我两天没能上班，在洗浴中心里住了两天，神奇的是两天的时间里我只喝了两瓶营养快线，但是一点都没有感觉到饥饿。<br/><br/>因为有了第一次的经历以及在吸食毒品后所带给我的奇妙感觉，那种浑身充满力量的快感，使我一有空脑子里就跳出吸毒时的景象。没过几天我就主动联系了那个损友，说想再玩一玩。钱我来出他们去拿货，并且负责场地。就这样，我吸毒的次数变得越来越多。<br/><br/>在这里我有必要跟大家普及一下毒品的知识，虽然我只懂得皮毛。首先我们大家最先想到的毒品就是白粉，学名海洛因，在吸毒者之间它被称为“四号”，它对人的毒害作用很大，人在吸食后会感觉浑身舒畅，就像进入了天堂，这段时间内闭上眼睛慢慢的享受，每抽一颗烟，就是一次快感的升华，鼻腔会嗅到那种淡淡的兰花香的味道，这是因为这种毒品是靠植物提取加工的。海洛因大多是金三角，以及金新月两个地区生产制造，然后流往各国，因为这两个地区都和我国有毗邻，所以我国在打击毒品上是很坚决的。如今吸食海洛因的人少了一些，大多数是海南、广西、云南等地较多。<br/><br/>而最近这些年，冰毒的吸食者成倍的增长。冰毒学名甲基苯丙胺，在圈里被称为“3号”，一开始是二战中日本制造出来给战士特别是空军使用，可以抗疲劳，长时间保持精力充沛，后来就被别有用心之人制造出来为了个人利益而残害人类。一开始的冰毒都是由从植物麻黄草中提取的麻黄素提炼而成，然而，后来人们又有了新的途径，可以化学合成冰毒，长期吸食冰毒的人脑神经会遭到不同程度的损坏（海洛因戒断的时候是身体的疼痛），如今社会上最流行的毒品就是它。<br/><br/>至于其他的流行毒品还有很多种。一种叫“麻古”，它的成分里也含有冰毒，它的作用与功效在社会上也是众所周知的，它可以让一个听话懂事的小姑娘在一天内变成一个常时间涉足风月场合的毫无羞耻心的人。所以千万不要听损友的话：“吸一点没事，不会上瘾的。”现在的九零后零零后都很喜欢“打K”，K粉其实也含有冰毒的成分，它可以让吸食者产生幻觉，有点像以前的摇头丸，它们都是流行于KTV、酒吧、迪厅等娱乐场所，如今市场很少找到摇头丸了。还有一种毒品叫“大麻”，这种毒品吸食了以后，会使人精神亢奋，同时会使人笑点降低，在旁人看来很普通的小事，都可以让他哈哈大笑，很多明星以及富二代都喜欢抽大麻，因为他们在事业上生活中遇到不开心的事，压抑苦恼时，吸食大麻就会缓和而开心起来。现在的市场又出现了很多新型毒品，如从欧洲传过来的“浴盐”、“丧尸药”、“神仙水”，这些新兴毒品也在不断地蔓延市场，经常出入酒吧迪厅的人要做好自我防备。还有既是药品又是毒品的，如“美沙酮”，是海洛因吸食者的一种替代品。可能有很多人会知道，那是政府为了一些没有被处理过的吸食者而发放的。美沙酮可以起到间接戒毒的作用：很多的城市中都有发放美沙酮的地点，价格很便宜。其实在这其中考虑的是害怕那些没钱买毒品的人，为了获得毒资挺而走险，而往往有些人在这当中又中了喝美沙酮的瘾。我见过一些人有钱时吸海洛因，没钱就去喝美沙酮，这样反而没帮上忙又害了他们。再如医院里常用的杜冷丁也是一种毒品。所以现在的医院对于这两种药品进行了严格的审查。<br/><br/>讲完了这些毒品的知识，我接着上边说的老婆怀孕后我开始吸毒的那一段。吸食冰毒后的这段时间我经常夜不归宿，每天都活跃在买货、吸毒的路上，对家人的关心疼爱少了很多。记得儿子出生那天，我因为毒瘾发作，甚至都没有到医院看看，说起这事我感觉自己太没人情味了。吸毒这件事，我从没告诉过她，她也没问过我，那时全家人的眼里和心里都是我们的孩子。我们很少聊起过双方，话题总是在孩子的身上，彼此心照不宣，小心翼翼地维护着表面的和平，有一种从宇宙的中心被流放到了犄角旮旯的感觉，每每想起都黯然神伤。从之前的二人世界变成了现在的三口之家，那种柴米油盐的繁琐使我们俩开始了无休止的争吵。在孩子两岁多的时候，我们的婚姻走到了尽头，两人坐在一起心平气和地在离婚协议书上签了字。从我和她认识到走进婚姻再到离婚，整好七年的时间。这段感情中我亏欠了她，如果我不曾吸毒，这个家不会散。<br/><br/>我离婚以后，爸妈知道了我吸毒，这对于他们是一个更加沉重的打击。本来离婚的事他们就有很多的话压在心中，爸妈没有干扰我们离婚，全凭我们俩的决定。对于吸毒，我的爸妈坚决要我断绝与之前朋友的联系，逼我只身来到南京，投奔我的二姐。<br/><br/>我二姐夫凭借关系给我安排了工作。在南京刚开始我还是很老实的，不过经过一段时间的认识与熟悉，错交了几个投脾气的人，每天又在一起花天酒地，每个晚上都不回公司，在外边玩赌博机，出入洗浴场所。很快不到半年的时间，在南京我又结识了几条卖毒品的资源，结果可想而知。我二姐每天都为我提心吊胆，就这样我的第一次尝试戒毒失败了。在南京的工作薪酬还算可以，所以我和另外两个本公司的员工整天都吸得头晕脑胀。也就是在那段时间里，我感到吸毒让我大脑的思考方式和为人处世的方式有了巨大的改变，因为那个和我一起的同事也是离异的，所以我们每次吸过毒后，不像之前那种轻飘飘的快感，反而都出现了遇事爱钻牛角尖、容易冲动等坏现象。这就符合了那个原理，一个吸食两年冰毒的人，如果说不是很能调理自己的情绪，都会有不同程度的大脑损害，思维方式会变得完全不正常，所以我们每个晚上都在外边度过，打过许多架，也受了不少罪。不仅是这些，在南京的那段时间我还染上了酒瘾，每天晚上都要喝。常在河边走，哪有不失鞋的，有一次，我和朋友在大排档为了争一个卖唱的女孩，发生了冲突和辱骂。但是在老板的劝解下，双方都罢手，最后他们走了，我们还傻傻地继续喝酒，万万没想到他们杀了个回马枪，我们三个人手无寸铁的情况下，被砍的遍体鳞伤，我是伤势最重的一个，因失血过多昏迷了十二个小时。等我醒来后，医生告诉我要不是出事地点与医院近的话，我早就挂了。当然还得感激我的一个同事，他在出事后很迅速地为我打电话求救，自己也受了伤。这件事让我二姐吓着了，最后还是和我大姐商量，告诉了我爸妈，为了不给我二姐继续添麻烦，我告别了在那堕落了三年的南京。<br/><br/>从南京回来以后，我还是整天惦记着去哪里买点货，时间已经来到了2013年，很快我又找回了以前朋友的联系方式，父亲与亲人苦苦相劝，也没能让我下定决心戒毒。但是这时的我，每次吸了毒以后都感觉很痛苦，因为我脑子总是想些不开心的事，所以我已经到了行业里面所说的“叉道”了，也有的说“拐”了。2013年底，我被家人送到了济宁的戒毒所，连过年也是从那里过，在戒毒所的时间里，我和其他人一样，整天除了吃就是睡，小小的房间里，除了中间的路，没有坐的地方，只能打坐式的坐床上；很小的一间房，十张床排的满满的，每天除了打扑克就没有什么娱乐活动了。里面还允许抽烟，内设了一个小超市，让我们购买吃的，这也是戒毒所创收的一种手段。第一次我们待满了一个疗程，当时是三月一疗程，然后又巩固一个月，四个月花了家里四万块钱。他们除了给吃些药，就没有别的事了。那里还有心理干预和品格重塑等这些课，因为我假装配合，所以我只被关了四个月，但我的心里还是想着出来以后再吸。就这样我回家不到一个月就又被送了进去，当时我就告诉我爸说这里不会管什么用，我爸也是无可奈何，因为他被我逼的不知道哪里才能治好我，在这样的情况下，我又在里面待了五个月，不仅毒品没戒成，反而从里面结识了更多我们这里和我一样去戒毒的人，互相还留了号码，出去之后好联系哪里的货便宜去哪买，所以我在里面就听说戒毒所的这种方式对戒毒没什么帮助。关在戒毒所里，只能是强制性的使人与毒品分离，但是他们没想这些人心里其实都在盼着回家如何复吸，并没有对这些吸毒人员进行深刻的了解，只会一味对过去的行为批判而忽略了这些人的品格思想的重塑。所以像戒毒所这种戒毒的方式成功率基本为零。<br/><br/>??这次从戒毒所出来后，我在家里只坚持了三四个月的时间，因为当时爸妈轮流在家里陪我，所以我不好发作，但过了段时间后，家人对我放松了警惕，我终究还是没能把持住自己。这样家人对我是彻底失望了，当时爸妈真的有了放弃我的念想。<br/><br/>这时，我舅舅家的表弟来到我家，当时我感觉很意外，为什么呢，因为他们家和我们家已经没有任何来往了，因为他们信奉基督教，就连我姥姥去世时都没能见到他们的身影，所以他的到来我很难解。那时我们每年过节去他们家时买的任何礼物，他们都会原封不动地送到我姥姥家退还给我们。他们只和信耶稣的人有来往，我们这些不信的人，在他们眼里都成了外邦人。他来我们家的目的就是想让我去信耶稣，开始我很抵触，但是他说信耶稣能帮助戒毒，我爸妈就同意了，就这样我被家人送到了温州，在一个山沟里开始了我的宗教戒毒之旅。<br/><br/>我开始很抵触，我不愿意信宗教，但爸妈不愿意让我回家，说就算我不信耶稣也不让我回家，如此我又被送到了温州的一家精卫中心，在医院里我每天和一些真正的精神病人住在一起，没人说话，没有烟，住了一个多月我就受不了了，打电话给我爸让他接我回家。但是我爸说接我出来可以，但是不是回家，而是再送到福音戒毒所，二者只能选一，要不就住在精神病院里，最后我还是选择了回到山上。这次上来我就要接受信基督，每天跟里面的兄弟姐妹一起读经祷告，还有很多传道人来给我们讲课分享。一天四节课，和那些同伴也就是那些五湖四海的同路人一起共学共勉，做饭，洗衣，玩乐，唱诗敬拜，生活得倒也很充实。慢慢地我不再像之前那样抵触了。因为我从这里的学习中了解到，圣经里很多的话和事例，其实都是好的，它教人学好，只不过是每个人对圣经的了解不同。如果没有一个正确的引导，很容易走偏。通过在温州近一年的基督教学习，几位月工决定让我去海南继续学习，这样我就坐上了温州到海口的飞机，这里是福音戒毒的另一个点。<br/><br/>在海南我表现的不错，教友们捐钱捐物，再加上自家教会的供养，我们的生活还算可以，当然我们也想了很多自立的方法，我们在温州那边的山上养鸡、养羊、喂猪，在海南的中心我们种了很多果树，莲雾、菠萝。每天下课后我们都会坐在一起和一些外国来的牧师一起讨论圣经，让他为我们按手祷告。很快在海南的日子也快一年了，队长支持我去读神学，学成后回来帮助学员，我爸也支持，我就去温州的一家神学班，可是后来因为我和班里的一个学员闹了矛盾，又对负责人恶语相向，我被迫回家了。<br/><br/>其实这次回来我心里是有很多愧意的，我辜负了那些帮助过我的人，还有我的父母，回到家后休息了几天，我就到公司上班了，说句实在话，我真的感谢这段时间帮助过我的每一个人。不可否认这个福音戒毒的方法比起政府性质的戒毒所成功率大得多，但也不是那么神奇，在这两年的时间我也见过很多的人进进出出，有成功，当然也有失败。听到过这样一个比例，政府的强制戒毒所成功率是零，而福音戒毒所的成功率为30%。<br/><br/>时间来到今年的春节期间，我因为个人问题处理，以及在孩子的教育问题上和家人产生了严重分歧。更为严重的是，由于之前长期吸毒对我身心的危害，特别是脑神经的伤害，使得我做事极端，性格偏执，思维和行为方式已经严重异于常人，与家庭社会格格不入。今年的2月12日，我被父母送到了临沂网戒中心。<br/><br/>我记得很清楚，刚开始我很抵触，对待盟友的态度非常不好，初来的一个多月里，我劣迹难改，惹了很多麻烦，多次想冲动伤人。但是这一次最大的不同是我爸在这里全程、全天候的陪着我，这让我始料未及，我也在每次的周六分享中，看到了那么多成功的范例，才开始慢慢从杨医生以及其他医生的讲课和关心中不断的对我们这个网戒中心有了客观的认识。<br/><br/>临沂网戒中心的特色，就是通过震撼人心的心理点评和一系列的环节，对我们这些问题青少年进行性格重塑和行为矫正。如心理点评课，就是让我们纠偏错误认知，重树理想信念，强化道德修养；日记课让我们学会反思，演讲和辩论就是用正确的思想强化我们的理性思辨能力，内务训练就是让我们养成良好的行为习惯。再如《阳光心态养成要领》，那是杨医生和他的团队对人生修养的智慧提炼，为的是使我们养成生活和工作中必需的健康心态，让我们在任何情况下保持理性，自我调适，坦然面对。中心正是用“阳光心态”，“劝悟之歌”，“挑战宿命”“60字心诀”“远离自私”“交友歌”等来提升我们的人生智慧，尤其是让像我这样的人不会再误入歧途。<br/><br/>另外，临沂网戒中心不仅是针对上网有瘾的孩子，也有其他不同原因的盟友，尤其是一些心理和人格有障碍的患者，中心和其他一些机构的不同之处在于，这里解决盟友的问题不是单一的就其个人，而是以全方位立体的方式解决，包括孩子父母给孩子造成的不良影响等。在这里治疗的这段时间，不仅让我们盟友有了质的改变，重塑了人格，同时家庭中的其他问题也在这里得到改善。比如说父母教育孩子的问题，夫妻之间的相处关系对孩子的影响等等，让我们知道问题并不是出在孩子一人身上。同时在这里，杨医生和其他的点评师都会对病友们出了中心之后所要面对的问题一一进行分析，并且给病友来中心充电、分享的机会，这是一个有机循环的过程，好的经验可以用来鼓励在院的病友，出现了难题可以由杨医生和在院的盟友一起来为自己分析、解答。尤其让我惊讶的是，在中心，我还看到了几位在这里戒毒成功的人，一开始来的时候我是不相信的，但是通过在这里将近七个月的学习，我认为只要能相信中心，相信杨永信医生的教诲，有了健全的人格，成功戒毒就能水到渠成。<br/><br/>临沂网戒中心，让我完成了浴火重生。这里让我拥有了人生的智慧，也让我更加自信的面对未来。<br/><br/>我想，我们每一位在这里走向重生道路的人都该感恩中心、感恩杨医生、感恩杨医生的医护团队，还有那些热心的家长。杨医生为了这个大家庭、为了中心、为了我们每一个家庭的付出，或许能从他头上的白发中显现出来。<br/><br/>乌云遮不住阳光，谣言掩盖不了真相，正义终将战胜邪恶。因为我们是真相的见证者，所以在中心遭到不良媒体的诋毁攻击时，我们坚定地与中心站在了一起，用自己的亲身经历来证实、来反驳那些来自不良媒体的诬陷！<br/><br/>能来到临沂网戒中心我算是幸运的，社会上还有很多像我一样不幸的人或家庭，需要临沂网戒中心这样的机构去救助。愿我们的网戒中心越来越好，让爱在这里得到最好的诠释。<br/><p
