倡导健康娱乐》）；之后，新华社也于7月7日至11日连刊三文（《“王者荣耀”还是“王者农药”：健康游戏不止于防沉迷》《手游不该“游戏”历史》《新华社再评王者荣耀：暴露游戏行业野蛮生长的痛点》。两家央媒痛批《王者荣耀》歪曲历史，诱导沉迷，殃及青少年健康成长与社会稳定，“我们需要认清的是，手机和游戏没有生命力，责任意识更应战胜商业利益。”<br/><br/>本来，腾讯股价进入2017年以来持续上涨，在已累计上涨50%的情况下，就在7月4日一天之内，腾讯股价突然下跌4.13%。<br/><br/>但随着腾讯一系列快速的危机公关，马化腾高调做客人民网，股价迅疾反弹。<br/><br/>仅仅一个月后的8月7日，据福布斯实时富豪排行榜，马化腾的个人财富361亿美元，超过了马云的356亿美元。<br/><br/>新首富的诞生，《王者荣耀》自然功不可没。<br/><br/>更令人瞠目的是，据中新网12月19日电：中国音数协游戏工委(GPC)、伽马数据(CNG)、国际数据公司(IDC)联合发布了《2017年中国游戏产业报告》。中国游戏产业年创收2036.1亿元，手游收入占比57%。<br/><br/>2000亿元是什么概念？从国家经济构成的宏观层面上看，我国文化产业的整体规模去年已达4万亿元，而网络游戏的产值已占其5%。因此，网游事实上已成为我国文化产业的支柱！<br/><br/>网游，被称为“指尖上的印钞机”，辉煌的GDP背后，可以想见的是，老师在课堂上面对着多少空位和低头族？又有多少学龄少年荒废青春昼夜不分疯狂盗刷着父母的血汗？又有多少父母的心在滴血？那些长发披肩、目光呆滞、自卑自闭、强迫焦虑，因人格扭曲而社会功能严重丧失被父母送到精神病院或戒网瘾学校的孩子又有多少？更何况因连续上网猝死网吧的血案屡屡曝光! 在游戏产业为国家贡献的GDP中，究竟流淌着多少道德的血液？!<br/><br/>说到底，究竟是谁妖魔化了游戏？！<br/><br/>然而，在网瘾被无视的同时，网瘾戒治的的确确被妖魔化了。不管你成功救治了多少网瘾少年，一起安全事件就足以毁灭一所戒网瘾学校，即使是没有出过一起医疗安全事故且成功救治了数千例网瘾孩子的临沂网戒中心，也因在专业背景下使用了“低频脉冲治疗”而被诬为“奥斯维辛集中营”！<br/><br/>今天的马首富，与当年不可同日而语，在慈善与公益的聚光灯下可谓鲜花簇拥掌声如潮。然而，在一款游戏能让无数少年沉迷却开发不出一款有效的防沉迷系统的窘迫中，马首富居然走进学校宣传戒网瘾了！做慈善也好，做公益也罢，难道马首富真不清楚游戏沦陷了和正在沦陷着无数青少年的事实，而你不正是当下首屈一指的游戏大佬吗？当你在两会期间大谈提高网络素养的时候，真的脸不变色心不跳吗？！<br/><br/>苦海无边回头是岸，但愿马首富莫负所言，带个好头让游戏产业尽快纳入健康规范的道路。至于说如何面对曾经因游戏沦陷的孩子和患难家庭，那就凭良心了！<br/><p
