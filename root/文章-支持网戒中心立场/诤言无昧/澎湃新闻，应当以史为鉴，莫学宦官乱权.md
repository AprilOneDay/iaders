/><br/><br/>――面对恶媒谣记的谎言，谁才是真相的见证者？（之十二）<br/><br/><!--more--><br/><br/>网络的普及，拓宽了人们行使民主权利的渠道，既可以发声表达诉求，又可以行使舆论监督。从辩证法的角度讲，任何事物都存在两面性，网络舆论同样在此范畴。我不否认网络舆论的积极作用，就如同我不否认地球是圆的，硬币有正反两面一样。它的好处就在那里，无论见与不见。举个例子来讲，如果没有网络舆论，“微笑局长”可能继续戴着金表笑到抽筋，“香烟局长”可能继续抽“九五至尊”抽到肺癌。从此角度来看，网络舆论可以申张民意，讨伐贪官，保护公民基本的民主权利，进而发展人民民主。<br/><br/>而人往往有一种能力就是变坏事为好事，而另一种能力就是变好事为坏事。我曾对比这两种能力，然而发现并不是人人都能变坏事为好事。<br/><br/>还记得那些因网络暴力而失去美好生命的人吗？有多少花季少年少女因抵制不住人肉搜索和网络暴力草草结束了生命。细细数来，投河的----有，上吊的----有，喝药的---有，跳楼的---有，谁知道还会发生什么呢？可能大多数的网民在发表言论时并没有考虑太多，热血澎湃、义愤填庸，亦或者小手一抖，承载着负面情绪充斥着毁灭能量的消息就发出去了，此后便不再过问，殊不知成为了“间接杀手”。“人多力量大”，一招手呼呼啦啦一群人，也确实很有力量，正所谓“唾沫星子淹死人”。网络暴力作为热门名词还没有引起足够多和足够高的重视，但既然能成为高考热点，便值得我们深思。<br/><br/><b>“不知者无罪”再溯源看“知者”</b><br/><br/>一般而言，“恶意攻击者”必先有一定的目的性----如打击、报复、炒作、盈利等等，当然这也就只有作者最清楚了，接下来，便是套路。搜集信息、拼接信息，把不需要的和有违目的的部分删掉；添油加醋，不断往自己的主观意愿靠拢，拼凑成表面上天衣无缝、能激起群众共鸣和响应的作品；继而煽风点火，利用网民的同情、爱国或盲从心理让他们变成“无意攻击者”。<br/><br/>这样捋一遍后，很多真相便浮出水面，清晰可见了。<br/><br/>澎湃新闻或者说澎湃新闻的两位记者张维、贾亚男出于什么目的不得而知，但他们的做法便略显老套了。<br/><br/>第一步：收集信息<br/><br/>即便再怎样虚张夸大，一个基础的信息是很必要的，几张图片，便证明自己“到此一游”为下文胡编乱造做了完美的铺垫。<br/><br/>第二步：断章取义<br/><br/>仿佛特异性变体一样，专门接受自己想要的信息，想写对方好，就抓住好的写，处处溢美之词；想写对方坏，就往死里写，反正和自己没有太大关系。网戒中心成功改变的案例视而不见，偏偏几个不学无术的未改变的盟友深深地吸引了他们的眼球，可谓臭味相投。<br/><br/>第三步：虚张声势<br/><br/>捏造几个路人甲乙丙丁，再赋予他们几句台词，便是天大的铁证了，不得不说，“人证”果然一抓一大把。<br/><br/>第四步：升华作品<br/><br/>此点最为重要，往往有能量的虚假文章多精于此道，芝麻大小的事轻者上升到道德与法制，重者升华到国家和民族。哎呀呀，想一想，舆论的影响力究竟有多大啊，连社会秩序都能冲击，更不用说对未成年人的欺骗与煽动了！<br/><br/>第五步：煽风点火<br/><br/>拍拍手，轻点鼠标，看着无数不知情的人被蒙在鼓里，暗暗窃笑的同时，不忘发几句“兄弟们、跟我冲”。<br/><br/>作为一名理科生，我几乎患上了逻辑推理的绝症，忍不住想再跟澎湃新闻理论理论。<br/><br/>先不说那“五根针”和“三月、五月”了，我相信以两位记者能造出如此文章的高水平，定是笔误！我最感兴趣的是那位准备用三个月写书的盟友，他太有魄力了！不知大家有没有见过这种人，反正孤陋寡闻的我是闻所未闻，见所未见。试问哪一个高三学生临近高考认为自己能考上大学了便不学习而去写书了？这是已达到清华北大哈佛牛津的水平，还是准备成为韩寒第二？无论作者怎样加工，这里也显得很突兀，我建议两位记者还是把这段删了为好。<br/><br/>再者，这两位记者是文科的高材生吧？以点带面用的炉火纯青，几个从网戒中心未改变好的盟友，你一言我一语，便三人成虎了，用这种<b>定向的抽样调查，总能得到自己想要的结果，却无法反应真实的概率，如此看来，像我这样的盟友是永远不会被你们采访到了。</b><br/><br/>当然，鲁迅先生的“拿来主义”也是屡试不爽，用的那几张网上已烂大街的图片和一段不知从哪个影片下载来的视频，便装模作样成了罪行的“铁证”，看来先生的“人血馒头”二位也应吃得津津有味吧！<br/><br/>细细品味整篇文章，虽然略有疵病，但也不仅暗暗称奇，抛开各种自圆其说，单就文章的主观倾向和感染力，便不是一般人所能做到的，不是一般人所能造出的假。<br/><br/>文章虽精彩，但是，我想请问二位记者，你们的爱国情怀和道德底线在何处！<br/><br/>泱泱华夏，源远流长。无数朝代更替，由盛到衰，需“资前人之鉴，以为明镜”。而二位的行为又与封建王朝时的宦官乱权有何不同？为了自己的私人目的，不顾社会和国家利益，凭借自己的手段和话语权，进献谗言，蒙蔽国人，毁一国之社稷！<br/><br/>青少年当居安思危。现今的祖国无论从经济实力还是政治地位，可以说已跻身世界前列，靠的是党的正确领导和无数先辈的努力，可当我们这一代青少年即将成为祖国的支柱时，情况如何，谁也难以预料。我不是一个悲观的人，也相信未来充满光明。可是，<b>看看我们的身边吧。学校中有多少抽烟喝酒、好勇斗狠、不学无术的问题学生？社会上有多少放弃学业、游手好闲、浑浑噩噩的问题青年？我们身边又有多少价值扭曲、一点即炸的游走于法律边缘的安全隐患？最保守的估计，问题青少年应该占十分之一。这看似微不足道的比例乘上中国庞大的人口基数，是一个令人咋舌的数字！这样的青少年，像活在一个没有约束的世界，就和以前的我一样，父母管不了，学校管不了，少管所进不去，监狱里更暂时没有我们的位置，难道要任由这样的问题孩子自生自灭么？临沂网戒中心存在的意义之一，便是解决这个“尴尬”的社会问题。</b><br/><br/>当两位记者高举“保护未成年人”的旗帜时，我不禁嗤笑一声，难道放任问题青少年随心所欲，便是“保护”？现在孩子的思想可“成熟”得很呢。<br/><br/><b>两位口中所要保护的“未成年人”，他们看着美国大片同时高呼反美帝国主义，尽情于日制游戏、动漫却高喊抵制日货，屋里挂满了韩国明星的海报还要推崇“限韩令”。2017年的国际局势风起云涌，各类“黑天鹅”事件纷至沓来，倘若真有一天爆发战争，那些网瘾患者、问题少年，有几个人能坚守阵地，又有几个人不会临阵脱逃？当然二位记者是肯定不会的，因为你们本就是敌方的伪军！</b><br/><br/>从中心出院至今，已有半年多光景，经历了炒作风波，又再次回归学校生活，发现了太多太多令人纠心的社会问题，也可能是我杞人忧天，但千里之堤溃于蚁穴，青蛙总是在温水中被煮死的。最后，我想说一段课本上的话：公民应依法行使政治权利，自觉履行政治义务，遵守宪法和法律，坚持实事求是的原则，正确行使监督权，否则，国家的民主与法治将再次受到创伤。<br/><p
