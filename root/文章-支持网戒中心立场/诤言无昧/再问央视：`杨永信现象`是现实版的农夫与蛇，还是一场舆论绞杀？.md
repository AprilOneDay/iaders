这个概念的真伪，我们也不反对你去探究问题孩子背后的教育失败，可问题的关键在于，在相当长的一段时间内，社会上早已客观存在这样的一个特殊群体：“学校往外撵，父母无法管，监狱不能留，痴迷网络游戏与学校家庭决裂早恋同居自卑自闭离家出走浪迹街头浑身刺青寻衅滋事涉赌涉黑涉黄人见人怕的问题少年”。<br/><br/>这些被社会遗弃又让人产生恐惧的孩子，接下来他们会走向何方？下一步会是什么结局？有一个答案是明确的：他们不会是维护社会和谐的力量，甚至相反；他们不会是社会经济文化发展的中坚，甚至相反；他们最大的可能，就是报复社会，走向毁灭！<br/><br/>谁来拯救他们？<br/><br/>当然，最割舍不下的，就是他们的生身父母。他们又会是一种什么心境？送给学校？他们已经被撵出来了；劝其回家？绝无可能；送进监狱，舍不得！说服教育他们，但他们已经用仇视、用凶器面对父母与老师了！他们已经叛逆扭曲极端到拒绝任何的教育与感化的地步！<br/><br/>你应该想象，作为家长，什么叫生不如死！<br/><br/>这个时候，你再去纠结于网瘾的存在与否，你再去追究探析夸夸其谈家庭教育的失败，管用吗？<br/><br/>但是，就是有这样一个敢吃螃蟹的人，承受巨大风险并用一腔热情与担当改变了或正在改变着他们！――这就是杨永信，这就是杨永信存在的价值。<br/><br/>在学校之外，在家庭之外，在离监狱最近的地方，他成立了临沂网络成瘾戒治中心。说是戒治网瘾，这只是杨永信迎合社会流行看法的一个牌子，更确切的说法，应该是性格缺陷矫治、问题少年救治，或人格重塑和社会功能康复。在这里，游走于毁灭边缘或者已经涉嫌犯罪的什么问题孩子都有。<br/><br/>十年来，尤其是经过2009年那场舆论炒作，杨永信不是不明白自己选择这条道路的风险与艰辛，他感动于父母的眼泪，感动于父母的泣血求助，这是一种在社会期盼中，在政府支持下的一种使命担当。十年来，他与同事们一道，栉风沐雨，构建起100余环节的网瘾戒治和问题少年救治模式。他的心理点评课充满了辩证法的智慧，充盈着与社会主义核心价值观一致的积极向上的力量，让无数浪子回头，让众多濒临毁灭的家庭重拾幸福。<br/><br/>但他有一个最大的遗憾，就是至今没有找出一种更科学更理想的方法与手段取代目前大中小型医院和家庭普遍使用的康复理疗产品――低频脉冲治疗仪，让那些充满暴力倾向极端狂躁的孩子能冷静地接受教育疏导，以摆脱长期以来备受别有用心之徒夸大扭曲中伤诟病的所谓“电击”！<br/><br/>今天的中国教育现状，无论是大中小学，总有那么一部分学生难于管教，对抗老师，仇视学校，报复同学。锤杀同学的马加爵，投毒害命的xxx,……,无论我们对教育如何反思纠正，这种现象过去就有，似乎未来也很难消失，但没有听说谁会因此而否定学校存在的合理性，而死在手术台上的医疗事故更是多到早已被人视为常态。<br/><br/>试问？与学校和医院相比，杨永信收治的是一群什么样的孩子？在这个充满危险的群体中，有那么一少部分没改变好甚至拒绝改变的少年，不是太正常了吗？<br/><br/>我们为什么对杨永信成功改变的数以千计的孩子故意忽略，而去抓住那几个个案和所谓的 “电击”死死不放？甚至绑架“道义”肆意中伤，去否定杨永信的存在？！<br/><br/>是这个社会缺少包容，还是国民浮躁的心态，还是险恶的利益攸关者在用“道义”绑架舆论，强奸民意？！或者是背后那根见不得人的利益链在进行一场蓄谋已久的舆论绞杀？！<br/><br/>究竟是谁在导演一场现实版的农夫与蛇？究竟是谁在被不良舆论诱导下扮演那条恩将仇报的蛇？不应该是我们的媒体，不应该是我们的民众，更不应该是曾经被杨永信施以热诚救治而因为种种因素不成功的个别孩子或家长。<br/><br/>确切一点说，这更像是一场在话语霸权的优势下蓄谋已久的舆论绞杀。<br/><br/>扪心自问，苍天在上，但愿我们的良心不会在某一天会受到审判。<br/><br/>我们也热切期盼，那些拒绝改变提前离院的孩子多回忆杨永信在点评课上对你倾心掏肺的呼唤，和在你拒绝改变执意离院时对你的善意规劝、热情挽留和后果警示，早一天回头，融入社会，明辨是非，不要去充当不良媒体误导下的炮灰。<br/><br/>我们更热切期盼，作为媒体人，要深明社会道义，恪守职业操守，不要盲目跟风，在不知不觉中去毁灭一个利国利民的正义事业，去毁灭一个在救治问题孩子和问题家庭领域不懈探索并取得重大成就的杨永信。<br/><br/>正义不灭，良心永恒，真相总会大白于天下！<br/>首次转载时间:
