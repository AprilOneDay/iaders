<br/><p style="text-align: center;">网瘾孩子家长?<wbr />小阳爸爸</p><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><br/><div id="sina_keyword_ad_area2" class="articalContent   "><br/><br/>?<wbr />?<wbr />?<wbr />?提起杨永信，在2008年7月4日以前，知道的人可能不多，但这之后，其名气却与日俱增，势不可挡，这一切都源于百度“杨永信贴吧”。至今为止，该吧回帖数已达62万多条。如此高的增长速度本该令人羡慕，可遗憾的是，这“名气”却与“骂名”连在一起。那么，杨永信到底是怎样一个“恶人”，竟招来如此众多的骂声呢？<br/><br/><wbr />?<wbr />?<wbr />?杨永信，山东临沂市第四人民医院副院长，兼网络成瘾戒治中心主任，主任医师，享受国务院政府特殊津贴专家，从事心理精神卫生工作近30年，积累了丰富的临床经验，他的出名是源于网络成瘾的戒治。数年前，他在坐诊过程中发现，很多青少年因痴迷网络游戏导致其心理、行为改变，表现为逃学厌学、孤独自闭、离家出走、打架斗殴、仇视父母、性格扭曲、人性尽失，以至于学业荒废、青春迷失，给父母带来了伤害，给家庭带来了灾难，给社会带来了不稳定因素。作为一名心理卫生工作者，他深感自己责任的重大。他说，成功救治一个孩子，就是挽救了一个或多个家庭，乃至在一定程度上消除了社会不安全隐患，是一件利国利民、功德无量的好事。从2004年始，他就有意识探索戒治网瘾的方法，并经过一番努力，于2006年1月成立了临沂网戒中心，正式开始对网瘾进行专业化、科学化、规范化、模式化、特色化的诊疗。随着时间的推移，经验的积累，和数千例网瘾孩子的成功救治，他总结出了一套全方位，多角度，以家长与孩子一同改变为特色，以长效机制为疗效保障，“心理+药物+物理+工娱”四种方法并用，贯穿78个环节的网络成瘾戒治模式，并被家长们冠以“永信模式”。<br/><br/><wbr />?<wbr />?<wbr />?杨永信认为，网瘾患者问题不在网瘾本身，其根本原因在于患者的认知偏差、性格缺陷、不恰当养育方式和社会不良环境等，基于这一观点，他避开了以戒网瘾而戒网瘾的误区（这也是许多机构或专家戒治网瘾成功率低下的主要原因），而从矫治性格缺陷入手，以“心理+药物+物理+工娱”之综合治疗模式来破解网瘾这一世界性难题。<br/><br/><wbr />?<wbr />?<wbr />?一、心理点评是整个模式的灵魂性环节。心理点评室被家长们看作创造了无数奇迹的地方，在这里，点评师、家长、孩子共聚一堂，相互影响，相互带动，相互激励，知无不言，言无不尽，是打开孩子心结的“心理手术室”，是化解孩子心理痼疾的“中药汤锅”，点评师专业而睿智的引导性语言就是彰显奇效的灵丹妙药。在这里，通过点评师的巧妙诱导，孩子们积压心底的各种隐私，都能和盘托出，而这些暗不见光的隐私则如同毒蛇或魔鬼，长期噬咬或扭曲着孩子的心灵，是导致孩子走偏的罪魁祸首，隐私说完了，心结打开了，心里亮堂了，阳光和微笑立马就写在了孩子的脸上，父母亲眼目睹着孩子奇迹般的变化，唯有啧啧称奇，感叹不已。杨永信常讲，我不是神仙，之所以屡见奇效，应该归功于大家共同营造的充满博爱、感动和积极向上的良好氛围，若让我一对一给孩子治病，我也比他人强不了多少。由此可见心理点评是“永信模式”的核心和灵魂。<br/><br/><wbr />?<wbr />?<wbr />?二、药物治疗必不可少。药物作用与心理点评一脉相承。通过一定时间小剂量中、西药物的药理作用，逐步改善或缓解了网瘾患者因长期上网而造成的焦虑、抑郁、恐惧、狂躁等情绪，和心理点评相得益彰。<br/><br/><wbr />?<wbr />?<wbr />?三、由低频脉冲治疗仪实现的心理导入治疗，能够快速稳定情绪并得以实现有效的心理沟通，是一种基础性、辅助性治疗手段。该仪器作为一种康复理疗保健仪器被大中小型医院和家庭广泛使用。<br/><br/><wbr />?<wbr />?<wbr />?四、工娱和感悟类活动也是“永信模式”的重要组成部分。通过各种活动，如早操晨练、军训拓展、亲近自然、角色体验、情景再现、感悟人生、郊游野炊、歌咏联欢、下棋打球等，以愉悦孩子身心，培养健康兴趣，适应社会环境，塑造健全人格。<br/><br/><wbr />?<wbr />?<wbr />?以上四种方法，78个环节，环环相扣，相互影响，相互作用，构建了一个完善而安全高效的治疗体系，极大地提高了网瘾戒治的成功率和精品率。<br/><br/><wbr />?<wbr />?<wbr />?临沂网戒中心的迅猛发展，得益于杨永信及其团队能够与时俱进，创造性地将各种有效方法应用于网瘾治疗。他集中医、西医、心理、物理、体能、娱乐于一体，极高的人气和90％以上的治愈率，使其名声远扬，一时间，患者家长奔走相告，人满为患。2009年最高峰时，在院人数达180人左右，整整占了3个半楼层。临沂网戒中心无意之中成了国内网戒行业的一面旗帜，各种媒体争相报道，山东卫视、三联周刊、东方早报、新华社、中央电视台等众多媒体纷至沓来。<br/><br/><wbr />?<wbr />?<wbr />?木秀于林，风必摧之。就在中央电视台宣传临沂网戒中心七集系列节目《战网魔》第四集《谁把天才变成了魔兽》播出的第二天，也就是2008年7月4日，百度贴吧里突然出现了一个“杨永信贴吧”，对杨永信疯狂地进行污辱、谩骂、歪曲和诽谤，对临沂网戒中心完整的78个环节的治疗体系，断章取义，攻其一点，不及其余，把普遍应用于各大医院精神科的“低剂量电刺激”，夸大成令人恐怖的“电击”，并以此为突破口，凭借一个曾在临沂网戒中心只住了11天，没有改变好的吴某某的《心理日记》进行大肆渲染，把临沂网戒中心说成是“集中营”、“受刑房”，把杨永信说成是“魔鬼”、“法西斯”。短短几周时间内，“杨永信贴吧”的内容迅速蔓延到成百上千个论坛里面，帖子数量和回帖数迅速攀升至几十万。同时，《中国杨永信网络成瘾戒治中心网站》屡屡被黑，《临沂网戒家长博客》被恶意篡改，《反杨博客》、《电休克博客》、《反杨联盟》、《杀羊用牛刀QQ群》等不期而至，恶搞杨永信的游戏、视频和电影被疯狂转载和点击。就这样，杨永信及其临沂网戒中心在网上被负面消息炒得天翻地覆。那么，在这么短的时间内就造成如此大的影响，该是如何运作的呢？<br/><br/><wbr />?<wbr />?<wbr />?不知您是否听过“网络黑社会”这个名词，它俗称叫“<a href="http://baike.baidu.com/view/186895.htm" target="_blank" rel="noopener">网络推手</a>”、“网络打手”、“发帖水军”，也叫“网络公关公司”、“网络营销公司”。之所以称其“黑社会”，主要在于，它们不仅能为客户提供<a href="http://baike.baidu.com/view/2408213.htm" target="_blank" rel="noopener">品牌炒作</a>、产品营销、口碑维护、危机公关等服务，更能按客户指令进行密集发帖，诋毁、诽谤竞争对手，使其无法正常运营，甚至控制舆论，左右法院判决。国内一批著名企业，如<a href="http://baike.baidu.com/view/30188.htm" target="_blank" rel="noopener">新东方</a>、<a href="http://baike.baidu.com/view/338652.htm" target="_blank" rel="noopener">万科</a>、<a href="http://baike.baidu.com/view/79757.htm" target="_blank" rel="noopener">康师傅</a>等等，都曾遭遇过“网络黑社会”的毒手。他们的运作流程一般是接受订单―分析心理―制作帖子―雇用“水军”―密集发帖。接受订单时，只要客户付得起费用，他们根本不管什么法律、道德，会完全按照客户指令去做。为了取得预期的炒作效果，他们往往会利用网民仇富、同情弱者等心理制作帖子。每个帖子还要有一定的错别字，以增加其真实性。雇用的“水军”一般是大学生、残疾人或闲散人员，100人为1组，公司中1人负责10组，通常掌握五、六十个水军小组。因发一贴所需费用多为0.5元，故又称水军为五毛党。有的公司掌握着50多万个网络<a href="http://baike.baidu.com/view/4394.htm" target="_blank" rel="noopener">论坛</a>地址，很容易让一张帖子出现在数千个论坛中，形成集束效应。<br/><br/><wbr />?<wbr />?<wbr />?看了上面的介绍，相信明智的人一定会对网上疯炒的杨永信及其临沂网戒中心的负面消息打上一个问号。如果真的是网络黑社会所为，那么，究竟是什么人或什么机构，出于什么目的，雇用这些“网络黑社会”的呢？<br/><br/>稍有头脑的人不用猜也能知道，这种力度、这种规模的恶意攻击和炒作，如果不是有实力的机构花大价钱在背后推动，某一个个体是绝对做不到的。那么杨永信及其临沂网戒中心到底动了谁的奶酪，断了谁的财路？怎么那么巧合，就在《战网魔・谁把天才变成了魔兽》播出的第二天就被人注册了杨永信贴吧呢？<br/><br/><wbr />?<wbr />?<wbr />?如果仅仅局限于贴吧，倒也无妨大局，更令人匪夷所思的是，随着贴吧炒作的步步升温，伴随着中国杨永信网络成瘾戒治中心网站论坛莫明其妙被黑，个别传统媒体竟也相信贴吧里这些胡编乱造、空穴来风、无中生有的骂街式的内容，随便剪接拚凑成章发文，以语不惊人死不休的醒目标题抓取读者的眼球，如《信息时报》2009年4月8日A20版刊载的，记者张志超、何剑辉的署名文章，主标题为《戒瘾过程如在集中营受刑》，《都市时报》2009年4月22日A22版刊登了记者李国豪的署名报道，题为《杨永信网络成瘾戒治中心网站论坛被黑，网友质疑――电击、殴打、下跪能治网瘾？》。随后2009年5月7日，《中国青年报》发表了三篇与临沂网戒中心有关的文章，题目分别是《一个网戒中心的生态系统》、《谁都想在网瘾治疗市场分杯羹》、《“戒网专家”杨永信电击治网瘾引发争议》。事情远非到此为止，更令人震惊的事情还在后面：7月8日卫生部在强大的舆论压力下颇为无奈的叫停“电击”治疗网瘾，通知称：“我部组织专家就“网瘾”诊疗相关问题进行了充分研究和论证。专家一致认为，电刺激（或电休克）治疗网瘾技术的安全性、有效性尚不确切，国内外并无相关临床研究和循证医学依据，暂不宜常规应用于临床。”从而也使得对这件事情的炒作达到了顶点；8月15日晚上9点半，在2007、2008年度曾经以16集专题片报道临沂网戒中心戒网瘾事迹的中央电视台，播出了柴静主持采访的专题片《新闻调查・网瘾之戒》，此节目一播出更是将杨永信和临沂网戒中心推入了争论和质疑的漩涡。在整个过程中，“杨永信贴吧”更是推波助澜，回帖数从10万上升到30万、60万……。<br/><br/><wbr />?<wbr />?<wbr />?但是，多行不义必自毙，最近蒙牛和伊利的恶性竞争，已经体现了法律的公正性。据报道，2010年 10月20日，呼和浩特警方证实，蒙牛高管因涉嫌在网上策划诋毁伊利产品已被正式批捕，负责策划的网络公关公司老总和一批高管也被批捕。蒙牛的高管想打击竞争对手，最终却偷鸡不成蚀把米。而炒作杨永信及临沂网戒中心，非但没达到预期目的，而更让他们没有想到的是，引发了全社会对淫秽、色情及低俗网络游戏毒害青少年的高度重视，使他们成为过街老鼠，人人喊打，文化部也针对当前网络游戏管理中存在的突出问题，专门下发了通知，从严格主体准入、改进内容审查、加强市场监管、建立自我约束机制等方面提出了加强和改进网络游戏管理的措施，对网络游戏进行了严厉打击整治，对网络游戏中含有的成瘾性、不良诱导性等内容进行严格管理，一向以犀利、敏锐、坚定和正义著称的著名主持人柴静因为《网瘾之戒》的不客观、不公正，也遭遇了网瘾孩子家长及其社会各界的空前质疑，许多为炒作新闻卖点，不顾客观事实，不惜出卖道德、良知，极尽歪曲、夸张之能事，或一味盲目跟风的媒体，其公信力也遭遇了空前的危机，正所谓搬起石头砸了自己的脚！<br/><br/><wbr />?<wbr />?<wbr />?要将一个充满社会责任感的人打入地狱，要将一个利国利民的机构打垮没那么容易，因为这里是患者家长的希望，是病患孩子重生的摇篮；这里有治愈的孩子不断出院，就像种子一样播撒在全国各地，成了口口相传的免费广告；这里有每周返院回来分享的精品，给在院的盟友和家长以极大的精神支持；这里有医务人员、同盟家委和同盟班委的密切配合和空前团结，积极开展家园自救，形成了一股强大的正向力量和相亲相爱的大氛围；这里有全国各地赶来的孩子家长参观考察，比起网上的谣传，他们更相信口口相传的疗效，更相信亲眼所看到的事实。存在即有其合理性，相信不久的将来，真正有良知、有责任感的媒体一定会还原杨永信及其临沂网戒中心事实真相，网戒行业的春天不会太远！<br/><br/></div><br/><div
