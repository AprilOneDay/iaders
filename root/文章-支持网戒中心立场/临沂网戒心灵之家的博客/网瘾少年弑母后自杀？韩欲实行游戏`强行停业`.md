<br/><p align="center">来源：中青在线 2010年11月23日16:47</p><br/><p align="center">　　本报特约记者　强恩芳</p><br/>http://blog.sina.com.cn/jiazhangtongmeng<!--more--><br/><br/>近日在韩国发生了一起震惊社会的少年弑母自杀案，究其原因，还是网络游戏惹的祸。2010年韩国宽带覆盖率达到100％。曾被视为又一个“汉江奇迹”的韩国互联网事业，近年来也给社会带来不少问题，其中最突出的便是被媒体形象地称为“游戏中毒”的青少年网瘾现象。<br/><br/><b>网瘾少年对母亲痛下毒手</b><br/><br/>据韩国《中央日报》近日报道，11月16日上午7点左右，家住釜山市的11岁李姓少女发现其母金某（43岁）死在卧室的床上。警方赶到现场后，又在阳台上发现了死者15岁儿子李某的尸体，经调查证实，这位少年掐死了自己的母亲，后因自责在阳台的燃气排管上用电线上吊自杀了。在李某的电脑旁，警方发现了一张便签纸，上面写着：“妈妈因为我玩游戏责骂了我，我做了无法原谅的事情。对不起。”<br/><br/>据警方透露，这一家人是去年11月从庆尚南道金海市搬到釜山的，李某的父亲从2000年起长期在海外出差，其母金某也因在照相馆工作很晚才能下班回家，两个孩子平时缺乏父母关心。李某从小学五年级开始沉迷电脑游戏，尤其喜好动枪动刀的暴力游戏。为了玩游戏，李某经常不去补习班上课，周末更是玩到凌晨两三点。为此，母子二人常在深夜争吵。去年李某曾三次接受专业机构的“深度游戏中毒”咨询，但成效不大。金某女儿也反映，母亲和哥哥“在前一天晚11时左右吵过架，比平时吵得更凶”。<br/><br/>这起悲剧事件一经公开即在韩国掀起轩然大波，人们在对李某的行为进行批判的同时，更多地反思和重新审视困扰韩国多年的“网络游戏中毒”现象。<br/><br/><b>“</b><b>游戏中毒</b><b>”</b><b>成韩国社会问题</b><br/><br/>在1995年至2000年的金融危机漩涡中，韩国成长为互联网强国，这又是一个韩国人引以为傲的“汉江奇迹”。之后，韩国的互联网继续高速发展，英国牛津大学和思科公司不久前联合发布的调查报告显示，韩国的宽带网络覆盖率从2009年的99．55％提升至100％，宽带网络的下载速度比去年增长了55％，宽带质量连续三年排名全球第一。哈佛大学伯克曼互联网与社会研究中心学者罗伯特?法瑞斯认为，这是由韩国的互联网行业竞争激烈、政府的鼓励政策、网络开放程度高、新技术的应用等因素决定的。<br/><br/>互联网的高速发展也带动了韩国在线游戏产业。2000年以来，韩国一直是在线游戏的主要出口国，获得了全球在线游戏市场1／4的份额。但同时，韩国青少年“游戏中毒”问题也日益严重，一些孩子甚至到了“没有游戏就不能活”的程度。<br/><br/>对于学生来说，“网络中毒”症不仅影响学习成绩，还会扭曲其性格和心理。据韩国国民健康保险公团统计，2009年因“注意力缺乏和过多行动障碍”而进行治疗的患者创纪录地达64066人，比2003年增加了238％。其中，10到14岁年龄段的有29679人，15到19岁的有10207人，分别增加了375％和765．7％，造成这种现象的一个重要原因就是“游戏中毒”。<br/><br/>去年3月，韩国的下一代国民健康机构表示，在过去10年间，韩国的游戏中毒问题呈持续上升趋势。全国大约有200万名互联网“中毒”者，其中100万名是不满19岁的儿童和青少年。报道称，青少年最喜欢的游戏已经由暴力游戏上升到杀人游戏。有神经精神科专家分析说：“对于中毒的孩子们来说，游戏不是娱乐，而是人生的全部。所以有的孩子不能区分现实和虚拟空间，很轻易地使用暴力，触犯法律。”数据显示，韩国每年因沉迷网络游戏导致犯罪的青少年达5000多名。今年以来，韩国已接连发生3起因沉溺网络游戏导致的孩子杀死母亲、父母饿死孩子的事件。<br/><br/><b>韩国欲实行游戏</b><b>“</b><b>强行停业制</b><b>”</b><br/><br/>针对游戏中毒现象及由此引发的悲剧事件，韩国社会开始反思，政府部门首先行动了起来。韩国女性家庭部白喜英部长于事发当日即登门吊唁了死者，并表示，这一事件表明网络游戏中毒加剧了社会的危险性，青少年政策的各主管机关应该感受到这一问题的严重性。<br/><br/>事实上，根据2009年8月23日开始实行的《韩国国家信息化基本法》，在主管网络中毒问题的行政安全部带领下，韩国信息化振兴院、保健福利部、文化观光部等已开始了防止网络中毒方面的合作，展开与网络中毒者进行面谈、制定网络中毒的标准并进行分类、培养治疗网瘾的专门人才等活动。保健福利部自2009年起，就以4年级小学生为对象进行网络中毒检查，并逐步推广到全部小学生，检查出的中毒较重的儿童将接受各所在地区精神保健中心的治疗。<br/><br/>由于韩国实行网络实名制，国会目前正考虑制定相关法律，推行禁止深夜时段向青少年提供游戏服务的“强行停业制”。专家指出，父母也要和子女加强沟通与交流，让孩子得到必要的关爱和关注，降低网络游戏对他们的吸引力。网络游戏中毒是一个社会现象，需要家长、邻居、老师、游戏开发商、当地的志愿者、政府相关部门等共同努力，才能防止由此引发的恶性事件再次上演。<br/><br/>(责任编辑：news1)<br/>首次转载时间:
