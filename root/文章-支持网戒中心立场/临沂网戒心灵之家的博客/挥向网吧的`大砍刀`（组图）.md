/><br/><br/><strong>案件频发!</strong><br/><br/><wbr />?<wbr />?<wbr />?2008年4月1日凌晨1时，燕都金帝城迪生网吧门口，发生一起抢劫案，四名犯罪嫌疑人持砍刀将受害人张某头部和左腿砍伤，抢走手机一部，现金890元。<br/><br/><wbr />?<wbr />?<wbr />?凌晨2时，党家庄村迪生网吧门口发生抢劫案，受害人被嫌疑人叫出网吧后，被砍伤左背，抢走诺基亚手机一部，现金30元。<br/><br/><wbr />?<wbr />?<wbr />?4月5日21时，南高基村金太阳网吧门口，正在上网的萧某被不认识的人叫出网吧，背部被砍伤，抢走手机一部，现金60元，银行卡一张，并被逼说出银行密码。<br/><br/><wbr />?<wbr />?<wbr />?4月7日凌晨1时30分，四名男子来到南高基化机桥附近，将陈某头部砍伤，右腿砍断，抢走手机一部，现金80元。<br/><br/><wbr />?<wbr />?<wbr />?4月12日凌晨2时50分，三名20岁模样的男子手持黑色双刃长刀将正在上网的宋某和王某砍伤。<br/><br/>……<br/><br/><wbr />?<wbr />?<wbr />?今年进入4月份以来，石家庄市新华区辖区连续发生多起持刀抢劫案,较去年同期大幅攀升，从案件特点分析，均系二人以上合伙作案，犯罪嫌疑人手持砍刀、弹簧刀、匕首或铁链子等作案工具先将受害人捅伤、打伤后，抢走手机、现金等财物，还威胁受害人胆敢报警就会一命呜呼。严峻的治安形势引起了网民的极大恐慌，同时也引起了新华分局领导的高度重视。<br/><br/><wbr />?<wbr />?<wbr />?为尽快消除在社会上产生的负面影响，为奥运安保创造良好的社会环境，局长康云良、政委马友林及时召开党委会，分析案情、研究应对措施，并迅速成立由副局长全卫强任组长的专案组，组织刑警大队长赵玉虎、副大队长闫利民、白宗臣及刑警六中队民警在初步掌握嫌疑人体貌特征和并案分析的基础上，展开了针对辖区及周边网吧等部位大规模摸排和侦破工作。<br/><br/><wbr /><br/><br/><strong>突发命案!</strong><br/><br/><wbr />?<wbr />?<wbr />?就在该局专案组部署警力全力侦破系列抢劫案的同时，4月19日凌晨3时许，专案组又接到报案称在党家庄村附近某网吧发生一起持刀抢劫案，三名犯罪嫌疑人手持砍刀在网吧内先后将三名正在上网的男子捅伤，抢走两部手机、现金30余元后迅速逃离现场，其中一名伤者黎某（男，20岁，湖南人）因伤势过重，经医院抢救无效死亡。<br/><br/><wbr />?<wbr />?<wbr />?专案组被震怒了，犯罪嫌疑人竟和警察叫板，在警察介入侦查此案后，竟又发命案？!局长康云良气愤之下把茶几拍得啪啪作响，“一夜之间竟然连续作案6起，一定要不惜一切代价生擒‘大砍刀’……”<br/><br/><wbr />?<wbr />?<wbr />?案情重大，局长康云良果断决策，抽调大案中队、四中队、六中队、八中队等精干警力采取超常措施，强力攻坚，全力侦破此案，彻底摧毁犯罪团伙。根据少数受害人对犯罪嫌疑人体貌特征的描述，专案组分析该团伙成员应该在6至7人左右，从他们的作案手法上看，应该是辍学后在社会上游荡的闲散人员，作事不计后果，年龄应在20岁左右的年青人。专案组根据报案人全部受伤情况分析，该团伙下手狠，都是不由分说先砍再抢，决不久留，流窜作案可能性较大。从作案时间和地点分析，该团伙选择作案地点随机性大，这也给民警破案带来了困难。<br/><br/><wbr />?<wbr />?<wbr />?于是专案组兵分三路，第一路由刑警大队大队长赵玉虎亲自率领，调取所有案发地网吧监控录像，围绕受害人、嫌疑人及其他所队破获的网吧犯罪案件中展开调查，并组织受害人秘密辨认，从中寻找突破口；第二路由刑警大队副大队长白宗臣率领，对案发现场周围群众与网民进行秘密调查，寻找目击证人，从中发现线索，锁定嫌疑人；第三路由副大队长闫利民率领，采取便衣查控、地毯摸排、连夜蹲守等措施，争取抓获犯罪。副局长全卫强同时命令各派出所迅速成立网吧打现行小组，专门在本辖区网吧周边巡逻，以防不测再次发生，一切都在紧锣密鼓地进行着，接连三天时间专案组民警并没有接到网吧抢劫案报警，也没有发现嫌疑人的踪迹，难道犯罪嫌疑人闻着什么味，躲起来不成？专案组民警没有因此掉以轻心。<br/><br/><wbr /><br/><br/><strong>果断出击!</strong><br/><br/><wbr />?<wbr />?<wbr />?但犯罪嫌疑人还是顶风作案，4月25日凌晨2时50分许，四名男子手持砍刀在高柱村一胡同内，将受害人连砍七刀后逃之夭夭。凌晨3时许，专案组民警得到线索，正在联盟路新世纪网吧上网的一男一女体貌特征与嫌疑人极其相似。民警立即向专案组领导汇报，分局副局长全卫强迅速带领六中队中队长马健、副中队长王志强、狄春一以及民警赵文跃、张殿峰、夏勇斌、田克俭等人迅速赶往新世纪网吧，经过周密部署，于当日15时许将犯罪嫌疑人赵男（男，20岁，重庆市永川市玉峰村人）、赵女（女，22岁，平山县回舍镇白塔坡村人）一举抓获。<br/><br/><wbr />?<wbr />?<wbr />?经审讯，两名犯罪嫌疑人赵男、赵女在证据面前交待了4月19日凌晨伙同贾某在党家庄村附近迪生网吧持匕首、砍刀将正在上网的三名男子捅伤，其中赵男手持砍刀向黎某腹部、腿部、后背等处连捅数刀，抢走两部手机及少量现金后迅速逃离现场的犯罪事实，并供述了贾某的暂住地。<br/><br/><wbr /><br/><br/><strong>乘胜追击!</strong><br/><br/><wbr />?<wbr />?<wbr />?办案民警根据嫌疑人提供的情况，于4月25日22时许在北焦村将涉嫌抢劫杀人正欲逃跑的犯罪嫌疑人贾某（男，22岁，邯郸馆陶县人）成功抓获。办案民警乘胜追击，不断加大审讯力度，巧妙运用审讯手段，一举挖出了一个两个月时间疯狂抢劫40余起，涉案价值数万元的特大抢劫团伙，并掌握了该团伙其他5名嫌疑人的基本情况。<br/><br/><wbr />?<wbr />?<wbr />?专案组马不停蹄，连续奋战，经过两天两夜的艰苦工作，至4月26日8时许，抓捕组民警先后在友谊大街、裕华路、小安舍等地附近网吧将该团伙张某（男，18岁，新乐市人）、孙某（男，18岁，湖北省汉川市人）、吕某（女，20岁，邢台市柏乡县人）、王某（男，20岁，四川省遂宁市莲溪县人）、闫某（男，21岁，邯郸馆陶县房寨镇人）5名成员全部抓获。<br/><br/><wbr /><br/><br/><wbr /><br/><br/><strong>真相大白!</strong><br/><br/><wbr />?<wbr />?<wbr />?案件破了，8名涉案人员全部归案。那么，抢劫动机是什么呢？为什么要手持砍刀先将人砍伤后动财呢？这伙人究竟想干什么？这还得从网络游戏说起。<br/><br/><wbr /><br/><br/><strong>受网络游戏的毒害！</strong><br/><br/><wbr />?<wbr />?<wbr />?原来，该团伙成员大多因与父母关系不好从家跑出来，终日沉迷上网，打网络游戏，长期吃住在网吧。相同的爱好和“遭遇”让他们成了好哥们儿。3月下旬以来，为了维持“生计”和支付高额网费，他们受网络游戏中厮杀场景的启发，萌生了抢劫生财之道。8人中的赵男被尊“老大”，“老大”命令手下购买了砍刀、匕首等工具，每当他们“囊中羞涩”时，就会趁夜深人静溜出暂住地--网吧，随意选择一个网吧中正在上网的男子，采用先用刀伤人后抢劫的作法，多次预谋持刀抢劫网吧上网者，抢劫完后拿着“胜利果实”继续上网打游戏。据统计该团伙以同样手段，或两人结伙，或三人成群先后在新华区赵陵铺、联盟路、小安舍、友谊大街以及保定、邯郸等地疯狂抢劫、交叉作案40余起的犯罪事实。仅3月27日至4月26日一个月时间就疯狂作案30余起，而警方接到报警的只有10余起。<br/><br/><wbr />?<wbr />?<wbr />?目前，8名犯罪嫌疑人已被刑事拘留，此案正在进一步审理之中。<br/><br/><wbr /><br/><br/><wbr /><br/><br/><strong>痛定思痛!</strong><br/><br/><wbr />?<wbr />?<wbr />?这个危害一方的抢劫团伙经新华分局民警三十多个昼夜的艰苦奋战一举打掉，案件成功告破，但民警们的心始终十分沉重。近几年来，随着计算机和网络技术的蓬勃发展，网络时代来临了，网络的影响不断扩大，已成为人们不可或缺的工具，上网成了人们最时髦的嗜好，受网络游戏的影响，网吧也就成了青少年经常光顾的场所。尽管有关部门下了很大功夫检查各网吧禁止未成年人上网情况，但不少经营者仍然在利益的驱动下，违背国家法律法规，暗中收留未成年人通宵达旦地上网玩游戏，有的甚至为网民提供“吃、住、玩一条龙”服务，青少年往往沉浸于网吧的“优质”服务和网络游戏不能自拔，哪儿还想着回家。沉溺于网络世界的青少年，荒废了学业，没有了正确的人生观、价值观和世界观，极有可能走上违法犯罪的道路，网吧已成为违法犯罪的集结地。<br/><br/><wbr />?<wbr />?<wbr />?针对本案，民警有三点提示：一是有关部门要加大对网吧的管理力度，对网吧营业时间予以限制，有条件的网吧都要安装监控系统；二是家庭、学校和社会都有责任的义务引导青少年正确看待“网络成瘾症”，学校可以开设网络教育课，对学生进行网络安全防范教育，让青少年认识到网络的双刃性，也可以请民警到学校现身说法，宣传网络法制教育知识；家长不能刻意遏制孩子接触网络的好奇心，应正确引导孩子，适时给予保护。三是网民最好不要在网吧深夜上网，在受到侵害时要及时报警，保护好现场，给警方破案提供有价值的线索。<br/>首次转载时间:
