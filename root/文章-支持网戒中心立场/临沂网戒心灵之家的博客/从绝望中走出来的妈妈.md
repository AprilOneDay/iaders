lang="EN-US">14</span>年前，满怀无限的期盼和喜悦迎来了可爱的儿子，儿子的降生虽然给我带来了肉体上的痛苦，但是更多的是给我们带来了欢乐和幸福，以及儿子给予我们的自豪感。然而，这一切都在<span lang="EN-US">14</span>年后破灭了。网魔迷住了儿子的心智，夺走了我积极阳光的儿子，夺走了我们家庭的幸福，让我承受了精神上的无限痛苦和折磨。一个<span lang="EN-US">14</span>岁的阳光少年竟然放弃学业，沉迷在网络中无法自拔，漠视亲情，对生活失去兴趣，对未来毫无目标、毫无理想。在我们千方百计寻找儿子、唤醒儿子的同时，发现更多类似的孩子，这让我们做父母的痛彻心肺。这是一代人的问题，更是一代人的悲哀，是关系到整个社会的现象。我们曾经试图通过执法机关整顿非法网吧，以及通过媒体威海电视台最受关注的栏目“威海直播”呼吁社会共同努力挽救孩子，但是都没有得到支持，毕竟我们个人的力量实在微薄。我们只能眼睁睁地看着儿子不断沉迷，只能默默地承受儿子将要毁掉的痛苦和无奈。<br/><br/>然而，身为父母我们要对孩子负责，我们不能放弃儿子！不能让网魔毁掉儿子！怀着这种坚定信念，我们苦苦地寻找着、祈求着、盼望着，谁来救救我的儿子！救救我们这个家！<br/><br/>冥冥之中，也许我们的诚意感动了上苍，让我们找到了临沂市网戒中心，找到了我们的救星――杨叔。<br/><br/>来到网戒中心后，儿子和我沐浴在杨叔独创的爱的氛围中。孩子们以老带新，以先进带后进，积极地相互帮助，共同改变；家长们都充满着爱心，将所有孩子看成是自己的孩子，共同爱护；而且从自身做起，改变自己，为孩子们树立榜样。同时，大家共同营造了一个具有极强的凝聚力、向心力的良好氛围；在一堂堂生动感人的心理点评课上，每一位盟友走偏的经历及父母悲痛伤心的泪水震撼着孩子们的心灵；每天的军训课锻炼着孩子们的体质、毅力；贫困生活体验、纠偏操培养信念、感悟人生路、回归自然、母子连心等七十多个环节紧密相扣，让孩子们感悟着一个个人生哲理。儿子也在其中逐步从消极沉迷、目光呆滞、玩世不恭、仇视父母变的积极阳光、充满爱心、懂得感恩。看着儿子一步步的改变，我高兴！我感激！高兴的是那个期盼已久的儿子在向我走来，幸福的家庭在回归！感激的是杨叔，是他拯救了我的儿子，拯救了我们的家庭！杨叔不仅戒除了儿子的网瘾，同时重塑了儿子的灵魂，改变了我们全家人的性格和命运。感谢您――杨叔！<br/><br/>我们敬爱的杨叔以他那特有的博爱之心和人格魅力影响着孩子们，又以渊博的专业知识点拨着孩子们，驱散孩子们心中的乌云，同时以他那无私奉献、自我牺牲的精神感动着孩子们及家长们。杨叔为了拯救这些迷途羔羊煞费苦心、呕心沥血，策划了一场场感人的亲情活动，创立了网戒事业独树一帜的“永信模式”，为网戒事业做出了不朽的贡献。<br/><br/>我们的孩子得到了救治，我们心里高兴。但是，社会上还有多少个孩子仍然沉迷在网游中，还有多少个父母在承受着我们曾经承受过的痛苦，想想这些，我们深感杨叔的伟大以及自己身上应该肩负的责任。因此，我们所有家长有一个共同的心愿，那就是尽我们所能支持杨叔、支持“永信模式”、支持“网戒和谐行”、支持网戒事业。同时由于我们的能力有限，希望社会各界给予我们网戒中心大力的关注和支持，让更多的孩子走出网魔的阴影，让更多的家庭找回幸福。我们一定紧跟杨叔的脚步，共同创建我们美好和谐的社会。<br/><br/><span
