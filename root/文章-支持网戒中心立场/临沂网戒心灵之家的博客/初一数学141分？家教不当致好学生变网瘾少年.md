/>?沙坪坝区初三学生小嘉(化名)整晚通宵上网玩游戏，白天就呼呼大睡。开学已两周，这个15岁男孩，为什么还不能收心，为什么他管不住自己，痴迷于网游？<br/><br/>记者通过走访小嘉本人及其父母、老师、同学，试图还原小嘉从初一到初三，为什么越来越不“听话”的变化过程。<br/><br/><b>初一 数学曾考过141分</b><br/><br/>“他非常聪明，数学成绩非常好，初一时还考过141分呢！”在父亲李永发看来，儿子天生聪颖，理科方面的知识一点就通，尤其在数学方面颇有天赋。但就是这样一个聪明的孩子，进入初中后，很快就开始拒绝完成作业。<br/><br/>李永发认为，不完成作业是因为儿子懒、不听老师的话，为此他不知被班主任庞老师请到学校多少次。每次老师告诉他，小嘉又不完成作业，李永发的处理方式就是当着老师和同学的面，对小嘉一阵怒骂甚至大打出手。<br/><br/>小嘉和父亲关系十分紧张。在庞老师的印象中，有一次，面对不断责骂他的父亲，小嘉突然捡起地上的砖头，狠狠地朝父亲砸过去。这样过激的反应，让学校师生都感到惊异，“这个孩子在班里人缘很好，和老师的关系也比较融洽，但对自己的家人，他的反应是暴躁和不受控制的。”<br/><br/>“他就知道打麻将。”小嘉曾非常不满地向老师和同学提起父亲。对此，李先生满腹委屈。“我10多岁就到重庆打工，从卖小菜，到如今有了自己的鸡鸭店铺，靠的是每天睡4个小时。”对儿子的不理解，李先生气得怒吼，“我挣这些家业是为了谁，我拖出一身毛病，每天到点吃不上饭，胃痛得要命，你还不听话来气我。”李先生怄得眼泪直流，但小嘉神情冷漠。<br/><br/>小嘉说，他不做作业，主要是因为偏科，除了数学能跟得上，语文和英语如听天书。“为了完成作业，我也想了办法的，晚上状态不好做不起，我就调闹钟，半夜三四点起来写。”对小嘉的这些努力，李先生一脸惊诧地表示根本不知情。<br/><br/><b>初二 近视了妈妈骂活该</b><br/><br/>“已经学不走了，那你为什么后来不听课了，在课堂上睡觉？”面对李先生的质问，小嘉突然暴跳如雷，站起来冲着父亲大吼，身体颤抖、表情歇斯底里，“那是因为我近视了，你们不给我配眼镜，一直拖到初三，整整一年我都看不清黑板。”<br/><br/>小嘉愤怒地告诉我们，他的座位在最后一排，初二时眼睛突然近视了，老师的板书一点都看不清。“我告诉妈妈要配眼镜，她却说‘谁让你玩电脑，眼睛近视了活该！’”就因为妈妈的这句话，小嘉不再提配眼镜的事，看不清黑板，干脆就放弃听课，在课堂上睡大觉。<br/><br/>庞老师向记者证实，小嘉说的情况的确存在，直到升上初三，老师了解到他近视后，主动与李先生夫妇沟通，夫妇俩才给了小嘉200元钱，请老师带他去配眼镜。<br/><br/>“在我们班上，像小嘉这样的孩子不少，父母都是进城务工人员，每天忙于生计根本没时间管孩子。”庞老师说，忙碌的父母往往疏于和孩子沟通，又不肯听孩子的意见，很多这种家庭的孩子，往往变得很叛逆，小嘉就是典型的例子。<br/><br/><b>初三 他干脆学都不上了</b><br/><br/>升上初三，小嘉与父母的关系更加恶劣。老师和同学们都发现，父母对小嘉已完全失去约束，父母的训导在他耳中毫无意义，反而表现出他的脾气比父母更大。<br/><br/>“初一时，小嘉埋怨爸爸，和他对嘴、对打，但说起妈妈，他有时会掉眼泪，觉得自己不听话，对不起妈妈。”庞老师说，从初二开始，小嘉对父亲完全失去尊重，李先生说什么，小嘉都跟他对着干。这时，小嘉和母亲的关系也紧张起来，面对妈妈的唠叨，小嘉开始从内疚变得麻木和厌烦。<br/><br/>到了初三，小嘉的父母已完全拿他没办法，管教不听反而激起强烈的对抗情绪，父母于是任其“自由发挥”。“寒假他每晚通宵游戏，已形成习惯，再加上觉得书反正读不进去，上不上学也无所谓。”庞老师说，就这样，小嘉不完成寒假作业，把在课堂上睡觉，干脆变成在家睡觉，越发不再约束自己。<br/><br/><b>现在 下周父母陪他报到</b><br/><br/>小嘉为什么会变成家长管不了的孩子，小嘉父母对小嘉玩网游就没有进行过干涉吗？李先生说，为了让小嘉戒网游，他们曾把家里的网络断掉，把电脑藏起来，但小嘉直接离家出走，每次好几天，不跟任何人联系。<br/><br/>“每次都是发动亲戚朋友去网吧挨家找，我们怕他被坏人带坏，被人教唆去犯罪，不能把他推到社会上去流浪啊！”李先生和妻子想来想去，与其让小嘉在外面混，不如把他留在家里上网，起码不会出事。<br/><br/>就这样，小嘉回家玩游戏、吃饭、睡觉，随心所欲，他曾在床上玩了一个月游戏没离开过屋子。<br/><br/>到上周，表哥看着小嘉越来越沉溺，一气之下砍伤他的手，让他无法上网，谁知小嘉缝合伤口后，继续单手战斗。小嘉的冷漠，让家人伤透了心。<br/><br/>这就是小嘉的故事，直到昨天，小嘉依然在家睡觉拒绝返校。小嘉的父母和他商量后，决定再给他几天时间收心，下周一家人陪他一起去学校报到。重庆晨报<br/>首次转载时间:
