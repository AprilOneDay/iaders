align="left"></p><br/><p align="left">?<wbr />?<wbr />?<wbr />?热闻快读 “我不该打他”，事情已过去多天，曾小华仍不断地重复着这句话。19日中午，曾小华拧着13岁儿子莫康生的耳朵把他从网吧拉回来，训斥并打了儿子两巴掌，然后出去干农活。随后，莫康生在门前的菜地拿起一瓶“敌敌畏”农药服毒身亡。在曾小华眼中，网吧是夺去儿子性命的恶魔，也是夺去全家人希望的刽子手。大亚湾宣教局表示，将联合文化、工商等部门对学校附近网吧进行严格管理，杜绝类似悲剧发生。</p><br/><p align="left">　　儿子突然迷上了上网</p><br/><p align="left">　　46岁的曾小华和丈夫是广西玉林人，膝下有三个儿女，13岁的莫康生排行老二，是家中唯一的男孩。3年前，莫康生随着父母来到大亚湾西区街道办塘横村，父母在该村承包了几亩菜地，以种菜为生。莫康生就在附近的塘横小学就读，今年读四年级。</p><br/><p align="left">　　曾小华说，儿子在学校读书，学习用功，成绩也从一年前的每科平均分60多分上升到80多分。但是，前一段时间儿子突然迷上了上网。“不想做作业，不想学习，花钱多了，成绩也下降了。”曾小华对儿子的变化忧心忡忡。“我原先不知道他迷上了上网，他的表妹有一次对我说，‘表哥在网吧里面呆了一天’。”曾小华得知儿子在上网后，从网吧把儿子拉回家，她心情很糟，但没有打骂儿子。</p><br/><p align="left">　　“我喝了敌敌畏，快打110！”</p><br/><p align="left">　　19日中午，曾小华再次发现儿子中午放学没有直接回家，而是逗留在网吧。曾小华把儿子叫出来，拧着儿子的耳朵。“你这样下去怎么行？”曾小华对儿子的行为极其失望，盛怒之下，她打了儿子两巴掌。“没有用力，轻轻地打了他。”曾小华没想到这两巴掌让母子从此阴阳相隔。“他没有哭，也没有反抗，一切都毫无征兆……”随后，曾小华和丈夫在几公里外的菜地给菜浇水。</p><br/><p align="left">　　曾小华租来的房屋前是一片绿色的菜地，田埂上摆满了农药瓶。21日中午12时许，正在给菜地浇水的邻居贺女士向记者描述了当日的情景。“当时，我正在和老公在菜地干活，突然，康生大声说，‘大叔，我喝了敌敌畏，头晕，快打110’。”贺女士说，此时的莫康生与他们只有几步之遥，身体已摇摇欲坠，随后一头栽倒在田地里。一个灰色的瓶子躺在一边，黑色的农药汩汩地流出。</p><br/><p align="left">　　贺女士见状，立刻与丈夫以及邻居们把莫康生抬起来送往附近的诊所，在途中，莫康生已不能言语，脸色发黑。闻讯的父母亲赶过来，用三轮车把莫康生转送往深圳坪山医院，此时莫已深度昏迷。后经坪山医院近两个小时的抢救之后，莫康生终不治身亡。</p><br/><p align="left">　　宣教局：将教育学生远离网吧</p><br/><p align="left">　　塘横小学校长华红先说，19日中午，他从学生处听闻莫康生服毒自尽，立刻赶往莫康生的家中，而莫康生已被送往深圳救治，随后传来其死讯。</p><br/><p align="left">　　对于莫康生迷上网吧的事情，华红先说，因学生周末不在学校，进网吧上网，学校也无法查证与跟踪，“虽然莫康生的死发生在校外，但学校的监管不力，也有一定的责任”。班主任陈老师说，他知道莫康生喜欢上网，也教育过他，但效果不是很好。大亚湾宣教局局长涂文子说，得知学生因上网被家长教训随后自杀的消息后，他们将加大宣传，让学生远离网吧。该局也将联合文化、工商等部门对学校附近网吧进行严格管理，杜绝类似悲剧发生。</p><br/><p align="left">　　21日，记者从大亚湾公安分局西区派出所了解到，莫康生自杀之后，西区派出所民警也赶到现场了解事件的详情。“他们家属已承认儿子是自杀的，不需要警方进行立案调查，也不需要鉴定。”该负责人的话得到了曾小华的证实，“人都死了，调查还有什么用。”</p><br/><p align="left">?<wbr /></p><br/>懊悔<br/><br/>自责的母亲“我真后悔，不该打他”<br/><br/>我很少打他，有时他调皮，我打他，他会笑嘻嘻地跑开，是一个懂事的孩子。以前，他回到家中就做作业，做完作业之后，就去菜地帮我们干活。<br/><br/>“我真后悔，不该打他。”曾小华头发凌乱，面容憔悴，眼睛一直漠视着前方，讲述着儿子的生前身后事。21日，丈夫把莫康生的遗体运到惠州市殡仪馆火化后，回到家中，把半瓶“敌敌畏”狠狠地摔在地上，然后坐上了回老家的列车准备把儿子的骨灰送往老家，记者联系到他时，他已经到了半途中。<br/><br/>“大女儿已出嫁，小女儿已送给他人抚养，我唯一的希望就在儿子身上。”曾小华把头埋在膝盖上，身体不停地颤抖，掩饰着泪水。“为了让儿子读书，我们才来到这里种菜，大部分收入都给儿子作为学费和生活费。”言语中，曾小华无法掩饰对儿子的疼爱，“他从小多病，经历了很多的磨难，我对他格外关爱。”曾小华说，儿子与父母感情融洽，父母极少辱骂和殴打儿子。<br/><br/>三年前，一家人来到大亚湾，蜗居在一间几十平米的小房子里面，还是和他人合租，每月40元的租金。即使是大白天，房间也暗无天日。几平米的卧室，两张床，其中一张是莫康生生前的，房间堆满了杂物，一个破旧的书包孤独地躺在一旁。房间里没有一张莫康生的照片，“没想到他走得这么匆忙，一张全家福也没来得及照。”<br/><br/>“我很少打他，有时他调皮，我打他，他会笑嘻嘻地跑开。”在曾小华的眼中，莫康生是一个懂事的孩子“以前，他回到家中就做作业，做完作业之后，就去菜地帮我们干活。”曾小华反复念叨着，犹如鲁迅笔下的祥林嫂，“我真不该打他，早知道，我连骂都不会骂他……”<br/><br/>愤怒的母亲“网吧，该死的网吧！”<br/><br/>自从儿子迷上网吧之后，就如同变了一个人，花的钱多了，回家的时间晚了，农活也不插手了，作业一拖再拖。回家就要钱，不给，他就不高兴。<br/><br/>21日下午，塘横小学附近，记者看到，一间网吧正在开放，狭小的空间里面挤满了一群少年，那里曾经是莫康生生前经常出没的地方。小学大门紧锁，无保安看守。<br/><br/>“网吧，该死的网吧！”曾小华说这句话的时候，眼中含着怒火。在曾小华眼中，网吧是万恶之源，是夺去儿子性命的恶魔，也是夺去全家人希望的刽子手。<br/><br/>曾小华说，自从儿子迷上网吧之后，就如同变了一个人，花的钱多了，回家的时间晚了，农活也不插手了，作业一拖再拖。“天黑了，回到家，不做作业，就要钱。我不给，他就不高兴。”曾小华对儿子的行为大失所望。“我不断地教育他远离网吧，但没什么效果。”<br/><br/>“我告诉他，好好学习，以后考上大学，我给他买一台电脑，他天天可以上网。”曾小华绞尽脑汁地说服儿子，也收效甚微。“但我们白天要在外面干活，根本没时间管他”，曾小华认为自己的管教疏忽也是酿成悲剧的原因之一。<br/><br/>“希望网吧都关掉，不要让它们再毒害孩子了。”对于网吧，曾小华深恶痛绝。<br/><br/>他们眼中的小华<br/><br/>老师<br/><br/>他是一个自尊心很强的孩子<br/><br/>21日下午，记者进入了塘横小学。在该校四年级一班墙上的座位表上，莫康生的名字赫然在列，身材高大的他坐在最后一排，但桌椅已被搬走。几名小学生正在打扫卫生，他们已得知自己的同学离去，但在老师的叮嘱下，他们对莫康生的死仍三缄其口。不过，一名男同学还是偷偷地告诉记者，莫康生是个性格外向、懂事的学生，平时喜欢玩游戏，喜欢上网。<br/><br/>“他(莫康生)身体不好，有淋巴病。不过，学习一直不错，而且很懂事。”华红先评价莫康生。莫康生的班主任陈老师说，莫康生在学校很懂事，学习比较勤奋，成绩也不错。“他性格外向，与同学相处融洽。而且，他的自尊心很强，老师批评他时，他就会低下头，表示不会再犯类似的错误。”陈老师说，莫从未有过异常的举动。“一切都在预料之外，或许他只是为了吓唬一下父母，没有想到送掉了性命。”陈老师对于学生的死深感意外和惋惜。<br/><br/>邻居他是一个很懂事的孩子<br/><br/>曾小华一家居住的房子是一座住着6户人家的集体出租屋，他们在一起生活了3年。曾小华的邻居胡先生说，莫康生是一个很懂事的孩子，很有礼貌，而且很勤劳，经常帮助家人干农活。“他很少与我们说话，但在我们的印象中，这个高大的男孩很外向，很懂事，没有反常的举动，父母亲也很少打他。”邻居贺女士说。<br/><br/>和莫康生一起生活了3年的16岁男孩胡伟强说，“虽然我经常和他吵架，最近和他也有矛盾，但是，我还是觉得他是一个不错的学生，很懂事。”<br/><br/>专家支招<br/><br/>家长要了解<br/><br/>孩子的内心世界<br/><br/>对于莫康生自杀的事件，惠州蓝天心理科学研究中心首席心理指导专家肖永柏认为，莫康生自杀的原因可能是因为与家人缺乏沟通，而且，父母不懂得因材施教，不知道外向开朗的他可能内心很脆弱。周永柏建议，现代社会，“棍棒下出秀才”的方式已不合时宜，如今的青少年有个性且信息来源广，不可能所有想法都与家人一致。因此，家长要善于了解孩子的内心，和孩子成为好朋友。<br/><br/>“家长要让孩子把自己的想法说出来，而不是憋在心里，无处发泄，以致酿成悲剧。”肖永柏说。<br/>首次转载时间:
