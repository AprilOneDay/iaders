/>?游戏口角引发现实血案<br/><br/>6月19日凌晨零时许，昆明市闹市区的一家网吧前，17岁的少年李英杰身中十数刀，倒在血泊中。当警察赶到时，李英杰已经死亡。<br/><br/>这件命案不难侦破，昆明市公安局五华刑侦大队四中队的办案民警很快就锁定了犯罪嫌疑人。案发后第三天，犯罪嫌疑人施应昌投案自首。7月11日，另两名嫌疑人张雨和谢海在浙江被当地警方抓获。<br/><br/>令警察们感到不可思议的是3名嫌疑人的犯罪动机：<br/><br/>根据警方记录，发生冲突的双方分属网络游戏“劲舞团”的不同“家族”。6月18日,由于多次遭到“流氓家族”的玩家蔡天星在游戏和电话中的辱骂，“昆明联盟”玩家施应昌和张雨“实在咽不下这口气”了，打算“给他点颜色看看”。他们打听到蔡天星正在某网吧上网，便叫上“兄弟”谢海径直闯了进去。正要动手时，发现该网吧安装了摄像头，因此决定换个地点。<br/><br/>一番计议后，施应昌通过其他玩家约对方到几百米外的另一网吧见面。晚上12点多，当蔡天星等4人在网吧遍寻不见正欲离去时，守候在网吧外的张雨和谢海上前围住他们并谩骂起来，几句话之后，便掏出了刀子。蔡天星与其他两名玩家见状逃跑，只有李英杰逃脱不及，冤魂西归。<br/><br/>而在此之前，除了熟知网名外，3名犯罪嫌疑人在现实生活中从未见过李英杰。<br/><br/>“网瘾少年将虚拟世界中的恩怨转移到现实生活中来解决，这类事件越来越多。”办案警官姜华说，“但他们一般私下解决，伤得再重也不惊动警察。一出事就是出大事。”<br/><br/>“家族利益”至高无上<br/><br/>与一些以暴力著称的网络游戏不同，“劲舞团”崇尚的是炫目的“舞技”。玩家通过键盘操纵虚拟人物扭腰踢腿，与别人比“舞姿”。在这款游戏中，分设了不同的频道和“家族”，同一“家族”的玩家在虚拟世界中称兄道弟，“同仇敌忾”。<br/><br/>犯罪嫌疑人施应昌说，虽然“劲舞团”本身并没有打打杀杀，但是“跳”得好看，在圈子里很有面子，“跳”得不好就会被嘲笑。在游戏圈中，嘲笑技术差的玩家叫做“踩人”。一个人被“踩”，丢的是整个“家族”的脸。<br/><br/>在这个强调“有福同享、有难同当、有仇必报”的江湖义气的虚拟世界里，所谓“家族利益”被放大到至高无上的地位。“昆明联盟”和“流氓家族”成员之间积怨已久，多次聚众打架，也动过刀，幸未酿成大祸。<br/><br/>今年6月以来，“流氓家族”中的玩家蔡天星自恃游戏装备好、技术高，屡屡通过网络服务器里的“小喇叭”，对“昆明联盟”的成员进行辱骂。他不仅“跳”得好看，而且打字也快，施应昌和张雨都骂不赢他。<br/><br/>施应昌说：“蔡天星不但在游戏里‘踩’我们，还经常打电话来骂我们，说在现实生活中也要找人打我们。他骂的脏话让我们无法忍受。”<br/><br/>年龄最小的犯罪嫌疑人谢海并没有与蔡天星直接打过交道，但是，“一个‘家族’就是一个整体，‘流氓家族’‘踩’我们的话，我们就要反抗。”为了“哥们儿义气”，他毫不犹豫地找了一把刀，跟着两个“兄弟”一起去“教训”蔡天星。<br/><br/>“我们等在网吧外面，他们出来以后，我们就互相对骂，用脏话骂。骂了几句话以后，就动手了。”<br/><br/>“当时我心里也犹豫过，但是我看见张雨动手了，如果我不动手，不知道以后怎么面对他。”<br/><br/>对同伴的逝去表现冷漠<br/><br/>谢海当时没有意识到，唯一真正见过蔡天星的施应昌并没有参与这场斗殴，他甚至没有带刀。当张雨和谢海挥刀砍向李英杰时，施应昌坐在一辆出租车里等着他的同伴，对此，他始终无法自圆其说。3人逃到贵阳以后，施应昌决定回来自首，便悄悄地一个人回到昆明。张雨和谢海至今不知道他为何不告而别。<br/><br/>“他的行为恰好为他们的所谓‘义气’写了个注脚。”一名警官说。<br/><br/>警察们无法理解，“家族利益”至上的玩家们在一名同伴逝去后表现出来的冷漠。办案警官周磊说：“当时网吧里有双方的人在玩游戏，他们知道外面发生了什么，但仍在继续玩，完全事不关己的样子。这件事情在游戏圈子里很快就传开了。哪些人打架，谁被害了，谁跑了，大家都知道。但是他们都无动于衷，没有触动，日子照过，游戏照玩。我们询问的十多名知情人，都表现得很冷漠。”<br/><br/>甚至连蔡天星本人，明知道自己才是矛头所在，李英杰不过是“替死鬼”，也仍然很淡漠，极力撇开与此事的联系。“他们号称‘家族观念重’，但是真的发生什么事的时候，又都不想牵涉其中，不愿意站出来帮忙，说不清楚是什么心理，很奇怪。”<br/><br/>“这些孩子大都是独生子女，以自我为中心。他们的逻辑似乎很简单：谁侵犯我了，我就要报复；但是闯了祸以后，又‘明哲保身’。”<br/><br/>生活已经高度网络化<br/><br/>更值得注意的，是犯罪嫌疑人和其他玩家们已经高度网络化的语言和行为方式。<br/><br/>案发后，民警询问了十多名知情的玩家，也审讯了嫌疑人，感觉“他们和我们的思维不一样”。“我们问话的时候，很平常的话他们一下子理解不了。他们说的话，我们也要‘处理一下’才明白。他们的语言已经很网络化了，与正常的社会交流有一定的差距。”<br/><br/>周磊举例说，这些游戏玩家之间互相称呼网名，提起真名反而一脸茫然。他们称女玩家为“婆娘”，用惯了网络语言，包括网络上骂人的脏话和网络符号。一名玩家的网名叫做“爹”，这些同龄人照喊不误，丝毫不以为怪。<br/><br/>周磊描述了这个“圈子”里的人的行为特征：十几、二十岁，社会闲散人员；找不着工作或懒得找工作；没有多少钱，沉溺网络聊天和游戏；可以没日没夜地泡在网吧里，将键盘敲得“啪啪”响；家长要么太忙、没功夫管孩子，要么离异、管不了；都说脏话，女孩子都抽烟；没有正常的社交圈子，朋友几乎都是网络上认识的；很少回家，有时累了就几个人一起找个小旅馆睡一夜；穿着也挺时尚，但因为不常回家洗澡换衣服，网吧里的烟味、汗味都沾在身上，很邋遢。<br/><br/>“有一个知情人，是个女孩。我们找到她时，她已经在网吧待了3天3夜。”周磊说。<br/><br/>3名犯罪嫌疑人中，谢海不满17岁，施、张二人21岁，都是初中毕业或肄业后便闲在家里，终日上网。今年3月，三人在现实生活中见面后，经常一同上网、喝酒、切磋游戏技术。<br/><br/>谢海说：“我们有时候玩通宵，就在网吧里吃点方便面什么的。‘施施’（施应昌的网名）的瘾最大，他也最有钱，经常付账。张雨的性格冲动一些。我觉得他们两个比较讲义气，是我的朋友。”<br/><br/>谢海有10多个这样的“朋友”。现在，他非常后悔那一刻为了“义气”冲上前去：“我现在想，他们不是真正的朋友，其实义气不是那么重要。如果现实生活中遇到别人这样羞辱我的朋友，我不会生气到动刀子的地步。”（文中涉案人员均为化名）?<wbr
