18岁男孩40多刀捅死亲妈，仅仅为了方便回家偷钱，养网恋同居的女友。涉嫌故意杀人罪，他被押上了刑事审判席.昨日，在郑州市中级人民法院开庭审理的这起案件，让人唏嘘不已.<br/><br/><wbr />?<wbr />?<wbr />?网恋男孩频繁回家偷钱朱陵杰生于1990年10月，浙江永嘉人，自幼随经商的父母在新郑市生活。其父母在新郑市八千乡开了一家超市，家境还算不错。<br/><br/><wbr />?<wbr />?<wbr />?朱林杰初中辍学后，迷恋上了网络。他开始从家里偷钱上网，为此受到父母严厉的批评和斥责。朱林杰非但没有收敛，反而经常一连几天泡在网吧不回家。<br/><br/><wbr />?<wbr />?<wbr />?没钱的时候，朱林杰就回家，目的很明确――偷钱，但被发现后，换来的依然是一顿臭骂。<br/><wbr />?<wbr />?<wbr />?父母对朱林杰的防范也越来越严密，有时候，朱林杰偷不到钱，他就偷偷将超市的物品通过窗户扔到外面地上，然后再悄悄捡走贱卖换钱花。<br/><br/><wbr />?<wbr />?<wbr />?2008年5月份，朱林杰认识了一个网名叫“虞冰冰”的呼和浩特女孩，两人聊的热火朝天，难舍难分。<br/>当年9月20日，朱林杰拿着从家里偷来的9000元钱，迫不及待的乘车来到呼和浩特，找到了虞冰冰。他在呼市住了一个星期后，9月27日，两人一起返回新郑，租了一套房子住在了一起。<br/><p align="center">朱林杰和虞冰冰都没有工作，每天除了上网就是睡觉。坐吃山空，俩人很快就没钱花了。</p><br/><b>?<wbr />?<wbr />?<wbr />?40</b><b>多刀捅死亲妈</b><br/><br/><wbr />?<wbr />?<wbr />?去年12月2日，朱林杰再次回家偷钱，这次，他到新郑市炎黄广场附近买了一个电警棍，打算将家人电晕。但仔细研究了说明书后，他看出，电警棍不能将人击晕。<br/>他又买了一把黑色折叠匕首，打算杀了家人。他天真的认为，“把家人杀了就可以偷钱出来过日子”。<br/>去年5月，朱在网上认识了15岁的内蒙古女孩冰冰(化名)，并于当年9月20日到呼和浩特找到了冰冰。在呼和浩特住了一个星期后，两人一起返回新郑，并租房子同居。两人整天沉湎网络，但均没有工作，又没有经济来源，朱陵杰只好回家偷钱，偷超市的货物、营业款。几个月时间，朱就偷了9000多元现金，香烟更是整条整条偷。朱的父母发现后，多次对其进行批评教育，但朱却仍旧我行我素。<br/>后来，父母为防范朱盗窃，在超市里安装了防盗网，朱的日子也日渐窘迫。去年12月初，女友说要回家，让朱拿路费。由于父母对其防范很严，朱拿不到钱。在女友“拿不到回家的钱，就死给你看”的威胁下，朱竟萌生了将家人杀死偷钱的恶念。<br/>去年12月2日，朱买了一把匕首。次日晚9时，朱携匕首回家，先到三楼住室睡觉，想等父母睡着后下楼偷钱。晚上10时许，朱父到其住室找其谈话，两人说话到次日凌晨零时许，朱父睡着了。朱下楼以喝水为由喊母亲开门，开门后他又在母亲的住室泡方便面吃，吃了面后便躺在母亲身边假装睡觉。到凌晨2时30分，朱看母亲睡着了，就左手拿枕头捂着母亲的脸，右手拿匕首朝母亲身上猛刺了几十刀。随后，朱陵杰开始满屋找钱，他先用匕首撬保险柜门，但直到匕首撬断仍未撬开保险柜。他只在超市内搜到1000余元现金，随后，朱林杰洗了洗手，开始满屋子找钱，在搜到1000余元现金后，他与女友会合，连夜逃往呼市虞冰冰家藏匿了起来。<br/>事发两天后，新郑警方在呼和浩特市将朱陵杰抓获。<br/><br/><b>?<wbr />?<wbr />?<wbr />?为讨女友欢心而弑母</b><br/><wbr />?<wbr />?<wbr />?亲人请求给他一个重新做人的机会<br/><br/><wbr />?<wbr />?<wbr />?事隔两天，新郑警方就追踪到了呼市，将朱林杰抓获。<br/><wbr />?<wbr />?<wbr />?经法医鉴定，朱林杰的母亲系被单刃刺器刺破双侧肺脏及肝脏，致失血性休克死亡。头部、胸背部、腿上等中40多刀。<br/><br/><wbr />?<wbr />?<wbr />?“网络到底是个啥？能让孩子杀死自己的母亲！”说起妻子被害，朱林杰的父亲痛心不已。<br/><br/><wbr />?<wbr />?<wbr />?昨日，因涉嫌故意杀人罪，朱林杰被押上了郑州市中级人民法院刑事审判席。<br/><br/><wbr />?<wbr />?<wbr />?庭审中，朱林杰的辩护律师李胜利宣读了一份申请，是朱林杰的外公、外婆、父亲和弟弟写的。他们说，他们对此事的发生感到万分震惊。一方面他们对朱林杰的行为感到愤怒无比，另一方面对女儿的死亡感到悲痛欲绝。<br/><br/><wbr />?<wbr />?<wbr />?他们说，朱林杰在生活中并不是善恶不分之人，考虑到朱建丹已不能复活，而朱林杰也多次表示忏悔，请求法官考虑本案的实际情况和两位古稀老人的心情，能给朱林杰一个重新做人的机会，同时，他们也放弃对朱林杰民事赔偿责任的追究。<br/><br/><wbr />?<wbr />?<wbr />?整个庭审过程，朱林杰表现平静。<br/><br/><wbr />?<wbr />?<wbr />?为满足女友的开销杀死亲生母亲的朱某被押上法庭。当日，郑州市中级人民法院开庭审理一起故意杀人案件，为满足女友的开销，朱某不断盗窃家里的财物，而为了不让父母发现自己的偷窃行为，年满18岁的浙江青年朱某用匕首往母亲身上连刺几十刀，致使母亲当场死亡，后被公安机关抓获，公诉机关认为，被告人朱某故意伤害他人身体，并致人死亡，其行为应以故意杀人罪追究其刑事责任，法庭将择日宣判。<br/><br/><wbr />?“现在我还稀里糊涂的，不知为啥就把我妈给杀了……”站在被告人席上的朱陵杰至今仍对自己的杀母行为困惑不解。他曾是个善良的孩子，但自从迷上了网络、有了网恋，他多次从家中偷钱，受到父母的严厉管教后，他非但没有收敛，反而产生了强烈的叛逆心理，出走和女网友租房同居。为了弄钱养女友，他竟然趁母亲熟睡时，持刀将母亲杀死。<br/>昨日上午，郑州市中级人民法院一审公开审理这起罕见的因上网引起的杀母案。为达到教育社会的目的，法院专门邀请了社会各界人士旁听此案，并现场对庭审情况进行评判。<br/><br/>庭审现场：外公外婆写信为外孙求情<br/><br/>在庭审现场，朱陵杰的辩护律师表示，对公诉人认为朱陵杰构成故意杀人罪的指控不持异议，仅对量刑方面进行辩护。辩护律师先当庭提交了一份朱陵杰外公、外婆提交的申请，其外公外婆表示，一方面对外孙的行为感到愤怒，一方面感到不解，因为在现实生活中，这个外孙曾经是个善良的孩子，不知道外孙为什么会变成这样。作为被害人的直系亲属，他们愿意放弃对被告人民事赔偿方面的责任追究。同时，也希望法院给外孙一个重新做人的机会。<br/>辩护律师认为，朱陵杰刚满18周岁，就其犯罪行为来说，社会和家庭都有责任。其刚踏入社会就迷恋上网络，后认识了女友，多次偷家里的钱供女友吃喝、上网。其伤害亲生母亲，一是怕母亲发现自己偷钱行为受责怪，二是在女友弄不来钱以死相逼的情况下实施犯罪。尽管不能因此就可以说其犯罪情有可原，但不能不说悲剧和被告人受的家庭教育不无关系。首先是其父母的溺爱式教育让其走了极端，其次，其父母从不在心灵上和嫌疑人进行沟通，整天只是忙于做生意，对儿子不管不问，即便在儿子出现早恋的情况下，也没进行正确引导，最终导致悲剧发生。<br/>庭审结束后，当法官让朱陵杰当庭做最后发言时，这个不擅言谈的孩子表示：尽管外公外婆已原谅了他，但他却无法原谅自己，他陷入虚拟的网络世界，最终因此走上了犯罪道路……<br/>“网络到底是个啥！能让孩子杀死自己的母亲？”坐在旁听席上的受害人家属极为困惑。其姑姑在庭审结束后，一直询问朱陵杰：“你以前是个好孩子，那么善良，你咋会杀死你亲妈？”朱陵杰无言以对。<br/>郑州市中级人民法院刑事审判庭法官张鹏飞认为：很多孩子学习不好，没有成就感，就沉浸在网络游戏当中，能打败别人，成就感或者说虚荣心就满足了。很多暴力游戏出现的血腥场面，令上网者感到特别刺激，这样的孩子容易产生暴力倾向。<br/>张鹏飞还表示，预防因上网引发的违法犯罪，需要家庭、学校、社会等多方面共同参与。政府相关部门特别是公安机关必须加强管理，坚决杜绝涉黄、涉暴等游戏、信息的登录和传播；要强化对网吧经营业主的教育管理和处罚力度，引导业主诚信经营，守法经营。此外，还要加强学校和家庭的教育引导，利用家庭和社会的力量使青少年摆脱网瘾。<br/>法庭宣布，本案将择日宣判。（记者韩景玮通讯员张鹏飞文记者许俊文图）<br/>首次转载时间:
