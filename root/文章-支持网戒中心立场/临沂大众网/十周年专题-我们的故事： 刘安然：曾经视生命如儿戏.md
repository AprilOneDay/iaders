src="http://linyi.dzwww.com/2015/jwzx/wmdgs/201601/W020160111324680127205.jpg" alt="" /></p><br/><p align="center"></p><br/>　　我是离院盟友刘安然。2014年10月4日在来到中心时，我已近一年没有上学，更是拒绝参加应届的高考，每天都宅在家中玩游戏，不思进取。网络游戏侵蚀了原本篷勃上进的我，原本我向爸爸妈妈承诺等到开学便回到学校复习准备高考，但我却没有遵守自己的诺言，一次次地欺骗爸爸妈妈，用新的承诺来搪塞父母，那个虚无缥缈缈的日期便越推越迟，不仅是欺骗爸妈，我更是在欺骗自己。我渐渐变成了一个没有目标，没有斗志，每天用谎言糊弄着父母。<br/><br/>在家中玩游戏无所事事的日子里，我越发的迷茫，不仅仅是斗志渐渐地消散，更是在日复一日的单调生活中觉得生命都变得无趣，没有意义，视生命如儿戏。而后来爸爸妈妈劝阻我，希望我能够回到学校，或是到社会上找份工作，但我却在这时萌生出对父母的仇视，面对父母的期盼和恨铁不成钢的失落，我却将他们视为仇人，没有一丝丝的悔意，反而是用厌烦和不屑来面对他们的关爱，发展到后来，我开始绝食，完全不计后果，甚至到医院检查身体已经出现了问题我都没有在意，我认识不到自己的问题和错误，只觉得自己委屈，到了危及生命的程度时我认知上的偏差都没有丝毫的改观，甚至愈演愈烈。爸爸妈妈也曾找过其他人来劝阻过我，学校的老师，家族里的亲戚，甚至请来了心理咨询师，开始时我还会和他们有些交流，但渐渐地，我对他们也觉得不屑甚至直接冷眼相对，当所有人都再没有办法来改变我的时候，爸爸经人介绍，听说了中心，没有一丝迟疑马上带我来到了中心。<br/><br/>初到中心时，我在心里是排斥的，还想这里就是骗人的地方，跟传销一样的地方。依然想通过绝食来逃避，却为此接受了心理导入，我因此而难得冷静了下来，才对自己之前的所作所为感到了后怕，也开始理解了爸爸妈妈始终是为了我着想，但我仍不想留在中心，并以高考为借口来忽悠爸爸妈妈来带我回去，但父母却都非常坚定，他们的坚决打消了我的幻想，只能安下心来改变。<br/><br/>在中心，我经历着杨叔精心设计的各个环节，我没有感受到在接受治疗，但却能真切地感受到自己的变化。<br/><br/>在大点评课堂上，首先课前的励志歌曲，使我被网络腐蚀的心慢慢的被亲情和爱融化；再听着杨叔的苦口婆心点评，我从被点盟友的身上找到了自己身上的不足，通过自醒、自悟、自纠，慢慢的理性在回归。通过精品盟友的改变经验，不断的总结自己的不足、学习他们的经验，在杨叔和各位点评师的精心指导下，更是学到许多人生的道理和智慧。在课下，也有着盟友与家长评们相互激励，相互学习。每天下午的军训我没有感觉是负担，而是锻炼体能和意志力的机会。回想以前的我，从来不锻炼和参加活动，一有时间就是泡在游戏里，体重超重，体能低下。学校的体育测试从来不及格。现在通过在中心的锻炼，我在大学开学的军训中被评为优秀学员，这都是得益于在中心的锻炼和培养。<br/><br/>在中心，盟友们同堂上课，收获却不尽相同。抱着怀疑的态度，只能一直浑浑噩噩，只有相信并学习才能得到更多。爱与信的力量是不可估量的。入院两个月后，非常感谢杨叔给了我学习的机会，我加入了中高考备考组，每天一个小时的学习时间是不够的，然而恰恰是在这短短的一小时中，我懂得了时间的宝贵，学会了如何高效地利用时间，更明白了我是在为自己学习。<br/><br/>在离开中心后，我面临的是三个月之后的高考。离开了中心我有些许的不安也有着迷茫，我不知道我的努力会得到什么样的结果，我不知道放下课本一年的我在陌生的班级里能用三个月的时间取得怎样的成绩，但是杨叔的“全力以赴、内心无愧”的理念不断的激励着我。我不断地探索，去努力，每天我都会利用日记来总结自己的不足并制定之后一天的计划，而且我保持着同中心相仿的作息时间并坚持一定量的运动，用这种方法来保持课堂上的注意力。老师曾预测我也许能达到二本线，但我的成绩从班级的下游不断进步，最终在高考时得到了662的分数，在班上是第二名，我的许多老师和亲戚都觉得这是个奇迹，但我知道，不管结果是什么，这不是奇迹，这就是杨叔所教导的“全力以赴，问心无愧”、相信、配合、坚持在我身上的体现。<br/><br/>如今，我在北京理工大学学习，运用在中心学到的军事技能在军训中表现出色，担任了排长并在班级中承担班长的工作，并努力激励同学们积极向上，我自己也没有放松过对自己的要求，在工图考试中取得了97的高分，并每天坚持运动，还自己接触一些实验课程。从一个废人到如今的积极向上，是杨叔和中心改变了我，是杨叔和中心让我找回了生命的意义，带给我重生。我知道，我现在才是在人生道路上迈出了一小步，在以后的人生路上肯定会遇到更多的挫折，相信我是杨叔学生，在中心锻炼的抗挫抗压的能力会为我以后的人生助力。我不会辜负杨叔和中心的栽培，我会向着人生正确的目标不断前进的。<br/><br/>中心九周年的纪念日还历历在目，而今十周年也已临近，短短的一年，因杨叔及各位点评师及医护人员的辛勤付出和千百家庭的重生而珍重。<br/><br/>祝中心在新的一年里蓬勃发展，让爱与信的力量传遍四方，为更多家庭重新带来希望幸福。<br/>首次转载时间:
